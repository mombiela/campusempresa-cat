Configurar el codi de seguiment
Següent ►

En aquest tema, aprendrem com configurar el codi de seguiment de Google Analytics al vostre lloc web. Aquest és un pas fonamental per començar a recopilar dades sobre els visitants del vostre lloc web. A continuació, es detallen els passos per configurar el codi de seguiment.

  1. Crear una propietat a Google Analytics

Abans de configurar el codi de seguiment, necessitem crear una propietat a Google Analytics. Una propietat representa el vostre lloc web o aplicació dins del compte de Google Analytics.

Passos per crear una propietat:

  1. Inicieu sessió a Google Analytics.
  2. Feu clic a l'opció Administrador a la part inferior esquerra.
  3. A la columna Propietat, feu clic a Crear propietat.
  4. Introduïu el nom de la propietat, la zona horària i la moneda.
  5. Feu clic a Següent i completeu la configuració segons les vostres necessitats.
  6. Feu clic a Crea.

  1. Obtenir el codi de seguiment

Un cop creada la propietat, necessitem obtenir el codi de seguiment que s'ha d'inserir al vostre lloc web.

Passos per obtenir el codi de seguiment:

  1. A la pàgina de l'administrador, a la columna Propietat, feu clic a Informació de seguiment.
  2. Seleccioneu Codi de seguiment.
  3. Copieu el fragment de codi de seguiment que apareix a la secció Etiqueta global del lloc web (gtag.js).

  1. Inserir el codi de seguiment al vostre lloc web

El codi de seguiment s'ha d'inserir a totes les pàgines del vostre lloc web que voleu seguir. Normalment, es col·loca a la secció <head> del codi HTML.

Exemple de codi de seguiment:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>El meu lloc web</title>
    <!-- Inseriu el codi de seguiment aquí -->
    <script async src="https://www.googletagmanager.com/gtag/js?id=UA-XXXXXXXXX-X"></script>
    <script>
        window.dataLayer = window.dataLayer || [];
        function gtag(){dataLayer.push(arguments);}
        gtag('js', new Date());

        gtag('config', 'UA-XXXXXXXXX-X');
    </script>
</head>
<body>
    <h1>Benvinguts al meu lloc web!</h1>
    <!-- Contingut del lloc web -->
</body>
</html>

Explicació del codi:

  • Etiqueta global del lloc web (gtag.js): Aquesta és la biblioteca de JavaScript que Google Analytics utilitza per recopilar dades.
  • ID de seguiment: Substituïu UA-XXXXXXXXX-X amb el vostre ID de seguiment únic proporcionat per Google Analytics.
  • gtag('config', 'UA-XXXXXXXXX-X'): Aquesta línia inicialitza Google Analytics amb el vostre ID de seguiment.

  1. Verificar la instal·lació del codi de seguiment

Després d'inserir el codi de seguiment, és important verificar que està funcionant correctament.

Passos per verificar la instal·lació:

  1. Obriu el vostre lloc web en un navegador.
  2. Inicieu sessió a Google Analytics.
  3. Aneu a Informes > Temps real > Visió general.
  4. Si el codi de seguiment està configurat correctament, hauríeu de veure la vostra visita en temps real.

Exercici pràctic

Objectiu:

Configurar el codi de seguiment de Google Analytics en un lloc web de prova.

Passos:

  1. Creeu una propietat a Google Analytics seguint els passos descrits anteriorment.
  2. Obteniu el codi de seguiment.
  3. Creeu un fitxer HTML senzill i inseriu el codi de seguiment a la secció <head>.
  4. Obriu el fitxer HTML en un navegador i verifiqueu la instal·lació a Google Analytics.

Solució:

<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Lloc web de prova</title>
    <!-- Inseriu el codi de seguiment aquí -->
    <script async src="https://www.googletagmanager.com/gtag/js?id=UA-XXXXXXXXX-X"></script>
    <script>
        window.dataLayer = window.dataLayer || [];
        function gtag(){dataLayer.push(arguments);}
        gtag('js', new Date());

        gtag('config', 'UA-XXXXXXXXX-X');
    </script>
</head>
<body>
    <h1>Benvinguts al lloc web de prova!</h1>
    <!-- Contingut del lloc web -->
</body>
</html>

Errors comuns i consells

Errors comuns:

  • No inserir el codi de seguiment a totes les pàgines: Assegureu-vos que el codi de seguiment està present a totes les pàgines que voleu seguir.
  • Inserir el codi de seguiment en una ubicació incorrecta: El codi de seguiment ha d'estar a la secció <head> del codi HTML.
  • No substituir l'ID de seguiment: Assegureu-vos de substituir UA-XXXXXXXXX-X amb el vostre ID de seguiment únic.

Consells:

  • Utilitzeu Google Tag Manager: Per a una gestió més fàcil del codi de seguiment i altres etiquetes, considereu utilitzar Google Tag Manager.
  • Verifiqueu regularment: Verifiqueu regularment que el codi de seguiment està funcionant correctament, especialment després de fer canvis al lloc web.

Conclusió

En aquesta secció, hem après com configurar el codi de seguiment de Google Analytics al vostre lloc web. Aquest és un pas crucial per començar a recopilar dades valuoses sobre els visitants del vostre lloc web. En la propera secció, explorarem els informes en temps real per veure com es poden utilitzar aquestes dades per obtenir informació immediata sobre l'activitat del vostre lloc web.
