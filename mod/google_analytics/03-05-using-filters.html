Ús de filtres
Següent ►

Els filtres a Google Analytics són una eina poderosa que permet als usuaris modificar i limitar les dades que es mostren en els informes. Això és especialment útil per assegurar-se que les dades analitzades siguin rellevants i precises. En aquesta secció, aprendrem com configurar i utilitzar filtres per millorar la qualitat de les dades.

  1. Què són els filtres?

Els filtres són regles que s'apliquen a les dades de Google Analytics per incloure, excloure o modificar la informació que es recull. Els filtres es poden aplicar a nivell de vista (view) i poden afectar com es mostren les dades en els informes.

Tipus de filtres comuns:

  • Incloure: Només inclou les dades que compleixen amb els criteris especificats.
  • Excloure: Exclou les dades que compleixen amb els criteris especificats.
  • Minúscules/Majúscules: Converteix totes les dades a minúscules o majúscules.
  • Cercar i substituir: Cerca un patró específic i el substitueix per un altre.
  • Avançat: Permet crear filtres personalitzats amb expressions regulars.

  1. Configurar un filtre

Pas 1: Accedir a l'administració de filtres

  1. Inicia sessió a Google Analytics.
  2. Selecciona el compte i la propietat on vols aplicar el filtre.
  3. A la columna "Vista", fes clic a "Filtres".

Pas 2: Crear un nou filtre

  1. Fes clic a "Afegeix un filtre".
  2. Assigna un nom al filtre.
  3. Selecciona el tipus de filtre (predefinit o personalitzat).

Pas 3: Configurar el filtre

  1. Filtre predefinit:

    • Selecciona el tipus de filtre (incloure, excloure, etc.).
    • Defineix els criteris del filtre (per exemple, incloure només trànsit de cert país).

  2. Filtre personalitzat:

    • Selecciona "Personalitzat".
    • Tria el tipus de filtre (excloure, incloure, minúscules, majúscules, etc.).
    • Defineix el camp del filtre (per exemple, URL de la pàgina, adreça IP).
    • Especifica el patró del filtre (per exemple, una expressió regular).

Exemple de filtre personalitzat:

Nom del filtre: Excloure trànsit intern
Tipus de filtre: Personalitzat
Excloure: Sí
Camp del filtre: Adreça IP
Patró del filtre: ^123\.456\.789\.0$

  1. Exemples pràctics

Exemple 1: Excloure trànsit intern

Per assegurar que les dades no es vegin distorsionades pel trànsit generat pels empleats de l'empresa, pots crear un filtre per excloure les adreces IP internes.

Nom del filtre: Excloure trànsit intern
Tipus de filtre: Personalitzat
Excloure: Sí
Camp del filtre: Adreça IP
Patró del filtre: ^123\.456\.789\.0$

Exemple 2: Incloure només trànsit d'un país específic

Si vols analitzar només el trànsit d'un país específic, pots crear un filtre per incloure només les visites d'aquell país.

Nom del filtre: Incloure trànsit d'Espanya
Tipus de filtre: Predefinit
Incloure: Sí
Camp del filtre: País
Patró del filtre: Espanya

  1. Consells i millors pràctiques

  • Prova els filtres en una vista de prova: Abans d'aplicar filtres a la vista principal, crea una vista de prova per assegurar-te que els filtres funcionen correctament.
  • Documenta els filtres: Mantingues un registre dels filtres aplicats i els seus objectius per facilitar la gestió i la revisió futura.
  • Revisa regularment els filtres: Assegura't que els filtres continuen sent rellevants i ajusta'ls segons sigui necessari.

  1. Exercici pràctic

Exercici:

Crea un filtre per excloure el trànsit generat per una adreça IP específica (per exemple, 192.168.1.1).

Solució:

  1. Accedeix a l'administració de filtres.
  2. Fes clic a "Afegeix un filtre".
  3. Assigna el nom "Excloure IP 192.168.1.1".
  4. Selecciona "Personalitzat".
  5. Marca "Excloure".
  6. Al camp del filtre, selecciona "Adreça IP".
  7. Al patró del filtre, introdueix ^192\.168\.1\.1$.
  8. Desa el filtre.

Conclusió

Els filtres són una eina essencial per assegurar que les dades de Google Analytics siguin precises i rellevants. Amb una configuració adequada, els filtres poden ajudar a excloure dades no desitjades, incloure només les dades rellevants i modificar la informació per adaptar-se millor a les necessitats d'anàlisi. Practica la configuració de filtres per familiaritzar-te amb les seves capacitats i millorar la qualitat de les teves dades.
