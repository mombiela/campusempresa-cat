Publicitat a YouTube: YouTube Ads

Següent

Introducció a YouTube Ads

YouTube és una de les plataformes de vídeo més grans del món, amb milers de milions d'usuaris actius mensuals. La publicitat a YouTube ofereix una oportunitat única per arribar a una audiència global amb contingut visual atractiu. En aquesta secció, aprendrem a crear, gestionar i optimitzar campanyes publicitàries a YouTube.

Tipus de Publicitat a YouTube

YouTube ofereix diversos formats publicitaris per adaptar-se a diferents objectius de màrqueting:

  1. TrueView Ads:

    • In-stream Ads: Es reprodueixen abans, durant o després d'un vídeo. Els espectadors poden saltar-los després de 5 segons.
    • Discovery Ads: Apareixen als resultats de cerca de YouTube, a la pàgina d'inici o a la barra lateral de vídeos relacionats.

  2. Bumper Ads:

    • Anuncis de 6 segons que no es poden saltar. Ideals per a missatges curts i impactants.

  3. Non-skippable In-stream Ads:

    • Anuncis de fins a 15-20 segons que no es poden saltar. Es reprodueixen abans, durant o després d'un vídeo.

  4. Masthead Ads:

    • Anuncis que apareixen a la part superior de la pàgina d'inici de YouTube. Són ideals per a grans campanyes de marca.

Creació de Campanyes a YouTube Ads

Pas 1: Configuració del Compte de Google Ads

Abans de crear una campanya a YouTube, necessites un compte de Google Ads. Si no en tens un, segueix aquests passos per crear-lo:

  1. Visita Google Ads.
  2. Fes clic a "Comença ara" i segueix les instruccions per configurar el teu compte.

Pas 2: Creació de la Campanya

  1. Accedeix a Google Ads:

    • Inicia sessió al teu compte de Google Ads.
    • Fes clic a "Campanyes" al menú de l'esquerra.
    • Fes clic al botó blau "+" i selecciona "Nova campanya".

  2. Selecciona l'Objectiu de la Campanya:

    • Tria l'objectiu que millor s'adapti a les teves necessitats (per exemple, "Consideració de marca i producte" o "Conscienciació de marca").

  3. Selecciona el Tipus de Campanya:

    • Tria "Vídeo" com a tipus de campanya.

  4. Defineix la Estratègia de Licitació:

    • Selecciona l'estratègia de licitació adequada (per exemple, CPV - Cost per Visualització).

  5. Configura el Pressupost i les Dates:

    • Defineix el pressupost diari o total de la campanya.
    • Estableix les dates d'inici i finalització de la campanya.

  6. Defineix la Segmentació:

    • Selecciona la ubicació geogràfica, l'idioma i altres criteris de segmentació per arribar a la teva audiència objectiu.

  7. Crea els Anuncis:

    • Tria el format d'anunci (TrueView, Bumper, etc.).
    • Puja el vídeo que vols utilitzar com a anunci.
    • Completa els detalls de l'anunci (títol, descripció, URL de destinació).

Exemple de Codi per a la Creació d'un Anunci TrueView

// Exemple de codi per a la creació d'un anunci TrueView a Google Ads API
const googleAds = require('google-ads-api');

const client = new googleAds.GoogleAdsClient({
  client_id: 'YOUR_CLIENT_ID',
  client_secret: 'YOUR_CLIENT_SECRET',
  developer_token: 'YOUR_DEVELOPER_TOKEN',
  refresh_token: 'YOUR_REFRESH_TOKEN',
});

async function createTrueViewAd() {
  const campaign = await client.campaigns.create({
    name: 'YouTube TrueView Campaign',
    advertising_channel_type: 'VIDEO',
    status: 'PAUSED',
    bidding_strategy_type: 'MANUAL_CPV',
    budget: {
      amount_micros: 1000000, // 1 USD
    },
    video_campaign: {
      video_campaign_type: 'TRUE_VIEW',
    },
  });

  const adGroup = await client.adGroups.create({
    campaign_id: campaign.id,
    name: 'TrueView Ad Group',
    status: 'ENABLED',
    cpc_bid_micros: 1000000, // 1 USD
  });

  const ad = await client.ads.create({
    ad_group_id: adGroup.id,
    ad: {
      video_ad: {
        video_id: 'YOUR_VIDEO_ID',
        in_stream: {
          action: {
            call_to_action: 'Learn More',
            final_url: 'https://www.example.com',
          },
        },
      },
    },
  });

  console.log('TrueView Ad created:', ad);
}

createTrueViewAd().catch(console.error);

Optimització de Campanyes

Anàlisi de Mètriques Clau

Per assegurar-te que les teves campanyes a YouTube Ads són efectives, és important monitoritzar i analitzar les mètriques clau:

  1. CPV (Cost per Visualització): El cost mitjà per cada visualització de l'anunci.
  2. CTR (Click-Through Rate): El percentatge de persones que fan clic a l'anunci després de veure'l.
  3. VTR (View-Through Rate): El percentatge de persones que veuen l'anunci complet.
  4. Conversions: Accions específiques que els usuaris realitzen després de veure l'anunci (per exemple, compres, registres).

Consells per a l'Optimització

  1. Segmentació Precisa:

    • Utilitza opcions de segmentació avançades per arribar a l'audiència més rellevant.

  2. Creativitat en els Anuncis:

    • Crea anuncis atractius i rellevants que captin l'atenció de l'audiència en els primers segons.

  3. Proves A/B:

    • Realitza proves A/B amb diferents versions dels anuncis per determinar quins funcionen millor.

  4. Anàlisi i Ajustaments Constants:

    • Revisa les mètriques regularment i ajusta les campanyes segons els resultats obtinguts.

Exercici Pràctic

Creació d'una Campanya TrueView

  1. Objectiu: Crear una campanya TrueView per promocionar un nou producte.
  2. Requisits:
    • Pressupost diari: 50 EUR.
    • Ubicació: Espanya.
    • Idioma: Català.
    • Vídeo: Enllaç al vídeo promocional.

Passos a Seguir

  1. Configura el teu compte de Google Ads.
  2. Crea una nova campanya de vídeo amb l'objectiu de "Consideració de marca i producte".
  3. Defineix el pressupost diari de 50 EUR.
  4. Selecciona Espanya com a ubicació i Català com a idioma.
  5. Crea un anunci TrueView utilitzant el vídeo proporcionat.

Solució

  1. Configura el compte de Google Ads:

    • Inicia sessió a Google Ads i segueix els passos per configurar el compte.

  2. Crea la campanya:

    • Accedeix a "Campanyes" i crea una nova campanya de vídeo.
    • Selecciona "Consideració de marca i producte" com a objectiu.
    • Defineix el pressupost diari de 50 EUR.
    • Selecciona Espanya com a ubicació i Català com a idioma.

  3. Crea l'anunci TrueView:

    • Tria el format TrueView In-stream.
    • Puja el vídeo promocional.
    • Completa els detalls de l'anunci (títol, descripció, URL de destinació).

Conclusió

La publicitat a YouTube ofereix una plataforma potent per arribar a una audiència global amb contingut visual atractiu. Mitjançant la comprensió dels diferents formats publicitaris, la creació de campanyes efectives i l'optimització constant, pots maximitzar l'impacte de les teves campanyes publicitàries a YouTube. Practica amb els exercicis proporcionats i utilitza les eines d'anàlisi per millorar contínuament els teus resultats.
