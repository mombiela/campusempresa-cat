Afegir filtres i controls interactius
Següent ►

En aquest tema, aprendrem a afegir filtres i controls interactius als nostres informes de Google Data Studio. Els filtres i controls interactius permeten als usuaris personalitzar la visualització de les dades segons les seves necessitats específiques, millorant així la interactivitat i l'ús dels informes.

Objectius

  • Comprendre la importància dels filtres i controls interactius.
  • Aprendre a afegir diferents tipus de filtres i controls.
  • Configurar i personalitzar els filtres per a una millor experiència d'usuari.

1. Importància dels filtres i controls interactius

Els filtres i controls interactius són eines essencials per a:

  • Permetre als usuaris explorar les dades de manera dinàmica.
  • Facilitar la segmentació de dades per períodes de temps, categories, regions, etc.
  • Millorar la comprensió de les dades mostrant només la informació rellevant.

2. Tipus de filtres i controls interactius

Google Data Studio ofereix diversos tipus de filtres i controls interactius, incloent:

  • Controls de data: Permeten seleccionar intervals de dates.
  • Controls de filtre: Permeten seleccionar valors específics d'un camp.
  • Controls de barra de cerca: Permeten buscar valors específics dins d'un camp.
  • Controls de llista desplegable: Permeten seleccionar un valor d'una llista.

3. Afegir un control de data

Pas a pas:

  1. Obre l'informe on vols afegir el control de data.
  2. Fes clic a "Afegeix un control" a la barra d'eines superior.
  3. Selecciona "Control de data".
  4. Dibuixa el control a l'àrea de treball de l'informe.
  5. Configura el control seleccionant el camp de data que vols utilitzar.

Exemple pràctic:

1. Obre l'informe "Vendes 2023".
2. Fes clic a "Afegeix un control" > "Control de data".
3. Dibuixa el control a la part superior de l'informe.
4. Selecciona el camp "Data de venda" per al control de data.

4. Afegir un control de filtre

Pas a pas:

  1. Obre l'informe on vols afegir el control de filtre.
  2. Fes clic a "Afegeix un control" a la barra d'eines superior.
  3. Selecciona "Control de filtre".
  4. Dibuixa el control a l'àrea de treball de l'informe.
  5. Configura el control seleccionant el camp que vols utilitzar per al filtre.

Exemple pràctic:

1. Obre l'informe "Vendes 2023".
2. Fes clic a "Afegeix un control" > "Control de filtre".
3. Dibuixa el control a la part esquerra de l'informe.
4. Selecciona el camp "Categoria de producte" per al control de filtre.

5. Personalització dels filtres i controls

Configuració avançada:

  • Estil: Pots personalitzar l'aparença dels controls (colors, fonts, mides).
  • Interacció: Pots definir com els filtres afecten els diferents components de l'informe.
  • Valors predeterminats: Pots establir valors predeterminats per als filtres.

Exemple pràctic:

1. Selecciona el control de filtre "Categoria de producte".
2. Fes clic a "Estil" a la barra lateral dreta.
3. Canvia el color de fons a blau clar.
4. Estableix la font a Arial, mida 12.
5. Defineix el valor predeterminat com "Totes les categories".

Exercici pràctic

Objectiu:

Crear un informe amb un control de data i un control de filtre per a un camp de categoria.

Instruccions:

  1. Crea un nou informe en Google Data Studio.
  2. Afegeix una font de dades que contingui camps de data i categoria.
  3. Afegeix un control de data i configura'l per al camp de data.
  4. Afegeix un control de filtre i configura'l per al camp de categoria.
  5. Personalitza els controls segons les teves preferències.

Solució:

1. Crea un nou informe i afegeix la font de dades "Vendes 2023".
2. Afegeix un control de data i selecciona el camp "Data de venda".
3. Afegeix un control de filtre i selecciona el camp "Categoria de producte".
4. Personalitza els controls canviant els colors i les fonts.

Errors comuns i consells

Errors comuns:

  • No seleccionar el camp correcte: Assegura't de seleccionar el camp adequat per als filtres.
  • No configurar els valors predeterminats: Estableix valors predeterminats per millorar l'experiència d'usuari.

Consells:

  • Prova els filtres: Abans de compartir l'informe, prova els filtres per assegurar-te que funcionen correctament.
  • Utilitza noms descriptius: Dona noms clars i descriptius als teus filtres per facilitar-ne l'ús.

Conclusió

Afegir filtres i controls interactius als teus informes de Google Data Studio és una manera excel·lent de millorar la interactivitat i la usabilitat dels teus informes. Amb aquests coneixements, pots permetre als usuaris explorar les dades de manera dinàmica i obtenir informació rellevant segons les seves necessitats específiques.
