Personalització de gràfics i taules
Següent ►

En aquest tema, aprendrem a personalitzar els gràfics i les taules en Google Data Studio per fer que els informes siguin més atractius i informatius. La personalització permet adaptar els informes a les necessitats específiques dels usuaris i fer que les dades siguin més comprensibles.

1. Introducció a la personalització de gràfics i taules

Conceptes clau:

  • Gràfics: Representacions visuals de dades que poden ser de diferents tipus com barres, línies, pastissos, etc.
  • Taules: Formats tabulars que mostren dades en files i columnes.

2. Tipus de gràfics i taules

Gràfics disponibles:

  • Gràfics de barres
  • Gràfics de línies
  • Gràfics de pastís
  • Gràfics de taula
  • Gràfics de dispersió
  • Mapes geogràfics

Taules disponibles:

  • Taules simples
  • Taules pivot

3. Personalització de gràfics

Passos per personalitzar un gràfic:

  1. Seleccionar el gràfic: Feu clic sobre el gràfic que voleu personalitzar.
  2. Accedir a les opcions de personalització: A la barra lateral dreta, trobareu les opcions de "Dades" i "Estil".

Opcions de "Dades":

  • Dimensió: Seleccioneu la dimensió que voleu mostrar.
  • Mètrica: Seleccioneu la mètrica que voleu visualitzar.
  • Filtre: Afegiu filtres per restringir les dades mostrades.

Opcions d'"Estil":

  • Colors: Trieu els colors per a les barres, línies, etc.
  • Etiquetes: Activeu o desactiveu les etiquetes de dades.
  • Format de l'eix: Personalitzeu els eixos X i Y (escala, títols, etc.).

Exemple pràctic:

Suposem que tenim un gràfic de barres que mostra les vendes mensuals per producte. Volem personalitzar-lo per destacar els productes amb més vendes.

1. Seleccionem el gràfic de barres.
2. A "Dades", seleccionem "Mes" com a dimensió i "Vendes" com a mètrica.
3. A "Estil", canviem els colors de les barres per destacar els productes amb més vendes en un color més intens.
4. Activem les etiquetes de dades per mostrar les vendes exactes sobre cada barra.
5. Personalitzem l'eix Y per mostrar només els intervals de 100 en 100.

4. Personalització de taules

Passos per personalitzar una taula:

  1. Seleccionar la taula: Feu clic sobre la taula que voleu personalitzar.
  2. Accedir a les opcions de personalització: A la barra lateral dreta, trobareu les opcions de "Dades" i "Estil".

Opcions de "Dades":

  • Columnes: Seleccioneu les columnes que voleu mostrar.
  • Filtre: Afegiu filtres per restringir les dades mostrades.

Opcions d'"Estil":

  • Colors de les cel·les: Trieu els colors per a les cel·les.
  • Format de text: Personalitzeu el tipus de lletra, mida, color, etc.
  • Format de les cel·les: Personalitzeu l'alineació, marges, etc.

Exemple pràctic:

Suposem que tenim una taula que mostra les vendes per producte i per regió. Volem personalitzar-la per fer-la més llegible i atractiva.

1. Seleccionem la taula.
2. A "Dades", seleccionem les columnes "Producte", "Regió" i "Vendes".
3. A "Estil", canviem el color de fons de les cel·les per alternar files de diferents colors.
4. Personalitzem el text per fer-lo més gran i en negreta.
5. Afegim un filtre per mostrar només les regions amb vendes superiors a 1000 unitats.

5. Exercicis pràctics

Exercici 1: Personalitzar un gràfic de línies

  1. Creeu un gràfic de línies que mostri les visites mensuals al vostre lloc web.
  2. Personalitzeu els colors de les línies per diferenciar els mesos.
  3. Afegiu etiquetes de dades per mostrar el nombre de visites sobre cada punt de la línia.
  4. Personalitzeu l'eix X per mostrar només els mesos de gener a desembre.

Exercici 2: Personalitzar una taula pivot

  1. Creeu una taula pivot que mostri les vendes per producte i per trimestre.
  2. Personalitzeu els colors de les cel·les per alternar files de diferents colors.
  3. Personalitzeu el text per fer-lo més gran i en negreta.
  4. Afegiu un filtre per mostrar només els productes amb vendes superiors a 500 unitats.

Solucions:

Solució Exercici 1:

  1. Seleccioneu el gràfic de línies.
  2. A "Dades", seleccioneu "Mes" com a dimensió i "Visites" com a mètrica.
  3. A "Estil", canvieu els colors de les línies per diferenciar els mesos.
  4. Activeu les etiquetes de dades per mostrar el nombre de visites sobre cada punt de la línia.
  5. Personalitzeu l'eix X per mostrar només els mesos de gener a desembre.

Solució Exercici 2:

  1. Seleccioneu la taula pivot.
  2. A "Dades", seleccioneu les columnes "Producte" i "Trimestre" i la mètrica "Vendes".
  3. A "Estil", canvieu el color de fons de les cel·les per alternar files de diferents colors.
  4. Personalitzeu el text per fer-lo més gran i en negreta.
  5. Afegiu un filtre per mostrar només els productes amb vendes superiors a 500 unitats.

Conclusió

La personalització de gràfics i taules en Google Data Studio és una eina poderosa per fer que els informes siguin més atractius i fàcils d'entendre. Amb les opcions de personalització, podeu adaptar els informes a les necessitats específiques dels usuaris i fer que les dades siguin més comprensibles. Practiqueu amb els exercicis proporcionats per consolidar els vostres coneixements i habilitats.
