Introducció

Abans de començar a utilitzar Google Data Studio, és important assegurar-se que es compleixen certs requisits previs i que es realitza una configuració inicial adequada. Aquest mòdul proporcionarà una guia pas a pas per garantir que estiguis preparat per començar a crear informes i dashboards amb Google Data Studio.

Requisits previs

1. Compte de Google

Per utilitzar Google Data Studio, necessitaràs un compte de Google. Si no en tens un, pots crear-lo de manera gratuïta seguint aquests passos:

  1. Visita accounts.google.com.
  2. Fes clic a "Crear compte".
  3. Completa el formulari amb la teva informació personal.
  4. Segueix les instruccions per verificar el teu compte.

2. Accés a les fonts de dades

Google Data Studio permet connectar-se a diverses fonts de dades. Assegura't de tenir accés a les dades que vols visualitzar. Algunes de les fonts de dades més comunes inclouen:

  • Google Analytics
  • Google Sheets
  • Google Ads
  • BigQuery
  • Bases de dades SQL
  • Fitxers CSV

3. Navegador web compatible

Google Data Studio funciona millor amb els navegadors web més actuals. Es recomana utilitzar:

  • Google Chrome (versió més recent)
  • Mozilla Firefox (versió més recent)
  • Microsoft Edge (versió més recent)
  • Safari (versió més recent)

Configuració inicial

1. Accés a Google Data Studio

  1. Obre el teu navegador web i visita Google Data Studio.
  2. Inicia sessió amb el teu compte de Google.

2. Familiarització amb la interfície

Quan accedeixis a Google Data Studio per primera vegada, veuràs la pàgina d'inici. Aquesta pàgina inclou:

  • Plantilles: Google Data Studio ofereix diverses plantilles predefinides que pots utilitzar com a punt de partida.
  • Informes recents: Una llista dels informes que has creat o als quals has accedit recentment.
  • Botó "Crear": Permet crear un nou informe o una nova font de dades.

3. Creació d'una font de dades

Per començar a treballar amb Google Data Studio, necessitaràs connectar una font de dades. Segueix aquests passos per crear una font de dades:

  1. Fes clic al botó "Crear" i selecciona "Font de dades".
  2. Tria el tipus de font de dades que vols connectar (per exemple, Google Analytics, Google Sheets, etc.).
  3. Segueix les instruccions per autoritzar l'accés a les teves dades i configurar la connexió.

4. Configuració de permisos

És important configurar els permisos adequats per a les fonts de dades i els informes. Pots compartir els teus informes amb altres usuaris i permetre'ls veure o editar les dades. Per configurar els permisos:

  1. Obre l'informe que vols compartir.
  2. Fes clic a "Compartir" a la part superior dreta.
  3. Afegeix les adreces de correu electrònic dels usuaris amb els quals vols compartir l'informe.
  4. Selecciona el nivell de permís (visualització o edició).

Exercici pràctic

Objectiu

Configurar Google Data Studio amb una font de dades de Google Sheets.

Passos

  1. Crea un compte de Google (si no en tens un).
  2. Accedeix a Google Data Studio.
  3. Crea una nova font de dades seleccionant Google Sheets.
  4. Autoritza l'accés a Google Sheets i selecciona un full de càlcul existent.
  5. Configura els permisos per compartir l'informe amb un company.

Solució

  1. Visita accounts.google.com i crea un compte de Google.
  2. Accedeix a Google Data Studio i inicia sessió.
  3. Fes clic a "Crear" i selecciona "Font de dades".
  4. Tria "Google Sheets" i autoritza l'accés.
  5. Selecciona un full de càlcul existent i segueix les instruccions per completar la configuració.
  6. Obre l'informe, fes clic a "Compartir", afegeix l'adreça de correu electrònic del teu company i selecciona el nivell de permís.

Conclusió

En aquest tema, hem cobert els requisits previs i la configuració inicial necessària per començar a utilitzar Google Data Studio. Ara estàs preparat per connectar les teves fonts de dades i començar a crear informes interactius. En el següent mòdul, aprendrem a connectar i gestionar diferents tipus de fonts de dades.
