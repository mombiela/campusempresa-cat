Ús de camps calculats

Següent

Els camps calculats són una eina poderosa en Google Data Studio que permet crear noves dimensions i mètriques basades en les dades existents. Aquestes noves dades es poden utilitzar per personalitzar i enriquir els informes, oferint una visió més profunda i detallada de les dades.

Què són els camps calculats?

Els camps calculats són expressions personalitzades que creen noves columnes de dades a partir de les dades existents. Aquestes expressions poden incloure funcions matemàtiques, lògiques i de text, així com altres operacions.

Exemples de camps calculats:

  • Càlcul de percentatges: Crear un camp que mostri el percentatge d'una mètrica respecte al total.
  • Classificació de dades: Crear un camp que classifiqui les dades en categories basades en certs criteris.
  • Transformació de dades: Convertir formats de data, unir textos, etc.

Com crear un camp calculat

Pas a pas per crear un camp calculat:

  1. Accedir a l'editor de dades:

    • Obre el teu informe a Google Data Studio.
    • Fes clic a la font de dades que vols modificar.
    • A la part superior dreta, fes clic a "Edit Data Source".

  2. Afegir un nou camp calculat:

    • A l'editor de dades, fes clic a "Add a Field".
    • Es mostrarà una finestra emergent per definir el nou camp.

  3. Definir el camp calculat:

    • Nom del camp: Assigna un nom descriptiu al nou camp.
    • Fórmula: Escriu la fórmula que defineix el càlcul. Pots utilitzar funcions predefinides, operadors matemàtics i lògics, i referències a altres camps.

Exemple pràctic:

Suposem que tenim una taula amb les vendes totals i el nombre de transaccions. Volem crear un camp calculat per determinar la venda mitjana per transacció.

Nom del camp: Venda Mitjana per Transacció
Fórmula: SUM(Vendes Totals) / COUNT(Transaccions)

Fórmules i funcions comunes:

  • Funcions matemàtiques: SUM, AVG, MIN, MAX, COUNT
  • Funcions de text: CONCAT, UPPER, LOWER, TRIM
  • Funcions lògiques: IF, CASE, AND, OR
  • Funcions de data: TODAY, DATE_DIFF, PARSE_DATE

Exercici pràctic

Objectiu:

Crear un camp calculat que mostri el marge de benefici com a percentatge de les vendes totals.

Dades disponibles:

  • Vendes Totals: Total de vendes en euros.
  • Costos Totals: Total de costos en euros.

Pas a pas:

  1. Accedir a l'editor de dades:

    • Obre el teu informe a Google Data Studio.
    • Fes clic a la font de dades que vols modificar.
    • A la part superior dreta, fes clic a "Edit Data Source".

  2. Afegir un nou camp calculat:

    • Fes clic a "Add a Field".

  3. Definir el camp calculat:

    • Nom del camp: Marge de Benefici (%)
    • Fórmula: ((SUM(Vendes Totals) - SUM(Costos Totals)) / SUM(Vendes Totals)) * 100

  4. Guardar i aplicar:

    • Fes clic a "Save" per guardar el nou camp calculat.
    • Utilitza aquest camp en els teus gràfics i taules per visualitzar el marge de benefici com a percentatge.

Errors comuns i consells

Errors comuns:

  • Referències incorrectes: Assegura't que els noms dels camps utilitzats a la fórmula coincideixin exactament amb els noms dels camps a la font de dades.
  • Divisió per zero: Quan es fan divisions, assegura't que el denominador no sigui zero per evitar errors.
  • Tipus de dades incompatibles: Assegura't que les operacions matemàtiques es facin sobre dades numèriques i les operacions de text sobre dades de text.

Consells:

  • Utilitza comentaris: Pots afegir comentaris a les teves fórmules per fer-les més comprensibles. Utilitza /* comentari */ per afegir comentaris.
  • Prova les fórmules: Abans d'aplicar una fórmula complexa, prova-la amb dades simples per assegurar-te que funciona correctament.
  • Documenta els camps calculats: Mantingues un registre dels camps calculats creats i les seves fórmules per facilitar el manteniment i la col·laboració.

Conclusió

Els camps calculats són una eina essencial per personalitzar i enriquir els informes a Google Data Studio. Amb una comprensió clara de com crear i utilitzar aquests camps, pots transformar les teves dades en informació valuosa i accionable. Practica amb diferents fórmules i funcions per descobrir tot el potencial dels camps calculats en els teus informes.
