Creació d'un informe des de zero
Següent ►

En aquest tema, aprendrem a crear un informe des de zero a Google Data Studio. Aquest procés inclou la configuració inicial, la connexió de fonts de dades, l'afegiment de components bàsics i la personalització de l'informe per satisfer les nostres necessitats específiques.

1. Configuració inicial

1.1. Accedir a Google Data Studio

  1. Obre el navegador web i ves a Google Data Studio.
  2. Inicia sessió amb el teu compte de Google.

1.2. Crear un nou informe

  1. A la pàgina principal de Google Data Studio, fes clic a "Blanc" per crear un nou informe.
  2. Se't demanarà que afegeixis una font de dades. Pots seleccionar una font de dades existent o crear-ne una de nova.

2. Connexió de fonts de dades

2.1. Seleccionar una font de dades

  1. Fes clic a "Afegeix una font de dades".
  2. Tria una de les fonts de dades disponibles (per exemple, Google Analytics, Google Sheets, BigQuery, etc.).
  3. Autoritza l'accés a la font de dades si és necessari.

2.2. Configurar la font de dades

  1. Selecciona les mètriques i dimensions que vols incloure a l'informe.
  2. Fes clic a "Afegeix" per afegir la font de dades a l'informe.

3. Afegir components bàsics

3.1. Afegir un títol

  1. Fes clic a "Afegeix un component" a la barra d'eines.
  2. Selecciona "Text" i arrossega'l a la part superior de l'informe.
  3. Escriu el títol de l'informe (per exemple, "Informe de Vendes").

3.2. Afegir un gràfic

  1. Fes clic a "Afegeix un component".
  2. Selecciona el tipus de gràfic que vols afegir (per exemple, "Gràfic de barres").
  3. Arrossega el gràfic a l'informe i ajusta la mida segons sigui necessari.
  4. Configura les dimensions i mètriques del gràfic utilitzant el panell de propietats a la dreta.

3.3. Afegir una taula

  1. Fes clic a "Afegeix un component".
  2. Selecciona "Taula".
  3. Arrossega la taula a l'informe i ajusta la mida.
  4. Configura les dimensions i mètriques de la taula utilitzant el panell de propietats.

4. Personalització de l'informe

4.1. Canviar el tema i l'estil

  1. Fes clic a "Tema i Estil" a la barra lateral dreta.
  2. Selecciona un tema predefinit o personalitza els colors, les fonts i altres elements d'estil segons les teves preferències.

4.2. Afegir filtres

  1. Fes clic a "Afegeix un component".
  2. Selecciona "Control de dades" i tria el tipus de filtre (per exemple, "Interval de dates").
  3. Arrossega el filtre a l'informe i configura les opcions segons sigui necessari.

5. Guardar i visualitzar l'informe

5.1. Guardar l'informe

  1. Fes clic a "Fitxer" a la barra de menú superior.
  2. Selecciona "Desa" i dona un nom a l'informe.

5.2. Visualitzar l'informe

  1. Fes clic a "Visualitza" a la part superior dreta de la pantalla.
  2. Revisa l'informe per assegurar-te que tots els components es mostren correctament i que les dades es visualitzen com esperaves.

Exercici pràctic

Objectiu

Crear un informe bàsic que mostri les vendes per producte utilitzant dades de Google Sheets.

Passos

  1. Crea un nou informe a Google Data Studio.
  2. Connecta una font de dades de Google Sheets que contingui informació de vendes per producte.
  3. Afegeix un títol a l'informe.
  4. Afegeix un gràfic de barres que mostri les vendes per producte.
  5. Afegeix una taula que llisti els productes i les seves vendes.
  6. Personalitza l'informe canviant el tema i afegint un filtre d'interval de dates.
  7. Guarda l'informe i visualitza'l.

Solució

  1. Accedeix a Google Data Studio i crea un nou informe.
  2. Connecta la font de dades de Google Sheets seleccionant el fitxer adequat.
  3. Afegeix un títol com "Informe de Vendes per Producte".
  4. Afegeix un gràfic de barres i configura'l per mostrar les vendes per producte.
  5. Afegeix una taula amb les dimensions i mètriques corresponents.
  6. Canvia el tema a un que t'agradi i afegeix un filtre d'interval de dates.
  7. Desa l'informe amb un nom adequat i visualitza'l per assegurar-te que tot es mostra correctament.

Resum

En aquesta secció, hem après a crear un informe des de zero a Google Data Studio. Hem cobert la configuració inicial, la connexió de fonts de dades, l'afegiment de components bàsics com gràfics i taules, i la personalització de l'informe. També hem practicat aquests conceptes amb un exercici pràctic. Ara estàs preparat per crear els teus propis informes bàsics a Google Data Studio.
