Programa Hello World

Següent ►
►

En aquest tema, aprendrem a escriure el nostre primer programa en Perl: el clàssic "Hello World". Aquest programa és un punt de partida ideal per familiaritzar-nos amb la sintaxi bàsica de Perl i el procés d'execució de scripts.

Objectius

  • Escriure un programa senzill en Perl.
  • Executar el programa des de la línia de comandes.
  • Entendre la sintaxi bàsica de Perl.

  1. Creació del Fitxer del Programa

Pas 1: Obrir un Editor de Text

Per començar, necessitem un editor de text. Podeu utilitzar qualsevol editor de text com ara Notepad, Sublime Text, Visual Studio Code, o qualsevol altre que preferiu.

Pas 2: Escriure el Codi

Escriviu el següent codi en el vostre editor de text:

#!/usr/bin/perl
# Aquest és el nostre primer programa en Perl

print "Hello, World!\n";

Explicació del Codi

  • #!/usr/bin/perl: Aquesta línia és coneguda com a "shebang" i indica al sistema operatiu que utilitzi l'intèrpret de Perl per executar aquest script.
  • # Aquest és el nostre primer programa en Perl: Aquesta línia és un comentari. Els comentaris en Perl comencen amb el símbol # i són ignorats per l'intèrpret.
  • print "Hello, World!\n";: Aquesta línia utilitza la funció print per mostrar el text "Hello, World!" a la pantalla. El \n és un caràcter de nova línia que fa que el cursor es mogui a la següent línia després de imprimir el text.

Pas 3: Guardar el Fitxer

Guardeu el fitxer amb el nom hello_world.pl. L'extensió .pl és una convenció per indicar que es tracta d'un script Perl.

  1. Executar el Programa

Pas 1: Obrir la Línia de Comandes

Obriu la línia de comandes del vostre sistema operatiu. A Windows, podeu utilitzar el "Símbol del sistema" o "PowerShell". A macOS i Linux, podeu utilitzar el "Terminal".

Pas 2: Navegar al Directori del Fitxer

Utilitzeu la comanda cd per navegar al directori on heu guardat el fitxer hello_world.pl. Per exemple:

cd /path/to/your/file

Pas 3: Executar el Fitxer

Executeu el fitxer amb la següent comanda:

perl hello_world.pl

Resultat Esperat

Si tot ha anat bé, hauríeu de veure el següent resultat a la pantalla:

Hello, World!

Exercici Pràctic

Exercici 1: Modificar el Missatge

Modifiqueu el programa per imprimir un missatge diferent, per exemple, "Hola, món!". Guardeu el fitxer i executeu-lo de nou per veure el nou missatge.

Solució

#!/usr/bin/perl
# Aquest és el nostre primer programa en Perl

print "Hola, món!\n";

Errors Comuns i Consells

Error 1: Permisos d'Execució (Linux/Mac)

Si obteniu un error de permisos, assegureu-vos que el fitxer té permisos d'execució. Podeu canviar els permisos amb la següent comanda:

chmod +x hello_world.pl

Error 2: Ruta Incorrecta de Perl

Si obteniu un error que indica que no es pot trobar l'intèrpret de Perl, assegureu-vos que la ruta especificada en la línia shebang (#!/usr/bin/perl) és correcta. Podeu trobar la ruta correcta executant which perl a la línia de comandes.

Conclusió

En aquest tema, hem après a escriure i executar un programa senzill en Perl. Aquest és el primer pas per familiaritzar-nos amb la sintaxi i l'entorn de Perl. En els següents temes, aprofundirem en conceptes més avançats i funcionalitats de Perl.
