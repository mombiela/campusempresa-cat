Configuració de l'Entorn
Següent ►

En aquesta secció, aprendrem com configurar l'entorn necessari per començar a programar en Perl. Això inclou la instal·lació de Perl al teu sistema, la configuració d'un editor de text adequat i la verificació de la instal·lació.

  1. Instal·lació de Perl

1.1. Windows

  1. Descarregar Strawberry Perl:

    • Visita el lloc web oficial de Strawberry Perl: Strawberry Perl.
    • Descarrega l'instal·lador adequat per a la teva versió de Windows (32-bit o 64-bit).

  2. Instal·lar Strawberry Perl:

    • Executa l'instal·lador descarregat.
    • Segueix les instruccions de l'assistent d'instal·lació. Accepta els valors per defecte.

  3. Verificar la Instal·lació:

    • Obre una finestra de símbol del sistema (cmd).
    • Escriu perl -v i prem Enter.
    • Hauries de veure informació sobre la versió de Perl instal·lada.

1.2. macOS

  1. Utilitzar Perl Preinstal·lat:

    • macOS ve amb Perl preinstal·lat. Per verificar-ho, obre el Terminal i escriu perl -v.

  2. Instal·lar Perl amb Homebrew (Opcional):

    • Si vols una versió més recent de Perl, pots utilitzar Homebrew.
    • Instal·la Homebrew si no el tens: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instal·la Perl: brew install perl
    • Verifica la instal·lació: perl -v

1.3. Linux

  1. Utilitzar Perl Preinstal·lat:

    • La majoria de distribucions Linux venen amb Perl preinstal·lat. Per verificar-ho, obre el terminal i escriu perl -v.

  2. Instal·lar Perl amb el Gestor de Paquets:

    • Debian/Ubuntu: sudo apt-get install perl
    • Fedora: sudo dnf install perl
    • Arch Linux: sudo pacman -S perl
    • Verifica la instal·lació: perl -v

  1. Configuració de l'Editor de Text

2.1. Editors Recomanats

  • Visual Studio Code (VS Code):

    • Descarrega i instal·la VS Code des del lloc oficial: Visual Studio Code.
    • Instal·la l'extensió "Perl" per a suport de sintaxi i altres funcionalitats.

  • Sublime Text:

    • Descarrega i instal·la Sublime Text des del lloc oficial: Sublime Text.
    • Instal·la el paquet "Perl" per a suport de sintaxi.

  • Atom:

    • Descarrega i instal·la Atom des del lloc oficial: Atom.
    • Instal·la el paquet "language-perl" per a suport de sintaxi.

2.2. Configuració Bàsica

  1. Visual Studio Code:

    • Obre VS Code.
    • Ves a Extensions (Ctrl+Shift+X).
    • Cerca "Perl" i instal·la l'extensió "Perl" de Gerald Richter.

  2. Sublime Text:

    • Obre Sublime Text.
    • Ves a Preferences > Package Control.
    • Cerca "Perl" i instal·la el paquet.

  3. Atom:

    • Obre Atom.
    • Ves a File > Settings > Install.
    • Cerca "language-perl" i instal·la el paquet.

  1. Verificació de la Configuració

3.1. Crear un Programa Senzill

  1. Obre el teu editor de text preferit.
  2. Crea un nou fitxer i anomena'l hello_world.pl.
  3. Escriu el següent codi:
#!/usr/bin/perl
use strict;
use warnings;

print "Hello, World!\n";

3.2. Executar el Programa

  1. Obre una terminal o símbol del sistema.
  2. Navega fins al directori on has guardat el fitxer hello_world.pl.
  3. Executa el programa amb la comanda:
perl hello_world.pl
  1. Hauries de veure la sortida:
Hello, World!

  1. Resum

En aquesta secció, hem après com instal·lar Perl en diferents sistemes operatius, configurar un editor de text adequat i verificar la instal·lació amb un programa senzill. Ara estàs preparat per començar a programar en Perl!

En el següent tema, explorarem el nostre primer programa en Perl amb més detall.
