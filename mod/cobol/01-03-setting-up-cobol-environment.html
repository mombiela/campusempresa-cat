Configuració de l'Entorn COBOL
Següent ►

En aquesta secció, aprendrem com configurar l'entorn necessari per començar a programar en COBOL. Aquest procés inclou la instal·lació del compilador COBOL, la configuració de l'editor de codi i la verificació de la instal·lació.

  1. Instal·lació del Compilador COBOL

Opcions de Compiladors COBOL

Hi ha diversos compiladors COBOL disponibles, tant comercials com de codi obert. A continuació, es presenten algunes opcions populars:

Compilador Tipus Descripció
GnuCOBOL Codi obert Un compilador COBOL gratuït i de codi obert.
Micro Focus COBOL Comercial Un dels compiladors COBOL més utilitzats en la indústria.
IBM COBOL Comercial Utilitzat principalment en sistemes mainframe.
Fujitsu COBOL Comercial Ofereix eines avançades per al desenvolupament COBOL.

Instal·lació de GnuCOBOL

Per a aquest curs, utilitzarem GnuCOBOL, ja que és gratuït i fàcil d'instal·lar. A continuació es mostren els passos per instal·lar GnuCOBOL en diferents sistemes operatius.

Instal·lació en Windows

  1. Descarregar GnuCOBOL:

    • Visita la pàgina oficial de GnuCOBOL: GnuCOBOL
    • Descarrega l'instal·lador per a Windows.

  2. Instal·lar GnuCOBOL:

    • Executa l'instal·lador descarregat.
    • Segueix les instruccions de l'assistent d'instal·lació.

  3. Verificar la Instal·lació:

    • Obre una finestra de comandaments (cmd).
    • Escriu cobc -v i prem Enter.
    • Hauries de veure la versió de GnuCOBOL instal·lada.

Instal·lació en Linux

  1. Instal·lar GnuCOBOL:

    • Obre una terminal.
    • Executa la següent comanda per instal·lar GnuCOBOL: 
      sudo apt-get install open-cobol

  2. Verificar la Instal·lació:

    • Escriu cobc -v a la terminal i prem Enter.
    • Hauries de veure la versió de GnuCOBOL instal·lada.

Instal·lació en macOS

  1. Instal·lar Homebrew (si no el tens instal·lat):

    • Obre una terminal.
    • Executa la següent comanda: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instal·lar GnuCOBOL:

    • Executa la següent comanda: 
      brew install gnu-cobol

  3. Verificar la Instal·lació:

    • Escriu cobc -v a la terminal i prem Enter.
    • Hauries de veure la versió de GnuCOBOL instal·lada.

  1. Configuració de l'Editor de Codi

Per escriure codi COBOL, necessitem un editor de codi. Hi ha molts editors disponibles, però recomanem Visual Studio Code (VS Code) per la seva flexibilitat i extensibilitat.

Instal·lació de Visual Studio Code

  1. Descarregar VS Code:

    • Visita la pàgina oficial de Visual Studio Code: Visual Studio Code
    • Descarrega l'instal·lador per al teu sistema operatiu.

  2. Instal·lar VS Code:

    • Executa l'instal·lador descarregat.
    • Segueix les instruccions de l'assistent d'instal·lació.

Configuració de l'Entorn COBOL en VS Code

  1. Instal·lar l'Extensió COBOL:

    • Obre VS Code.
    • Ves a la pestanya d'extensions (icona de quadrat amb quatre quadrats petits).
    • Cerca "COBOL" i instal·la l'extensió "Bitlang COBOL".

  2. Configurar l'Extensió:

    • Després d'instal·lar l'extensió, VS Code hauria de reconèixer els fitxers COBOL (.cbl, .cob).
    • Pots personalitzar la configuració de l'extensió segons les teves necessitats a la secció de configuració de VS Code.

  1. Verificació de la Instal·lació

Per assegurar-nos que tot està configurat correctament, crearem i compilarem un programa COBOL senzill.

Crear un Programa COBOL Senzill

  1. Crear un Fitxer COBOL:

    • Obre VS Code.
    • Crea un nou fitxer i anomena'l hello.cob.

  2. Escriure el Codi:

    • Copia i enganxa el següent codi al fitxer hello.cob: 
      IDENTIFICATION DIVISION.
PROGRAM-ID. HELLO-WORLD.
PROCEDURE DIVISION.
DISPLAY 'Hello, World!'.
STOP RUN.

Compilar i Executar el Programa

  1. Compilar el Programa:

    • Obre una terminal dins de VS Code (Terminal > New Terminal).
    • Navega fins al directori on has guardat el fitxer hello.cob.
    • Executa la següent comanda per compilar el programa: 
      cobc -x -o hello hello.cob

  2. Executar el Programa:

    • Executa el fitxer compilat: 
      ./hello
    • Hauries de veure el missatge Hello, World! a la terminal.

Conclusió

Ara que has configurat l'entorn COBOL, estàs preparat per començar a programar. En la següent secció, explorarem l'estructura bàsica d'un programa COBOL. Assegura't de tenir el teu entorn configurat correctament per seguir els exemples i exercicis pràctics que vindran.
