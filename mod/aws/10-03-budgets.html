Introducció a AWS Budgets

AWS Budgets és una eina poderosa que permet als usuaris d'AWS establir pressupostos personalitzats per controlar i gestionar els costos i l'ús dels seus serveis AWS. Aquesta eina proporciona alertes quan els costos o l'ús superen els llindars establerts, ajudant a mantenir el control financer i a evitar sorpreses desagradables a la factura.

Objectius d'aprenentatge

En aquesta secció, aprendràs a:

  1. Crear i configurar un pressupost a AWS Budgets.
  2. Establir alertes per a costos i ús.
  3. Monitoritzar i gestionar els pressupostos.
  4. Utilitzar AWS Budgets per optimitzar els costos.

Crear i Configurar un Pressupost

Pas 1: Accedir a AWS Budgets

  1. Inicia sessió a la Consola de Gestió d'AWS.
  2. Navega a la secció de Billing and Cost Management.
  3. Selecciona Budgets al menú lateral.

Pas 2: Crear un Nou Pressupost

  1. Fes clic a Create a budget.
  2. Selecciona el tipus de pressupost que vols crear:
    • Cost budget: Per controlar els costos.
    • Usage budget: Per controlar l'ús dels serveis.
    • Reservation budget: Per controlar les reserves de recursos.

Pas 3: Configurar els Detalls del Pressupost

  1. Budget Name: Introdueix un nom descriptiu per al pressupost.
  2. Period: Selecciona la freqüència del pressupost (mensual, trimestral, anual).
  3. Start and End Date: Defineix les dates d'inici i finalització del pressupost.
  4. Amount: Estableix l'import màxim del pressupost.

Pas 4: Establir Alertes

  1. Thresholds: Defineix els llindars per a les alertes (per exemple, 80% i 100% del pressupost).
  2. Notification Type: Selecciona el tipus de notificació (correu electrònic, SNS).
  3. Recipients: Introdueix les adreces de correu electrònic dels destinataris de les alertes.

Pas 5: Revisar i Crear

  1. Revisa tots els detalls del pressupost.
  2. Fes clic a Create budget per finalitzar la configuració.

Monitoritzar i Gestionar els Pressupostos

Visualitzar els Pressupostos

  1. A la pàgina principal d'AWS Budgets, pots veure una llista de tots els pressupostos creats.
  2. Selecciona un pressupost per veure els detalls, incloent-hi els costos actuals, l'ús i les alertes.

Actualitzar un Pressupost

  1. Selecciona el pressupost que vols actualitzar.
  2. Fes clic a Edit per modificar els detalls del pressupost, com ara l'import, els llindars d'alerta o els destinataris de les notificacions.

Eliminar un Pressupost

  1. Selecciona el pressupost que vols eliminar.
  2. Fes clic a Delete i confirma l'eliminació.

Optimitzar els Costos amb AWS Budgets

Anàlisi de Costos

  1. Utilitza AWS Budgets per identificar els serveis que generen més costos.
  2. Revisa els informes de costos per detectar patrons i tendències.

Ajustar els Pressupostos

  1. Basant-te en l'anàlisi de costos, ajusta els pressupostos per reflectir millor les necessitats del teu projecte.
  2. Estableix nous llindars d'alerta per a un control més estricte.

Consells Addicionals

  • Utilitza AWS Cost Explorer: Per obtenir una visió més detallada dels costos i l'ús.
  • Revisa Regularment: Assegura't de revisar els pressupostos regularment per ajustar-los segons sigui necessari.
  • Automatitza les Alertes: Configura alertes automàtiques per rebre notificacions immediates quan s'assoleixin els llindars.

Exercici Pràctic

Objectiu

Crear un pressupost mensual de $100 per als serveis d'AWS i configurar alertes al 80% i 100% del pressupost.

Passos

  1. Accedeix a la Consola de Gestió d'AWS i navega a Billing and Cost Management.
  2. Selecciona Budgets i fes clic a Create a budget.
  3. Tria Cost budget i introdueix els detalls següents:
    • Budget Name: "Monthly AWS Budget"
    • Period: Monthly
    • Start Date: Today
    • End Date: No end date
    • Amount: 100
  4. Configura les alertes:
    • 80% Threshold: Notificació per correu electrònic a tu mateix.
    • 100% Threshold: Notificació per correu electrònic a tu mateix.
  5. Revisa els detalls i fes clic a Create budget.

Solució

1. Accedeix a la Consola de Gestió d'AWS.
2. Navega a **Billing and Cost Management** > **Budgets**.
3. Fes clic a **Create a budget**.
4. Selecciona **Cost budget**.
5. Introdueix els detalls:
   - **Budget Name**: "Monthly AWS Budget"
   - **Period**: Monthly
   - **Start Date**: Today
   - **End Date**: No end date
   - **Amount**: 100
6. Configura les alertes:
   - **80% Threshold**: Notificació per correu electrònic a tu mateix.
   - **100% Threshold**: Notificació per correu electrònic a tu mateix.
7. Revisa els detalls i fes clic a **Create budget**.

Conclusió

AWS Budgets és una eina essencial per gestionar i controlar els costos dels serveis AWS. Amb la capacitat de crear pressupostos personalitzats i establir alertes, pots assegurar-te que els teus projectes es mantenen dins del pressupost i evitar sorpreses a la factura. Practica la creació i gestió de pressupostos per familiaritzar-te amb aquesta eina i optimitzar els teus costos a AWS.
