Consola de gestió d'AWS
Següent ►

Introducció

La Consola de gestió d'AWS és una interfície web que permet als usuaris accedir i gestionar els serveis d'AWS. Aquesta eina és essencial per a qualsevol persona que vulgui treballar amb AWS, ja que proporciona una manera intuïtiva i visual de gestionar els recursos i serveis.

Objectius d'aprenentatge

En aquesta secció, aprendràs:

  • Com accedir a la Consola de gestió d'AWS.
  • Les funcionalitats bàsiques de la consola.
  • Com navegar per la interfície.
  • Com utilitzar la consola per gestionar els serveis d'AWS.

Accés a la Consola de gestió d'AWS

Passos per accedir a la consola

  1. Inicia sessió al teu compte d'AWS:

    • Ves a AWS Management Console.
    • Fes clic a "Sign In to the Console".
    • Introdueix les teves credencials (usuari i contrasenya).

  2. Pantalla d'inici:

    • Un cop iniciada la sessió, seràs redirigit a la pantalla d'inici de la consola.

Navegació per la interfície

Components principals de la consola

  1. Barra de cerca:

    • Situada a la part superior de la pantalla, permet buscar serveis ràpidament.

  2. Menú de serveis:

    • A l'esquerra de la barra de cerca, aquest menú desplegable llista tots els serveis d'AWS disponibles.

  3. Panell de navegació:

    • A la part esquerra de la pantalla, proporciona accés ràpid als serveis utilitzats recentment i als recursos favorits.

  4. Panell principal:

    • La part central de la pantalla on es mostren els detalls dels serveis seleccionats.

  5. Barra de notificacions:

    • A la part superior dreta, mostra notificacions importants sobre el teu compte i els teus serveis.

Exemple pràctic: Navegar a Amazon S3

  1. Utilitza la barra de cerca:

    • Escriu "S3" a la barra de cerca i selecciona "S3" dels resultats.

  2. Explora el servei:

    • Seràs redirigit a la pàgina principal d'Amazon S3, on podràs veure els teus buckets i gestionar els teus objectes.

Funcionalitats bàsiques

Crear un bucket a Amazon S3

  1. Accedeix a Amazon S3:

    • Utilitza la barra de cerca o el menú de serveis per accedir a Amazon S3.

  2. Crea un bucket:

    • Fes clic a "Create bucket".
    • Introdueix un nom únic per al bucket.
    • Selecciona la regió on vols crear el bucket.
    • Configura les opcions addicionals segons les teves necessitats.
    • Fes clic a "Create bucket".
# Exemple de codi per pujar un fitxer a S3 utilitzant boto3 (SDK per a Python)

import boto3

# Crea una sessió de boto3
s3 = boto3.client('s3')

# Defineix el nom del bucket i el fitxer a pujar
bucket_name = 'el-meu-bucket'
file_name = 'path/al/meu/fitxer.txt'
object_name = 'fitxer.txt'

# Puja el fitxer
s3.upload_file(file_name, bucket_name, object_name)

Gestió d'instàncies EC2

  1. Accedeix a Amazon EC2:

    • Utilitza la barra de cerca o el menú de serveis per accedir a Amazon EC2.

  2. Llança una nova instància:

    • Fes clic a "Launch Instance".
    • Selecciona una AMI (Amazon Machine Image).
    • Tria un tipus d'instància.
    • Configura els detalls de la instància.
    • Afegeix emmagatzematge.
    • Configura els grups de seguretat.
    • Revisa i llança la instància.
# Exemple de codi per llançar una instància EC2 utilitzant boto3

import boto3

# Crea una sessió de boto3
ec2 = boto3.client('ec2')

# Llança una nova instància
response = ec2.run_instances(
    ImageId='ami-0abcdef1234567890',  # Substitueix per una AMI vàlida
    InstanceType='t2.micro',
    MinCount=1,
    MaxCount=1,
    KeyName='el-meu-key-pair'  # Substitueix per un key pair vàlid
)

# Mostra l'ID de la instància llançada
print('Instància llançada amb ID:', response['Instances'][0]['InstanceId'])

Exercicis pràctics

Exercici 1: Crear un bucket a Amazon S3

  1. Accedeix a la Consola de gestió d'AWS.
  2. Navega a Amazon S3.
  3. Crea un nou bucket amb un nom únic.
  4. Puja un fitxer al bucket utilitzant la consola.

Exercici 2: Llançar una instància EC2

  1. Accedeix a la Consola de gestió d'AWS.
  2. Navega a Amazon EC2.
  3. Llança una nova instància utilitzant una AMI de la teva elecció.
  4. Configura els detalls de la instància i llança-la.

Resum

En aquesta secció, has après a accedir i navegar per la Consola de gestió d'AWS. Has après a utilitzar les funcionalitats bàsiques de la consola per gestionar serveis com Amazon S3 i Amazon EC2. A més, has practicat la creació de buckets i el llançament d'instàncies. Aquestes habilitats són fonamentals per treballar amb AWS de manera eficient i efectiva.
