Reserves d'Azure

Següent

Introducció

Les reserves d'Azure permeten als usuaris estalviar diners en els seus serveis d'Azure mitjançant el pagament anticipat per un període d'un o tres anys. Aquest enfocament és ideal per a càrregues de treball que tenen un ús constant i previsible. En aquesta secció, explorarem com funcionen les reserves d'Azure, els seus beneficis, i com configurar-les.

Conceptes Clau

  1. Reserves de màquines virtuals (VMs): Permet estalviar fins a un 72% en comparació amb els preus de pagament per ús.
  2. Reserves de bases de dades: Inclou serveis com Azure SQL Database, Azure Cosmos DB, i altres.
  3. Reserves de capacitat de computació: Per a serveis com Azure Synapse Analytics, Azure Databricks, etc.
  4. Períodes de reserva: Opcions de reserva per un o tres anys.
  5. Flexibilitat de reserva: Capacitat per canviar o cancel·lar reserves segons les necessitats.

Beneficis de les Reserves d'Azure

  • Estalvi de costos: Reducció significativa en els costos en comparació amb els preus de pagament per ús.
  • Previsibilitat de costos: Facilita la planificació financera amb costos fixos.
  • Flexibilitat: Opcions per canviar o cancel·lar reserves si les necessitats canvien.

Configuració de Reserves d'Azure

Pas 1: Accedir al Portal d'Azure

  1. Inicia sessió al Portal d'Azure.
  2. Navega a la secció de "Reserves" des del menú principal.

Pas 2: Seleccionar el Tipus de Reserva

  1. Tria el tipus de reserva que necessites (per exemple, màquines virtuals, bases de dades, etc.).
  2. Selecciona la regió i la configuració específica del servei.

Pas 3: Configurar els Detalls de la Reserva

  1. Període de reserva: Tria entre un o tres anys.
  2. Quantitat: Defineix el nombre d'instàncies o la capacitat que vols reservar.
  3. Opcions de pagament: Selecciona el mètode de pagament (pagament únic o mensual).

Pas 4: Revisar i Confirmar

  1. Revisa els detalls de la reserva.
  2. Confirma la compra per completar la reserva.

Exemple Pràctic

A continuació, es mostra un exemple de com reservar una màquina virtual a Azure:

1. Inicia sessió al Portal d'Azure.
2. Navega a "Reserves" i selecciona "Afegeix".
3. Tria "Màquines Virtuals".
4. Selecciona la regió (per exemple, "East US").
5. Tria el tipus de màquina virtual (per exemple, "Standard_D2s_v3").
6. Defineix el període de reserva (per exemple, "1 any").
7. Especifica la quantitat (per exemple, "5 instàncies").
8. Selecciona el mètode de pagament.
9. Revisa els detalls i confirma la compra.

Exercici Pràctic

Exercici 1: Reserva d'una Màquina Virtual

Objectiu: Reserva una màquina virtual a Azure per un període d'un any.

Passos:

  1. Inicia sessió al Portal d'Azure.
  2. Navega a "Reserves".
  3. Selecciona "Afegeix" i tria "Màquines Virtuals".
  4. Selecciona la regió "West Europe".
  5. Tria el tipus de màquina virtual "Standard_B2s".
  6. Defineix el període de reserva com "1 any".
  7. Especifica la quantitat com "3 instàncies".
  8. Selecciona el mètode de pagament.
  9. Revisa els detalls i confirma la compra.

Solució: Segueix els passos indicats per completar la reserva. Si tens dubtes, revisa l'exemple pràctic anterior.

Errors Comuns i Consells

  • Error: No revisar els detalls de la reserva abans de confirmar.
    • Consell: Sempre revisa els detalls per assegurar-te que la configuració és correcta.
  • Error: No considerar la flexibilitat necessària.
    • Consell: Avalua si necessites la capacitat de canviar o cancel·lar la reserva abans de fer la compra.

Conclusió

Les reserves d'Azure són una eina poderosa per optimitzar els costos i garantir la previsibilitat financera. Amb una comprensió clara dels conceptes clau i els passos per configurar-les, pots aprofitar al màxim els beneficis que ofereixen. En la següent secció, explorarem l'Assessor d'Azure per ajudar-te a optimitzar encara més els teus costos.
