Configuració del compte d'Azure
Següent ►

En aquest tema, aprendràs a configurar un compte d'Azure, que és el primer pas per començar a utilitzar els serveis d'Azure. Aquest procés inclou la creació d'un compte, la configuració de la subscripció i la comprensió dels diferents tipus de subscripcions disponibles.

  1. Creació d'un compte d'Azure

Pas 1: Accedir al lloc web d'Azure

  1. Obre el teu navegador web preferit.
  2. Ves a Azure Portal.

Pas 2: Registrar-se o iniciar sessió

  1. Si ja tens un compte de Microsoft, fes clic a "Sign in" i introdueix les teves credencials.
  2. Si no tens un compte de Microsoft, fes clic a "Create one!" i segueix les instruccions per crear un compte nou.

Pas 3: Completar el registre

  1. Introdueix la teva informació personal, com ara el nom, l'adreça de correu electrònic i el número de telèfon.
  2. Verifica el teu compte mitjançant el codi que rebràs per correu electrònic o SMS.
  3. Proporciona la informació de pagament. Azure ofereix una prova gratuïta amb un crèdit inicial, però necessitaràs introduir una targeta de crèdit per verificar la teva identitat. No se't cobrarà durant el període de prova gratuïta.

  1. Configuració de la subscripció

Tipus de subscripcions

Azure ofereix diversos tipus de subscripcions, cadascuna amb diferents característiques i preus. Els tipus més comuns són:

  • Prova gratuïta: Inclou un crèdit inicial que pots utilitzar durant un període limitat.
  • Pay-As-You-Go: Pagues només pels recursos que utilitzes.
  • Enterprise Agreement: Per a grans empreses amb necessitats específiques i un volum alt de consum.

Pas 1: Seleccionar una subscripció

  1. Després de completar el registre, se't demanarà que seleccionis una subscripció.
  2. Tria la subscripció que millor s'adapti a les teves necessitats. Si ets nou a Azure, et recomanem començar amb la prova gratuïta.

Pas 2: Configurar la subscripció

  1. Un cop seleccionada la subscripció, se't demanarà que configuris alguns paràmetres bàsics, com ara la regió on es desplegaran els teus recursos.
  2. Revisa i accepta els termes i condicions.

  1. Exploració del portal d'Azure

Pas 1: Accedir al portal

  1. Un cop configurada la teva subscripció, seràs redirigit al portal d'Azure.
  2. El portal és la interfície principal on gestionaràs tots els teus recursos d'Azure.

Pas 2: Familiaritzar-se amb el portal

  1. Dedica uns minuts a explorar el portal. Observa les diferents seccions, com ara el tauler de control, els serveis disponibles i les opcions de configuració.
  2. Prova de crear un recurs senzill, com ara una màquina virtual o un compte d'emmagatzematge, per familiaritzar-te amb el procés.

  1. Exercici pràctic

Objectiu

Configurar un compte d'Azure i crear una màquina virtual bàsica.

Instruccions

  1. Segueix els passos descrits anteriorment per crear un compte d'Azure i configurar la teva subscripció.
  2. Un cop al portal d'Azure, fes clic a "Create a resource".
  3. Selecciona "Virtual Machine" i segueix les instruccions per crear una màquina virtual bàsica.
  4. Utilitza les opcions predeterminades per a la majoria dels paràmetres, però assegura't de seleccionar una regió propera a la teva ubicació.
  5. Un cop creada la màquina virtual, accedeix-hi mitjançant RDP (Remote Desktop Protocol) o SSH, segons el sistema operatiu que hagis seleccionat.

Solució

  1. Després de crear la màquina virtual, hauràs de veure-la llistada al teu tauler de control del portal d'Azure.
  2. Fes clic a la màquina virtual per veure els seus detalls i opcions de gestió.
  3. Utilitza les credencials que vas configurar durant la creació per accedir-hi.

Conclusió

En aquesta secció, has après a configurar un compte d'Azure, seleccionar una subscripció i familiaritzar-te amb el portal d'Azure. També has realitzat un exercici pràctic per crear una màquina virtual bàsica. Aquestes habilitats són fonamentals per començar a treballar amb Azure i aprofitar els seus serveis al màxim. En el següent tema, explorarem més a fons el portal d'Azure i les seves funcionalitats.
