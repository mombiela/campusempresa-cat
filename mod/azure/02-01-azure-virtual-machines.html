Màquines virtuals d'Azure
Següent ►

Introducció

Les màquines virtuals (VMs) d'Azure són un dels serveis més utilitzats dins de la plataforma de núvol d'Azure. Proporcionen una infraestructura com a servei (IaaS) que permet als usuaris crear i gestionar màquines virtuals a la carta. Aquest mòdul cobreix els conceptes bàsics de les VMs d'Azure, incloent-hi la seva configuració, gestió i ús.

Objectius del Mòdul

  1. Entendre què són les màquines virtuals d'Azure.
  2. Aprendre a crear i configurar una màquina virtual.
  3. Conèixer les opcions de gestió i manteniment de les VMs.
  4. Explorar les millors pràctiques per a l'ús de VMs en Azure.

Contingut

  1. Què són les màquines virtuals d'Azure?

Les màquines virtuals d'Azure són instàncies de servidors virtuals que es poden utilitzar per executar aplicacions, emmagatzemar dades, i molt més. Ofereixen flexibilitat i escalabilitat, permetent als usuaris pagar només pel que utilitzen.

Característiques clau:

  • Escalabilitat: Les VMs es poden escalar verticalment (augmentant els recursos) o horitzontalment (afegint més instàncies).
  • Flexibilitat: Suporten diversos sistemes operatius, incloent-hi Windows i Linux.
  • Integració: Es poden integrar fàcilment amb altres serveis d'Azure com ara xarxes virtuals, emmagatzematge, i bases de dades.

  1. Creació d'una màquina virtual

Pas 1: Accedir al portal d'Azure

  1. Inicia sessió al portal d'Azure.
  2. Al menú de l'esquerra, selecciona "Màquines virtuals".

Pas 2: Crear una nova màquina virtual

  1. Fes clic a "Afegeix" i selecciona "Màquina virtual".
  2. Omple els detalls bàsics:
    • Subscripció: Selecciona la subscripció d'Azure.
    • Grup de recursos: Crea un nou grup de recursos o selecciona un existent.
    • Nom de la màquina virtual: Assigna un nom a la VM.
    • Regió: Selecciona la regió on vols desplegar la VM.
    • Imatge: Tria el sistema operatiu (Windows, Linux, etc.).
    • Mida: Selecciona la mida de la VM segons les necessitats de recursos.

Pas 3: Configurar les opcions de seguretat i xarxa

  1. Autenticació: Tria entre autenticació per contrasenya o clau SSH.
  2. Xarxa virtual: Selecciona una xarxa virtual existent o crea'n una de nova.
  3. Grup de seguretat de xarxa (NSG): Configura les regles de seguretat per permetre o denegar trànsit.

Pas 4: Revisar i crear

  1. Revisa la configuració.
  2. Fes clic a "Crea" per desplegar la màquina virtual.

  1. Gestió de màquines virtuals

Accions bàsiques:

  • Iniciar/Aturar: Pots iniciar o aturar la VM des del portal d'Azure.
  • Redimensionar: Canvia la mida de la VM per ajustar els recursos.
  • Monitoratge: Utilitza Azure Monitor per supervisar el rendiment i l'ús de la VM.

Exemples de codi per gestionar VMs amb Azure CLI:

# Iniciar una màquina virtual
az vm start --resource-group myResourceGroup --name myVM

# Aturar una màquina virtual
az vm stop --resource-group myResourceGroup --name myVM

# Redimensionar una màquina virtual
az vm resize --resource-group myResourceGroup --name myVM --size Standard_DS3_v2

  1. Millors pràctiques

  • Seguretat: Utilitza NSGs i Azure Security Center per protegir les teves VMs.
  • Còpies de seguretat: Configura còpies de seguretat regulars per evitar pèrdues de dades.
  • Escalabilitat: Utilitza escales automàtiques per ajustar els recursos segons la demanda.
  • Costos: Monitoritza i optimitza els costos utilitzant Azure Cost Management.

Exercicis Pràctics

Exercici 1: Crear una màquina virtual

  1. Accedeix al portal d'Azure.
  2. Crea una nova màquina virtual amb les següents especificacions:
    • Sistema operatiu: Ubuntu 20.04 LTS
    • Mida: Standard_B1s
    • Autenticació: Clau SSH
    • Xarxa virtual: Nova xarxa virtual amb subxarxa per defecte

Exercici 2: Gestionar una màquina virtual amb Azure CLI

  1. Inicia sessió a Azure CLI.
  2. Inicia la màquina virtual creada en l'exercici anterior.
  3. Redimensiona la màquina virtual a la mida Standard_DS1_v2.
  4. Atura la màquina virtual.

Solucions

Solució a l'Exercici 1:

  1. Accedeix al portal d'Azure.
  2. Segueix els passos descrits a la secció "Creació d'una màquina virtual" per crear una VM amb les especificacions donades.

Solució a l'Exercici 2:

# Iniciar sessió a Azure CLI
az login

# Iniciar la màquina virtual
az vm start --resource-group myResourceGroup --name myVM

# Redimensionar la màquina virtual
az vm resize --resource-group myResourceGroup --name myVM --size Standard_DS1_v2

# Aturar la màquina virtual
az vm stop --resource-group myResourceGroup --name myVM

Conclusió

En aquest mòdul, hem après què són les màquines virtuals d'Azure, com crear-les i gestionar-les, i hem explorat algunes de les millors pràctiques per al seu ús. Les VMs són una eina poderosa dins de l'ecosistema d'Azure, oferint flexibilitat i escalabilitat per a una àmplia varietat d'aplicacions. En el proper mòdul, explorarem els serveis d'aplicacions d'Azure, que proporcionen una plataforma gestionada per desplegar i escalar aplicacions web.
