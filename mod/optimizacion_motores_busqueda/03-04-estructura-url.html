Estructura de URL
Següent ►

Introducció

L'estructura de les URL és un component fonamental del SEO On-Page. Una URL ben estructurada no només ajuda els motors de cerca a comprendre millor el contingut d'una pàgina, sinó que també millora l'experiència de l'usuari. En aquesta secció, explorarem les millors pràctiques per crear URL amigables per als motors de cerca i els usuaris.

Conceptes Clau

  1. URL amigables: URL que són fàcils de llegir i entendre tant per als usuaris com per als motors de cerca.
  2. Paraules clau en la URL: Incloure paraules clau rellevants a la URL per millorar la seva visibilitat en els resultats de cerca.
  3. Estructura jeràrquica: Organitzar les URL de manera que reflecteixin l'estructura del lloc web.
  4. Longitud de la URL: Mantenir les URL curtes i descriptives.
  5. Separadors de paraules: Utilitzar guions (-) en lloc de guions baixos (_) per separar paraules.

Millors Pràctiques per a l'Estructura de URL

  1. Crear URL amigables

Les URL amigables són fàcils de llegir i entendre. Això significa evitar l'ús de caràcters especials, números i lletres sense sentit. Per exemple:

  • No amigable: https://www.exemple.com/p?12345
  • Amigable: https://www.exemple.com/productes/ordinadors-portatils

  1. Incloure Paraules Clau

Incloure paraules clau rellevants a la URL pot ajudar els motors de cerca a comprendre el contingut de la pàgina i millorar el seu posicionament. Per exemple:

  • Sense paraules clau: https://www.exemple.com/pagina1
  • Amb paraules clau: https://www.exemple.com/serveis/disseny-web

  1. Utilitzar una Estructura Jeràrquica

Organitzar les URL de manera que reflecteixin l'estructura del lloc web ajuda tant els usuaris com els motors de cerca a navegar pel lloc. Per exemple:

  • Estructura plana: https://www.exemple.com/pagina1, https://www.exemple.com/pagina2
  • Estructura jeràrquica: https://www.exemple.com/categoria/subcategoria/pagina

  1. Mantenir les URL Curtes

Les URL curtes són més fàcils de llegir, recordar i compartir. A més, els motors de cerca prefereixen URL més curtes. Per exemple:

  • Llarga: https://www.exemple.com/categoria-de-productes/ordinadors-portatils-de-gaming
  • Curta: https://www.exemple.com/productes/ordinadors-gaming

  1. Utilitzar Guions com a Separadors

Els guions (-) són preferits pels motors de cerca per separar paraules en una URL. Els guions baixos (_) no són tan fàcils de llegir per als motors de cerca. Per exemple:

  • Amb guions baixos: https://www.exemple.com/ordinadors_portatils
  • Amb guions: https://www.exemple.com/ordinadors-portatils

Exemples Pràctics

Exemple 1: URL per a una Pàgina de Producte

  • No amigable: https://www.exemple.com/p?12345
  • Amigable: https://www.exemple.com/productes/ordinadors-portatils

Exemple 2: URL per a una Pàgina de Serveis

  • Sense paraules clau: https://www.exemple.com/pagina1
  • Amb paraules clau: https://www.exemple.com/serveis/disseny-web

Exemple 3: URL per a una Pàgina de Categoria

  • Llarga: https://www.exemple.com/categoria-de-productes/ordinadors-portatils-de-gaming
  • Curta: https://www.exemple.com/productes/ordinadors-gaming

Exercicis Pràctics

Exercici 1: Convertir URL No Amigables a Amigables

Converteix les següents URL no amigables en URL amigables:

  1. https://www.exemple.com/p?12345
  2. https://www.exemple.com/categoria-de-productes/ordinadors-portatils-de-gaming
  3. https://www.exemple.com/pagina1

Solucions:

  1. https://www.exemple.com/productes/ordinadors-portatils
  2. https://www.exemple.com/productes/ordinadors-gaming
  3. https://www.exemple.com/serveis/disseny-web

Exercici 2: Crear URL per a Noves Pàgines

Crea URL amigables per a les següents pàgines:

  1. Una pàgina de serveis de màrqueting digital.
  2. Una pàgina de categoria de productes de telèfons mòbils.
  3. Una pàgina de contacte.

Solucions:

  1. https://www.exemple.com/serveis/marketing-digital
  2. https://www.exemple.com/productes/telefons-mobils
  3. https://www.exemple.com/contacte

Errors Comuns i Consells Addicionals

Errors Comuns

  1. Ús de caràcters especials: Evita l'ús de caràcters especials com &, %, $, etc.
  2. URL massa llargues: Evita URL massa llargues que siguin difícils de llegir i recordar.
  3. No incloure paraules clau: No oblidis incloure paraules clau rellevants a la URL.

Consells Addicionals

  1. Consistència: Mantingues una estructura consistent per a totes les URL del teu lloc web.
  2. Redireccions 301: Si canvies una URL, assegura't de configurar una redirecció 301 des de l'antiga URL a la nova per no perdre trànsit ni posicionament.

Conclusió

Una bona estructura de URL és essencial per a l'optimització del SEO On-Page. Seguint les millors pràctiques esmentades, pots assegurar-te que les teves URL siguin amigables tant per als motors de cerca com per als usuaris, millorant així la visibilitat i l'experiència de navegació del teu lloc web.
