Introducció

Haskell, un llenguatge de programació purament funcional i amb tipatge estàtic, ofereix una varietat de tipus de dades bàsiques que són essencials per a qualsevol programador de Haskell. Aquesta secció cobrirà els tipus de dades fonamentals en Haskell, proporcionant explicacions clares, exemples i exercicis per solidificar la teva comprensió.

Tipus de Dades Clau

Enters

  • Int: Un tipus d'enter de precisió fixa.
  • Integer: Un tipus d'enter de precisió arbitrària.
-- Exemple de Int
let x :: Int
x = 42

-- Exemple de Integer
let y :: Integer
y = 12345678901234567890

Nombres de Punt Flotant

  • Float: Un nombre de punt flotant de precisió simple.
  • Double: Un nombre de punt flotant de precisió doble.
-- Exemple de Float
let a :: Float
a = 3.14

-- Exemple de Double
let b :: Double
b = 2.718281828459045

Caràcters i Cadenes

  • Char: Representa un sol caràcter.
  • String: Una llista de caràcters, essencialment [Char].
-- Exemple de Char
let c :: Char
c = 'H'

-- Exemple de String
let s :: String
s = "Hola, Haskell!"

Booleà

  • Bool: Representa un valor booleà, que pot ser True o False.
-- Exemple de Bool
let isHaskellFun :: Bool
isHaskellFun = True

Tuples

  • Les tuples són col·leccions de mida fixa d'elements, que poden ser de diferents tipus.
-- Exemple d'una tuple
let person :: (String, Int)
person = ("Alice", 30)

Llistes

  • Les llistes són col·leccions d'elements del mateix tipus.
-- Exemple d'una llista
let numbers :: [Int]
numbers = [1, 2, 3, 4, 5]

Comparació de Tipus de Dades

Tipus de Dada Descripció Exemple

Integer Enter de precisió arbitrària 12345678901234567890
Float Punt flotant de precisió simple 3.14
Double Punt flotant de precisió doble 2.718281828459045

Exercicis

  1. Definir Variables:

    • Defineix una variable Int amb el valor 10.
    • Defineix una variable Double amb el valor 3.14159.
    • Defineix una variable String amb el valor "Haskell".

  2. Crear Tuples i Llistes:

    • Crea una tuple que representi un llibre amb un títol (String) i el nombre de pàgines (Int).
    • Crea una llista de valors Bool que representin els resultats d'algunes condicions: [True, False, True].

  3. Manipular Dades:

    • Escriu una funció que prengui una llista d'enters i retorni la suma dels elements.
    • Escriu una funció que prengui una tuple de dos enters i retorni el seu producte.
-- Exercici 1
let myInt :: Int
myInt = 10

let myDouble :: Double
myDouble = 3.14159

let myString :: String
myString = "Haskell"

-- Exercici 2
let book :: (String, Int)
book = ("Learn You a Haskell", 400)

let results :: [Bool]
results = [True, False, True]

-- Exercici 3
sumList :: [Int] -> Int
sumList xs = sum xs

productTuple :: (Int, Int) -> Int
productTuple (a, b) = a * b

Conclusió

Entendre els tipus de dades bàsics de Haskell és crucial per escriure codi efectiu i eficient. Aquesta secció t'ha introduït als principals tipus de dades, incloent enters, nombres de punt flotant, caràcters, cadenes, booleans, tuples i llistes. Practicant els exercicis proporcionats, obtindràs una base sòlida per manejar aquests tipus de dades en Haskell.
