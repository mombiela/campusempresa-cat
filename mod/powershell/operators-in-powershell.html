Introducció

Els operadors a PowerShell són caràcters especials o paraules clau que s'utilitzen per realitzar operacions sobre variables i valors. Són essencials per manipular dades i controlar el flux dels scripts. Aquesta secció cobrirà els diversos tipus d'operadors disponibles a PowerShell, des de l'aritmètica bàsica fins als operadors avançats de comparació i lògics.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics.

  • + : Addició
  • - : Substracció
  • * : Multiplicació
  • / : Divisió
  • % : Mòdul (resta)

Exemples

# Addició
$sum = 5 + 3
Write-Output $sum  # Sortida: 8

# Substracció
$difference = 10 - 4
Write-Output $difference  # Sortida: 6

# Multiplicació
$product = 7 * 6
Write-Output $product  # Sortida: 42

# Divisió
$quotient = 20 / 4
Write-Output $quotient  # Sortida: 5

# Mòdul
$remainder = 10 % 3
Write-Output $remainder  # Sortida: 1

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • = : Assignació simple
  • += : Sumar i assignar
  • -= : Restar i assignar
  • *= : Multiplicar i assignar
  • /= : Dividir i assignar
  • %= : Mòdul i assignar

Exemples

# Assignació simple
$a = 5

# Sumar i assignar
$a += 3  # Equivalent a $a = $a + 3
Write-Output $a  # Sortida: 8

# Restar i assignar
$a -= 2  # Equivalent a $a = $a - 2
Write-Output $a  # Sortida: 6

# Multiplicar i assignar
$a *= 4  # Equivalent a $a = $a * 4
Write-Output $a  # Sortida: 24

# Dividir i assignar
$a /= 3  # Equivalent a $a = $a / 3
Write-Output $a  # Sortida: 8

# Mòdul i assignar
$a %= 5  # Equivalent a $a = $a % 5
Write-Output $a  # Sortida: 3

Operadors de Comparació

Els operadors de comparació s'utilitzen per comparar valors i retornar un resultat booleà ($true o $false).

  • -eq : Igual a
  • -ne : No igual a
  • -gt : Major que
  • -ge : Major o igual a
  • -lt : Menor que
  • -le : Menor o igual a

Exemples

# Igual a
Write-Output (5 -eq 5)  # Sortida: True

# No igual a
Write-Output (5 -ne 3)  # Sortida: True

# Major que
Write-Output (7 -gt 4)  # Sortida: True

# Major o igual a
Write-Output (7 -ge 7)  # Sortida: True

# Menor que
Write-Output (3 -lt 5)  # Sortida: True

# Menor o igual a
Write-Output (3 -le 3)  # Sortida: True

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples condicions.

  • -and : Lògic I
  • -or : Lògic O
  • -not : Lògic NO

Exemples

# Lògic I
Write-Output ((5 -eq 5) -and (3 -lt 4))  # Sortida: True

# Lògic O
Write-Output ((5 -eq 5) -or (3 -gt 4))  # Sortida: True

# Lògic NO
Write-Output (-not (5 -eq 3))  # Sortida: True

Operadors Especials

PowerShell també inclou alguns operadors especials per a tasques específiques.

  • .. : Operador de rang
  • -contains : Operador de conté
  • -match : Coincidència d'expressió regular
  • -replace : Operador de reemplaçament

Exemples

# Operador de rang
$range = 1..5
Write-Output $range  # Sortida: 1 2 3 4 5

# Operador de conté
$array = @(1, 2, 3, 4, 5)
Write-Output ($array -contains 3)  # Sortida: True

# Coincidència d'expressió regular
$string = "PowerShell"
Write-Output ($string -match "Power")  # Sortida: True

# Operador de reemplaçament
$string = "Hello World"
$newString = $string -replace "World", "PowerShell"
Write-Output $newString  # Sortida: Hello PowerShell

Conclusió

Els operadors a PowerShell són eines fonamentals que et permeten realitzar una àmplia gamma d'operacions, des de l'aritmètica simple fins a avaluacions lògiques complexes. Entendre com utilitzar aquests operadors de manera efectiva millorarà molt la teva capacitat per escriure scripts de PowerShell potents i eficients. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un expert en la programació de PowerShell.
