Introducció

El Registre de Windows és una base de dades jeràrquica que emmagatzema configuracions i opcions per al sistema operatiu i les aplicacions instal·lades. PowerShell proporciona un conjunt robust de cmdlets per interactuar amb el registre, facilitant l'automatització de tasques relacionades amb la configuració del sistema.

Conceptes Clau

  • Hive del Registre: Les seccions de nivell superior del registre, com ara HKEY_LOCAL_MACHINE (HKLM) i HKEY_CURRENT_USER (HKCU).
  • Clau del Registre: Un node en el registre que pot contenir subclaus i valors.
  • Valor del Registre: Les dades emmagatzemades dins d'una clau, que poden ser de diferents tipus com String, Binary, DWORD, etc.

Accedint al Registre

PowerShell utilitza un model de proveïdor per accedir al registre. El proveïdor Registry et permet navegar pel registre com si fos un sistema de fitxers.

Navegant pel Registre

Pots utilitzar Get-ChildItem (àlies ls o dir) per llistar claus i Set-Location (àlies cd) per canviar la ubicació actual.

# Llistar totes les subclaus sota HKEY_LOCAL_MACHINE\Software
Get-ChildItem -Path "HKLM:\Software"

# Canviar la ubicació a HKEY_CURRENT_USER\Software
Set-Location -Path "HKCU:\Software"

Creant i Eliminant Claus

Pots crear noves claus utilitzant New-Item i eliminar-les utilitzant Remove-Item.

# Crear una nova clau sota HKEY_CURRENT_USER\Software
New-Item -Path "HKCU:\Software\MyApp"

# Eliminar la clau
Remove-Item -Path "HKCU:\Software\MyApp"

Llegint i Escrivent Valors

Utilitza Get-ItemProperty per llegir valors i Set-ItemProperty per escriure valors.

# Llegir un valor del registre
$regValue = Get-ItemProperty -Path "HKCU:\Software\MyApp" -Name "MyValue"
Write-Output $regValue.MyValue

# Escriure un valor al registre
Set-ItemProperty -Path "HKCU:\Software\MyApp" -Name "MyValue" -Value "NewValue"

Operacions Avançades

Exportant i Important Claus del Registre

Pots exportar i importar claus del registre utilitzant la utilitat de línia de comandes reg.exe dins de PowerShell.

# Exportar una clau del registre a un fitxer
reg export "HKCU\Software\MyApp" "C:\Backup\MyApp.reg"

# Importar una clau del registre des d'un fitxer
reg import "C:\Backup\MyApp.reg"

Gestionant Diferents Tipus de Valors

El registre suporta diversos tipus de dades. Aquí tens com gestionar alguns tipus comuns:

  • String (REG_SZ): Valor de text simple.
  • Binary (REG_BINARY): Dades binàries.
  • DWORD (REG_DWORD): Enter de 32 bits.
# Establir un valor String
Set-ItemProperty -Path "HKCU:\Software\MyApp" -Name "MyString" -Value "Hello"

# Establir un valor DWORD
Set-ItemProperty -Path "HKCU:\Software\MyApp" -Name "MyDWORD" -Value 1 -Type DWord

# Establir un valor Binari
$binaryData = [byte[]](1, 2, 3, 4)
Set-ItemProperty -Path "HKCU:\Software\MyApp" -Name "MyBinary" -Value $binaryData -Type Binary

Resum

En aquest tema, hem cobert els conceptes bàsics de treballar amb el Registre de Windows utilitzant PowerShell. Hem après com navegar pel registre, crear i eliminar claus, llegir i escriure valors, i gestionar diferents tipus de valors. Aquestes habilitats són essencials per automatitzar tasques de configuració del sistema i gestionar configuracions d'aplicacions.

Dominant aquestes tècniques, pots gestionar eficientment el registre i aprofitar les capacitats de PowerShell per optimitzar els teus fluxos de treball.
