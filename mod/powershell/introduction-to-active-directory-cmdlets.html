Introducció als Cmdlets d'Active Directory
Gestionar Usuaris i Grups ►

Visió general

Active Directory (AD) és un servei de directori desenvolupat per Microsoft per a xarxes de dominis de Windows. PowerShell proporciona un conjunt de cmdlets específicament dissenyats per interactuar amb Active Directory. Aquesta secció t'introduirà a aquests cmdlets, explicarà el seu ús i proporcionarà exemples pràctics per ajudar-te a entendre com gestionar objectes AD utilitzant PowerShell.

Conceptes clau

  • Active Directory: Un servei de directori per a xarxes de dominis de Windows.
  • Cmdlets: Comandes especialitzades en PowerShell que realitzen operacions específiques.
  • Mòdul AD: Un mòdul de PowerShell que conté cmdlets per gestionar Active Directory.

Prerequisits

Abans d'endinsar-te en els cmdlets d'Active Directory, assegura't de tenir:

  • Coneixement bàsic de la sintaxi i les comandes de PowerShell.
  • Accés a un servidor Windows amb Active Directory instal·lat.
  • Privilegis administratius per realitzar operacions AD.

Instal·lació del mòdul d'Active Directory

Per utilitzar els cmdlets AD, necessites instal·lar el mòdul d'Active Directory. Aquest mòdul està disponible a les eines d'administració de servidor remot (RSAT) per a Windows.

Passos per instal·lar RSAT

  1. Obre Configuració.
  2. Ves a Aplicacions > Funcions opcionals.
  3. Fes clic a Afegeix una funció.
  4. Cerca RSAT: Active Directory Domain Services and Lightweight Directory Tools.
  5. Instal·la la funció.

Cmdlets comuns d'Active Directory

Aquí tens alguns dels cmdlets d'Active Directory més utilitzats:

| Cmdlet | Descripció | |-----------------------|--------------------------------------------------| | Get-ADUser | Recupera objectes d'usuari d'AD. | | New-ADUser | Crea un nou usuari a AD. | | Set-ADUser | Modifica les propietats d'un usuari existent. | | Remove-ADUser | Elimina un usuari d'AD. | | Get-ADGroup | Recupera objectes de grup d'AD. | | New-ADGroup | Crea un nou grup a AD. | | Add-ADGroupMember | Afegeix un usuari a un grup. | | Remove-ADGroupMember| Elimina un usuari d'un grup. |

Ús dels cmdlets d'Active Directory

Recuperació d'informació d'usuari

Per recuperar informació sobre un usuari, pots utilitzar el cmdlet Get-ADUser.

Get-ADUser -Identity "jdoe"

Explicació:

  • Get-ADUser: El cmdlet per recuperar informació d'usuari.
  • -Identity "jdoe": Especifica l'usuari a recuperar pel seu nom d'usuari.

Creació d'un nou usuari

Per crear un nou usuari a Active Directory, utilitza el cmdlet New-ADUser.

New-ADUser -Name "John Doe" -GivenName "John" -Surname "Doe" -SamAccountName "jdoe" -UserPrincipalName "jdoe@example.com" -Path "OU=Users,DC=example,DC=com" -AccountPassword (ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force) -Enabled $true

Explicació:

  • New-ADUser: El cmdlet per crear un nou usuari.
  • -Name "John Doe": El nom complet de l'usuari.
  • -GivenName "John": El nom de pila de l'usuari.
  • -Surname "Doe": El cognom de l'usuari.
  • -SamAccountName "jdoe": El nom de compte SAM.
  • -UserPrincipalName "jdoe@example.com": El UPN de l'usuari.
  • -Path "OU=Users,DC=example,DC=com": La unitat organitzativa on es crearà l'usuari.
  • -AccountPassword (ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force): Estableix la contrasenya del compte.
  • -Enabled $true: Activa el compte d'usuari.

Modificació de propietats d'usuari

Per modificar les propietats d'un usuari existent, utilitza el cmdlet Set-ADUser.

Set-ADUser -Identity "jdoe" -Title "Senior Developer" -Department "IT"

Explicació:

  • Set-ADUser: El cmdlet per modificar les propietats d'usuari.
  • -Identity "jdoe": Especifica l'usuari a modificar.
  • -Title "Senior Developer": Estableix el títol de l'usuari.
  • -Department "IT": Estableix el departament de l'usuari.

Eliminació d'un usuari

Per eliminar un usuari d'Active Directory, utilitza el cmdlet Remove-ADUser.

Remove-ADUser -Identity "jdoe"

Explicació:

  • Remove-ADUser: El cmdlet per eliminar un usuari.
  • -Identity "jdoe": Especifica l'usuari a eliminar.

Conclusió

En aquesta secció, has après els conceptes bàsics dels cmdlets d'Active Directory en PowerShell. Ara saps com instal·lar el mòdul AD, recuperar informació d'usuari, crear nous usuaris, modificar propietats d'usuari i eliminar usuaris. Aquests cmdlets són eines poderoses per gestionar objectes d'Active Directory de manera eficient. Practica utilitzant aquests cmdlets per sentir-te més còmode gestionant AD a través de PowerShell.
