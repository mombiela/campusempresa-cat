Sentències Switch a PowerShell
Gestió d'Errors ►

Les sentències switch a PowerShell proporcionen una manera d'executar diferents blocs de codi basats en el valor d'una variable o expressió. Són particularment útils per gestionar múltiples condicions de manera més eficient que utilitzant múltiples sentències if.

Conceptes Clau

  • Sintaxi de la Sentència Switch: L'estructura bàsica d'una sentència switch.
  • Coincidència de Patrons: Utilització de comodins i expressions regulars dins de les sentències switch.
  • Condicions Múltiples: Gestió de múltiples casos i casos per defecte.
  • Funcions Avançades: Ús de les opcions -file, -regex i -wildcard.

Sintaxi de la Sentència Switch

La sintaxi bàsica d'una sentència switch a PowerShell és la següent:

switch ($variable) {
    case 'value1' {
        # Codi a executar si $variable és 'value1'
    }
    case 'value2' {
        # Codi a executar si $variable és 'value2'
    }
    default {
        # Codi a executar si $variable no coincideix amb cap cas
    }
}

Exemple

$day = 'Monday'

switch ($day) {
    'Monday' { "Start of the work week." }
    'Friday' { "End of the work week." }
    default { "Midweek day." }
}

En aquest exemple:

  • Si $day és 'Monday', mostrarà "Start of the work week."
  • Si $day és 'Friday', mostrarà "End of the work week."
  • Per qualsevol altre valor de $day, mostrarà "Midweek day."

Coincidència de Patrons

Les sentències switch de PowerShell suporten la coincidència de patrons utilitzant comodins i expressions regulars.

Comodins

$fruit = 'apple'

switch -Wildcard ($fruit) {
    'a*' { "Starts with 'a'" }
    '*e' { "Ends with 'e'" }
    default { "No match" }
}

Expressions Regulars

$fruit = 'banana'

switch -Regex ($fruit) {
    '^b.*a$' { "Starts with 'b' and ends with 'a'" }
    'n{2}' { "Contains double 'n'" }
    default { "No match" }
}

Condicions Múltiples

Pots gestionar múltiples condicions especificant múltiples valors en un sol cas.

$number = 3

switch ($number) {
    1 { "One" }
    2 { "Two" }
    3 { "Three" }
    4 { "Four" }
    default { "Number not in range 1-4" }
}

Funcions Avançades

Ús de `-file`

L'opció -file et permet llegir l'entrada des d'un fitxer i processar cada línia.

switch -File "C:\path\to\file.txt" {
    'Error' { "Found an error" }
    'Warning' { "Found a warning" }
    default { "General information" }
}

Ús de `-regex` i `-wildcard`

Pots combinar les opcions -regex i -wildcard per gestionar escenaris més complexos.

$input = 'error: file not found'

switch -Regex ($input) {
    'error:.*' { "An error occurred" }
    'warning:.*' { "A warning occurred" }
    default { "No issues found" }
}

Conclusió

Les sentències switch a PowerShell són una eina poderosa per gestionar múltiples condicions i coincidència de patrons. Poden simplificar els teus scripts reduint la necessitat de múltiples sentències if i fent el teu codi més llegible. Entenent la sintaxi bàsica, les capacitats de coincidència de patrons i les funcions avançades, pots utilitzar efectivament les sentències switch per controlar el flux dels teus scripts de PowerShell.
