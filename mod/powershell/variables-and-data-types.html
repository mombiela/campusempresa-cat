Variables i tipus de dades a PowerShell
Operadors a PowerShell ►

Introducció

En aquesta secció, cobrirem els conceptes bàsics de les variables i els tipus de dades a PowerShell. Entendre aquests conceptes fonamentals és crucial per escriure scripts efectius i automatitzar tasques.

Variables a PowerShell

Les variables a PowerShell s'utilitzen per emmagatzemar dades que es poden referenciar i manipular al llarg dels teus scripts.

Declarar variables

  • Les variables a PowerShell es declaren utilitzant el símbol $ seguit del nom de la variable.
  • Els noms de les variables no distingeixen entre majúscules i minúscules.
  • No cal especificar un tipus de dada quan es declara una variable.
# Declarar una variable
$myVariable = "Hola, PowerShell!"

Utilitzar variables

  • Les variables poden emmagatzemar diferents tipus de dades com cadenes de text, enters, arrays i objectes.
  • Pots referenciar una variable utilitzant el seu nom.
# Utilitzar una variable
$greeting = "Hola, Món!"
Write-Output $greeting

Àmbit de les variables

  • Les variables a PowerShell tenen diferents àmbits: Global, Local, Script i Privat.
  • Per defecte, les variables són locals a l'àmbit en què es creen.
# Exemple d'àmbit de variables
$global:globalVar = "Sóc global"
$localVar = "Sóc local"

Tipus de dades a PowerShell

PowerShell admet una varietat de tipus de dades, que es poden categoritzar àmpliament en tipus simples i complexos.

Tipus de dades simples

  • String: Una seqüència de caràcters.
  • Integer: Nombres enters.
  • Boolean: Representa True o False.
  • Float/Double: Nombres amb punts decimals.
# Exemples de tipus de dades simples
$stringVar = "Això és una cadena de text"
$intVar = 42
$boolVar = $true
$floatVar = 3.14

Tipus de dades complexes

  • Array: Una col·lecció de valors.
  • Hash Table: Una col·lecció de parells clau-valor.
  • Object: Instàncies de classes .NET.
# Exemples de tipus de dades complexes
$arrayVar = @(1, 2, 3, 4, 5)
$hashTableVar = @{Name="John"; Age=30}
$objectVar = New-Object -TypeName PSObject -Property @{Name="Jane"; Age=25}

Conversió de tipus

  • PowerShell pot convertir automàticament entre tipus de dades quan sigui necessari.
  • La conversió explícita de tipus es pot fer utilitzant el càsting.
# Conversió automàtica de tipus
$number = "123"
$sum = $number + 10  # Converteix la cadena de text a enter

# Conversió explícita de tipus
$explicitInt = [int]"456"

Exemples i exercicis

Exemple 1: Ús bàsic de variables

# Declarar i utilitzar una variable
$message = "Benvingut a PowerShell!"
Write-Output $message

Exemple 2: Treballar amb arrays

# Declarar un array i accedir als seus elements
$numbers = @(10, 20, 30, 40, 50)
Write-Output $numbers[2]  # Mostra 30

Exercici 1: Crear i manipular variables

  1. Declara una variable per emmagatzemar el teu nom.
  2. Declara una variable per emmagatzemar la teva edat.
  3. Mostra un missatge que inclogui ambdues variables.
# Solució
$name = "Alice"
$age = 28
Write-Output "Em dic $name i tinc $age anys."

Exercici 2: Treballar amb taules de hash

  1. Crea una taula de hash per emmagatzemar informació sobre un llibre (títol, autor, any).
  2. Mostra el títol i l'autor del llibre.
# Solució
$book = @{Title="1984"; Author="George Orwell"; Year=1949}
Write-Output "El llibre '$($book.Title)' va ser escrit per $($book.Author)."

Conclusió

En aquesta secció, hem cobert els conceptes bàsics de les variables i els tipus de dades a PowerShell. Has après com declarar i utilitzar variables, entendre diferents tipus de dades i realitzar conversions de tipus. Aquestes habilitats fonamentals són essencials per escriure scripts efectius a PowerShell. Practica els exercicis proporcionats per reforçar la teva comprensió, i estaràs en bon camí per dominar la programació a PowerShell.
