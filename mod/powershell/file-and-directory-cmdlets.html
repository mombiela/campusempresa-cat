Introducció

PowerShell proporciona un conjunt robust de cmdlets per gestionar fitxers i directoris. Aquests cmdlets et permeten realitzar una varietat de tasques com crear, copiar, moure i eliminar fitxers i directoris. Aquesta secció et guiarà a través dels cmdlets essencials per a la gestió de fitxers i directoris, des de l'ús bàsic fins a l'avançat.

Cmdlets bàsics per a la gestió de fitxers i directoris

Get-ChildItem

El cmdlet Get-ChildItem recupera els elements i els elements fills en una o més ubicacions especificades. És similar a l'ordre dir al Símbol del sistema.

# Llistar tots els fitxers i directoris al directori actual
Get-ChildItem

# Llistar tots els fitxers i directoris en un camí especificat
Get-ChildItem -Path "C:\Example\Path"

New-Item

El cmdlet New-Item crea un nou element, com ara un fitxer o directori.

# Crear un nou directori
New-Item -Path "C:\Example\NewDirectory" -ItemType Directory

# Crear un nou fitxer
New-Item -Path "C:\Example\NewFile.txt" -ItemType File

Remove-Item

El cmdlet Remove-Item elimina un o més elements.

# Eliminar un fitxer
Remove-Item -Path "C:\Example\NewFile.txt"

# Eliminar un directori i el seu contingut
Remove-Item -Path "C:\Example\NewDirectory" -Recurse

Copy-Item

El cmdlet Copy-Item copia un element d'una ubicació a una altra.

# Copiar un fitxer a una nova ubicació
Copy-Item -Path "C:\Example\SourceFile.txt" -Destination "C:\Example\DestinationFile.txt"

# Copiar un directori i el seu contingut a una nova ubicació
Copy-Item -Path "C:\Example\SourceDirectory" -Destination "C:\Example\DestinationDirectory" -Recurse

Move-Item

El cmdlet Move-Item mou un element d'una ubicació a una altra.

# Moure un fitxer a una nova ubicació
Move-Item -Path "C:\Example\SourceFile.txt" -Destination "C:\Example\DestinationFile.txt"

# Moure un directori a una nova ubicació
Move-Item -Path "C:\Example\SourceDirectory" -Destination "C:\Example\DestinationDirectory"

Cmdlets i tècniques avançades

Get-Content i Set-Content

El cmdlet Get-Content recupera el contingut d'un fitxer, mentre que el cmdlet Set-Content escriu o reemplaça el contingut en un fitxer.

# Recuperar el contingut d'un fitxer
Get-Content -Path "C:\Example\File.txt"

# Escriure contingut a un fitxer
Set-Content -Path "C:\Example\File.txt" -Value "Aquest és un nou contingut."

Add-Content

El cmdlet Add-Content afegeix contingut a un fitxer sense sobreescriure el contingut existent.

# Afegir contingut a un fitxer
Add-Content -Path "C:\Example\File.txt" -Value "Aquest és un contingut addicional."

Get-Item i Set-Item

El cmdlet Get-Item recupera les propietats d'un element especificat, mentre que el cmdlet Set-Item canvia el valor d'un element.

# Recuperar propietats d'un fitxer
Get-Item -Path "C:\Example\File.txt"

# Canviar el valor d'un element (per exemple, canviar el nom d'un fitxer)
Set-Item -Path "C:\Example\File.txt" -NewName "NewFileName.txt"

Ús de filtres i comodins

Els cmdlets de PowerShell admeten filtres i comodins per refinar la selecció d'elements.

# Llistar tots els fitxers .txt en un directori
Get-ChildItem -Path "C:\Example" -Filter "*.txt"

# Llistar tots els fitxers que comencen amb 'Report'
Get-ChildItem -Path "C:\Example" -Filter "Report*"

Resum

En aquesta secció, hem cobert els cmdlets essencials per gestionar fitxers i directoris a PowerShell. Hem començat amb operacions bàsiques com llistar, crear, copiar, moure i eliminar fitxers i directoris. Després hem explorat cmdlets i tècniques més avançades, incloent recuperar i establir contingut, afegir contingut i utilitzar filtres i comodins.

En dominar aquests cmdlets, podràs gestionar eficientment fitxers i directoris en els teus scripts de PowerShell, automatitzant tasques rutinàries i millorant la teva productivitat.
