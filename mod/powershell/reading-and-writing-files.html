Llegir i Escriure Fitxers a PowerShell

Introducció

Llegir i escriure fitxers són tasques fonamentals en la creació de scripts i l'automatització. PowerShell proporciona cmdlets robustos per gestionar operacions de fitxers de manera eficient. Aquesta secció et guiarà des de les tècniques bàsiques fins a les avançades per gestionar fitxers a PowerShell.

Operacions Bàsiques de Fitxers

Llegir Fitxers

PowerShell facilita la lectura del contingut d'un fitxer utilitzant el cmdlet Get-Content.

Exemple:

# Llegeix el contingut d'un fitxer i el mostra
Get-Content -Path "C:\example\file.txt"
  • Get-Content: Llegeix el contingut del fitxer especificat.
  • -Path: Especifica el camí al fitxer.

Escriure Fitxers

Per escriure contingut a un fitxer, pots utilitzar els cmdlets Set-Content o Add-Content.

Exemple:

# Escriu contingut a un fitxer nou o sobreescriu un fitxer existent
Set-Content -Path "C:\example\file.txt" -Value "Hello, World!"

# Afegeix contingut a un fitxer existent
Add-Content -Path "C:\example\file.txt" -Value "This is an additional line."
  • Set-Content: Escriu o reemplaça el contingut d'un fitxer.
  • Add-Content: Afegeix contingut a un fitxer existent.
  • -Value: Especifica el contingut a escriure o afegir.

Operacions Intermèdies de Fitxers

Llegir Fitxers Grans

Per a fitxers grans, Get-Content pot ser ineficient. En canvi, utilitza el paràmetre -ReadCount per llegir el fitxer en blocs.

Exemple:

# Llegeix un fitxer gran en blocs de 100 línies
Get-Content -Path "C:\example\largefile.txt" -ReadCount 100 | ForEach-Object {
    # Processa cada bloc aquí
    $_
}
  • -ReadCount: Especifica el nombre de línies a llegir alhora.

Escriure Dades Estructurades

PowerShell pot gestionar formats de dades estructurades com CSV, JSON i XML.

Exemple: Escriure CSV

# Crea una matriu d'objectes
$data = @(
    [PSCustomObject]@{Name="John"; Age=30},
    [PSCustomObject]@{Name="Jane"; Age=25}
)

# Exporta la matriu a un fitxer CSV
$data | Export-Csv -Path "C:\example\data.csv" -NoTypeInformation
  • Export-Csv: Converteix objectes en un fitxer CSV.
  • -NoTypeInformation: Omiteix la informació de tipus del fitxer CSV.

Llegir Dades Estructurades

De manera similar, pots importar dades estructurades des de fitxers.

Exemple: Llegir CSV

# Importa dades d'un fitxer CSV
$data = Import-Csv -Path "C:\example\data.csv"

# Mostra les dades importades
$data
  • Import-Csv: Llegeix un fitxer CSV i el converteix en objectes.

Operacions Avançades de Fitxers

Gestió de Fitxers Binari

Per a fitxers binaris, utilitza Get-Content amb el paràmetre -AsByteStream.

Exemple:

# Llegeix un fitxer binari
$binaryData = Get-Content -Path "C:\example\file.bin" -AsByteStream

# Escriu dades binàries a un nou fitxer
[System.IO.File]::WriteAllBytes("C:\example\newfile.bin", $binaryData)
  • -AsByteStream: Llegeix el fitxer com una matriu de bytes.

Fluxos de Fitxers

PowerShell et permet treballar amb fluxos de fitxers per a un control més granular.

Exemple:

# Obre un flux de fitxer per a lectura
$fileStream = [System.IO.File]::OpenRead("C:\example\file.txt")
$reader = New-Object System.IO.StreamReader($fileStream)

# Llegeix el contingut del fitxer
$content = $reader.ReadToEnd()

# Tanca el flux
$reader.Close()
$fileStream.Close()

# Mostra el contingut
$content
  • System.IO.File::OpenRead: Obre un flux de fitxer per a lectura.
  • System.IO.StreamReader: Llegeix caràcters d'un flux de bytes.

Conclusió

PowerShell proporciona un conjunt complet de cmdlets i mètodes per llegir i escriure fitxers, des de fitxers de text simples fins a dades estructurades complexes i fitxers binaris. En dominar aquestes tècniques, pots automatitzar una àmplia gamma d'operacions de fitxers de manera eficient.

Resum

  • Operacions Bàsiques: Get-Content, Set-Content, Add-Content.
  • Operacions Intermèdies: Gestió de fitxers grans, dades estructurades (CSV, JSON, XML).
  • Operacions Avançades: Fitxers binaris, fluxos de fitxers.

Amb aquestes eines, pots gestionar pràcticament qualsevol tasca relacionada amb fitxers a PowerShell, millorant les teves capacitats de creació de scripts i automatització.
