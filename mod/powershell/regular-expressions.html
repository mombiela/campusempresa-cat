Introducció a les expressions regulars

Les expressions regulars (regex) són seqüències de caràcters que defineixen un patró de cerca. S'utilitzen comunament per a la coincidència i manipulació de cadenes. A PowerShell, les regex es poden utilitzar per cercar, reemplaçar i validar cadenes.

Conceptes clau

  • Coincidència de patrons: Identificar si una cadena conté un patró especificat.
  • Classes de caràcters: Conjunts de caràcters tancats entre claudàtors [] que coincideixen amb qualsevol caràcter dins del conjunt.
  • Quantificadors: Especifiquen el nombre d'ocurrències d'un caràcter o grup.
  • Ancoratges: Afirmen la posició dins d'una cadena (per exemple, inici o final).
  • Grups i captura: Tanquen part de la regex entre parèntesis () per crear grups.

Sintaxi bàsica i ús

Coincidència de patrons

Per comprovar si una cadena coincideix amb un patró, utilitza l'operador -match.

$string = "Hello World"
if ($string -match "World") {
    Write-Output "Pattern found!"
}

Classes de caràcters

Les classes de caràcters et permeten coincidir amb qualsevol dels diversos caràcters.

$string = "Hello World"
if ($string -match "[aeiou]") {
    Write-Output "String contains a vowel!"
}

Quantificadors

Els quantificadors especifiquen quantes vegades un caràcter o grup ha de coincidir.

  • * : 0 o més vegades
  • + : 1 o més vegades
  • ? : 0 o 1 vegada
  • {n} : Exactament n vegades
  • {n,} : n o més vegades
  • {n,m} : Entre n i m vegades
$string = "Helloooo World"
if ($string -match "o{2,}") {
    Write-Output "String contains two or more consecutive 'o's!"
}

Ancoratges

Els ancoratges s'utilitzen per especificar la posició dins de la cadena.

  • ^ : Inici de la cadena
  • $ : Final de la cadena
$string = "Hello World"
if ($string -match "^Hello") {
    Write-Output "String starts with 'Hello'!"
}
if ($string -match "World$") {
    Write-Output "String ends with 'World'!"
}

Grups i captura

Els grups s'utilitzen per crear sub-patrons dins d'una regex.

$string = "Hello123"
if ($string -match "([A-Za-z]+)(\d+)") {
    Write-Output "Letters: $($matches[1]), Numbers: $($matches[2])"
}

Tècniques avançades

Lookahead i Lookbehind

Les afirmacions Lookahead i Lookbehind et permeten coincidir amb un patró només si està (o no està) seguit o precedit per un altre patró.

  • Lookahead positiu: (?=...)
  • Lookahead negatiu: (?!...)
  • Lookbehind positiu: (?<=...)
  • Lookbehind negatiu: (?<!...)
$string = "foo123bar"
if ($string -match "foo(?=\d{3})") {
    Write-Output "Found 'foo' followed by three digits!"
}

Grups no capturadors

Els grups no capturadors s'utilitzen per agrupar part d'una regex sense capturar-la.

$string = "Hello123"
if ($string -match "(?:Hello)(\d+)") {
    Write-Output "Numbers: $($matches[1])"
}

Grups de captura amb nom

Els grups de captura amb nom et permeten assignar un nom a un grup.

$string = "Hello123"
if ($string -match "(?<letters>[A-Za-z]+)(?<numbers>\d+)") {
    Write-Output "Letters: $($matches['letters']), Numbers: $($matches['numbers'])"
}

Exemples pràctics

Validació d'adreces de correu electrònic

$email = "example@example.com"
$regex = "^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}$"
if ($email -match $regex) {
    Write-Output "Valid email address!"
} else {
    Write-Output "Invalid email address!"
}

Extracció de dades de text

$text = "Order ID: 12345, Date: 2023-10-01"
if ($text -match "Order ID: (\d+), Date: (\d{4}-\d{2}-\d{2})") {
    $orderId = $matches[1]
    $date = $matches[2]
    Write-Output "Order ID: $orderId, Date: $date"
}

Resum

Les expressions regulars són eines poderoses per al processament de text a PowerShell. Entenent la sintaxi bàsica i les tècniques avançades, pots realitzar manipulacions i validacions de cadenes complexes de manera eficient. Practica amb diferents patrons i casos d'ús per convertir-te en un expert en l'ús de regex als teus scripts de PowerShell.
