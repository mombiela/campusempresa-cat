Introducció

PowerShell és un marc d'automatització de tasques i gestió de configuracions de Microsoft, que consisteix en una línia de comandes i el llenguatge de script associat. Aquesta secció us introduirà algunes de les comandes bàsiques de PowerShell que són essencials per a principiants.

Conceptes clau

  • Cmdlets: Les comandes de PowerShell es coneixen com a cmdlets. Segueixen una convenció de nomenclatura Verb-Noun.
  • Pipelines: PowerShell utilitza pipelines per passar la sortida d'un cmdlet com a entrada a un altre.
  • Objectes: A diferència de les shells tradicionals que treballen amb text, PowerShell treballa amb objectes.

Cmdlets bàsics

Get-Command

El cmdlet Get-Command recupera totes les comandes disponibles a la vostra sessió de PowerShell.

Get-Command
  • Explicació: Aquest cmdlet llista tots els cmdlets, funcions, àlies i scripts disponibles.

Get-Help

El cmdlet Get-Help proporciona informació detallada sobre els cmdlets i conceptes de PowerShell.

Get-Help Get-Command
  • Explicació: Aquesta comanda mostrarà informació d'ajuda sobre el cmdlet Get-Command.

Get-Process

El cmdlet Get-Process recupera informació sobre els processos que s'executen al vostre sistema.

Get-Process
  • Explicació: Aquest cmdlet llista tots els processos que s'estan executant actualment a la vostra màquina.

Get-Service

El cmdlet Get-Service recupera l'estat dels serveis en una màquina local o remota.

Get-Service
  • Explicació: Aquest cmdlet llista tots els serveis i els seus estats.

Set-Location

El cmdlet Set-Location canvia el directori actual.

Set-Location -Path "C:\Users"
  • Explicació: Aquesta comanda canvia el directori actual a C:\Users.

Get-ChildItem

El cmdlet Get-ChildItem recupera els fitxers i directoris en una ubicació especificada.

Get-ChildItem -Path "C:\Users"
  • Explicació: Aquest cmdlet llista tots els fitxers i directoris dins de C:\Users.

Copy-Item

El cmdlet Copy-Item copia un element d'una ubicació a una altra.

Copy-Item -Path "C:\Users\example.txt" -Destination "C:\Backup\example.txt"
  • Explicació: Aquesta comanda copia example.txt de C:\Users a C:\Backup.

Remove-Item

El cmdlet Remove-Item elimina un element.

Remove-Item -Path "C:\Users\example.txt"
  • Explicació: Aquesta comanda elimina example.txt de C:\Users.

New-Item

El cmdlet New-Item crea un nou element, com ara un fitxer o directori.

New-Item -Path "C:\Users\newfile.txt" -ItemType "File"
  • Explicació: Aquesta comanda crea un nou fitxer anomenat newfile.txt a C:\Users.

Rename-Item

El cmdlet Rename-Item canvia el nom d'un element.

Rename-Item -Path "C:\Users\newfile.txt" -NewName "renamedfile.txt"
  • Explicació: Aquesta comanda canvia el nom de newfile.txt a renamedfile.txt.

Resum

En aquesta secció, hem cobert algunes de les comandes bàsiques de PowerShell que són essencials per a principiants. Aquestes comandes inclouen Get-Command, Get-Help, Get-Process, Get-Service, Set-Location, Get-ChildItem, Copy-Item, Remove-Item, New-Item i Rename-Item. Entendre aquests cmdlets bàsics proporcionarà una base sòlida per a tasques més avançades de scripting i automatització amb PowerShell.
