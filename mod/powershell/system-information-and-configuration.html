Introducció

PowerShell és un llenguatge de script i una shell de línia de comandes potent dissenyat per a l'administració i l'automatització del sistema. Una de les seves principals fortaleses és la capacitat de recollir i configurar informació del sistema. Aquest tema et guiarà des de les tècniques bàsiques fins a les avançades per recuperar i gestionar informació del sistema utilitzant PowerShell.

Recollida d'Informació del Sistema

Informació Bàsica del Sistema

PowerShell proporciona diversos cmdlets per recuperar informació bàsica del sistema.

  • Get-ComputerInfo: Recupera un conjunt complet de propietats del sistema.
  • Get-WmiObject: Recupera informació de gestió d'ordinadors locals i remots.
  • Get-Process: Recupera informació sobre els processos en execució.

Exemples:

# Recuperar informació completa del sistema
Get-ComputerInfo

# Recuperar informació específica del sistema utilitzant WMI
Get-WmiObject -Class Win32_OperatingSystem

# Recuperar informació sobre els processos en execució
Get-Process

Informació de Maquinari

Pots recollir informació detallada del maquinari utilitzant classes WMI específiques.

  • Win32_Processor: Informació sobre la CPU.
  • Win32_PhysicalMemory: Informació sobre la memòria física.
  • Win32_DiskDrive: Informació sobre les unitats de disc.

Exemples:

# Recuperar informació de la CPU
Get-WmiObject -Class Win32_Processor

# Recuperar informació de la memòria física
Get-WmiObject -Class Win32_PhysicalMemory

# Recuperar informació de les unitats de disc
Get-WmiObject -Class Win32_DiskDrive

Informació de Xarxa

PowerShell també es pot utilitzar per recollir informació relacionada amb la xarxa.

  • Get-NetIPAddress: Recupera la configuració de l'adreça IP.
  • Get-NetAdapter: Recupera informació de l'adaptador de xarxa.
  • Test-Connection: Prova la connectivitat de la xarxa.

Exemples:

# Recuperar la configuració de l'adreça IP
Get-NetIPAddress

# Recuperar informació de l'adaptador de xarxa
Get-NetAdapter

# Provar la connectivitat de la xarxa amb un host remot
Test-Connection -ComputerName google.com

Configuració de la Configuració del Sistema

Gestió de Serveis

PowerShell et permet gestionar els serveis de Windows de manera eficient.

  • Get-Service: Recupera l'estat dels serveis.
  • Start-Service: Inicia un servei.
  • Stop-Service: Atura un servei.
  • Restart-Service: Reinicia un servei.

Exemples:

# Recuperar l'estat de tots els serveis
Get-Service

# Iniciar un servei específic
Start-Service -Name "wuauserv"

# Aturar un servei específic
Stop-Service -Name "wuauserv"

# Reiniciar un servei específic
Restart-Service -Name "wuauserv"

Gestió de Processos

També pots gestionar processos utilitzant PowerShell.

  • Start-Process: Inicia un nou procés.
  • Stop-Process: Atura un procés en execució.
  • Get-Process: Recupera informació sobre els processos en execució.

Exemples:

# Iniciar un nou procés (per exemple, Notepad)
Start-Process -FilePath "notepad.exe"

# Aturar un procés en execució pel nom
Stop-Process -Name "notepad"

# Recuperar informació sobre els processos en execució
Get-Process

Configuració de la Xarxa

PowerShell proporciona cmdlets per configurar la xarxa.

  • New-NetIPAddress: Configura una nova adreça IP.
  • Set-DnsClientServerAddress: Configura les adreces dels servidors DNS.
  • Set-NetIPInterface: Configura les propietats de la interfície IP.

Exemples:

# Configurar una nova adreça IP
New-NetIPAddress -InterfaceAlias "Ethernet" -IPAddress "192.168.1.100" -PrefixLength 24 -DefaultGateway "192.168.1.1"

# Configurar les adreces dels servidors DNS
Set-DnsClientServerAddress -InterfaceAlias "Ethernet" -ServerAddresses ("8.8.8.8", "8.8.4.4")

# Configurar les propietats de la interfície IP
Set-NetIPInterface -InterfaceAlias "Ethernet" -Dhcp Disabled

Tècniques Avançades

Ús de Cmdlets CIM

Els cmdlets CIM (Common Information Model) proporcionen un enfocament més modern per accedir a la informació del sistema.

  • Get-CimInstance: Recupera informació de gestió.
  • New-CimSession: Crea una nova sessió CIM.
  • Invoke-CimMethod: Invoca un mètode en una instància CIM.

Exemples:

# Recuperar informació utilitzant CIM
Get-CimInstance -ClassName Win32_OperatingSystem

# Crear una nova sessió CIM
$cimSession = New-CimSession -ComputerName "RemoteComputer"

# Invocar un mètode en una instància CIM
Invoke-CimMethod -CimSession $cimSession -ClassName Win32_Process -MethodName Create -Arguments @{CommandLine="notepad.exe"}

Scripts i Automatització

Els scripts de PowerShell poden automatitzar la recuperació i configuració de la informació del sistema.

Exemple de Script:

# Script per recollir i registrar informació del sistema
$systemInfo = Get-ComputerInfo
$cpuInfo = Get-WmiObject -Class Win32_Processor
$memoryInfo = Get-WmiObject -Class Win32_PhysicalMemory
$diskInfo = Get-WmiObject -Class Win32_DiskDrive

# Registrar la informació en un fitxer
$logFile = "C:\SystemInfoLog.txt"
$systemInfo | Out-File -FilePath $logFile -Append
$cpuInfo | Out-File -FilePath $logFile -Append
$memoryInfo | Out-File -FilePath $logFile -Append
$diskInfo | Out-File -FilePath $logFile -Append

Conclusió

PowerShell és una eina versàtil per recollir i configurar informació del sistema. Des de propietats bàsiques del sistema fins a configuracions avançades de maquinari i xarxa, els cmdlets de PowerShell proporcionen una interfície potent per a l'administració del sistema. Dominant aquests cmdlets i tècniques, pots gestionar i automatitzar tasques del sistema de manera eficient, millorant la teva productivitat i capacitats de gestió del sistema.
