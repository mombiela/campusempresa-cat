Declaracions condicionals a PowerShell
Bucles a PowerShell ►

Introducció

Les declaracions condicionals són fonamentals en la programació, ja que permeten executar codi basat en certes condicions. A PowerShell, les declaracions condicionals ajuden a automatitzar tasques prenent decisions dins dels scripts.

Conceptes clau

  • If Statement: Executa un bloc de codi si una condició especificada és certa.
  • Else Statement: Executa un bloc de codi si la condició en la declaració if és falsa.
  • ElseIf Statement: Afegeix condicions addicionals a una declaració if.
  • Switch Statement: Avalua una sola expressió contra múltiples valors possibles.

If Statement

La declaració if s'utilitza per provar una condició. Si la condició és certa, el bloc de codi dins de la declaració if s'executa.

Sintaxi

if (<condició>) {
    <codi a executar si la condició és certa>
}

Exemple

$number = 10

if ($number -gt 5) {
    Write-Output "El nombre és més gran que 5"
}

En aquest exemple, la condició $number -gt 5 comprova si $number és més gran que 5. Com que $number és 10, la condició és certa, i es mostra el missatge "El nombre és més gran que 5".

Else Statement

La declaració else s'utilitza per executar un bloc de codi si la condició en la declaració if és falsa.

Sintaxi

if (<condició>) {
    <codi a executar si la condició és certa>
} else {
    <codi a executar si la condició és falsa>
}

Exemple

$number = 3

if ($number -gt 5) {
    Write-Output "El nombre és més gran que 5"
} else {
    Write-Output "El nombre no és més gran que 5"
}

Aquí, la condició $number -gt 5 és falsa perquè $number és 3. Per tant, es mostra el missatge "El nombre no és més gran que 5".

ElseIf Statement

La declaració elseif permet provar múltiples condicions.

Sintaxi

if (<condició1>) {
    <codi a executar si la condició1 és certa>
} elseif (<condició2>) {
    <codi a executar si la condició2 és certa>
} else {
    <codi a executar si totes les condicions són falses>
}

Exemple

$number = 7

if ($number -gt 10) {
    Write-Output "El nombre és més gran que 10"
} elseif ($number -gt 5) {
    Write-Output "El nombre és més gran que 5 però menor o igual a 10"
} else {
    Write-Output "El nombre és 5 o menys"
}

En aquest exemple, la primera condició $number -gt 10 és falsa, però la segona condició $number -gt 5 és certa. Per tant, es mostra el missatge "El nombre és més gran que 5 però menor o igual a 10".

Switch Statement

La declaració switch avalua una sola expressió contra múltiples valors possibles i executa el bloc de codi corresponent.

Sintaxi

switch (<expressió>) {
    <valor1> { <codi a executar si l'expressió és igual a valor1> }
    <valor2> { <codi a executar si l'expressió és igual a valor2> }
    default { <codi a executar si l'expressió no coincideix amb cap valor> }
}

Exemple

$day = "Dimarts"

switch ($day) {
    "Dilluns" { Write-Output "Avui és Dilluns" }
    "Dimarts" { Write-Output "Avui és Dimarts" }
    "Dimecres" { Write-Output "Avui és Dimecres" }
    default { Write-Output "Avui no és Dilluns, Dimarts o Dimecres" }
}

En aquest exemple, l'expressió $day és avaluada. Com que $day és "Dimarts", es mostra el missatge "Avui és Dimarts".

Resum

Les declaracions condicionals a PowerShell són essencials per controlar el flux dels scripts basats en condicions específiques. Les declaracions if, else, elseif i switch proporcionen eines poderoses per prendre decisions i executar codi en conseqüència. Entendre i utilitzar aquestes declaracions de manera efectiva millorarà la teva capacitat per escriure scripts de PowerShell dinàmics i responsius.
