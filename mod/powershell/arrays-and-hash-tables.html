Arrays i Taules de Hash a PowerShell

Introducció

En aquesta secció, explorarem dues estructures de dades fonamentals a PowerShell: Arrays i Taules de Hash. Entendre aquestes estructures és crucial per a una manipulació i emmagatzematge eficient de dades en els teus scripts.

Arrays

Què és un Array?

Un array és una col·lecció d'elements emmagatzemats en ubicacions de memòria contigües. A PowerShell, els arrays poden emmagatzemar múltiples elements de diferents tipus.

Creació d'Arrays

Pots crear un array envoltant els elements en @().

# Creació d'un array d'enters
$numbers = @(1, 2, 3, 4, 5)

# Creació d'un array de cadenes
$fruits = @("Apple", "Banana", "Cherry")

# Creació d'un array mixt
$mixedArray = @(1, "Hello", 3.14, $true)

Accés als Elements d'un Array

Els elements d'un array s'accedeixen utilitzant el seu índex, començant des de 0.

# Accés al primer element
$firstFruit = $fruits[0]  # Sortida: Apple

# Accés al tercer element
$thirdNumber = $numbers[2]  # Sortida: 3

Modificació d'Arrays

Pots modificar els elements d'un array assignant nous valors a índexs específics.

# Canvi del segon fruit
$fruits[1] = "Blueberry"  # $fruits ara conté "Apple", "Blueberry", "Cherry"

Mètodes d'Array

Els arrays de PowerShell venen amb diversos mètodes útils.

# Afegir un element a l'array
$numbers += 6  # $numbers ara conté 1, 2, 3, 4, 5, 6

# Eliminar un element de l'array
$numbers = $numbers - 3  # $numbers ara conté 1, 2, 4, 5, 6

Iteració sobre Arrays

Pots utilitzar bucles per iterar sobre els elements d'un array.

# Utilitzant un bucle foreach
foreach ($fruit in $fruits) {
    Write-Output $fruit
}

Taules de Hash

Què és una Taula de Hash?

Una taula de hash és una col·lecció de parells clau-valor, on cada clau és única. Les taules de hash són útils per emmagatzemar dades que necessiten ser recuperades ràpidament utilitzant una clau.

Creació de Taules de Hash

Pots crear una taula de hash utilitzant @{}.

# Creació d'una taula de hash
$person = @{
    Name = "John"
    Age = 30
    Occupation = "Engineer"
}

Accés als Elements d'una Taula de Hash

Els elements d'una taula de hash s'accedeixen utilitzant les seves claus.

# Accés al valor associat amb la clau 'Name'
$name = $person["Name"]  # Sortida: John

# Accés al valor utilitzant la notació de punt
$age = $person.Age  # Sortida: 30

Modificació de Taules de Hash

Pots afegir o modificar elements en una taula de hash assignant valors a claus.

# Afegir un nou parell clau-valor
$person["City"] = "New York"

# Modificar un valor existent
$person.Age = 31

Mètodes de Taula de Hash

Les taules de hash venen amb diversos mètodes útils.

# Comprovar si una clau existeix
$exists = $person.ContainsKey("Name")  # Sortida: True

# Eliminar un parell clau-valor
$person.Remove("Occupation")

Iteració sobre Taules de Hash

Pots utilitzar bucles per iterar sobre els elements d'una taula de hash.

# Utilitzant un bucle foreach per iterar sobre les claus
foreach ($key in $person.Keys) {
    Write-Output "$key: $($person[$key])"
}

Conclusió

Els arrays i les taules de hash són estructures de dades potents a PowerShell que et permeten emmagatzemar i manipular col·leccions de dades de manera eficient. Els arrays són ideals per a col·leccions ordenades, mentre que les taules de hash són perfectes per a parells clau-valor. Entendre com crear, accedir, modificar i iterar sobre aquestes estructures millorarà significativament les teves capacitats de scripting a PowerShell.
