Introducció

PowerShell està construït sobre el framework .NET, cosa que significa que té accés a una vasta biblioteca de classes .NET. Entendre com utilitzar aquestes classes pot millorar significativament les teves capacitats de scripting. Aquesta secció et guiarà a través dels conceptes bàsics de l'ús de classes .NET en PowerShell, progressant cap a un ús més avançat.

Accés a classes .NET

Ús bàsic

  • Carregar classes .NET: Pots accedir a les classes .NET directament en PowerShell utilitzant la sintaxi [Namespace.ClassName].
  • Crear instàncies: Utilitza el cmdlet New-Object o el mètode ::new() per crear instàncies de classes .NET.

Exemple: Creació d'un objecte .NET

# Utilitzant New-Object
$object = New-Object System.DateTime

# Utilitzant el mètode ::new() (PowerShell 5.0+)
$object = [System.DateTime]::new()

Mètodes i propietats estàtiques

  • Mètodes estàtics: Es criden a la classe mateixa en lloc d'una instància.
  • Propietats estàtiques: S'accedeixen directament des de la classe.

Exemple: Ús de mètodes i propietats estàtiques

# Mètode estàtic
$currentDate = [System.DateTime]::Now

# Propietat estàtica
$maxValue = [System.Int32]::MaxValue

Classes .NET comunament utilitzades

System.String

  • Descripció: Representa text com una sèrie de caràcters Unicode.
  • Mètodes comuns: Substring(), Replace(), ToUpper(), ToLower()

Exemple: Manipulació de cadenes

$string = "Hello, World!"
$substring = $string.Substring(0, 5)  # Sortida: Hello
$replacedString = $string.Replace("World", "PowerShell")  # Sortida: Hello, PowerShell!

System.IO.File

  • Descripció: Proporciona mètodes estàtics per a la creació, còpia, eliminació, moviment i obertura d'arxius.
  • Mètodes comuns: ReadAllText(), WriteAllText(), Copy(), Delete()

Exemple: Operacions amb arxius

$filePath = "C:\example.txt"

# Escriure text a un arxiu
[System.IO.File]::WriteAllText($filePath, "Hello, PowerShell!")

# Llegir text d'un arxiu
$content = [System.IO.File]::ReadAllText($filePath)
Write-Output $content  # Sortida: Hello, PowerShell!

System.Net.WebClient

  • Descripció: Proporciona mètodes comuns per enviar dades a i rebre dades d'un recurs identificat per un URI.
  • Mètodes comuns: DownloadString(), UploadString()

Exemple: Descarregar contingut del web

$url = "http://example.com"
$webClient = [System.Net.WebClient]::new()
$content = $webClient.DownloadString($url)
Write-Output $content

Ús avançat

Ús de col·leccions .NET

  • System.Collections.Generic.List: Una llista fortament tipada d'objectes.
  • System.Collections.Hashtable: Representa una col·lecció de parells clau/valor.

Exemple: Ús d'una llista genèrica

# Crear una nova llista d'enters
$list = [System.Collections.Generic.List[int]]::new()
$list.Add(1)
$list.Add(2)
$list.Add(3)

# Iterar a través de la llista
foreach ($item in $list) {
    Write-Output $item
}

Gestió d'excepcions amb excepcions .NET

  • Try/Catch/Finally: PowerShell suporta la gestió estructurada d'excepcions utilitzant excepcions .NET.

Exemple: Gestió d'excepcions

try {
    # Intentar obrir un arxiu inexistent
    [System.IO.File]::Open("C:\nonexistentfile.txt", [System.IO.FileMode]::Open)
} catch [System.IO.FileNotFoundException] {
    Write-Output "Arxiu no trobat."
} finally {
    Write-Output "Gestió d'excepcions completada."
}

Conclusió

Utilitzar classes .NET en PowerShell obre una àmplia gamma de funcionalitats que poden millorar els teus scripts. Des de la creació bàsica d'objectes fins a la gestió avançada d'excepcions, entendre com aprofitar aquestes classes és crucial tant per a usuaris principiants com avançats de PowerShell. Practica utilitzant diferents classes .NET i explora la documentació de .NET per expandir encara més els teus coneixements i capacitats.
