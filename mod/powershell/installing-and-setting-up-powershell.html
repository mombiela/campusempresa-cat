Introducció

PowerShell és un marc d'automatització de tasques i gestió de configuració de Microsoft, que consisteix en una línia d'ordres i el llenguatge de script associat. Aquesta secció et guiarà a través del procés d'instal·lació i configuració de PowerShell al teu sistema, assegurant-te que tens l'entorn adequat per començar a escriure scripts i automatitzar tasques.

Instal·lació de PowerShell

Windows

PowerShell ve preinstal·lat a Windows 10 i versions posteriors. No obstant això, potser voldràs instal·lar la versió més recent de PowerShell (PowerShell Core) per obtenir funcions addicionals i suport multiplataforma.

  1. Descarregar PowerShell Core:

    • Visita la pàgina de llançaments de PowerShell a GitHub.
    • Descarrega el paquet .msi més recent per a Windows.

  2. Instal·lar PowerShell Core:

    • Executa el fitxer .msi descarregat.
    • Segueix els passos de l'assistent d'instal·lació.
    • Assegura't que l'opció d'afegir PowerShell al PATH del sistema estigui marcada.

  3. Verificar la Instal·lació:

    • Obre un símbol del sistema o el PowerShell de Windows existent.
    • Escriu pwsh i prem Enter.
    • Hauries de veure el prompt de PowerShell Core, indicant una instal·lació exitosa.

macOS

PowerShell Core està disponible per a macOS i es pot instal·lar utilitzant Homebrew.

  1. Instal·lar Homebrew (si no està ja instal·lat):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Instal·lar PowerShell Core:

    brew install --cask powershell

  3. Verificar la Instal·lació:

    • Obre Terminal.
    • Escriu pwsh i prem Enter.
    • Hauries de veure el prompt de PowerShell Core.

Linux

PowerShell Core es pot instal·lar en diverses distribucions de Linux. A continuació es mostren els passos per a Ubuntu:

  1. Descarregar i Instal·lar les claus GPG del repositori de Microsoft:

    wget -q https://packages.microsoft.com/config/ubuntu/20.04/packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb

  2. Actualitzar la llista de productes:

    sudo apt-get update

  3. Instal·lar PowerShell:

    sudo apt-get install -y powershell

  4. Verificar la Instal·lació:

    • Obre Terminal.
    • Escriu pwsh i prem Enter.
    • Hauries de veure el prompt de PowerShell Core.

Configuració de PowerShell

Configuració de la Política d'Execució

La política d'execució determina les condicions sota les quals PowerShell carrega fitxers de configuració i executa scripts. Per defecte, la política d'execució està establerta a Restricted, la qual cosa impedeix que s'executin scripts.

  1. Comprovar la Política d'Execució Actual:

    Get-ExecutionPolicy

  2. Establir la Política d'Execució:

    Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
    • RemoteSigned permet executar scripts creats localment però requereix que els scripts descarregats d'internet estiguin signats per un editor de confiança.
    • L'abast CurrentUser aplica la política només a l'usuari actual.

Instal·lació de Mòduls de PowerShell

Els mòduls són paquets de cmdlets, proveïdors, funcions, fluxos de treball, variables i àlies. La PowerShell Gallery és el repositori central per al contingut de PowerShell.

  1. Instal·lar un Mòdul:

    Install-Module -Name Az -Scope CurrentUser -Repository PSGallery -Force
    • Az és un mòdul per gestionar recursos d'Azure.
    • -Scope CurrentUser instal·la el mòdul per a l'usuari actual.
    • -Repository PSGallery especifica el repositori de fonts.

  2. Verificar la Instal·lació del Mòdul:

    Get-Module -ListAvailable -Name Az

Personalització del Perfil de PowerShell

El perfil de PowerShell és un script que s'executa cada vegada que PowerShell s'inicia. Pots utilitzar-lo per personalitzar el teu entorn.

  1. Localitzar la Ruta del Perfil:

    $PROFILE

  2. Crear o Editar el Perfil:

    if (!(Test-Path -Path $PROFILE)) {
    New-Item -ItemType File -Path $PROFILE -Force
}
notepad $PROFILE

  3. Afegir Personalitzacions:

    • Exemple: Afegir un àlies per a una comanda utilitzada freqüentment.
    Set-Alias ll Get-ChildItem

Conclusió

Seguint els passos descrits en aquesta secció, ara hauries de tenir PowerShell instal·lat i configurat al teu sistema. També has après a establir la política d'execució, instal·lar mòduls i personalitzar el teu perfil de PowerShell. Amb aquesta configuració, estàs preparat per començar a explorar i utilitzar PowerShell per a tasques d'automatització i scripting.
