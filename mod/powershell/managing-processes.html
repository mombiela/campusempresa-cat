Introducció

La gestió de processos és una habilitat crucial per als administradors de sistemes i desenvolupadors. PowerShell proporciona un conjunt robust de cmdlets per gestionar processos en una màquina local o remota. Aquesta secció et guiarà des dels conceptes bàsics fins a tècniques avançades per gestionar processos utilitzant PowerShell.

Conceptes Clau

  • Procés: Una instància d'un programa en execució.
  • Cmdlets: Comandes de PowerShell dissenyades per realitzar tasques específiques.
  • Pipelines: Un mètode per passar la sortida d'un cmdlet com a entrada a un altre.

Començant amb els Processos

Veure Processos

Per veure els processos que s'estan executant actualment, pots utilitzar el cmdlet Get-Process.

# Llistar tots els processos en execució
Get-Process

Pots filtrar processos per nom o altres propietats:

# Llistar processos amb el nom 'notepad'
Get-Process -Name notepad

Aturar Processos

Per aturar un procés en execució, utilitza el cmdlet Stop-Process.

# Aturar un procés pel seu nom
Stop-Process -Name notepad

# Aturar un procés pel seu ID
Stop-Process -Id 1234

Iniciar Processos

Per iniciar un nou procés, utilitza el cmdlet Start-Process.

# Iniciar Notepad
Start-Process notepad.exe

Gestió Avançada de Processos

Monitorització de Processos

Pots monitoritzar processos i el seu ús de recursos utilitzant Get-Process amb propietats addicionals.

# Obtenir informació detallada sobre un procés
Get-Process -Name notepad | Select-Object Name, Id, CPU, WS

Gestió de Processos Remots

PowerShell et permet gestionar processos en màquines remotes utilitzant Invoke-Command.

# Obtenir processos en una màquina remota
Invoke-Command -ComputerName RemotePC -ScriptBlock { Get-Process }

# Aturar un procés en una màquina remota
Invoke-Command -ComputerName RemotePC -ScriptBlock { Stop-Process -Name notepad }

Automatització de la Gestió de Processos

Pots automatitzar tasques de gestió de processos utilitzant scripts de PowerShell.

# Script per monitoritzar i reiniciar un procés si s'atura
$processName = "notepad"
while ($true) {
    $process = Get-Process -Name $processName -ErrorAction SilentlyContinue
    if (-not $process) {
        Start-Process $processName
        Write-Output "$processName restarted at $(Get-Date)"
    }
    Start-Sleep -Seconds 10
}

Resum

En aquesta secció, has après com gestionar processos utilitzant PowerShell. Pots:

  • Veure processos en execució amb Get-Process.
  • Aturar processos utilitzant Stop-Process.
  • Iniciar nous processos amb Start-Process.
  • Monitoritzar i gestionar processos en màquines remotes amb Invoke-Command.
  • Automatitzar tasques de gestió de processos utilitzant scripts de PowerShell.

Entendre aquests conceptes i cmdlets t'ajudarà a gestionar processos de manera eficient tant en sistemes locals com remots, fent que les teves tasques administratives siguin més fluides i automatitzades.
