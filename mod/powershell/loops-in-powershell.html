Introducció als Bucles

Els bucles són constructes fonamentals en la programació que permeten executar un bloc de codi múltiples vegades. En PowerShell, els bucles ajuden a automatitzar tasques repetitives de manera eficient. Aquesta secció cobrirà els diferents tipus de bucles disponibles en PowerShell i com utilitzar-los.

Tipus de Bucles en PowerShell

PowerShell admet diversos tipus de bucles:

  • for loop
  • foreach loop
  • while loop
  • do-while loop
  • do-until loop

`for` Loop

El bucle for s'utilitza quan saps per endavant quantes vegades vols executar una instrucció o un bloc d'instruccions.

Sintaxi

for ($i = 0; $i -lt 10; $i++) {
    Write-Output "Iteració $i"
}

Explicació

  • $i = 0: Inicialitza el comptador del bucle.
  • $i -lt 10: Condició per continuar el bucle.
  • $i++: Incrementa el comptador del bucle després de cada iteració.

Exemple

for ($i = 1; $i -le 5; $i++) {
    Write-Output "Aquesta és la iteració número $i"
}

`foreach` Loop

El bucle foreach s'utilitza per iterar sobre una col·lecció d'elements, com ara una matriu o una llista.

Sintaxi

foreach ($item in $collection) {
    Write-Output $item
}

Explicació

  • $item: Representa l'element actual de la col·lecció.
  • $collection: La col·lecció d'elements per iterar.

Exemple

$names = @("Alice", "Bob", "Charlie")
foreach ($name in $names) {
    Write-Output "Hola, $name!"
}

`while` Loop

El bucle while continua executant-se mentre la condició especificada sigui certa.

Sintaxi

while ($condition) {
    # Codi a executar
}

Explicació

  • $condition: La condició que es comprova abans de cada iteració.

Exemple

$i = 0
while ($i -lt 3) {
    Write-Output "Iteració $i"
    $i++
}

`do-while` Loop

El bucle do-while és similar al bucle while, però garanteix que el bloc de codi s'executarà almenys una vegada.

Sintaxi

do {
    # Codi a executar
} while ($condition)

Explicació

  • $condition: La condició que es comprova després de cada iteració.

Exemple

$i = 0
do {
    Write-Output "Iteració $i"
    $i++
} while ($i -lt 3)

`do-until` Loop

El bucle do-until és similar al bucle do-while, però continua executant-se fins que la condició especificada esdevingui certa.

Sintaxi

do {
    # Codi a executar
} until ($condition)

Explicació

  • $condition: La condició que es comprova després de cada iteració.

Exemple

$i = 0
do {
    Write-Output "Iteració $i"
    $i++
} until ($i -ge 3)

Conceptes Clau

  • Instruccions de Control de Bucle: PowerShell admet break i continue per controlar el flux dels bucles.
    • break: Surt del bucle immediatament.
    • continue: Salta la resta de la iteració actual i procedeix a la següent iteració.

Exemple de break

for ($i = 0; $i -lt 10; $i++) {
    if ($i -eq 5) { break }
    Write-Output "Iteració $i"
}

Exemple de continue

for ($i = 0; $i -lt 10; $i++) {
    if ($i -eq 5) { continue }
    Write-Output "Iteració $i"
}

Conclusió

Els bucles són essencials per automatitzar tasques repetitives en PowerShell. Entendre els diferents tipus de bucles i com controlar el seu flux pot millorar significativament les teves capacitats de scripting. Practica utilitzant aquests bucles amb diverses condicions i col·leccions per convertir-te en un expert en scripting de PowerShell.
