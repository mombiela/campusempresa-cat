Tipus i estructures de dades bàsiques
Següent ►

En aquest tema, aprendrem sobre els tipus de dades i les estructures de dades bàsiques que utilitzarem en Tableau. Comprendre aquests conceptes és fonamental per treballar de manera efectiva amb les dades i crear visualitzacions significatives.

Tipus de dades

Els tipus de dades en Tableau són similars als que es troben en altres eines de manipulació de dades. A continuació, es presenten els tipus de dades més comuns:

  1. Numèrics (Numeric)

    • Enter (Integer): Nombres sencers, com ara 1, 2, 3.
    • Decimal (Float): Nombres amb decimals, com ara 1.5, 2.75.

  2. Cadenes de text (String)

    • Text o caràcters, com ara "Barcelona", "Producte A".

  3. Data i hora (Date & Time)

    • Data (Date): Dates, com ara 2023-10-01.
    • Hora (Time): Hores, com ara 14:30:00.
    • Data i hora (DateTime): Combinació de data i hora, com ara 2023-10-01 14:30:00.

  4. Booleà (Boolean)

    • Valors veritables o falsos, com ara TRUE o FALSE.

  5. Geogràfic (Geographic)

    • Dades geogràfiques, com ara ciutats, països, codis postals.

Estructures de dades

Les estructures de dades en Tableau es refereixen a la manera com les dades estan organitzades i emmagatzemades. Les estructures més comunes són:

  1. Taules (Tables)

    • Les dades es presenten en format de taula, amb files i columnes. Cada fila representa un registre i cada columna representa un camp o atribut.

  2. Mida de les dades (Data Granularity)

    • La granularitat de les dades es refereix al nivell de detall de les dades. Per exemple, les dades poden estar a nivell de dia, mes, any, etc.

  3. Relacions (Relationships)

    • Les relacions defineixen com es connecten diferents taules entre si. En Tableau, podem utilitzar unions (joins) i combinacions (blends) per relacionar taules.

Exemples pràctics

Exemple 1: Tipus de dades

Suposem que tenim una taula amb les següents dades:

ID Nom Data de naixement Ingressos Actiu
1 Anna 1990-05-15 50000.50 TRUE
2 Bernat 1985-11-23 60000.00 FALSE
3 Carla 1992-07-30 55000.75 TRUE
  • ID: Enter
  • Nom: Cadena de text
  • Data de naixement: Data
  • Ingressos: Decimal
  • Actiu: Booleà

Exemple 2: Estructures de dades

Suposem que tenim dues taules: Vendes i Clients.

Taula Vendes:

VendaID ClientID DataVenda Import
1 101 2023-01-15 150.00
2 102 2023-02-20 200.00
3 101 2023-03-10 250.00

Taula Clients:

ClientID Nom Ciutat
101 Anna Barcelona
102 Bernat Madrid

Podem crear una relació entre aquestes dues taules utilitzant el camp ClientID.

Exercici pràctic

Exercici 1: Identificant tipus de dades

A continuació, es presenta una taula amb diferents tipus de dades. Identifica el tipus de dades de cada columna.

ProducteID NomProducte DataLlançament Preu Disponible
1 Producte A 2023-01-01 25.50 TRUE
2 Producte B 2023-02-15 30.00 FALSE
3 Producte C 2023-03-20 22.75 TRUE

Solució:

  • ProducteID: Enter
  • NomProducte: Cadena de text
  • DataLlançament: Data
  • Preu: Decimal
  • Disponible: Booleà

Exercici 2: Creant una relació

Suposem que tenim les següents taules:

Taula Comandes:

ComandaID ClientID DataComanda Total
1 201 2023-04-01 100.00
2 202 2023-04-05 150.00
3 201 2023-04-10 200.00

Taula Clients:

ClientID Nom Ciutat
201 Carla València
202 David Sevilla

Crea una relació entre aquestes dues taules utilitzant el camp ClientID.

Solució:

La relació es crea utilitzant el camp ClientID com a clau comuna entre les dues taules.

Conclusió

En aquest tema, hem après sobre els tipus de dades i les estructures de dades bàsiques en Tableau. Hem vist exemples pràctics i hem realitzat exercicis per reforçar els conceptes apresos. Ara estem preparats per avançar cap a tècniques més avançades de visualització i anàlisi de dades en Tableau.
