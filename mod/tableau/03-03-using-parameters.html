Utilitzant paràmetres
Següent ►

Els paràmetres a Tableau són valors dinàmics que es poden utilitzar per modificar visualitzacions, càlculs i filtres. Els paràmetres permeten als usuaris interactuar amb les visualitzacions de manera més flexible i personalitzada.

Què és un paràmetre?

Un paràmetre és un valor que es pot canviar per l'usuari per afectar la visualització. Pot ser un número, una data, un text o una llista de valors. Els paràmetres es poden utilitzar en càlculs, filtres, referències de línies, i molt més.

Creant un paràmetre

Passos per crear un paràmetre:

  1. Obre Tableau i carrega una font de dades.
  2. A la finestra de dades, fes clic amb el botó dret i selecciona "Crea paràmetre...".
  3. Defineix les propietats del paràmetre:
    • Nom: Dona un nom descriptiu al paràmetre.
    • Tipus de dades: Selecciona el tipus de dades (Número, Data, Text).
    • Permet valors: Defineix si el paràmetre permetrà qualsevol valor, una llista de valors o un rang de valors.
    • Valors per defecte: Estableix un valor per defecte per al paràmetre.

Exemple de creació d'un paràmetre:

Nom: Selecció de Categoria
Tipus de dades: String
Permet valors: Llista
Valors per defecte: Mobiliari, Tecnologia, Subministraments d'Oficina

Utilitzant paràmetres en càlculs

Un cop creat el paràmetre, es pot utilitzar en càlculs per modificar les visualitzacions. Per exemple, podem crear un càlcul que mostri les vendes només per a la categoria seleccionada.

Exemple de càlcul amb paràmetre:

IF [Categoria] = [Selecció de Categoria] THEN [Vendes] ELSE 0 END

Aquest càlcul mostrarà les vendes només per a la categoria seleccionada pel paràmetre.

Utilitzant paràmetres en filtres

Els paràmetres també es poden utilitzar per crear filtres dinàmics. Per exemple, podem utilitzar un paràmetre per filtrar les dades per any.

Exemple de filtre amb paràmetre:

  1. Crea un paràmetre per seleccionar l'any:

    • Nom: Selecció d'Any
    • Tipus de dades: Integer
    • Permet valors: Rang
    • Valors per defecte: 2018 a 2022

  2. Crea un càlcul per filtrar les dades per l'any seleccionat:

[Year] = [Selecció d'Any]
  1. Arrossega el càlcul a la targeta de filtres.

Utilitzant paràmetres en referències de línies

Els paràmetres també es poden utilitzar per crear línies de referència dinàmiques en els gràfics.

Exemple de línia de referència amb paràmetre:

  1. Crea un paràmetre per seleccionar el valor de la línia de referència:

    • Nom: Valor de Referència
    • Tipus de dades: Float
    • Permet valors: Qualsevol valor
    • Valor per defecte: 5000

  2. Afegeix una línia de referència al gràfic:

    • Fes clic amb el botó dret a l'eix on vols afegir la línia de referència.
    • Selecciona "Afegeix línia de referència".
    • Selecciona el paràmetre "Valor de Referència" com a valor de la línia de referència.

Exercici pràctic

Objectiu:

Crear un paràmetre que permeti als usuaris seleccionar una categoria de producte i mostrar les vendes només per a la categoria seleccionada.

Passos:

  1. Crea un paràmetre anomenat "Selecció de Categoria" amb els valors "Mobiliari", "Tecnologia" i "Subministraments d'Oficina".
  2. Crea un càlcul anomenat "Vendes per Categoria Seleccionada" que utilitzi el paràmetre per filtrar les vendes.
  3. Afegeix el càlcul a la visualització per mostrar les vendes per a la categoria seleccionada.

Solució:

  1. Crea el paràmetre:
Nom: Selecció de Categoria
Tipus de dades: String
Permet valors: Llista
Valors per defecte: Mobiliari, Tecnologia, Subministraments d'Oficina
  1. Crea el càlcul:
IF [Categoria] = [Selecció de Categoria] THEN [Vendes] ELSE 0 END
  1. Afegeix el càlcul a la visualització:
    • Arrossega el càlcul "Vendes per Categoria Seleccionada" a la targeta de text o a l'eix de la visualització.

Resum

Els paràmetres a Tableau són una eina poderosa per crear visualitzacions dinàmiques i interactives. Permeten als usuaris modificar visualitzacions, càlculs i filtres de manera flexible. En aquest tema, hem après a crear paràmetres, utilitzar-los en càlculs, filtres i línies de referència, i hem practicat amb un exercici pràctic per reforçar els conceptes apresos.
