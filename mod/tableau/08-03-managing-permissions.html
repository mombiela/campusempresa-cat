Gestionant permisos
Següent ►

En aquest tema, aprendrem com gestionar els permisos a Tableau Server i Tableau Online. La gestió de permisos és crucial per assegurar que només els usuaris autoritzats tinguin accés a les dades i visualitzacions adequades. Això ajuda a mantenir la seguretat i la integritat de les dades.

Objectius d'aprenentatge

Al final d'aquest tema, hauràs après:

  • Com funcionen els permisos a Tableau Server i Tableau Online.
  • Com assignar permisos a usuaris i grups.
  • Com configurar permisos a nivell de projecte, llibre de treball i font de dades.
  • Com utilitzar les vistes d'administració per monitoritzar els permisos.

  1. Comprenent els permisos a Tableau

1.1. Què són els permisos?

Els permisos a Tableau determinen qui pot veure, interactuar, modificar o administrar els continguts (com projectes, llibres de treball i fonts de dades) a Tableau Server o Tableau Online.

1.2. Nivells de permisos

Els permisos es poden configurar a diferents nivells:

  • Projecte: Conté llibres de treball i fonts de dades.
  • Llibre de treball: Conté fulls de càlcul, quadres de comandament i històries.
  • Font de dades: Conté les dades utilitzades per crear visualitzacions.

  1. Assignant permisos a usuaris i grups

2.1. Usuaris i grups

A Tableau, pots assignar permisos tant a usuaris individuals com a grups d'usuaris. Els grups permeten gestionar permisos de manera més eficient, especialment en organitzacions grans.

2.2. Tipus de permisos

Els permisos es divideixen en diverses categories:

  • Visualitzar: Permet veure el contingut.
  • Interactuar: Permet interactuar amb el contingut (per exemple, filtrar dades).
  • Modificar: Permet editar el contingut.
  • Administrar: Permet gestionar el contingut i els seus permisos.

2.3. Assignació de permisos

Per assignar permisos, segueix aquests passos:

  1. Accedeix a la pàgina de permisos:

    • Navega fins al projecte, llibre de treball o font de dades.
    • Fes clic a la icona de permisos (un escut) o selecciona "Permisos" des del menú desplegable.

  2. Afegeix usuaris o grups:

    • Fes clic a "Afegeix un usuari o grup".
    • Selecciona els usuaris o grups als quals vols assignar permisos.

  3. Configura els permisos:

    • Defineix els permisos per a cada categoria (Visualitzar, Interactuar, Modificar, Administrar).
    • Pots utilitzar plantilles de permisos predefinides o personalitzar els permisos segons les necessitats.

  4. Guarda els canvis:

    • Fes clic a "Aplica" o "Guarda" per confirmar els canvis.

  1. Configurant permisos a diferents nivells

3.1. Permisos a nivell de projecte

Els permisos a nivell de projecte s'apliquen a tots els llibres de treball i fonts de dades dins del projecte. Això facilita la gestió de permisos per a múltiples continguts.

3.2. Permisos a nivell de llibre de treball

Els permisos a nivell de llibre de treball permeten controlar qui pot veure, interactuar o modificar un llibre de treball específic.

3.3. Permisos a nivell de font de dades

Els permisos a nivell de font de dades determinen qui pot accedir i utilitzar una font de dades específica per crear visualitzacions.

  1. Utilitzant les vistes d'administració per monitoritzar els permisos

4.1. Vistes d'administració

Tableau Server i Tableau Online ofereixen vistes d'administració que permeten als administradors monitoritzar i gestionar els permisos de manera eficient.

4.2. Monitorització de permisos

Les vistes d'administració proporcionen informació detallada sobre:

  • Qui té accés a què.
  • Quins permisos estan assignats a cada usuari o grup.
  • Activitats recents relacionades amb els permisos.

Exercici pràctic

Exercici 1: Assignar permisos a un projecte

  1. Crea un nou projecte anomenat "Projecte de Vendes".
  2. Afegeix un grup anomenat "Analistes de Vendes".
  3. Assigna permisos al grup "Analistes de Vendes" per al projecte "Projecte de Vendes" amb els següents permisos:
    • Visualitzar: Permès
    • Interactuar: Permès
    • Modificar: No permès
    • Administrar: No permès

Solució

  1. Crea un nou projecte:

    • Navega fins a la pàgina principal de Tableau Server o Tableau Online.
    • Fes clic a "Nou projecte" i introdueix "Projecte de Vendes" com a nom.

  2. Afegeix un grup:

    • Navega fins a la pàgina d'administració d'usuaris i grups.
    • Fes clic a "Nou grup" i introdueix "Analistes de Vendes" com a nom.

  3. Assigna permisos:

    • Navega fins al projecte "Projecte de Vendes".
    • Fes clic a la icona de permisos.
    • Afegeix el grup "Analistes de Vendes".
    • Configura els permisos segons les instruccions (Visualitzar: Permès, Interactuar: Permès, Modificar: No permès, Administrar: No permès).
    • Fes clic a "Aplica" per guardar els canvis.

Conclusió

En aquest tema, hem après com gestionar els permisos a Tableau Server i Tableau Online. Hem vist com assignar permisos a usuaris i grups, configurar permisos a diferents nivells i utilitzar les vistes d'administració per monitoritzar els permisos. La gestió adequada dels permisos és essencial per mantenir la seguretat i la integritat de les dades a Tableau.
