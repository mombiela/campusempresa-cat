Programant extractes

Següent ►

En aquest tema, aprendrem com programar extractes a Tableau Server i Tableau Online. La programació d'extractes és una funcionalitat clau que permet actualitzar les dades de manera automàtica i periòdica, assegurant que les visualitzacions sempre mostrin informació actualitzada.

Objectius d'aprenentatge

  • Entendre què són els extractes i per què són importants.
  • Aprendre a programar extractes a Tableau Server i Tableau Online.
  • Configurar horaris de programació per a extractes.
  • Gestionar i monitoritzar extractes programats.

Què són els extractes?

Els extractes són còpies de les dades que es guarden en un format optimitzat per a l'anàlisi a Tableau. Aquests permeten millorar el rendiment de les visualitzacions i treballar amb dades fora de línia.

Avantatges dels extractes

  • Rendiment millorat: Els extractes estan optimitzats per a consultes ràpides.
  • Treball fora de línia: Pots treballar amb les dades sense necessitat d'una connexió constant a la font de dades.
  • Reducció de la càrrega en les fonts de dades: Les consultes es fan sobre l'extracte, no directament sobre la font de dades original.

Programant extractes a Tableau Server

Pas 1: Publicar el llibre de treball amb un extracte

  1. Obre Tableau Desktop i carrega el llibre de treball que vols publicar.
  2. Crea un extracte de les dades si encara no ho has fet:
    • Ves a la font de dades.
    • Selecciona "Extracte" en lloc de "En directe".
    • Fes clic a "Full Extract" i després a "Extract".
  3. Publica el llibre de treball a Tableau Server:
    • Fes clic a "Server" > "Publica llibre de treball".
    • Completa els detalls necessaris i fes clic a "Publica".

Pas 2: Configurar la programació de l'extracte

  1. Accedeix a Tableau Server i navega fins al llibre de treball publicat.
  2. Obre el llibre de treball i selecciona la font de dades que conté l'extracte.
  3. Configura la programació:
    • Fes clic a "Més accions" (els tres punts) al costat de la font de dades.
    • Selecciona "Programar actualització".
    • Tria un horari de programació existent o crea'n un de nou.
    • Defineix la freqüència (diària, setmanal, etc.) i l'hora de l'actualització.
    • Fes clic a "Desa".

Monitorització i gestió d'extractes programats

Monitorització

  • Tauler d'administració: Utilitza el tauler d'administració de Tableau Server o Tableau Online per veure l'estat dels extractes programats.
  • Notificacions: Configura notificacions per rebre alertes en cas d'errors en les actualitzacions dels extractes.

Gestió

  • Editar programacions: Pots editar les programacions existents per ajustar la freqüència o l'hora.
  • Eliminar programacions: Si ja no necessites una programació, pots eliminar-la per evitar actualitzacions innecessàries.

Exercici pràctic

Exercici 1: Programar un extracte

  1. Crea un extracte d'una font de dades en Tableau Desktop.
  2. Publica el llibre de treball a Tableau Server o Tableau Online.
  3. Configura una programació per actualitzar l'extracte diàriament a les 8:00 AM.
  4. Verifica que la programació s'ha creat correctament.

Solució

  1. Crea un extracte: 
    - Obre Tableau Desktop.
- Carrega la font de dades.
- Selecciona "Extracte" i fes clic a "Extract".
  2. Publica el llibre de treball: 
    - Fes clic a "Server" > "Publica llibre de treball".
- Completa els detalls i fes clic a "Publica".
  3. Configura la programació: 
    - Accedeix a Tableau Server o Tableau Online.
- Navega fins al llibre de treball publicat.
- Selecciona la font de dades i fes clic a "Més accions" > "Programar actualització".
- Tria "Diàriament" i estableix l'hora a les 8:00 AM.
- Fes clic a "Desa".
  4. Verifica: 
    - Accedeix al tauler d'administració.
- Comprova que la programació apareix a la llista d'actualitzacions programades.

Resum

En aquest tema, hem après què són els extractes i com programar-los a Tableau Server i Tableau Online. Hem vist com configurar horaris de programació per assegurar que les dades es mantinguin actualitzades automàticament. També hem après a monitoritzar i gestionar extractes programats per garantir un rendiment òptim i dades actualitzades en les nostres visualitzacions.
