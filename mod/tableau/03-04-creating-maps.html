Creant mapes

Següent ►

Els mapes són una eina poderosa per visualitzar dades geogràfiques i identificar patrons espacials. En aquest tema, aprendrem a crear mapes a Tableau, utilitzant diverses tècniques per representar dades geogràfiques de manera efectiva.

  1. Introducció als mapes a Tableau

Què és un mapa a Tableau?

Un mapa a Tableau és una visualització que utilitza coordenades geogràfiques per representar dades en un context espacial. Els mapes poden mostrar informació com ara ubicacions, densitats de població, vendes per regió, etc.

Tipus de mapes a Tableau

  • Mapes de punts: Mostren ubicacions específiques amb punts.
  • Mapes de calor: Representen la densitat de dades en una àrea.
  • Mapes de coropletes: Utilitzen colors per mostrar dades agregades per regions (comarques, països, etc.).

  1. Creant un mapa bàsic

Pas 1: Connectar-se a les dades

  1. Obre Tableau i connecta't a la teva font de dades.
  2. Assegura't que la teva font de dades conté camps geogràfics (com ara latitud, longitud, ciutat, país, etc.).

Pas 2: Crear la visualització del mapa

  1. Arrossega un camp geogràfic (per exemple, "Ciutat") a la vista.
  2. Tableau automàticament crearà un mapa de punts amb les ubicacions geogràfiques.
Exemple de codi:
1. Arrossega el camp "Ciutat" a la targeta "Rows".
2. Tableau crearà un mapa amb punts que representen cada ciutat.

Pas 3: Afegir dades al mapa

  1. Arrossega un camp de mesures (per exemple, "Vendes") a la targeta "Color".
  2. Els punts del mapa canviaran de color segons el valor de les vendes.
Exemple de codi:
1. Arrossega el camp "Vendes" a la targeta "Color".
2. Els punts del mapa es colorejaran segons el valor de les vendes.

  1. Personalitzant el mapa

Utilitzant capes de mapes

  1. A la targeta "Map Layers", pots activar o desactivar capes com ara noms de llocs, fronteres, etc.
  2. Pots ajustar la transparència, el color i altres propietats de les capes.

Afegint etiquetes

  1. Arrossega un camp (per exemple, "Ciutat") a la targeta "Label".
  2. Les etiquetes apareixeran al mapa, mostrant el nom de cada ciutat.
Exemple de codi:
1. Arrossega el camp "Ciutat" a la targeta "Label".
2. Les etiquetes amb els noms de les ciutats apareixeran al mapa.

Utilitzant filtres

  1. Arrossega un camp (per exemple, "Regió") a la targeta "Filters".
  2. Selecciona les regions que vols mostrar al mapa.
Exemple de codi:
1. Arrossega el camp "Regió" a la targeta "Filters".
2. Selecciona les regions que vols incloure al mapa.

  1. Creant mapes avançats

Mapes de calor

  1. Arrossega un camp geogràfic (per exemple, "Ciutat") a la vista.
  2. A la targeta "Marks", selecciona "Map".
  3. Arrossega un camp de mesures (per exemple, "Vendes") a la targeta "Color".
  4. Ajusta la intensitat del color per representar la densitat de les dades.

Mapes de coropletes

  1. Arrossega un camp geogràfic (per exemple, "País") a la vista.
  2. A la targeta "Marks", selecciona "Map".
  3. Arrossega un camp de mesures (per exemple, "Vendes") a la targeta "Color".
  4. Els països es colorejaran segons el valor de les vendes.

  1. Exercicis pràctics

Exercici 1: Crear un mapa de punts

  1. Connecta't a una font de dades que contingui informació geogràfica.
  2. Crea un mapa de punts que mostri les ubicacions de les ciutats.
  3. Afegiu una mesura (per exemple, "Vendes") per canviar el color dels punts segons el valor de les vendes.

Exercici 2: Crear un mapa de calor

  1. Utilitza la mateixa font de dades de l'exercici anterior.
  2. Crea un mapa de calor que mostri la densitat de les vendes per regió.

Exercici 3: Crear un mapa de coropletes

  1. Utilitza la mateixa font de dades de l'exercici anterior.
  2. Crea un mapa de coropletes que mostri les vendes agregades per país.

Solucions dels exercicis

Solució 1: Crear un mapa de punts

  1. Connecta't a la font de dades.
  2. Arrossega el camp "Ciutat" a la targeta "Rows".
  3. Arrossega el camp "Vendes" a la targeta "Color".

Solució 2: Crear un mapa de calor

  1. Connecta't a la font de dades.
  2. Arrossega el camp "Ciutat" a la targeta "Rows".
  3. A la targeta "Marks", selecciona "Map".
  4. Arrossega el camp "Vendes" a la targeta "Color".

Solució 3: Crear un mapa de coropletes

  1. Connecta't a la font de dades.
  2. Arrossega el camp "País" a la targeta "Rows".
  3. A la targeta "Marks", selecciona "Map".
  4. Arrossega el camp "Vendes" a la targeta "Color".

Conclusió

En aquest tema, hem après a crear mapes a Tableau, des de mapes bàsics de punts fins a mapes avançats com mapes de calor i mapes de coropletes. Els mapes són una eina poderosa per visualitzar dades geogràfiques i identificar patrons espacials. Amb la pràctica, podràs crear visualitzacions geogràfiques efectives que ajudin a comunicar informació de manera clara i impactant.
