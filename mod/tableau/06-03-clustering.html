Introducció a l'Agrupament

L'agrupament (clustering) és una tècnica d'anàlisi de dades que permet agrupar elements similars en grups (clusters) basant-se en les seves característiques. En Tableau, l'agrupament es pot utilitzar per identificar patrons i tendències en les dades, facilitant la presa de decisions informades.

Objectius d'aquest tema:

  1. Entendre què és l'agrupament i com funciona.
  2. Aprendre a crear agrupaments en Tableau.
  3. Interpretar els resultats dels agrupaments.
  4. Aplicar l'agrupament a casos pràctics.

Què és l'Agrupament?

L'agrupament és una tècnica d'aprenentatge automàtic no supervisat que classifica les dades en grups basant-se en la similitud de les seves característiques. Els elements dins d'un mateix grup són més similars entre ells que amb els elements d'altres grups.

Tipus d'Agrupament:

  • K-means clustering: Un dels mètodes més comuns, que divideix les dades en k grups basant-se en la distància mitjana.
  • Hierarchical clustering: Crea una jerarquia de grups, que pot ser aglomerativa (comença amb cada punt com un grup i els combina) o divisiva (comença amb tots els punts en un grup i els divideix).

Creant Agrupaments en Tableau

Pas 1: Preparar les Dades

Abans de crear un agrupament, assegura't que les dades estiguin netes i preparades. Pots utilitzar tècniques de neteja de dades vistes en mòduls anteriors.

Pas 2: Crear un Agrupament

  1. Obre Tableau i connecta't a la teva font de dades.
  2. Crea una visualització amb les dades que vols agrupar. Per exemple, pots crear un gràfic de dispersió amb dues variables.
  3. Selecciona la pestanya "Analytics" a la part dreta de la interfície.
  4. Arrossega l'opció "Cluster" a la visualització.

Pas 3: Configurar l'Agrupament

  1. Defineix el nombre de grups (k): Tableau suggereix un nombre de grups basat en les dades, però pots ajustar-lo segons les teves necessitats.
  2. Ajusta les variables: Pots seleccionar quines variables utilitzar per a l'agrupament. Tableau utilitza les variables de la visualització per defecte, però pots afegir-ne o eliminar-ne.

Exemple Pràctic

Suposem que tenim un conjunt de dades de vendes amb les següents variables:
- Ingressos
- Nombre de clients
- Nombre de productes venuts

Volem agrupar les botigues en funció d'aquestes variables per identificar patrons de rendiment.
  1. Crea un gràfic de dispersió amb "Ingressos" a l'eix X i "Nombre de clients" a l'eix Y.
  2. Arrossega "Cluster" des de la pestanya "Analytics" a la visualització.
  3. Configura l'agrupament per utilitzar "Ingressos", "Nombre de clients" i "Nombre de productes venuts".
  4. Ajusta el nombre de grups a 3.

Interpretació dels Resultats

Després de crear l'agrupament, Tableau mostrarà els grups amb diferents colors. Pots analitzar els grups per identificar patrons, com ara botigues amb alts ingressos però pocs clients, o botigues amb molts clients però baixos ingressos.

Exercici Pràctic

Dades:

  • Utilitza un conjunt de dades de vendes amb les següents variables: "Ingressos", "Nombre de clients", "Nombre de productes venuts".

Objectiu:

  • Crea un agrupament per identificar diferents segments de botigues.

Passos:

  1. Connecta't a les dades de vendes.
  2. Crea una visualització amb "Ingressos" a l'eix X i "Nombre de clients" a l'eix Y.
  3. Arrossega "Cluster" a la visualització.
  4. Configura l'agrupament per utilitzar "Ingressos", "Nombre de clients" i "Nombre de productes venuts".
  5. Ajusta el nombre de grups a 4.
  6. Interpreta els resultats i descriu els diferents segments de botigues.

Solució

  1. Visualització: Gràfic de dispersió amb "Ingressos" a l'eix X i "Nombre de clients" a l'eix Y.
  2. Agrupament: Utilitza "Ingressos", "Nombre de clients" i "Nombre de productes venuts".
  3. Nombre de grups: 4.
  4. Resultats:
    • Grup 1: Botigues amb alts ingressos i molts clients.
    • Grup 2: Botigues amb baixos ingressos i pocs clients.
    • Grup 3: Botigues amb alts ingressos però pocs clients.
    • Grup 4: Botigues amb molts clients però baixos ingressos.

Conclusió

L'agrupament és una eina poderosa per identificar patrons i tendències en les dades. En aquest tema, hem après a crear agrupaments en Tableau, configurar-los i interpretar els resultats. Aquesta tècnica és especialment útil per a l'anàlisi de segments de mercat, rendiment de botigues i altres aplicacions pràctiques.

Resum

  • L'agrupament classifica les dades en grups basant-se en la similitud de les seves característiques.
  • Tableau facilita la creació d'agrupaments amb una interfície intuïtiva.
  • La interpretació dels resultats permet identificar patrons i prendre decisions informades.

Amb aquests coneixements, estàs preparat per aplicar l'agrupament a les teves pròpies dades i descobrir nous insights.
