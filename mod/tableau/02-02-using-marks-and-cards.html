Utilitzant marques i targetes

Següent ►
►

Introducció

En aquest tema, aprendrem a utilitzar marques i targetes a Tableau per personalitzar i enriquir les nostres visualitzacions. Les marques i targetes són eines fonamentals que ens permeten controlar l'aparença i el comportament dels nostres gràfics, afegint detalls visuals que poden ajudar a comunicar millor la informació.

Objectius

  • Entendre què són les marques i les targetes a Tableau.
  • Aprendre a utilitzar les targetes de marques per personalitzar visualitzacions.
  • Explorar diferents tipus de marques i com afecten la visualització de dades.

Què són les marques i les targetes?

Marques

Les marques són elements visuals que representen les dades en una visualització. Poden ser punts, línies, barres, àrees, etc. Cada tipus de marca té les seves pròpies propietats i usos específics.

Targetes

Les targetes són panells a la interfície de Tableau que permeten controlar diferents aspectes de les marques, com ara el color, la mida, la forma, l'etiqueta, el detall i el camí. Utilitzant aquestes targetes, podem personalitzar com es mostren les dades a la visualització.

Utilitzant la targeta de marques

La targeta de marques és una de les eines més potents a Tableau per personalitzar visualitzacions. A continuació, veurem com utilitzar les diferents opcions disponibles a la targeta de marques.

Opcions de la targeta de marques

  1. Tipus de marca: Permet seleccionar el tipus de marca que es vol utilitzar (punt, línia, barra, etc.).
  2. Color: Permet assignar colors a les marques basant-se en dimensions o mesures.
  3. Mida: Permet ajustar la mida de les marques.
  4. Forma: Permet seleccionar diferents formes per a les marques.
  5. Etiqueta: Permet afegir etiquetes a les marques per mostrar informació addicional.
  6. Detall: Permet afegir més dimensions a les marques per mostrar més detalls.
  7. Camí: Utilitzat principalment en gràfics de línies per definir l'ordre de les marques.

Exemple pràctic

Suposem que tenim un conjunt de dades de vendes per regió i volem crear una visualització que mostri les vendes totals per regió amb diferents colors per a cada regió.

  1. Tipus de marca: Seleccionem "Barra" com a tipus de marca.
  2. Color: Arrosseguem la dimensió "Regió" a la targeta de color.
  3. Mida: Ajustem la mida de les barres per fer-les més visibles.
  4. Etiqueta: Arrosseguem la mesura "Vendes totals" a la targeta d'etiqueta per mostrar les vendes a cada barra.
+-------------------+-------------------+
| Regió             | Vendes totals     |
+-------------------+-------------------+
| Nord              | 150,000           |
| Sud               | 200,000           |
| Est               | 175,000           |
| Oest              | 125,000           |
+-------------------+-------------------+

Codi d'exemple

1. Obrim Tableau i connectem-nos a la font de dades.
2. Arrosseguem la dimensió "Regió" a l'eix de columnes.
3. Arrosseguem la mesura "Vendes totals" a l'eix de files.
4. A la targeta de marques, seleccionem "Barra" com a tipus de marca.
5. Arrosseguem la dimensió "Regió" a la targeta de color.
6. Arrosseguem la mesura "Vendes totals" a la targeta d'etiqueta.
7. Ajustem la mida de les barres a la targeta de mida.

Exercici pràctic

Exercici 1: Creant una visualització amb marques i targetes

  1. Connecta't a una font de dades que contingui informació de vendes per producte i regió.
  2. Crea una visualització que mostri les vendes totals per producte.
  3. Utilitza la targeta de marques per assignar diferents colors a cada producte.
  4. Afegeix etiquetes per mostrar les vendes totals a cada barra.
  5. Ajusta la mida de les barres per fer-les més visibles.

Solució

  1. Connecta't a la font de dades.
  2. Arrossega la dimensió "Producte" a l'eix de columnes.
  3. Arrossega la mesura "Vendes totals" a l'eix de files.
  4. A la targeta de marques, selecciona "Barra" com a tipus de marca.
  5. Arrossega la dimensió "Producte" a la targeta de color.
  6. Arrossega la mesura "Vendes totals" a la targeta d'etiqueta.
  7. Ajusta la mida de les barres a la targeta de mida.

Conclusió

En aquest tema, hem après a utilitzar marques i targetes a Tableau per personalitzar les nostres visualitzacions. Hem vist com les targetes de marques ens permeten controlar diferents aspectes de les marques, com ara el color, la mida, la forma, l'etiqueta, el detall i el camí. A més, hem practicat amb un exemple pràctic i un exercici per reforçar els conceptes apresos. En el proper tema, explorarem com construir gràfics bàsics a Tableau.
