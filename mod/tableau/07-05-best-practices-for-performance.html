Millors pràctiques per al rendiment
Següent ►

En aquest tema, explorarem les millors pràctiques per optimitzar el rendiment dels teus llibres de treball a Tableau. Un rendiment òptim és crucial per assegurar que les teves visualitzacions siguin ràpides i responsives, especialment quan treballes amb grans volums de dades. A continuació, es detallen diverses estratègies i tècniques per millorar el rendiment.

1. Utilitza Extractes en lloc de Connexions en Viu

Avantatges dels Extractes:

  • Rendiment més ràpid: Els extractes són optimitzats per Tableau, el que permet una càrrega i processament de dades més ràpids.
  • Treball fora de línia: Pots treballar amb les dades sense necessitat d'estar connectat a la font de dades original.

Com crear un extracte:

  1. Connecta't a la teva font de dades.
  2. Fes clic a "Data" al menú superior.
  3. Selecciona "Extract Data".
  4. Configura les opcions de l'extracte (com filtres, agregacions, etc.).
  5. Fes clic a "Extract".
Data -> Extract Data -> Configura opcions -> Extract

2. Redueix la Quantitat de Dades

Estratègies per reduir dades:

  • Filtra les dades innecessàries: Utilitza filtres per incloure només les dades rellevants.
  • Agrega dades: Agrega les dades a un nivell més alt per reduir la quantitat de registres.
  • Utilitza mostres: Treballa amb una mostra representativa de les dades per a desenvolupament i proves.

Exemple de filtratge de dades:

Data -> Edit Data Source -> Add Filter

3. Optimitza les Consultes SQL

Consells per optimitzar consultes:

  • Utilitza índexs: Assegura't que les columnes utilitzades en filtres i joins tinguin índexs adequats.
  • Evita subconsultes complexes: Simplifica les subconsultes o utilitza vistes materialitzades si és possible.
  • Utilitza funcions de base de dades: Delega càlculs complexos a la base de dades quan sigui possible.

Exemple de consulta optimitzada:

SELECT 
    customer_id, 
    SUM(sales) AS total_sales 
FROM 
    sales_data 
WHERE 
    order_date >= '2023-01-01' 
GROUP BY 
    customer_id;

4. Simplifica les Visualitzacions

Consells per simplificar:

  • Redueix el nombre de marques: Evita visualitzacions amb un nombre excessiu de marques (punts, barres, etc.).
  • Utilitza gràfics senzills: Gràfics senzills com barres, línies i taules solen ser més ràpids que gràfics complexos.
  • Evita l'ús excessiu de càlculs: Redueix el nombre de càlculs en les visualitzacions.

Exemple de gràfic senzill:

Gràfic de barres per mostrar vendes per regió

5. Utilitza Càlculs de Taula amb Precaució

Consells per a càlculs de taula:

  • Limita l'ús de càlculs complexos: Els càlculs de taula poden ser costosos en termes de rendiment.
  • Precalcula si és possible: Si un càlcul es pot fer a la font de dades, fes-ho abans d'importar les dades a Tableau.

Exemple de càlcul de taula:

WINDOW_SUM(SUM([Sales]))

6. Optimitza els Quadres de Comandament

Consells per optimitzar quadres de comandament:

  • Redueix el nombre de fulls: Utilitza només els fulls necessaris en un quadre de comandament.
  • Evita l'ús excessiu de filtres: Els filtres poden afectar el rendiment, especialment si són complexos.
  • Utilitza accions de full: Les accions de full poden ser més eficients que els filtres globals.

Exemple d'acció de full:

Full -> Accions -> Afegeix Acció -> Filtre

7. Monitoritza i Ajusta Regularment

Consells per a la monitorització:

  • Utilitza la gravació del rendiment: Analitza els registres de rendiment per identificar colls d'ampolla.
  • Revisa les visualitzacions regularment: Ajusta les visualitzacions i les fonts de dades segons sigui necessari per mantenir un rendiment òptim.

Exemple de gravació del rendiment:

Help -> Settings and Performance -> Start Performance Recording

Conclusió

Seguint aquestes millors pràctiques, podràs optimitzar el rendiment dels teus llibres de treball a Tableau, assegurant que les teves visualitzacions siguin ràpides i eficients. Recorda que l'optimització és un procés continu, i és important revisar i ajustar les teves estratègies regularment per mantenir un rendiment òptim.
