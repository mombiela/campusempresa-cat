Utilitzant expressions LOD
Següent ►

Les expressions LOD (Level of Detail) són una característica avançada de Tableau que permeten controlar el nivell de detall en els càlculs. Això és especialment útil quan necessites fer càlculs que no es poden aconseguir fàcilment amb les funcions bàsiques de Tableau. Les expressions LOD permeten crear càlculs que es poden aplicar a diferents nivells de detall en les dades, independentment de la visualització actual.

Tipus d'expressions LOD

Hi ha tres tipus principals d'expressions LOD en Tableau:

  1. FIXED: Calcula un valor a un nivell de detall específic, independentment de les dimensions presents a la visualització.
  2. INCLUDE: Afegeix dimensions addicionals al nivell de detall de la visualització.
  3. EXCLUDE: Exclou dimensions del nivell de detall de la visualització.

FIXED

L'expressió FIXED permet definir un càlcul a un nivell de detall específic, independentment de les dimensions de la visualització.

Sintaxi:

{ FIXED [Dimension1], [Dimension2], ... : AGGREGATE_EXPRESSION }

Exemple: Suposem que volem calcular les vendes totals per a cada regió, independentment de les dimensions de la visualització actual.

{ FIXED [Regió] : SUM([Vendes]) }

INCLUDE

L'expressió INCLUDE afegeix dimensions addicionals al nivell de detall de la visualització.

Sintaxi:

{ INCLUDE [Dimension1], [Dimension2], ... : AGGREGATE_EXPRESSION }

Exemple: Suposem que volem calcular la mitjana de vendes per client dins de cada regió.

{ INCLUDE [Client ID] : AVG([Vendes]) }

EXCLUDE

L'expressió EXCLUDE exclou dimensions del nivell de detall de la visualització.

Sintaxi:

{ EXCLUDE [Dimension1], [Dimension2], ... : AGGREGATE_EXPRESSION }

Exemple: Suposem que volem calcular les vendes totals excloent la dimensió de la categoria de producte.

{ EXCLUDE [Categoria] : SUM([Vendes]) }

Exemples Pràctics

Exemple 1: FIXED

Volem calcular el total de vendes per a cada regió, independentment de les dimensions de la visualització.

  1. Crea un nou camp calculat anomenat Total Vendes per Regió.
  2. Utilitza la següent expressió LOD: 
    { FIXED [Regió] : SUM([Vendes]) }
  3. Afegeix aquest camp a la visualització per veure les vendes totals per regió.

Exemple 2: INCLUDE

Volem calcular la mitjana de vendes per client dins de cada regió.

  1. Crea un nou camp calculat anomenat Mitjana Vendes per Client.
  2. Utilitza la següent expressió LOD: 
    { INCLUDE [Client ID] : AVG([Vendes]) }
  3. Afegeix aquest camp a la visualització per veure la mitjana de vendes per client dins de cada regió.

Exemple 3: EXCLUDE

Volem calcular les vendes totals excloent la dimensió de la categoria de producte.

  1. Crea un nou camp calculat anomenat Total Vendes Excloent Categoria.
  2. Utilitza la següent expressió LOD: 
    { EXCLUDE [Categoria] : SUM([Vendes]) }
  3. Afegeix aquest camp a la visualització per veure les vendes totals excloent la categoria de producte.

Exercicis Pràctics

Exercici 1: FIXED

Objectiu: Calcula el total de vendes per a cada ciutat, independentment de les dimensions de la visualització.

  1. Crea un nou camp calculat anomenat Total Vendes per Ciutat.
  2. Utilitza la següent expressió LOD: 
    { FIXED [Ciutat] : SUM([Vendes]) }
  3. Afegeix aquest camp a la visualització i verifica els resultats.

Solució:

{ FIXED [Ciutat] : SUM([Vendes]) }

Exercici 2: INCLUDE

Objectiu: Calcula la mitjana de vendes per producte dins de cada regió.

  1. Crea un nou camp calculat anomenat Mitjana Vendes per Producte.
  2. Utilitza la següent expressió LOD: 
    { INCLUDE [Producte] : AVG([Vendes]) }
  3. Afegeix aquest camp a la visualització i verifica els resultats.

Solució:

{ INCLUDE [Producte] : AVG([Vendes]) }

Exercici 3: EXCLUDE

Objectiu: Calcula les vendes totals excloent la dimensió de la subcategoria de producte.

  1. Crea un nou camp calculat anomenat Total Vendes Excloent Subcategoria.
  2. Utilitza la següent expressió LOD: 
    { EXCLUDE [Subcategoria] : SUM([Vendes]) }
  3. Afegeix aquest camp a la visualització i verifica els resultats.

Solució:

{ EXCLUDE [Subcategoria] : SUM([Vendes]) }

Errors Comuns i Consells

  1. No oblidis les claus {}: Les expressions LOD sempre han d'estar envoltades per claus {}.
  2. Comprova les dimensions: Assegura't que les dimensions utilitzades en les expressions LOD existeixen en les teves dades.
  3. Entén el context: Les expressions LOD poden canviar el nivell de detall de la visualització, així que és important entendre com afecten el context de les dades.

Conclusió

Les expressions LOD són una eina poderosa en Tableau que permeten realitzar càlculs complexos a diferents nivells de detall. Amb la pràctica, podràs utilitzar aquestes expressions per crear visualitzacions més avançades i significatives. En el següent tema, explorarem com crear quadres de comandament per combinar múltiples visualitzacions en una sola vista coherent.
