Introducció a Tableau Server

Què és Tableau Server?

Tableau Server és una plataforma d'anàlisi empresarial que permet als usuaris compartir, distribuir i col·laborar en visualitzacions de dades i informes creats amb Tableau Desktop. Amb Tableau Server, les organitzacions poden centralitzar les seves dades i visualitzacions, permetent un accés segur i controlat a la informació.

Característiques clau de Tableau Server:

  • Compartició segura de dades: Permet compartir visualitzacions i dades de manera segura dins de l'organització.
  • Col·laboració: Facilita la col·laboració entre equips, permetent comentaris i discussions sobre les visualitzacions.
  • Automatització: Permet programar actualitzacions de dades i informes.
  • Escalabilitat: Pot gestionar grans volums de dades i usuaris simultanis.
  • Integració: Es pot integrar amb altres eines i sistemes empresarials.

Instal·lació de Tableau Server

Requisits del sistema:

  • Sistema operatiu: Windows Server o Linux.
  • Memòria RAM: Mínim 8 GB (recomanat 16 GB o més).
  • Espai en disc: Mínim 15 GB d'espai lliure.
  • Processador: CPU de 4 nuclis (recomanat 8 nuclis o més).

Passos per a la instal·lació:

  1. Descarregar Tableau Server: Visita el lloc web oficial de Tableau i descarrega la versió adequada per al teu sistema operatiu.
  2. Executar l'instal·lador: Segueix les instruccions de l'instal·lador per completar la instal·lació.
  3. Configuració inicial: Durant la instal·lació, se't demanarà que configuris paràmetres bàsics com el nom del servidor, la configuració de la base de dades i els usuaris administradors.
  4. Activació de la llicència: Introdueix la clau de llicència per activar Tableau Server.

Visió general de la interfície de Tableau Server

Components principals:

  • Inici: Pàgina principal on es mostren les visualitzacions i projectes recents.
  • Explorar: Permet navegar per les carpetes, projectes i visualitzacions disponibles.
  • Favorits: Accés ràpid a les visualitzacions i projectes marcats com a favorits.
  • Recents: Llista de les visualitzacions i projectes recentment accedits.
  • Cerca: Barra de cerca per trobar visualitzacions, projectes i usuaris.
  • Configuració: Opcions per configurar el servidor, gestionar usuaris i permisos.

Taula de comparació de rols d'usuari:

Rol d'usuari Descripció Permisos clau
Administrador Gestiona tot el servidor, incloent usuaris, permisos i configuracions. Crear, editar, eliminar usuaris i grups, configurar el servidor.
Editor Pot crear i editar visualitzacions i projectes. Crear, editar i eliminar visualitzacions i projectes.
Visualitzador Pot veure visualitzacions i projectes, però no editar-los. Veure visualitzacions i projectes, afegir comentaris.
Convidat Accés limitat a visualitzacions específiques sense necessitat de compte. Veure visualitzacions específiques.

Connectant-se a Tableau Server

Passos per connectar-se:

  1. Obrir Tableau Desktop: Inicia Tableau Desktop al teu ordinador.
  2. Seleccionar "Tableau Server": A la pantalla d'inici, selecciona l'opció "Tableau Server".
  3. Introduir les credencials: Introdueix l'adreça del servidor, el teu nom d'usuari i contrasenya.
  4. Navegar per les visualitzacions: Un cop connectat, podràs navegar per les visualitzacions i projectes disponibles al servidor.

Exercici pràctic

Objectiu:

Connectar-se a Tableau Server i explorar les visualitzacions disponibles.

Passos:

  1. Inicia Tableau Desktop.
  2. Selecciona "Tableau Server" a la pantalla d'inici.
  3. Introdueix les credencials del servidor.
  4. Navega per les visualitzacions i projectes disponibles.
  5. Marca una visualització com a favorita.

Solució:

  1. Inicia Tableau Desktop.
  2. Selecciona "Tableau Server" a la pantalla d'inici.
  3. Introdueix l'adreça del servidor, el teu nom d'usuari i contrasenya.
  4. Un cop connectat, utilitza la barra de navegació per explorar les visualitzacions i projectes disponibles.
  5. Per marcar una visualització com a favorita, fes clic a l'estrella al costat del nom de la visualització.

Resum

En aquesta secció, hem après què és Tableau Server, com instal·lar-lo, i hem explorat la seva interfície i funcionalitats bàsiques. També hem après a connectar-nos a Tableau Server des de Tableau Desktop i a navegar per les visualitzacions disponibles. Aquest coneixement és fonamental per a la col·laboració i la compartició de dades dins d'una organització. En el següent tema, aprofundirem en com publicar llibres de treball a Tableau Server.
