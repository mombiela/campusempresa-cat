Publicant llibres de treball
Següent ►

En aquest tema, aprendrem com publicar llibres de treball a Tableau Server o Tableau Online. Publicar els teus llibres de treball permet compartir les teves visualitzacions amb altres usuaris de la teva organització de manera segura i eficient.

  1. Preparació per a la publicació

1.1. Verificar les dades

Abans de publicar, assegura't que les dades i les visualitzacions estiguin correctes i actualitzades. Revisa els següents punts:

  • Fonts de dades: Verifica que les connexions a les fonts de dades siguin correctes.
  • Visualitzacions: Assegura't que totes les visualitzacions funcionin com s'espera.
  • Filtres i paràmetres: Comprova que els filtres i paràmetres estiguin configurats correctament.

1.2. Optimització del llibre de treball

Optimitza el rendiment del llibre de treball per assegurar una experiència d'usuari fluida:

  • Redueix la mida del llibre de treball: Elimina fulls i dades innecessàries.
  • Utilitza extractes: Considera utilitzar extractes en lloc de connexions en viu per millorar el rendiment.

  1. Publicació a Tableau Server o Tableau Online

2.1. Connexió a Tableau Server o Tableau Online

  1. Obre el teu llibre de treball a Tableau Desktop.
  2. Fes clic a "Server" al menú superior.
  3. Selecciona "Sign In" i introdueix les teves credencials per connectar-te a Tableau Server o Tableau Online.

2.2. Publicar el llibre de treball

  1. Un cop connectat, fes clic a "Server" al menú superior.
  2. Selecciona "Publish Workbook".
  3. Apareixerà una finestra de diàleg amb diverses opcions de configuració.

2.3. Configuració de la publicació

  • Projecte: Selecciona el projecte on vols publicar el llibre de treball.
  • Nom del llibre de treball: Introdueix un nom per al llibre de treball.
  • Descripció: Opcionalment, afegeix una descripció per proporcionar més context als usuaris.
  • Dades: Tria si vols publicar les dades com a extracte o mantenir la connexió en viu.
  • Permisos: Configura els permisos per determinar qui pot veure i editar el llibre de treball.

2.4. Publicar

  1. Revisa totes les opcions de configuració.
  2. Fes clic a "Publish" per completar el procés de publicació.

  1. Gestió de llibres de treball publicats

3.1. Actualització de llibres de treball

Per actualitzar un llibre de treball publicat:

  1. Obre el llibre de treball a Tableau Desktop.
  2. Realitza els canvis necessaris.
  3. Repeteix el procés de publicació, seleccionant el mateix projecte i nom de llibre de treball.

3.2. Eliminació de llibres de treball

Per eliminar un llibre de treball publicat:

  1. Accedeix a Tableau Server o Tableau Online.
  2. Navega fins al projecte on es troba el llibre de treball.
  3. Selecciona el llibre de treball i fes clic a "Delete".

Exercici pràctic

Exercici 1: Publicar un llibre de treball

  1. Crea una visualització senzilla a Tableau Desktop utilitzant una font de dades de mostra.
  2. Connecta't a Tableau Server o Tableau Online.
  3. Publica el llibre de treball seguint els passos descrits anteriorment.
  4. Configura els permisos perquè només els membres del teu equip puguin veure el llibre de treball.

Solució de l'exercici

  1. Crear la visualització:

    - Obre Tableau Desktop.
- Connecta't a la font de dades "Sample - Superstore".
- Crea una visualització senzilla, com un gràfic de barres que mostri les vendes per categoria de producte.

  2. Connectar-se a Tableau Server o Tableau Online:

    - Fes clic a "Server" al menú superior.
- Selecciona "Sign In" i introdueix les teves credencials.

  3. Publicar el llibre de treball:

    - Fes clic a "Server" al menú superior.
- Selecciona "Publish Workbook".
- Configura el projecte, nom del llibre de treball, descripció, dades i permisos.
- Fes clic a "Publish".

  4. Configurar els permisos:

    - A la finestra de configuració de la publicació, selecciona els permisos adequats.
- Assegura't que només els membres del teu equip tinguin accés al llibre de treball.

Conclusió

En aquesta secció, hem après com preparar, publicar i gestionar llibres de treball a Tableau Server o Tableau Online. La publicació de llibres de treball és una eina poderosa per compartir visualitzacions i col·laborar amb altres usuaris de manera eficient. Practica aquests passos per familiaritzar-te amb el procés i assegurar-te que les teves visualitzacions estiguin sempre disponibles per als teus col·legues.
