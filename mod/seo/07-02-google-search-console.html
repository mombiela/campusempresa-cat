Introducció a Google Search Console

Google Search Console (GSC) és una eina gratuïta proporcionada per Google que ajuda els propietaris de llocs web a monitoritzar, mantenir i solucionar problemes de presència del seu lloc en els resultats de cerca de Google. És una eina essencial per a qualsevol estratègia de SEO, ja que proporciona informació valuosa sobre com Google veu el teu lloc web i com els usuaris el troben.

Funcionalitats Clau de Google Search Console

  1. Monitorització del Rendiment de Cerca:

    • Impressions: Nombre de vegades que el teu lloc web apareix en els resultats de cerca.
    • Clics: Nombre de vegades que els usuaris fan clic en el teu lloc web des dels resultats de cerca.
    • CTR (Click-Through Rate): Percentatge de clics respecte a les impressions.
    • Posició Mitjana: Posició mitjana del teu lloc web en els resultats de cerca.

  2. Cobertura de l'Índex:

    • Pàgines Indexades: Nombre de pàgines del teu lloc web que Google ha indexat.
    • Errors d'Indexació: Problemes que impedeixen que les pàgines siguin indexades correctament.

  3. Millores:

    • Usabilitat Mòbil: Problemes d'usabilitat en dispositius mòbils.
    • Dades Estructurades: Errors i advertències en el marcatge de dades estructurades.

  4. Enllaços:

    • Enllaços Interns: Nombre d'enllaços interns dins del teu lloc web.
    • Enllaços Externs: Nombre d'enllaços que apunten al teu lloc web des d'altres llocs.

  5. Sitemaps:

    • Enviar Sitemaps: Permet enviar sitemaps XML per ajudar Google a indexar el teu lloc web de manera més eficient.

Configuració de Google Search Console

Pas 1: Verificació del Lloc Web

Per començar a utilitzar Google Search Console, primer has de verificar que ets el propietari del lloc web. Hi ha diverses maneres de fer-ho:

  1. Fitxer HTML: Descarrega un fitxer HTML proporcionat per Google i puja'l al directori arrel del teu lloc web.
  2. Etiqueta HTML: Afegeix una metaetiqueta proporcionada per Google a la secció <head> del teu lloc web.
  3. Google Analytics: Utilitza el teu compte de Google Analytics per verificar el lloc web.
  4. Google Tag Manager: Utilitza el teu compte de Google Tag Manager per verificar el lloc web.
  5. Proveïdor de Domini: Verifica el lloc web a través del teu proveïdor de domini.

Pas 2: Enviar un Sitemap

Un cop verificat el lloc web, és recomanable enviar un sitemap XML per ajudar Google a indexar les pàgines del teu lloc web de manera més eficient.

<!-- Exemple d'un sitemap XML -->
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://www.example.com/</loc>
    <lastmod>2023-10-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://www.example.com/about</loc>
    <lastmod>2023-10-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
</urlset>

Utilització de Google Search Console

Monitorització del Rendiment

  1. Accedir a la Secció de Rendiment:

    • Navega a la secció "Rendiment" per veure les dades sobre impressions, clics, CTR i posició mitjana.

  2. Filtrar Dades:

    • Pots filtrar les dades per data, tipus de cerca (web, imatges, vídeos), país, dispositiu, etc.

  3. Anàlisi de Paraules Clau:

    • Identifica les paraules clau que generen més trànsit al teu lloc web i optimitza el contingut en conseqüència.

Resolució d'Errors d'Indexació

  1. Accedir a la Secció de Cobertura:

    • Navega a la secció "Cobertura" per veure les pàgines indexades i els errors d'indexació.

  2. Solucionar Errors:

    • Revisa els errors d'indexació i segueix les recomanacions de Google per solucionar-los.

Millores d'Usabilitat Mòbil

  1. Accedir a la Secció d'Usabilitat Mòbil:

    • Navega a la secció "Usabilitat Mòbil" per veure els problemes d'usabilitat en dispositius mòbils.

  2. Solucionar Problemes:

    • Revisa els problemes d'usabilitat i segueix les recomanacions de Google per solucionar-los.

Exercici Pràctic

Exercici 1: Verificació del Lloc Web

  1. Verifica el teu lloc web utilitzant una de les opcions proporcionades per Google Search Console.
  2. Envia un sitemap XML del teu lloc web.

Exercici 2: Anàlisi de Rendiment

  1. Accedeix a la secció de rendiment de Google Search Console.
  2. Filtra les dades per les últimes 3 mesos.
  3. Identifica les 5 paraules clau que generen més trànsit al teu lloc web.

Solucions

Exercici 1: Verificació del Lloc Web

  1. Segueix els passos de verificació descrits anteriorment.
  2. Crea un fitxer sitemap XML i envia'l a través de la secció "Sitemaps" de Google Search Console.

Exercici 2: Anàlisi de Rendiment

  1. Navega a la secció "Rendiment".
  2. Utilitza el filtre de dates per seleccionar els últims 3 mesos.
  3. A la taula de paraules clau, ordena per clics per identificar les 5 paraules clau principals.

Conclusió

Google Search Console és una eina poderosa que proporciona informació crucial per optimitzar el teu lloc web per als motors de cerca. Des de la monitorització del rendiment fins a la resolució d'errors d'indexació, GSC és indispensable per a qualsevol estratègia de SEO. Assegura't d'utilitzar totes les funcionalitats disponibles per maximitzar la visibilitat i el rendiment del teu lloc web.
