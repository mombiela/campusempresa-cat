Dades estructurades i marcatge de l'esquema
Següent ►

Introducció

Les dades estructurades i el marcatge de l'esquema són elements clau del SEO tècnic que ajuden els motors de cerca a comprendre millor el contingut del teu lloc web. Aquest mòdul t'ensenyarà què són les dades estructurades, com implementar-les i quins beneficis poden aportar al teu SEO.

Què són les dades estructurades?

Les dades estructurades són un format estandarditzat per proporcionar informació sobre una pàgina i classificar el seu contingut. Aquestes dades ajuden els motors de cerca a comprendre millor el contingut de la pàgina i a mostrar-lo de manera més rica i informativa en els resultats de cerca.

Tipus de dades estructurades

  • JSON-LD (JavaScript Object Notation for Linked Data): És el format recomanat per Google. És fàcil d'implementar i mantenir.
  • Microdata: És un format que s'insereix directament en el codi HTML.
  • RDFa (Resource Description Framework in Attributes): És un format que utilitza atributs HTML per incrustar dades estructurades.

Què és el marcatge de l'esquema?

El marcatge de l'esquema és un tipus de dades estructurades que utilitza un vocabulari específic per definir diferents tipus de dades. Schema.org és el vocabulari més utilitzat i és suportat per Google, Bing, Yahoo i Yandex.

Exemples de marcatge de l'esquema

  • Articles: Per a notícies, blogs i articles.
  • Productes: Per a pàgines de productes en llocs de comerç electrònic.
  • Receptes: Per a llocs de cuina i receptes.
  • Esdeveniments: Per a pàgines d'esdeveniments.
  • Opinions: Per a ressenyes i valoracions de productes o serveis.

Beneficis del marcatge de l'esquema

  • Millora de la visibilitat: Els resultats enriquits (rich snippets) poden augmentar la visibilitat del teu lloc web en els resultats de cerca.
  • Millora del CTR (Click-Through Rate): Els resultats enriquits poden fer que els usuaris siguin més propensos a fer clic en el teu enllaç.
  • Millor comprensió del contingut: Els motors de cerca poden comprendre millor el contingut del teu lloc web, la qual cosa pot millorar el teu rànquing.

Com implementar dades estructurades

Pas 1: Selecciona el tipus de dades estructurades

Decideix quin tipus de dades estructurades és més adequat per al contingut de la teva pàgina. Per exemple, si tens una pàgina de producte, utilitza el tipus de dades estructurades "Product".

Pas 2: Genera el codi de dades estructurades

Pots utilitzar eines com el Google Structured Data Markup Helper per generar el codi de dades estructurades.

Pas 3: Afegeix el codi al teu lloc web

Afegeix el codi de dades estructurades al codi HTML de la teva pàgina. Aquí tens un exemple de codi JSON-LD per a una pàgina de producte:

<script type="application/ld+json">
{
  "@context": "https://schema.org/",
  "@type": "Product",
  "name": "Nom del Producte",
  "image": [
    "https://example.com/photos/1x1/photo.jpg",
    "https://example.com/photos/4x3/photo.jpg",
    "https://example.com/photos/16x9/photo.jpg"
   ],
  "description": "Descripció del producte",
  "sku": "0446310786",
  "mpn": "925872",
  "brand": {
    "@type": "Brand",
    "name": "Marca del Producte"
  },
  "offers": {
    "@type": "Offer",
    "url": "https://example.com/product",
    "priceCurrency": "USD",
    "price": "29.99",
    "priceValidUntil": "2023-11-05",
    "itemCondition": "https://schema.org/NewCondition",
    "availability": "https://schema.org/InStock",
    "seller": {
      "@type": "Organization",
      "name": "Nom del Venedor"
    }
  }
}
</script>

Pas 4: Valida el codi de dades estructurades

Utilitza l'eina de Prova de dades estructurades de Google per assegurar-te que el teu codi és correcte i no conté errors.

Exercici pràctic

Tasca

Implementa dades estructurades per a una pàgina d'article de blog utilitzant JSON-LD. Utilitza el següent contingut per a l'article:

  • Títol: "Com optimitzar el teu lloc web per a SEO"
  • Autor: "Joan Garcia"
  • Data de publicació: "2023-10-01"
  • Descripció: "Una guia completa per optimitzar el teu lloc web per a SEO i millorar el teu rànquing en els motors de cerca."
  • Imatge: "https://example.com/images/seo-guide.jpg"

Solució

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Com optimitzar el teu lloc web per a SEO",
  "author": {
    "@type": "Person",
    "name": "Joan Garcia"
  },
  "datePublished": "2023-10-01",
  "image": "https://example.com/images/seo-guide.jpg",
  "description": "Una guia completa per optimitzar el teu lloc web per a SEO i millorar el teu rànquing en els motors de cerca."
}
</script>

Resum

En aquest mòdul, has après què són les dades estructurades i el marcatge de l'esquema, així com els beneficis que poden aportar al teu SEO. També has après a implementar dades estructurades utilitzant JSON-LD i a validar el teu codi per assegurar-te que és correcte. Amb aquestes habilitats, podràs millorar la visibilitat i el rendiment del teu lloc web en els resultats de cerca.
