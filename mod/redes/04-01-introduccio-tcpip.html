Introducció al Model TCP/IP
Següent ►

El model TCP/IP (Transmission Control Protocol/Internet Protocol) és un conjunt de protocols de comunicació utilitzats per interconnectar dispositius en una xarxa, especialment a Internet. Aquest model és fonamental per comprendre com es transmeten les dades a través de les xarxes i com es garanteix la seva integritat i seguretat.

Objectius d'aquest tema

  • Entendre la història i l'evolució del model TCP/IP.
  • Conèixer les capes del model TCP/IP i les seves funcions.
  • Comparar el model TCP/IP amb el model OSI.
  • Aprendre els protocols principals de cada capa del model TCP/IP.

Història i Evolució del Model TCP/IP

El model TCP/IP va ser desenvolupat per l'Agència de Projectes de Recerca Avançada de Defensa (DARPA) dels Estats Units durant els anys 1970. El seu objectiu era crear una xarxa robusta i flexible que pogués resistir fallades parcials i permetre la comunicació entre diferents tipus de sistemes informàtics.

Fites Clau:

  • 1973: Vinton Cerf i Robert Kahn proposen el protocol TCP.
  • 1983: El protocol TCP/IP es converteix en l'estàndard per a ARPANET, la precursora d'Internet.
  • 1989: Tim Berners-Lee desenvolupa el World Wide Web, utilitzant TCP/IP com a base per a la transmissió de dades.

Capes del Model TCP/IP

El model TCP/IP està format per quatre capes principals, cadascuna amb funcions específiques per a la transmissió de dades. Aquestes capes són:

  1. Capa d'Accés a la Xarxa (Network Access Layer)
  2. Capa d'Internet (Internet Layer)
  3. Capa de Transport (Transport Layer)
  4. Capa d'Aplicació (Application Layer)

  1. Capa d'Accés a la Xarxa

Funcions:

  • Defineix com es transmeten les dades a través del mitjà físic (cablejat, Wi-Fi, etc.).
  • Inclou protocols com Ethernet i Wi-Fi.

Protocols Principals:

  • Ethernet: Utilitzat en xarxes locals (LAN).
  • Wi-Fi: Utilitzat en xarxes sense fils.

  1. Capa d'Internet

Funcions:

  • Gestiona l'adreçament i el encaminament de paquets de dades.
  • Assegura que els paquets arribin a la seva destinació correcta.

Protocols Principals:

  • IP (Internet Protocol): Responsable de l'adreçament i el encaminament.
  • ICMP (Internet Control Message Protocol): Utilitzat per enviar missatges d'error i informació operativa.

  1. Capa de Transport

Funcions:

  • Proporciona una comunicació fiable entre dispositius.
  • Gestiona el control de flux i la correcció d'errors.

Protocols Principals:

  • TCP (Transmission Control Protocol): Garanteix una transmissió fiable i ordenada de dades.
  • UDP (User Datagram Protocol): Proporciona una transmissió ràpida però no fiable.

  1. Capa d'Aplicació

Funcions:

  • Proporciona serveis de xarxa a les aplicacions.
  • Inclou protocols que suporten aplicacions com el correu electrònic, la navegació web, etc.

Protocols Principals:

  • HTTP (Hypertext Transfer Protocol): Utilitzat per a la navegació web.
  • SMTP (Simple Mail Transfer Protocol): Utilitzat per a l'enviament de correu electrònic.
  • FTP (File Transfer Protocol): Utilitzat per a la transferència de fitxers.

Comparació entre el Model TCP/IP i el Model OSI

Tot i que el model TCP/IP i el model OSI tenen objectius similars, hi ha diferències clau entre ells.

Aspecte Model TCP/IP Model OSI
Capacitat 4 capes 7 capes
Desenvolupament Basat en protocols existents Teòric, desenvolupat per ISO
Flexibilitat Més flexible i pràctic Més detallat i estructurat
Adopció Utilitzat àmpliament a Internet Utilitzat principalment com a model de referència

Resum

El model TCP/IP és essencial per a la comprensió de les xarxes modernes, especialment Internet. Consta de quatre capes que gestionen diferents aspectes de la transmissió de dades, des de l'accés al mitjà físic fins a la interacció amb aplicacions. Comparat amb el model OSI, el model TCP/IP és més pràctic i àmpliament adoptat, tot i que menys detallat.

En el següent tema, explorarem en profunditat la Capa d'Accés a la Xarxa del model TCP/IP, on veurem com es gestionen les connexions físiques i les tecnologies associades.
