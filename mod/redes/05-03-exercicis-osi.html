Exercicis del Model OSI
Següent ►

En aquesta secció, treballarem amb exercicis pràctics per reforçar els conceptes apresos sobre el Model OSI. Cada exercici inclou una descripció del problema, els passos per resoldre'l i la solució detallada. També proporcionarem retroalimentació sobre errors comuns i consells per ajudar-vos a superar obstacles.

Exercici 1: Identificació de Capes OSI

Descripció

Identifiqueu a quina capa del Model OSI pertanyen les següents funcions o protocols.

Preguntes

  1. HTTP
  2. IP
  3. Ethernet
  4. TLS/SSL
  5. TCP
  6. DNS
  7. MAC Address
  8. Routing
  9. Data Encoding
  10. Session Establishment

Solucions

  1. HTTP - Capa d'Aplicació (Capa 7)
  2. IP - Capa de Xarxa (Capa 3)
  3. Ethernet - Capa d'Enllaç de Dades (Capa 2)
  4. TLS/SSL - Capa de Presentació (Capa 6)
  5. TCP - Capa de Transport (Capa 4)
  6. DNS - Capa d'Aplicació (Capa 7)
  7. MAC Address - Capa d'Enllaç de Dades (Capa 2)
  8. Routing - Capa de Xarxa (Capa 3)
  9. Data Encoding - Capa de Presentació (Capa 6)
  10. Session Establishment - Capa de Sessió (Capa 5)

Retroalimentació

  • Error comú: Confondre protocols de la capa d'aplicació amb la capa de presentació. Recordeu que la capa d'aplicació és on es troben els protocols que interactuen directament amb les aplicacions de l'usuari.
  • Consell: Memoritzar les funcions clau de cada capa pot ajudar a identificar correctament a quina capa pertany un protocol o funció.

Exercici 2: Anàlisi de Trames

Descripció

Analitzeu la següent trama de xarxa i determineu a quina capa del Model OSI pertany cada part de la trama.

Trama

Ethernet Header | IP Header | TCP Header | HTTP Data

Solució

  1. Ethernet Header - Capa d'Enllaç de Dades (Capa 2)
  2. IP Header - Capa de Xarxa (Capa 3)
  3. TCP Header - Capa de Transport (Capa 4)
  4. HTTP Data - Capa d'Aplicació (Capa 7)

Retroalimentació

  • Error comú: No identificar correctament els encapçalaments de cada capa. Recordeu que cada capa afegeix el seu propi encapçalament a la dada que es transmet.
  • Consell: Practicar amb captures de trames reals utilitzant eines com Wireshark pot ajudar a familiaritzar-se amb l'estructura de les trames i els encapçalaments de cada capa.

Exercici 3: Resolució de Problemes de Xarxa

Descripció

Un usuari no pot accedir a una pàgina web. Determineu a quina capa del Model OSI podria estar el problema basant-vos en les següents descripcions.

Preguntes

  1. El cable de xarxa està desconnectat.
  2. L'adreça IP del servidor web no es pot resoldre.
  3. El servidor web està caigut.
  4. Hi ha una configuració incorrecta del protocol TCP.
  5. El navegador web no pot establir una connexió segura.

Solucions

  1. El cable de xarxa està desconnectat. - Capa Física (Capa 1)
  2. L'adreça IP del servidor web no es pot resoldre. - Capa d'Aplicació (Capa 7)
  3. El servidor web està caigut. - Capa d'Aplicació (Capa 7)
  4. Hi ha una configuració incorrecta del protocol TCP. - Capa de Transport (Capa 4)
  5. El navegador web no pot establir una connexió segura. - Capa de Presentació (Capa 6)

Retroalimentació

  • Error comú: Confondre problemes de la capa física amb problemes de la capa d'enllaç de dades. Recordeu que la capa física tracta amb el maquinari i les connexions físiques, mentre que la capa d'enllaç de dades tracta amb la transmissió de dades entre nodes adjacents.
  • Consell: Quan es resolen problemes de xarxa, és útil seguir un enfocament sistemàtic, començant per la capa física i avançant cap a les capes superiors.

Exercici 4: Mapeig de Capes OSI a Capes TCP/IP

Descripció

Mapegeu les capes del Model OSI a les capes del Model TCP/IP.

Preguntes

  1. Capa Física (OSI)
  2. Capa d'Enllaç de Dades (OSI)
  3. Capa de Xarxa (OSI)
  4. Capa de Transport (OSI)
  5. Capa de Sessió (OSI)
  6. Capa de Presentació (OSI)
  7. Capa d'Aplicació (OSI)

Solucions

  1. Capa Física (OSI) - Capa d'Accés a la Xarxa (TCP/IP)
  2. Capa d'Enllaç de Dades (OSI) - Capa d'Accés a la Xarxa (TCP/IP)
  3. Capa de Xarxa (OSI) - Capa d'Internet (TCP/IP)
  4. Capa de Transport (OSI) - Capa de Transport (TCP/IP)
  5. Capa de Sessió (OSI) - Capa d'Aplicació (TCP/IP)
  6. Capa de Presentació (OSI) - Capa d'Aplicació (TCP/IP)
  7. Capa d'Aplicació (OSI) - Capa d'Aplicació (TCP/IP)

Retroalimentació

  • Error comú: No tenir en compte que el Model TCP/IP combina algunes de les capes del Model OSI. La capa d'aplicació del Model TCP/IP inclou les funcions de les capes de sessió, presentació i aplicació del Model OSI.
  • Consell: Memoritzar les correspondències entre les capes dels dos models pot ajudar a comprendre millor com funcionen les xarxes en diferents nivells.

Conclusió

En aquesta secció, hem treballat amb diversos exercicis per reforçar els conceptes del Model OSI. Hem identificat capes, analitzat trames, resolt problemes de xarxa i mapejat capes entre els models OSI i TCP/IP. Aquests exercicis us ajudaran a comprendre millor com funcionen les xarxes i a aplicar aquest coneixement en situacions reals. En el següent tema, ens centrarem en exercicis relacionats amb el Model TCP/IP.
