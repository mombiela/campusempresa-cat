Exercicis de Conceptes Bàsics
Següent ►

En aquesta secció, treballarem amb exercicis pràctics per reforçar els conceptes bàsics de xarxes que hem après en el Mòdul 1. Aquests exercicis estan dissenyats per ajudar-te a comprendre millor els fonaments de les xarxes, els tipus de xarxes i les topologies de xarxes.

Exercici 1: Identificació de Tipus de Xarxes

Enunciat:

Classifica les següents xarxes segons el seu tipus (LAN, WAN, MAN, PAN):

  1. Una xarxa que connecta diversos ordinadors dins d'una oficina.
  2. Una xarxa que connecta diverses oficines d'una empresa situades en diferents ciutats.
  3. Una xarxa que connecta dispositius personals com telèfons mòbils, tauletes i ordinadors portàtils dins d'una habitació.
  4. Una xarxa que cobreix una àrea metropolitana com una ciutat.

Solució:

  1. LAN (Local Area Network)
  2. WAN (Wide Area Network)
  3. PAN (Personal Area Network)
  4. MAN (Metropolitan Area Network)

Exercici 2: Identificació de Topologies de Xarxes

Enunciat:

Relaciona les següents descripcions amb la topologia de xarxa corresponent (Bus, Estrella, Anell, Malla):

  1. Tots els dispositius estan connectats a un únic cable central.
  2. Tots els dispositius estan connectats a un dispositiu central com un switch o un hub.
  3. Cada dispositiu està connectat exactament a dos altres dispositius, formant un cercle.
  4. Cada dispositiu està connectat a tots els altres dispositius de la xarxa.

Solució:

  1. Bus
  2. Estrella
  3. Anell
  4. Malla

Exercici 3: Avantatges i Desavantatges de les Topologies

Enunciat:

Llista un avantatge i un desavantatge per a cada una de les següents topologies de xarxa: Bus, Estrella, Anell, Malla.

Solució:

  • Bus:

    • Avantatge: Fàcil d'implementar i econòmica.
    • Desavantatge: Si el cable central falla, tota la xarxa cau.

  • Estrella:

    • Avantatge: Fàcil de gestionar i afegir nous dispositius.
    • Desavantatge: Si el dispositiu central falla, tota la xarxa cau.

  • Anell:

    • Avantatge: El rendiment és consistent, ja que cada dispositiu té el mateix accés al cable.
    • Desavantatge: Si un dispositiu falla, pot afectar tota la xarxa.

  • Malla:

    • Avantatge: Alta redundància i fiabilitat.
    • Desavantatge: Costosa i complexa d'implementar.

Exercici 4: Dibuix de Topologies

Enunciat:

Dibuixa les següents topologies de xarxa:

  1. Topologia Bus
  2. Topologia Estrella
  3. Topologia Anell
  4. Topologia Malla

Solució:

  • Topologia Bus:

    [PC1]---[PC2]---[PC3]---[PC4]

  • Topologia Estrella:

          [PC1]
        |
[PC2]--[Switch]--[PC3]
        |
      [PC4]

  • Topologia Anell:

    [PC1]---[PC2]---[PC3]---[PC4]---[PC1]

  • Topologia Malla:

    [PC1]---[PC2]
 | \     |  
       |  \    |   
      [PC3]---[PC4]---[PC5]

Exercici 5: Cas Pràctic de Disseny de Xarxa

Enunciat:

Una petita empresa té una oficina amb 10 ordinadors, una impressora de xarxa i un servidor. Dissenya una xarxa adequada per a aquesta empresa, tenint en compte la facilitat de gestió i la fiabilitat.

Solució:

Per a una petita empresa amb aquests requisits, una topologia d'estrella seria adequada. Això permet una fàcil gestió i afegir nous dispositius de manera senzilla. El disseny seria el següent:

        [PC1] [PC2] [PC3] [PC4] [PC5]
          \    |    |    |    /
           \   |    |    |   /
            [Switch]
           /   |    |    |   \
          /    |    |    |    \
        [PC6] [PC7] [PC8] [PC9] [PC10]
                  |
               [Servidor]
                  |
              [Impressora]

Resum

En aquesta secció, hem treballat amb exercicis pràctics per reforçar els conceptes bàsics de xarxes, incloent-hi la identificació de tipus de xarxes, topologies de xarxes, avantatges i desavantatges de les topologies, i el disseny d'una xarxa per a una petita empresa. Aquests exercicis t'ajudaran a consolidar els teus coneixements i a preparar-te per a temes més avançats en els següents mòduls.
