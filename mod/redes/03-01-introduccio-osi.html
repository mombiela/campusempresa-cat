Introducció al Model OSI
Següent ►

El Model OSI (Open Systems Interconnection) és un model conceptual que descriu les funcions d'un sistema de comunicació o xarxa en termes de capes. Va ser desenvolupat per l'Organització Internacional per a la Normalització (ISO) i és àmpliament utilitzat per comprendre i dissenyar xarxes de comunicació. El model OSI divideix les funcions de xarxa en set capes, cadascuna amb responsabilitats específiques.

Objectius del Model OSI

  • Estandardització: Proporcionar un marc estàndard per a la interconnexió de sistemes de comunicació.
  • Interoperabilitat: Permetre que diferents sistemes i productes de diferents fabricants treballin junts.
  • Modularitat: Facilitar la comprensió i el desenvolupament de protocols de xarxa mitjançant la divisió de funcions en capes.

Les Set Capa del Model OSI

El Model OSI està compost per set capes, cadascuna de les quals té una funció específica:

  1. Capa Física (Physical Layer)
  2. Capa d'Enllaç de Dades (Data Link Layer)
  3. Capa de Xarxa (Network Layer)
  4. Capa de Transport (Transport Layer)
  5. Capa de Sessió (Session Layer)
  6. Capa de Presentació (Presentation Layer)
  7. Capa d'Aplicació (Application Layer)

Descripció de les Capa del Model OSI

1. Capa Física

  • Funció: Transmissió de bits bruts a través d'un mitjà físic.
  • Exemples: Cables, connectors, senyals elèctrics.

2. Capa d'Enllaç de Dades

  • Funció: Proporcionar un enllaç fiable entre dos nodes directament connectats.
  • Exemples: Ethernet, Wi-Fi.

3. Capa de Xarxa

  • Funció: Determinar el camí que els paquets han de seguir per arribar a la seva destinació.
  • Exemples: IP (Internet Protocol).

4. Capa de Transport

  • Funció: Assegurar la transferència de dades de manera fiable i ordenada.
  • Exemples: TCP (Transmission Control Protocol), UDP (User Datagram Protocol).

5. Capa de Sessió

  • Funció: Gestionar i controlar les connexions entre aplicacions.
  • Exemples: RPC (Remote Procedure Call).

6. Capa de Presentació

  • Funció: Traduir, xifrar i comprimir les dades.
  • Exemples: SSL/TLS (Secure Sockets Layer/Transport Layer Security).

7. Capa d'Aplicació

  • Funció: Proporcionar serveis de xarxa a les aplicacions de l'usuari.
  • Exemples: HTTP (HyperText Transfer Protocol), FTP (File Transfer Protocol).

Comparació entre el Model OSI i el Model TCP/IP

Tot i que el Model OSI és un model teòric, el Model TCP/IP és un model pràctic utilitzat en la majoria de les xarxes modernes. A continuació es mostra una taula comparativa entre els dos models:

Exercici Pràctic

Exercici 1: Identificació de Capa Determina a quina capa del Model OSI pertanyen les següents tecnologies o protocols:

  1. Ethernet
  2. IP
  3. HTTP
  4. TCP
  5. SSL/TLS

Solució:

  1. Ethernet - Capa d'Enllaç de Dades
  2. IP - Capa de Xarxa
  3. HTTP - Capa d'Aplicació
  4. TCP - Capa de Transport
  5. SSL/TLS - Capa de Presentació

Resum

El Model OSI és una eina fonamental per comprendre com funcionen les xarxes de comunicació. Divideix les funcions de xarxa en set capes, cadascuna amb responsabilitats específiques, facilitant la interoperabilitat i la modularitat en el disseny de sistemes de comunicació. En el proper tema, explorarem en detall la primera capa del Model OSI: la Capa Física.
