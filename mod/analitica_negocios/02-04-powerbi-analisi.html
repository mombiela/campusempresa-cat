Power BI: Anàlisi i Visualització
Següent ►

Introducció a Power BI

Power BI és una eina de Microsoft per a la visualització de dades i l'anàlisi interactiva. Permet als usuaris connectar-se a diverses fonts de dades, transformar-les i crear informes i dashboards interactius que poden compartir-se fàcilment amb altres membres de l'organització.

Característiques Clau de Power BI

  • Integració de Dades: Connecta amb una àmplia varietat de fonts de dades, incloent bases de dades, serveis en el núvol, fitxers Excel, i més.
  • Transformació de Dades: Utilitza Power Query per netejar, transformar i combinar dades.
  • Visualitzacions Interactives: Crea gràfics, mapes, taules i altres visualitzacions interactives.
  • Dashboards Compartibles: Publica i comparteix informes i dashboards amb altres usuaris.
  • Actualització Automàtica: Configura actualitzacions automàtiques per mantenir les dades actualitzades.

Connexió a Fonts de Dades

Passos per Connectar-se a una Font de Dades

  1. Obrir Power BI Desktop: Inicia l'aplicació Power BI Desktop.
  2. Seleccionar la Font de Dades: Fes clic a "Obtenir dades" i selecciona la font de dades desitjada (per exemple, Excel, SQL Server, etc.).
  3. Importar Dades: Segueix les instruccions per connectar-te a la font de dades i importa les dades a Power BI.
**Exemple: Connexió a un Fitxer Excel**

1. Fes clic a "Obtenir dades" i selecciona "Excel".
2. Navega fins al fitxer Excel i fes clic a "Obrir".
3. Selecciona les taules o fulls de càlcul que vols importar i fes clic a "Carregar".

Transformació de Dades amb Power Query

Neteja i Preparació de Dades

Power Query és una eina potent dins de Power BI que permet netejar i transformar dades abans d'utilitzar-les en visualitzacions.

Operacions Comunes amb Power Query

  • Eliminar Columnes: Elimina columnes innecessàries.
  • Filtrar Files: Filtra files basades en criteris específics.
  • Agrupar Dades: Agrupa dades per categories.
  • Crear Columnes Calculades: Afegeix columnes noves basades en càlculs.
**Exemple: Eliminar Columnes Innecessàries**

1. A la finestra de Power Query, selecciona les columnes que vols eliminar.
2. Fes clic amb el botó dret i selecciona "Eliminar columnes".

Creació de Visualitzacions

Tipus de Visualitzacions

Power BI ofereix una àmplia varietat de visualitzacions per representar dades de manera efectiva.

Visualitzacions Comunes

  • Gràfics de Barres: Per comparar valors entre categories.
  • Gràfics de Línies: Per mostrar tendències al llarg del temps.
  • Mapes: Per visualitzar dades geogràfiques.
  • Taules: Per mostrar dades detallades en format tabular.
**Exemple: Crear un Gràfic de Barres**

1. Arrossega un camp de categoria a l'àrea de "Eix" i un camp de valor a l'àrea de "Valors".
2. Selecciona el tipus de visualització "Gràfic de Barres" des del panell de visualitzacions.

Creació de Dashboards

Passos per Crear un Dashboard

  1. Crear Visualitzacions: Crea diverses visualitzacions a Power BI Desktop.
  2. Publicar l'Informe: Publica l'informe a Power BI Service.
  3. Crear el Dashboard: A Power BI Service, selecciona les visualitzacions publicades i afegeix-les al dashboard.
**Exemple: Publicar un Informe**

1. Fes clic a "Publicar" a Power BI Desktop.
2. Selecciona l'espai de treball a Power BI Service on vols publicar l'informe.
3. Un cop publicat, ves a Power BI Service i crea un nou dashboard.
4. Afegeix les visualitzacions de l'informe publicat al dashboard.

Exercici Pràctic: Creació d'un Dashboard en Power BI

Objectiu

Crear un dashboard interactiu que mostri les vendes per regió, producte i període de temps.

Passos

  1. Importar Dades: Importa un fitxer Excel amb dades de vendes.
  2. Transformar Dades: Utilitza Power Query per netejar i preparar les dades.
  3. Crear Visualitzacions: Crea gràfics de barres, línies i mapes per visualitzar les vendes.
  4. Crear el Dashboard: Publica l'informe i crea un dashboard a Power BI Service.

Solució

**Pas 1: Importar Dades**

1. Obre Power BI Desktop.
2. Fes clic a "Obtenir dades" i selecciona "Excel".
3. Navega fins al fitxer Excel i fes clic a "Obrir".
4. Selecciona les taules o fulls de càlcul que vols importar i fes clic a "Carregar".

**Pas 2: Transformar Dades**

1. A la finestra de Power Query, elimina les columnes innecessàries.
2. Filtra les files per eliminar dades incompletes.
3. Agrupa les dades per regió i producte.

**Pas 3: Crear Visualitzacions**

1. Crea un gràfic de barres per mostrar les vendes per regió.
2. Crea un gràfic de línies per mostrar les vendes al llarg del temps.
3. Crea un mapa per visualitzar les vendes per ubicació geogràfica.

**Pas 4: Crear el Dashboard**

1. Fes clic a "Publicar" a Power BI Desktop.
2. Selecciona l'espai de treball a Power BI Service on vols publicar l'informe.
3. Un cop publicat, ves a Power BI Service i crea un nou dashboard.
4. Afegeix les visualitzacions de l'informe publicat al dashboard.

Conclusió

Power BI és una eina potent per a l'anàlisi i visualització de dades que permet als usuaris transformar dades crues en informació valuosa. Amb la seva capacitat per connectar-se a diverses fonts de dades, transformar-les i crear visualitzacions interactives, Power BI és una eina essencial per a qualsevol professional d'analítica de negocis. En el proper mòdul, explorarem Google Analytics i com pot ser utilitzat per analitzar el rendiment web.
