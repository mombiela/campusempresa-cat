Exercici 2: Anàlisi Predictiu amb Excel

Següent

Objectiu de l'Exercici

L'objectiu d'aquest exercici és aplicar tècniques d'anàlisi predictiu utilitzant Microsoft Excel. Aprendràs a utilitzar funcions i eines d'Excel per crear models predictius que ajudin a prendre decisions basades en dades.

Descripció de l'Exercici

En aquest exercici, treballaràs amb un conjunt de dades de vendes històriques d'una empresa fictícia. Utilitzaràs aquestes dades per crear un model predictiu que estimi les vendes futures.

Conjunt de Dades

El conjunt de dades inclou les següents columnes:

  • Data: La data de la venda.
  • Producte: El nom del producte venut.
  • Quantitat: La quantitat de productes venuts.
  • Preu: El preu per unitat del producte.
  • Ingressos: Els ingressos generats per la venda (Quantitat * Preu).

Passos a Seguir

  1. Importació de Dades a Excel

    • Obre un nou llibre d'Excel.
    • Importa el conjunt de dades proporcionat (per exemple, un fitxer CSV) a una fulla de càlcul.

  2. Neteja i Preparació de Dades

    • Assegura't que no hi hagi valors nuls o errors en les dades.
    • Crea una columna addicional per calcular els ingressos si no està inclosa en el conjunt de dades.

  3. Anàlisi Descriptiu

    • Utilitza taules dinàmiques per resumir les dades.
    • Crea gràfics per visualitzar les tendències de vendes al llarg del temps.

  4. Creació del Model Predictiu

    • Utilitza la funció de regressió lineal d'Excel per crear un model predictiu.
    • Selecciona les variables independents (per exemple, Data) i la variable dependent (Ingressos).

  5. Validació del Model

    • Divideix les dades en un conjunt d'entrenament i un conjunt de prova.
    • Avalua la precisió del model utilitzant el conjunt de prova.

  6. Predicció de Vendes Futures

    • Utilitza el model per predir les vendes futures.
    • Interpreta els resultats i presenta les conclusions.

Exemple Pràctic

Importació de Dades a Excel

  1. Obre Excel i selecciona Dades > Des de Text/CSV.
  2. Selecciona el fitxer CSV amb les dades de vendes i fes clic a Importa.
  3. Assegura't que les dades s'importin correctament i es mostrin en una taula.

Neteja i Preparació de Dades

  1. Revisa les dades per assegurar-te que no hi hagi valors nuls.
  2. Si cal, crea una nova columna per calcular els ingressos: 
    =C2 * D2
    On C2 és la quantitat i D2 és el preu.

Anàlisi Descriptiu

  1. Crea una taula dinàmica per resumir les vendes per mes:
    • Selecciona les dades i ves a Insereix > Taula Dinàmica.
    • Arrossega la columna Data a l'àrea de Files i la columna Ingressos a l'àrea de Valors.
  2. Crea un gràfic de línies per visualitzar les tendències de vendes:
    • Selecciona la taula dinàmica i ves a Insereix > Gràfic de Línies.

Creació del Model Predictiu

  1. Selecciona les dades i ves a Dades > Anàlisi de Dades > Regressió.
  2. Selecciona la columna Ingressos com a variable dependent i la columna Data com a variable independent.
  3. Fes clic a D'acord per generar el model de regressió.

Validació del Model

  1. Divideix les dades en un conjunt d'entrenament (per exemple, els primers 80% de les dades) i un conjunt de prova (els últims 20%).
  2. Avalua la precisió del model utilitzant el conjunt de prova i calcula l'error mitjà absolut (MAE).

Predicció de Vendes Futures

  1. Utilitza el model de regressió per predir les vendes futures.
  2. Interpreta els resultats i presenta les conclusions en un informe.

Conclusió

En aquest exercici, has après a utilitzar Excel per crear un model predictiu de vendes. Has importat i netejat les dades, realitzat un anàlisi descriptiu, creat un model de regressió, validat el model i fet prediccions de vendes futures. Aquestes habilitats són essencials per a l'analítica de negocis i la presa de decisions basada en dades.
