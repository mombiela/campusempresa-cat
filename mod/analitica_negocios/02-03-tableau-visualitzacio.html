Tableau: Visualització de Dades
Següent ►

Introducció a Tableau

Tableau és una eina de visualització de dades que permet als usuaris crear gràfics, taules i altres tipus de visualitzacions de manera intuïtiva i interactiva. És àmpliament utilitzada en l'analítica de negocis per la seva capacitat de transformar dades complexes en visualitzacions comprensibles i atractives.

Característiques Clau de Tableau

  • Interfície intuïtiva: Permet crear visualitzacions arrossegant i deixant anar elements.
  • Integració de dades: Compatible amb diverses fonts de dades com Excel, bases de dades SQL, serveis en el núvol, etc.
  • Visualitzacions interactives: Els usuaris poden interactuar amb les visualitzacions per explorar les dades.
  • Compartició i col·laboració: Facilita la compartició de dashboards i informes amb altres usuaris.

Instal·lació i Configuració de Tableau

Requisits del Sistema

  • Sistema Operatiu: Windows 7 o superior, macOS 10.12 o superior.
  • Memòria RAM: Recomanable 8 GB o més.
  • Espai en Disc: Almenys 1,5 GB d'espai lliure.

Passos per Instal·lar Tableau

  1. Descarregar Tableau: Visita el lloc web oficial de Tableau i descarrega la versió adequada per al teu sistema operatiu.
  2. Instal·lar Tableau: Executa l'instal·lador i segueix les instruccions en pantalla.
  3. Activar Tableau: Introdueix la clau de llicència o utilitza la versió de prova.

Importació de Dades a Tableau

Fonts de Dades Compatibles

  • Fitxers: Excel, CSV, JSON, etc.
  • Bases de Dades: MySQL, PostgreSQL, SQL Server, etc.
  • Serveis en el Núvol: Google Analytics, Salesforce, Amazon Redshift, etc.

Importació de Dades des d'un Fitxer Excel

  1. Obrir Tableau: Inicia Tableau i selecciona "Connectar a un fitxer".
  2. Seleccionar Fitxer: Tria el fitxer Excel que vols importar.
  3. Seleccionar Fulls: Selecciona els fulls de càlcul que vols utilitzar.
  4. Carregar Dades: Tableau carregarà les dades i les mostrarà en una vista prèvia.

Creació de Visualitzacions

Tipus de Visualitzacions

  • Gràfics de Barres: Comparació de valors entre diferents categories.
  • Gràfics de Línies: Seguiment de tendències al llarg del temps.
  • Gràfics de Pastís: Distribució percentual de categories.
  • Mapes: Visualització de dades geogràfiques.
  • Taules: Presentació de dades en format tabular.

Exemple Pràctic: Creació d'un Gràfic de Barres

  1. Importar Dades: Importa un fitxer Excel amb dades de vendes.
  2. Crear una Nova Fulla: Fes clic a "Fulla 1" per crear una nova visualització.
  3. Arrossegar Dimensions i Mesures: Arrossega la dimensió "Producte" a l'eix X i la mesura "Vendes" a l'eix Y.
  4. Seleccionar Tipus de Gràfic: Fes clic a "Mostrar-me" i selecciona "Gràfic de Barres".
  5. Personalitzar Visualització: Ajusta els colors, etiquetes i títols segons les teves necessitats.
# Exemple de codi per crear un gràfic de barres en Tableau

1. Importa les dades de vendes des d'un fitxer Excel.
2. Arrossega la dimensió "Producte" a l'eix X.
3. Arrossega la mesura "Vendes" a l'eix Y.
4. Selecciona "Gràfic de Barres" des del panell "Mostrar-me".
5. Personalitza els colors, etiquetes i títols.

Dashboards i Històries

Creació de Dashboards

  1. Crear un Nou Dashboard: Fes clic a "Nou Dashboard".
  2. Afegir Fulles: Arrossega les fulles de visualització al dashboard.
  3. Ajustar Disseny: Organitza les visualitzacions i ajusta les mides.
  4. Afegir Interactivitat: Afegeix filtres i accions per permetre la interacció dels usuaris.

Creació d'Històries

  1. Crear una Nova Història: Fes clic a "Nova Història".
  2. Afegir Fulles i Dashboards: Arrossega les fulles i dashboards a les diapositives de la història.
  3. Narrar la Història: Afegeix text i descripcions per guiar els usuaris a través de les visualitzacions.

Exercici Pràctic: Creació d'un Dashboard en Tableau

Objectiu

Crear un dashboard interactiu que mostri les vendes per producte i regió.

Passos

  1. Importar Dades: Importa un fitxer Excel amb dades de vendes per producte i regió.
  2. Crear Visualitzacions:
    • Gràfic de barres per vendes per producte.
    • Mapa per vendes per regió.
  3. Crear Dashboard:
    • Afegeix les visualitzacions creades al dashboard.
    • Afegeix filtres per permetre la selecció de productes i regions.
  4. Personalitzar: Ajusta el disseny i afegeix títols i descripcions.

Solució

1. Importa el fitxer Excel amb dades de vendes.
2. Crea un gràfic de barres per vendes per producte:
   - Arrossega "Producte" a l'eix X.
   - Arrossega "Vendes" a l'eix Y.
   - Selecciona "Gràfic de Barres".
3. Crea un mapa per vendes per regió:
   - Arrossega "Regió" a l'eix X.
   - Arrossega "Vendes" a l'eix Y.
   - Selecciona "Mapa".
4. Crea un nou dashboard i afegeix les visualitzacions creades.
5. Afegeix filtres per producte i regió.
6. Personalitza el disseny i afegeix títols i descripcions.

Conclusió

Tableau és una eina poderosa per a la visualització de dades que permet als usuaris transformar dades complexes en visualitzacions comprensibles i interactives. Amb les seves característiques intuïtives i la seva capacitat d'integrar-se amb diverses fonts de dades, Tableau és una eina essencial per a qualsevol professional de l'analítica de negocis. En aquesta secció, hem après a importar dades, crear visualitzacions, i construir dashboards i històries interactives. A continuació, aplicarem aquests coneixements en exercicis pràctics per reforçar el que hem après.
