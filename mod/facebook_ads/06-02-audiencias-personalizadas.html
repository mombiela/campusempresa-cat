Audiències personalitzades
Següent ►

Introducció

Les audiències personalitzades són una eina poderosa dins de Facebook Ads que permet als anunciants arribar a grups específics de persones amb missatges publicitaris altament rellevants. Aquestes audiències es creen a partir de dades pròpies, com ara llistes de clients, visitants del lloc web o usuaris de l'aplicació.

Conceptes Clau

  1. Dades pròpies: Informació que l'empresa ja té sobre els seus clients o usuaris.
  2. Llistes de clients: Llistes de correus electrònics, números de telèfon o altres identificadors que es poden carregar a Facebook.
  3. Píxel de Facebook: Un fragment de codi que es col·loca al lloc web per rastrejar les accions dels visitants.
  4. Interaccions a Facebook: Activitats com ara likes, comentaris o comparticions que es poden utilitzar per crear audiències.

Tipus d'Audiències Personalitzades

1. Llistes de Clients

Descripció: Permet carregar una llista de clients existents per crear una audiència personalitzada.

Exemple:

Llista de correus electrònics dels clients que han comprat en els últims 6 mesos.

Passos per crear-la:

  1. Navega a l'Administrador d'Anuncis.
  2. Selecciona "Audiències" al menú.
  3. Fes clic a "Crea una audiència" i selecciona "Audiència personalitzada".
  4. Tria "Llista de clients".
  5. Carrega la llista de clients seguint les instruccions.

2. Visitants del Lloc Web

Descripció: Utilitza el Píxel de Facebook per crear audiències basades en les visites al lloc web.

Exemple:

Persones que han visitat la pàgina de productes però no han realitzat una compra.

Passos per crear-la:

  1. Assegura't que el Píxel de Facebook està instal·lat al teu lloc web.
  2. Navega a l'Administrador d'Anuncis.
  3. Selecciona "Audiències" al menú.
  4. Fes clic a "Crea una audiència" i selecciona "Audiència personalitzada".
  5. Tria "Trànsit del lloc web".
  6. Defineix les regles per a la teva audiència (per exemple, URL específica visitada).

3. Activitat a l'Aplicació

Descripció: Crea audiències basades en les accions que els usuaris realitzen dins de la teva aplicació mòbil.

Exemple:

Usuaris que han descarregat l'aplicació però no han completat el registre.

Passos per crear-la:

  1. Assegura't que el SDK de Facebook està integrat a la teva aplicació.
  2. Navega a l'Administrador d'Anuncis.
  3. Selecciona "Audiències" al menú.
  4. Fes clic a "Crea una audiència" i selecciona "Audiència personalitzada".
  5. Tria "Activitat a l'aplicació".
  6. Defineix les accions específiques que vols utilitzar per crear l'audiència.

4. Interaccions a Facebook

Descripció: Crea audiències basades en les interaccions dels usuaris amb la teva pàgina de Facebook, publicacions, vídeos, etc.

Exemple:

Persones que han interactuat amb els teus vídeos en els últims 30 dies.

Passos per crear-la:

  1. Navega a l'Administrador d'Anuncis.
  2. Selecciona "Audiències" al menú.
  3. Fes clic a "Crea una audiència" i selecciona "Audiència personalitzada".
  4. Tria "Interacció".
  5. Defineix el tipus d'interacció (per exemple, vídeos, pàgina de Facebook).

Exercici Pràctic

Objectiu: Crear una audiència personalitzada basada en els visitants del lloc web.

Passos:

  1. Accedeix a l'Administrador d'Anuncis.
  2. Selecciona "Audiències" al menú.
  3. Fes clic a "Crea una audiència" i selecciona "Audiència personalitzada".
  4. Tria "Trànsit del lloc web".
  5. Defineix les regles per a la teva audiència (per exemple, URL específica visitada).
  6. Guarda l'audiència amb un nom descriptiu.

Solució:

1. Accedeix a l'Administrador d'Anuncis.
2. Selecciona "Audiències" al menú.
3. Fes clic a "Crea una audiència" i selecciona "Audiència personalitzada".
4. Tria "Trànsit del lloc web".
5. Defineix les regles per a la teva audiència, per exemple, "Persones que han visitat la pàgina de productes".
6. Guarda l'audiència amb el nom "Visitants pàgina de productes".

Errors Comuns i Consells

  1. Error: No tenir el Píxel de Facebook correctament instal·lat.

    • Solució: Revisa la instal·lació del Píxel i assegura't que està funcionant correctament.

  2. Error: Carregar llistes de clients amb dades incompletes o incorrectes.

    • Solució: Verifica que les dades carregades són correctes i completen els requisits de Facebook.

  3. Error: No definir clarament les regles per a les audiències basades en el trànsit del lloc web.

    • Solució: Assegura't de definir regles específiques i rellevants per a la teva audiència.

Conclusió

Les audiències personalitzades són una eina essencial per a qualsevol estratègia de Facebook Ads, ja que permeten arribar a grups específics de persones amb missatges altament rellevants. Aprendre a crear i utilitzar aquestes audiències pot augmentar significativament l'eficàcia de les teves campanyes publicitàries.
