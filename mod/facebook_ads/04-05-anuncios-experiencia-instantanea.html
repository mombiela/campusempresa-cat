Anuncis d'experiència instantània

Següent ►
►

Els anuncis d'experiència instantània (anteriorment coneguts com a Canvas) són un tipus d'anunci interactiu i immersiu que permet als usuaris explorar contingut multimèdia dins de Facebook sense haver de sortir de l'aplicació. Aquests anuncis combinen imatges, vídeos, text i botons de crida a l'acció (CTA) per crear una experiència rica i atractiva.

Característiques dels Anuncis d'Experiència Instantània

  1. Interactivitat: Els usuaris poden interactuar amb diferents elements dins de l'anunci, com ara desplaçar-se, fer clic, ampliar imatges, etc.
  2. Velocitat de càrrega: Aquests anuncis es carreguen ràpidament, proporcionant una experiència fluida i sense interrupcions.
  3. Contingut multimèdia: Permeten la combinació de diversos formats de contingut, com ara imatges, vídeos, carrusels, i més.
  4. Personalització: Es poden personalitzar per adaptar-se a diferents objectius de màrqueting, com ara la generació de leads, la promoció de productes, o la narració de la història de la marca.

Avantatges dels Anuncis d'Experiència Instantània

  • Experiència immersiva: Capturen l'atenció dels usuaris amb una experiència de pantalla completa.
  • Major compromís: La interactivitat i la riquesa del contingut poden augmentar el compromís dels usuaris.
  • Mesurament detallat: Facebook proporciona mètriques detallades sobre com els usuaris interactuen amb els elements de l'anunci.

Creació d'un Anunci d'Experiència Instantània

Pas 1: Accedir a l'Administrador d'Anuncis

  1. Inicia sessió al teu compte de Facebook Business.
  2. Accedeix a l'Administrador d'Anuncis.

Pas 2: Crear una Nova Campanya

  1. Fes clic a "Crea" per iniciar una nova campanya.
  2. Selecciona l'objectiu de la campanya (per exemple, "Trafic", "Conversió", etc.).

Pas 3: Configurar l'Ad Set

  1. Defineix l'audiència, el pressupost i el calendari.
  2. Selecciona les ubicacions on vols que apareguin els teus anuncis.

Pas 4: Crear l'Anunci

  1. Selecciona el format "Experiència Instantània".
  2. Afegeix els elements multimèdia (imatges, vídeos, etc.).
  3. Personalitza el text i els botons de crida a l'acció (CTA).

Exemple de Configuració d'un Anunci d'Experiència Instantània

{
  "name": "Campanya de Promoció de Productes",
  "objective": "TRAFFIC",
  "status": "ACTIVE",
  "adsets": [
    {
      "name": "Ad Set 1",
      "daily_budget": 1000,
      "start_time": "2023-10-01T00:00:00-0700",
      "end_time": "2023-10-31T23:59:59-0700",
      "targeting": {
        "geo_locations": {
          "countries": ["ES"]
        },
        "age_min": 18,
        "age_max": 65,
        "genders": [1, 2]
      },
      "creative": {
        "title": "Descobreix els nostres nous productes!",
        "body": "Explora la nostra nova col·lecció amb una experiència immersiva.",
        "image_url": "https://example.com/image.jpg",
        "video_url": "https://example.com/video.mp4",
        "cta": "Més informació"
      }
    }
  ]
}

Pas 5: Publicar l'Anunci

  1. Revisa la configuració de l'anunci.
  2. Fes clic a "Publica" per llançar la campanya.

Exercici Pràctic

Exercici: Crear un Anunci d'Experiència Instantània

Objectiu: Crear un anunci d'experiència instantània per promocionar un nou producte.

Passos:

  1. Accedeix a l'Administrador d'Anuncis de Facebook.
  2. Crea una nova campanya amb l'objectiu "Trafic".
  3. Configura l'ad set amb una audiència específica, pressupost i calendari.
  4. Crea un anunci d'experiència instantània amb imatges, vídeos i botons de crida a l'acció.
  5. Publica l'anunci.

Solució: Segueix els passos detallats anteriorment per crear i publicar l'anunci.

Errors Comuns i Consells

  • Error: Contingut massa carregat: Evita afegir massa elements multimèdia que puguin sobrecarregar l'usuari. Mantingues un equilibri entre interactivitat i simplicitat.
  • Error: Crida a l'acció poc clara: Assegura't que els botons de crida a l'acció (CTA) siguin clars i visibles.
  • Consell: Utilitza contingut de qualitat alta per garantir una experiència visual atractiva.

Resum

Els anuncis d'experiència instantània són una eina poderosa per crear campanyes publicitàries immersives i interactives a Facebook. Aquests anuncis permeten combinar diversos formats de contingut per captar l'atenció dels usuaris i augmentar el compromís. Seguint els passos detallats, pots crear i publicar anuncis d'experiència instantània efectius per assolir els teus objectius de màrqueting.
