Píxel de Facebook
Següent ►

Introducció

El píxel de Facebook és una eina essencial per a qualsevol anunciant que vulgui mesurar, optimitzar i crear audiències per a les seves campanyes publicitàries. En aquesta secció, aprendrem què és el píxel de Facebook, com instal·lar-lo i com utilitzar-lo per millorar les nostres campanyes publicitàries.

Què és el Píxel de Facebook?

El píxel de Facebook és un fragment de codi JavaScript que col·loques al teu lloc web. Aquest codi recopila dades que t'ajuden a fer un seguiment de les conversions des de Facebook Ads, optimitzar els anuncis en funció d'aquestes conversions, crear audiències personalitzades per a futurs anuncis i tornar a comercialitzar a persones que ja han interactuat amb el teu lloc web.

Funcions del Píxel de Facebook

  1. Seguiment de conversions: Permet veure com les persones interactuen amb el teu lloc web després de veure els teus anuncis.
  2. Optimització d'anuncis: Ajuda a optimitzar els anuncis per a conversions específiques.
  3. Creació d'audiències personalitzades: Permet crear audiències personalitzades basades en el comportament dels usuaris al teu lloc web.
  4. Retargeting: Permet tornar a comercialitzar a persones que han visitat el teu lloc web.

Instal·lació del Píxel de Facebook

Pas 1: Crear el Píxel

  1. Accedeix al teu compte de Facebook Business.
  2. Ves a l'Administrador d'Anuncis.
  3. Selecciona "Píxels" al menú de l'esquerra.
  4. Fes clic a "Crea un píxel".
  5. Introdueix un nom per al teu píxel i el lloc web on el col·locaràs.
  6. Fes clic a "Crea".

Pas 2: Instal·lar el Píxel al teu lloc web

Opció 1: Instal·lació manual

  1. Copia el codi del píxel proporcionat per Facebook.
  2. Afegeix aquest codi a la capçalera del teu lloc web, just abans de la etiqueta </head>.

Opció 2: Utilitzar un gestor de continguts (CMS)

Molts CMS com WordPress, Shopify, i altres tenen plugins o integracions que faciliten la instal·lació del píxel de Facebook. Consulta la documentació del teu CMS per obtenir instruccions específiques.

Pas 3: Verificar la Instal·lació

  1. Utilitza l'extensió del navegador "Facebook Pixel Helper" per verificar que el píxel està instal·lat correctament.
  2. Navega pel teu lloc web i assegura't que el píxel està enviant dades a Facebook.

Ús del Píxel de Facebook

Seguiment d'Esdeveniments

Els esdeveniments són accions que les persones realitzen al teu lloc web. Alguns exemples inclouen:

  • PageView: Quan algú visita una pàgina del teu lloc web.
  • ViewContent: Quan algú visualitza una pàgina de contingut específic, com ara una pàgina de producte.
  • AddToCart: Quan algú afegeix un producte al carret de compra.
  • Purchase: Quan algú completa una compra.

Configuració d'Esdeveniments

Pots configurar esdeveniments addicionals per fer un seguiment de les accions específiques que t'interessen. Aquí tens un exemple de com configurar l'esdeveniment "AddToCart":

<script>
  fbq('track', 'AddToCart', {
    content_name: 'Product Name',
    content_ids: ['1234'],
    content_type: 'product',
    value: 19.99,
    currency: 'USD'
  });
</script>

Creació d'Audiències Personalitzades

Amb el píxel instal·lat, pots crear audiències personalitzades basades en les accions que les persones realitzen al teu lloc web. Per exemple, pots crear una audiència de persones que han visitat una pàgina específica o que han completat una compra.

  1. Ves a l'Administrador d'Anuncis.
  2. Selecciona "Audiències" al menú de l'esquerra.
  3. Fes clic a "Crea una audiència" i selecciona "Audiència personalitzada".
  4. Tria "Trànsit del lloc web" i defineix les condicions per a la teva audiència.

Exercici Pràctic

Objectiu

Configurar el píxel de Facebook al teu lloc web i crear una audiència personalitzada de persones que han visitat una pàgina específica.

Passos

  1. Crea un píxel de Facebook seguint els passos descrits anteriorment.
  2. Instal·la el píxel al teu lloc web.
  3. Verifica la instal·lació amb l'extensió "Facebook Pixel Helper".
  4. Configura un esdeveniment "ViewContent" per fer un seguiment de les visites a una pàgina de producte.
  5. Crea una audiència personalitzada de persones que han visitat aquesta pàgina de producte.

Solució

  1. Creació del píxel: Segueix els passos descrits a la secció "Crear el Píxel".
  2. Instal·lació del píxel: Afegeix el codi del píxel a la capçalera del teu lloc web.
  3. Verificació: Utilitza "Facebook Pixel Helper" per assegurar-te que el píxel està funcionant correctament.
  4. Configuració de l'esdeveniment:
<script>
  fbq('track', 'ViewContent', {
    content_name: 'Product Name',
    content_ids: ['1234'],
    content_type: 'product'
  });
</script>
  1. Creació de l'audiència personalitzada: Ves a l'Administrador d'Anuncis, selecciona "Audiències", fes clic a "Crea una audiència" i defineix les condicions per a la teva audiència basada en el trànsit del lloc web.

Conclusió

El píxel de Facebook és una eina poderosa que et permet fer un seguiment de les conversions, optimitzar els teus anuncis i crear audiències personalitzades. Instal·lar i configurar correctament el píxel és essencial per maximitzar l'eficàcia de les teves campanyes publicitàries a Facebook. Amb la pràctica i l'ús continuat, podràs aprofitar al màxim aquesta eina per millorar els teus resultats publicitaris.
