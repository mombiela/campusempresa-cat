Visió General del Projecte

Següent ►
►

En aquest mòdul final, aplicarem tots els coneixements adquirits al llarg del curs per construir un lloc web completament responsiu. Aquest projecte servirà com a pràctica integral per consolidar les habilitats de CSS que has après, des de les propietats bàsiques fins a les tècniques avançades i el disseny responsiu.

Objectius del Projecte

  1. Aplicar Conceptes de CSS: Utilitzar les propietats i tècniques de CSS per estilitzar un lloc web.
  2. Disseny Responsiu: Assegurar que el lloc web es vegi bé en diferents dispositius i mides de pantalla.
  3. Organització del Codi: Escriure codi CSS net i organitzat seguint les millors pràctiques.
  4. Utilitzar Preprocessadors i Frameworks: Integrar preprocessadors com Sass i frameworks com Bootstrap per millorar l'eficiència del desenvolupament.
  5. Optimització i Depuració: Minificar i optimitzar el codi CSS per a un millor rendiment i depurar qualsevol problema que pugui sorgir.

Estructura del Projecte

El projecte es dividirà en les següents etapes:

  1. Configuració del Projecte: Configurarem l'entorn de treball i crearem l'estructura bàsica del projecte.
  2. Crear el Disseny: Dissenyarem l'estructura del lloc web utilitzant HTML i CSS.
  3. Estilitzar els Components: Aplicarem estils CSS als diferents components del lloc web.
  4. Fer el Lloc Responsiu: Utilitzarem tècniques de disseny responsiu per assegurar que el lloc es vegi bé en diferents dispositius.
  5. Tocs Finals i Desplegament: Realitzarem els últims ajustos i desplegarem el lloc web.

Requisits del Projecte

  • HTML i CSS: Coneixements bàsics d'HTML i CSS.
  • Editor de Codi: Un editor de codi com Visual Studio Code, Sublime Text o Atom.
  • Navegador Web: Un navegador modern com Google Chrome, Firefox o Safari.
  • Preprocessador CSS: Coneixements bàsics de Sass (opcional però recomanable).
  • Framework CSS: Coneixements bàsics de Bootstrap (opcional però recomanable).

Descripció del Lloc Web

El lloc web que construirem serà una pàgina de perfil personal que inclourà les següents seccions:

  1. Capçalera (Header): Inclourà el nom i una imatge de perfil.
  2. Sobre Mi (About Me): Una breu descripció personal.
  3. Projectes (Projects): Una galeria de projectes amb imatges i descripcions.
  4. Habilitats (Skills): Una llista de les habilitats tècniques.
  5. Contacte (Contact): Un formulari de contacte.

Recursos Addicionals

  • Documentació de CSS: MDN Web Docs
  • Documentació de Sass: Sass Documentation
  • Documentació de Bootstrap: Bootstrap Documentation

Conclusió

Aquest projecte és una oportunitat per posar en pràctica tot el que has après i crear un lloc web atractiu i funcional. A mesura que avancem per les diferents etapes, assegura't de seguir les millors pràctiques i de revisar el teu codi per garantir que sigui net i eficient. Al final del projecte, tindràs un lloc web completament responsiu que podràs afegir al teu portafoli professional.

En el següent tema, configurarem l'entorn de treball i crearem l'estructura bàsica del projecte. Comencem!
