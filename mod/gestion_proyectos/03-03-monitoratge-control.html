Monitoratge i Control del Projecte
Següent ►

El monitoratge i control del projecte és una fase crítica en la gestió de projectes que assegura que el projecte es mantingui en el camí correcte per assolir els seus objectius. Aquesta fase implica la supervisió contínua de les activitats del projecte, la comparació del progrés real amb el pla de projecte, i la implementació de mesures correctives quan sigui necessari.

Objectius del Monitoratge i Control

  1. Assegurar l'Aliniació amb el Pla de Projecte:

    • Verificar que les activitats del projecte es compleixin segons el pla establert.
    • Identificar desviacions i prendre mesures correctives.

  2. Gestió de Riscos:

    • Monitoritzar els riscos identificats i implementar plans de resposta.
    • Identificar nous riscos i avaluar el seu impacte.

  3. Control de Qualitat:

    • Assegurar que els lliurables compleixin els estàndards de qualitat establerts.
    • Realitzar revisions i proves per verificar la qualitat.

  4. Gestió de Canvis:

    • Processar i aprovar sol·licituds de canvi.
    • Actualitzar el pla de projecte i els documents relacionats.

Processos Clau en el Monitoratge i Control

1. Seguiment del Progrés

  • Recopilació de Dades:

    • Recopilar dades sobre el progrés del projecte, incloent-hi l'estat de les tasques, els costos i els recursos utilitzats.
    • Utilitzar eines com fulls de temps, informes d'estat i reunions d'equip.

  • Anàlisi del Progrés:

    • Comparar el progrés real amb el pla de projecte.
    • Utilitzar tècniques com l'anàlisi de valor guanyat (EVA) per avaluar el rendiment del projecte.
Exemple d'Anàlisi de Valor Guanyat (EVA):
- Valor Planificat (PV): El valor del treball planificat fins a una data específica.
- Valor Guanyat (EV): El valor del treball realment completat fins a una data específica.
- Cost Real (AC): El cost realment incorregut fins a una data específica.

Indicadors Clau:
- Variància de Cost (CV) = EV - AC
- Variància de Programa (SV) = EV - PV
- Índex de Rendiment de Cost (CPI) = EV / AC
- Índex de Rendiment de Programa (SPI) = EV / PV

2. Control de Qualitat

  • Revisions de Qualitat:

    • Realitzar revisions periòdiques dels lliurables per assegurar que compleixin els estàndards de qualitat.
    • Utilitzar llistes de verificació i proves per avaluar la qualitat.

  • Auditories de Qualitat:

    • Realitzar auditories internes o externes per verificar la conformitat amb els processos de qualitat establerts.

3. Gestió de Canvis

  • Procés de Sol·licitud de Canvi:

    • Establir un procés formal per a la sol·licitud, avaluació i aprovació de canvis.
    • Utilitzar un registre de canvis per documentar totes les sol·licituds de canvi.

  • Avaluació de l'Impacte:

    • Avaluar l'impacte dels canvis proposats en el temps, el cost i l'abast del projecte.
    • Decidir si aprovar, rebutjar o modificar les sol·licituds de canvi.

4. Comunicació i Informes

  • Informes d'Estat:

    • Preparar informes d'estat regulars per comunicar el progrés del projecte als interessats.
    • Utilitzar gràfics, taules i altres eines visuals per presentar la informació de manera clara.

  • Reunions d'Estat:

    • Realitzar reunions periòdiques amb l'equip de projecte i els interessats per revisar l'estat del projecte i discutir problemes i solucions.

Eines i Tècniques de Monitoratge i Control

  • Eines de Programari:

    • Utilitzar eines de gestió de projectes com Microsoft Project, Trello, Asana, o Jira per monitoritzar el progrés i gestionar tasques.

  • Gràfics de Gantt:

    • Utilitzar gràfics de Gantt per visualitzar el progrés del projecte i identificar desviacions.

  • Quadres de Comandament:

    • Utilitzar quadres de comandament per presentar informació clau sobre el rendiment del projecte de manera visual i fàcilment comprensible.

Exercici Pràctic: Anàlisi de Valor Guanyat (EVA)

Enunciat

Suposem que estàs gestionant un projecte amb les següents dades fins a la data actual:

  • Valor Planificat (PV): 50,000 €
  • Valor Guanyat (EV): 45,000 €
  • Cost Real (AC): 48,000 €

Calcula els següents indicadors:

  1. Variància de Cost (CV)
  2. Variància de Programa (SV)
  3. Índex de Rendiment de Cost (CPI)
  4. Índex de Rendiment de Programa (SPI)

Solució

  1. Variància de Cost (CV):

    • CV = EV - AC
    • CV = 45,000 € - 48,000 €
    • CV = -3,000 €

  2. Variància de Programa (SV):

    • SV = EV - PV
    • SV = 45,000 € - 50,000 €
    • SV = -5,000 €

  3. Índex de Rendiment de Cost (CPI):

    • CPI = EV / AC
    • CPI = 45,000 € / 48,000 €
    • CPI = 0.9375

  4. Índex de Rendiment de Programa (SPI):

    • SPI = EV / PV
    • SPI = 45,000 € / 50,000 €
    • SPI = 0.9

Interpretació dels Resultats

  • CV Negatiu: El projecte està excedint el pressupost.
  • SV Negatiu: El projecte està endarrerit respecte al pla.
  • CPI Menor que 1: El projecte està gastant més del previst per unitat de treball completada.
  • SPI Menor que 1: El projecte està completant menys treball del previst per unitat de temps.

Resum

El monitoratge i control del projecte és essencial per assegurar que el projecte es mantingui en el camí correcte. Implica la supervisió contínua del progrés, la gestió de riscos, el control de qualitat i la gestió de canvis. Utilitzant eines i tècniques adequades, els gestors de projectes poden identificar desviacions i prendre mesures correctives per garantir l'èxit del projecte.
