Instal·lant NGINX

Següent ►
►

En aquesta secció, aprendrem com instal·lar NGINX en diferents sistemes operatius. NGINX és un servidor web i proxy invers molt popular, conegut per la seva alta rendibilitat i capacitat de gestió de càrrega. A continuació, veurem com instal·lar NGINX en sistemes operatius basats en Debian (com Ubuntu) i en sistemes operatius basats en Red Hat (com CentOS).

Instal·lació en sistemes basats en Debian (Ubuntu)

Pas 1: Actualitzar el sistema

Abans d'instal·lar NGINX, és una bona pràctica assegurar-se que el sistema està actualitzat. Obre una terminal i executa les següents comandes:

sudo apt update
sudo apt upgrade

Pas 2: Instal·lar NGINX

Un cop el sistema estigui actualitzat, pots instal·lar NGINX amb la següent comanda:

sudo apt install nginx

Pas 3: Verificar la instal·lació

Després d'instal·lar NGINX, pots verificar que s'ha instal·lat correctament i que està funcionant executant:

sudo systemctl status nginx

Si NGINX està funcionant correctament, hauries de veure una sortida similar a aquesta:

● nginx.service - A high performance web server and a reverse proxy server
   Loaded: loaded (/lib/systemd/system/nginx.service; enabled; vendor preset: enabled)
   Active: active (running) since ...

Pas 4: Accedir a la pàgina per defecte

Obre un navegador web i accedeix a http://localhost. Hauries de veure la pàgina per defecte de NGINX, la qual cosa indica que NGINX s'ha instal·lat correctament.

Instal·lació en sistemes basats en Red Hat (CentOS)

Pas 1: Actualitzar el sistema

Igual que en els sistemes basats en Debian, és recomanable actualitzar el sistema abans d'instal·lar NGINX. Obre una terminal i executa les següents comandes:

sudo yum update

Pas 2: Instal·lar el repositori EPEL

NGINX no està disponible en els repositoris per defecte de CentOS, així que necessitem instal·lar el repositori EPEL (Extra Packages for Enterprise Linux):

sudo yum install epel-release

Pas 3: Instal·lar NGINX

Un cop el repositori EPEL estigui instal·lat, pots instal·lar NGINX amb la següent comanda:

sudo yum install nginx

Pas 4: Iniciar i habilitar NGINX

Després d'instal·lar NGINX, necessitem iniciar el servei i configurar-lo perquè s'iniciï automàticament en arrencar el sistema:

sudo systemctl start nginx
sudo systemctl enable nginx

Pas 5: Verificar la instal·lació

Pots verificar que NGINX està funcionant correctament executant:

sudo systemctl status nginx

Si NGINX està funcionant correctament, hauries de veure una sortida similar a aquesta:

● nginx.service - The nginx HTTP and reverse proxy server
   Loaded: loaded (/usr/lib/systemd/system/nginx.service; enabled; vendor preset: disabled)
   Active: active (running) since ...

Pas 6: Accedir a la pàgina per defecte

Obre un navegador web i accedeix a http://localhost. Hauries de veure la pàgina per defecte de NGINX, la qual cosa indica que NGINX s'ha instal·lat correctament.

Resum

En aquesta secció, hem après com instal·lar NGINX en sistemes operatius basats en Debian (com Ubuntu) i en sistemes operatius basats en Red Hat (com CentOS). Hem cobert els passos per actualitzar el sistema, instal·lar NGINX, verificar la instal·lació i accedir a la pàgina per defecte de NGINX. Ara estem preparats per configurar NGINX i començar a utilitzar-lo com a servidor web o proxy invers.
