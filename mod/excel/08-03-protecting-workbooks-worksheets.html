Protegir Llibres de Treball i Fulls de Càlcul
Següent ►

En aquest tema, aprendrem com protegir els nostres llibres de treball i fulls de càlcul a Excel per assegurar-nos que la informació es mantingui segura i que només les persones autoritzades puguin fer canvis. La protecció és una eina essencial per a la col·laboració i la gestió de dades sensibles.

  1. Protegir un Llibre de Treball

1.1. Protegir l'Estructura del Llibre de Treball

Protegir l'estructura d'un llibre de treball impedeix que altres usuaris afegeixin, eliminin, moguin o renominin fulls de càlcul.

Passos per protegir l'estructura:

  1. Aneu a la pestanya Revisió.
  2. Feu clic a Protegir llibre.
  3. Marqueu l'opció Estructura.
  4. Introduïu una contrasenya (opcional) i confirmeu-la.
  5. Feu clic a D'acord.
Revisió -> Protegir llibre -> Estructura -> Contrasenya (opcional) -> D'acord

1.2. Protegir amb Contrasenya per Obrir o Modificar

Podeu establir una contrasenya per obrir o modificar un llibre de treball.

Passos per establir una contrasenya:

  1. Aneu a Fitxer -> Desa com.
  2. Seleccioneu la ubicació i el nom del fitxer.
  3. Feu clic a Eines (al costat del botó Desa) -> Opcions generals.
  4. Introduïu una contrasenya a Contrasenya per obrir i/o Contrasenya per modificar.
  5. Feu clic a D'acord i confirmeu la contrasenya.
  6. Feu clic a Desa.
Fitxer -> Desa com -> Eines -> Opcions generals -> Contrasenya per obrir/modificar -> D'acord -> Desa

  1. Protegir Fulls de Càlcul

2.1. Protegir un Full de Càlcul

Protegir un full de càlcul impedeix que altres usuaris facin canvis en les cel·les protegides.

Passos per protegir un full de càlcul:

  1. Aneu a la pestanya Revisió.
  2. Feu clic a Protegir full.
  3. Introduïu una contrasenya (opcional) i confirmeu-la.
  4. Seleccioneu les opcions de protecció desitjades (per exemple, permetre als usuaris seleccionar cel·les bloquejades o desbloquejades).
  5. Feu clic a D'acord.
Revisió -> Protegir full -> Contrasenya (opcional) -> Opcions de protecció -> D'acord

2.2. Bloquejar i Desbloquejar Cel·les

Per defecte, totes les cel·les estan bloquejades quan es protegeix un full de càlcul. Podeu desbloquejar cel·les específiques per permetre l'edició.

Passos per desbloquejar cel·les:

  1. Seleccioneu les cel·les que voleu desbloquejar.
  2. Feu clic amb el botó dret i seleccioneu Format de cel·les.
  3. Aneu a la pestanya Protecció.
  4. Desmarqueu l'opció Bloquejada.
  5. Feu clic a D'acord.
Seleccionar cel·les -> Botó dret -> Format de cel·les -> Protecció -> Desmarcar Bloquejada -> D'acord

  1. Protegir Elements Específics

3.1. Protegir Fórmules

Podeu ocultar fórmules per evitar que altres usuaris les vegin o les modifiquin.

Passos per ocultar fórmules:

  1. Seleccioneu les cel·les amb les fórmules que voleu ocultar.
  2. Feu clic amb el botó dret i seleccioneu Format de cel·les.
  3. Aneu a la pestanya Protecció.
  4. Marqueu l'opció Oculta.
  5. Feu clic a D'acord.
  6. Protegiu el full de càlcul (vegeu la secció 2.1).
Seleccionar cel·les -> Botó dret -> Format de cel·les -> Protecció -> Marcar Oculta -> D'acord -> Protegir full

3.2. Protegir Gràfics

Podeu protegir gràfics per evitar que altres usuaris els modifiquin.

Passos per protegir gràfics:

  1. Seleccioneu el gràfic.
  2. Feu clic amb el botó dret i seleccioneu Format de l'àrea del gràfic.
  3. Aneu a la pestanya Propietats.
  4. Marqueu l'opció No moure ni redimensionar amb cel·les.
  5. Protegiu el full de càlcul (vegeu la secció 2.1).
Seleccionar gràfic -> Botó dret -> Format de l'àrea del gràfic -> Propietats -> No moure ni redimensionar amb cel·les -> Protegir full

Exercicis Pràctics

Exercici 1: Protegir un Llibre de Treball

  1. Creeu un llibre de treball nou.
  2. Afegiu tres fulls de càlcul.
  3. Protegiu l'estructura del llibre de treball amb una contrasenya.

Exercici 2: Protegir un Full de Càlcul

  1. Creeu un full de càlcul amb dades fictícies.
  2. Bloquegeu totes les cel·les excepte una columna específica.
  3. Protegiu el full de càlcul amb una contrasenya.

Exercici 3: Ocultar Fórmules

  1. Creeu un full de càlcul amb diverses fórmules.
  2. Oculteu les fórmules de manera que no siguin visibles per altres usuaris.
  3. Protegiu el full de càlcul.

Solucions

Solució a l'Exercici 1

  1. Creeu un llibre de treball nou: Fitxer -> Nou.
  2. Afegiu tres fulls de càlcul: Feu clic a la icona "+" al costat dels noms dels fulls.
  3. Protegiu l'estructura: Revisió -> Protegir llibre -> Estructura -> Contrasenya -> D'acord.

Solució a l'Exercici 2

  1. Creeu un full de càlcul amb dades fictícies: Introduïu dades a les cel·les.
  2. Bloquegeu totes les cel·les excepte una columna específica:
    • Seleccioneu la columna específica.
    • Botó dret -> Format de cel·les -> Protecció -> Desmarcar Bloquejada -> D'acord.
  3. Protegiu el full de càlcul: Revisió -> Protegir full -> Contrasenya -> D'acord.

Solució a l'Exercici 3

  1. Creeu un full de càlcul amb diverses fórmules: Introduïu fórmules a les cel·les.
  2. Oculteu les fórmules:
    • Seleccioneu les cel·les amb fórmules.
    • Botó dret -> Format de cel·les -> Protecció -> Marcar Oculta -> D'acord.
  3. Protegiu el full de càlcul: Revisió -> Protegir full -> Contrasenya -> D'acord.

Conclusió

Protegir els llibres de treball i els fulls de càlcul és una habilitat essencial per mantenir la integritat i la seguretat de les dades a Excel. Hem après a protegir l'estructura dels llibres de treball, a establir contrasenyes per obrir o modificar, a protegir fulls de càlcul i a bloquejar o ocultar cel·les específiques. Aquestes tècniques són fonamentals per a la col·laboració segura i la gestió de dades sensibles.
