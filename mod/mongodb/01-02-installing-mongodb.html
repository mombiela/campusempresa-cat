Instal·lant MongoDB
Següent ►

En aquesta secció, aprendrem com instal·lar MongoDB al vostre sistema. MongoDB és una base de dades NoSQL que es pot instal·lar en diversos sistemes operatius com Windows, macOS i Linux. A continuació, es detallen els passos per a cada sistema operatiu.

Instal·lació a Windows

Pas 1: Descarregar MongoDB

  1. Visiteu el lloc web oficial de MongoDB: MongoDB Download Center.
  2. Seleccioneu la versió de MongoDB adequada per al vostre sistema operatiu (Windows).
  3. Feu clic a "Download" per descarregar l'instal·lador.

Pas 2: Executar l'Instal·lador

  1. Executeu el fitxer .msi descarregat.
  2. Seguiu les instruccions de l'assistent d'instal·lació.
  3. Seleccioneu "Complete" per a una instal·lació completa.
  4. Assegureu-vos de marcar l'opció "Install MongoDB as a Service" per configurar MongoDB com un servei de Windows.

Pas 3: Configurar les Variables d'Entorn

  1. Obriu el "Control Panel" i aneu a "System and Security" > "System" > "Advanced system settings".
  2. Feu clic a "Environment Variables".
  3. A la secció "System variables", seleccioneu la variable Path i feu clic a "Edit".
  4. Afegiu el camí del directori bin de MongoDB (per exemple, C:\Program Files\MongoDB\Server\<version>\bin).

Pas 4: Verificar la Instal·lació

  1. Obriu una finestra de terminal (Command Prompt).
  2. Executeu el següent comandament per verificar la instal·lació: 
    mongod --version
    Si tot està correcte, veureu la versió de MongoDB instal·lada.

Instal·lació a macOS

Pas 1: Utilitzar Homebrew

  1. Si no teniu Homebrew instal·lat, podeu instal·lar-lo executant el següent comandament al terminal: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Pas 2: Instal·lar MongoDB

  1. Executeu el següent comandament per afegir el repositori de MongoDB a Homebrew: 
    brew tap mongodb/brew
  2. Instal·leu MongoDB executant: 
    brew install mongodb-community@<version>

Pas 3: Iniciar MongoDB

  1. Inicieu el servei de MongoDB amb el següent comandament: 
    brew services start mongodb/brew/mongodb-community

Pas 4: Verificar la Instal·lació

  1. Executeu el següent comandament per verificar la instal·lació: 
    mongod --version
    Si tot està correcte, veureu la versió de MongoDB instal·lada.

Instal·lació a Linux (Ubuntu)

Pas 1: Importar la Clau Pública de MongoDB

  1. Executeu el següent comandament per importar la clau pública de MongoDB: 
    wget -qO - https://www.mongodb.org/static/pgp/server-<version>.asc | sudo apt-key add -

Pas 2: Crear el Fitxer de Llista per a MongoDB

  1. Creeu el fitxer de llista per a MongoDB executant: 
    echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu $(lsb_release -cs)/mongodb-org/<version> multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-<version>.list

Pas 3: Actualitzar el Repositori de Paquets

  1. Actualitzeu el repositori de paquets amb el següent comandament: 
    sudo apt-get update

Pas 4: Instal·lar MongoDB

  1. Instal·leu MongoDB executant: 
    sudo apt-get install -y mongodb-org

Pas 5: Iniciar MongoDB

  1. Inicieu el servei de MongoDB amb el següent comandament: 
    sudo systemctl start mongod

Pas 6: Verificar la Instal·lació

  1. Executeu el següent comandament per verificar la instal·lació: 
    mongod --version
    Si tot està correcte, veureu la versió de MongoDB instal·lada.

Conclusió

Ara que heu instal·lat MongoDB al vostre sistema, esteu preparats per començar a treballar amb aquesta potent base de dades NoSQL. En el següent mòdul, explorarem els conceptes bàsics de MongoDB, incloent-hi com crear, llegir, actualitzar i eliminar documents.
