Anuncis de Cerca Dinàmics

Següent

Els Anuncis de Cerca Dinàmics (DSA) són una eina poderosa dins de les campanyes SEM que permeten als anunciants mostrar anuncis rellevants basats en el contingut del seu lloc web. Aquests anuncis s'ajusten automàticament a les cerques dels usuaris, sense la necessitat de definir paraules clau específiques per a cada anunci.

Què són els Anuncis de Cerca Dinàmics?

Els Anuncis de Cerca Dinàmics són anuncis que es generen automàticament en funció del contingut del teu lloc web. Google Ads rastreja el teu lloc web per identificar les pàgines més rellevants per a les cerques dels usuaris i crea anuncis que coincideixen amb aquestes cerques.

Beneficis dels Anuncis de Cerca Dinàmics

  1. Cobertura Automàtica de Paraules Clau: Els DSA cobreixen una àmplia gamma de cerques rellevants sense la necessitat de definir cada paraula clau manualment.
  2. Estalvi de Temps: Redueixen el temps necessari per a la gestió de paraules clau i la creació d'anuncis.
  3. Anuncis Altament Rellevants: Els anuncis es generen en funció del contingut del teu lloc web, assegurant que siguin rellevants per a les cerques dels usuaris.
  4. Millora del Trànsit: Atrauran trànsit addicional al teu lloc web, especialment per a cerques de cua llarga que podrien no estar cobertes per les teves paraules clau tradicionals.

Com Funcionen els Anuncis de Cerca Dinàmics?

  1. Rastreig del Lloc Web: Google Ads rastreja el contingut del teu lloc web per identificar les pàgines més rellevants.
  2. Generació d'Anuncis: Quan un usuari realitza una cerca rellevant, Google Ads genera automàticament un anunci basat en el contingut de la pàgina més rellevant del teu lloc web.
  3. Títols Dinàmics: Els títols dels anuncis es generen automàticament per coincidir amb la cerca de l'usuari.
  4. URL de Destinació: L'URL de destinació de l'anunci és la pàgina més rellevant del teu lloc web.

Exemples de Codi

A continuació es mostra un exemple de com configurar una campanya d'Anuncis de Cerca Dinàmics a Google Ads:

# Exemple de configuració d'una campanya DSA amb Google Ads API

from google.ads.google_ads.client import GoogleAdsClient
from google.ads.google_ads.errors import GoogleAdsException

client = GoogleAdsClient.load_from_storage()

def create_dynamic_search_ads_campaign(client, customer_id):
    campaign_service = client.get_service("CampaignService", version="v6")
    campaign_operation = client.get_type("CampaignOperation", version="v6")
    campaign = campaign_operation.create

    campaign.name = "Dynamic Search Ads Campaign"
    campaign.advertising_channel_type = client.get_type(
        "AdvertisingChannelTypeEnum", version="v6"
    ).SEARCH
    campaign.status = client.get_type("CampaignStatusEnum", version="v6").PAUSED

    # Configuració de la campanya DSA
    campaign.dynamic_search_ads_setting.domain_name = "example.com"
    campaign.dynamic_search_ads_setting.language_code = "en"

    response = campaign_service.mutate_campaigns(customer_id, [campaign_operation])
    print(f"Created Dynamic Search Ads campaign with resource name: {response.results[0].resource_name}")

create_dynamic_search_ads_campaign(client, "INSERT_CUSTOMER_ID_HERE")

Millors Pràctiques per a Anuncis de Cerca Dinàmics

  1. Optimitza el Contingut del teu Lloc Web: Assegura't que el contingut del teu lloc web sigui rellevant i ben estructurat per facilitar el rastreig per part de Google Ads.
  2. Utilitza Exclusions de Pàgines: Exclou pàgines que no vols que es mostrin en els teus anuncis, com ara pàgines de política de privacitat o de contacte.
  3. Supervisa i Ajusta Regularment: Revisa el rendiment dels teus DSA i ajusta les exclusions i el contingut del teu lloc web segons sigui necessari.
  4. Combina amb Anuncis de Text: Utilitza DSA juntament amb anuncis de text tradicionals per cobrir una àmplia gamma de cerques i maximitzar el teu abast.

Exercici Pràctic

Exercici

Configura una campanya d'Anuncis de Cerca Dinàmics per a un lloc web fictici "www.exemple.com" utilitzant Google Ads. Assegura't d'incloure exclusions per a pàgines no rellevants.

Solució

from google.ads.google_ads.client import GoogleAdsClient
from google.ads.google_ads.errors import GoogleAdsException

client = GoogleAdsClient.load_from_storage()

def create_dynamic_search_ads_campaign_with_exclusions(client, customer_id):
    campaign_service = client.get_service("CampaignService", version="v6")
    campaign_operation = client.get_type("CampaignOperation", version="v6")
    campaign = campaign_operation.create

    campaign.name = "Dynamic Search Ads Campaign with Exclusions"
    campaign.advertising_channel_type = client.get_type(
        "AdvertisingChannelTypeEnum", version="v6"
    ).SEARCH
    campaign.status = client.get_type("CampaignStatusEnum", version="v6").PAUSED

    # Configuració de la campanya DSA
    campaign.dynamic_search_ads_setting.domain_name = "exemple.com"
    campaign.dynamic_search_ads_setting.language_code = "en"

    # Exclusions de pàgines
    exclusion = campaign.dynamic_search_ads_setting.excluded_page_urls.add()
    exclusion.url = "https://www.exemple.com/privacy-policy"
    exclusion = campaign.dynamic_search_ads_setting.excluded_page_urls.add()
    exclusion.url = "https://www.exemple.com/contact"

    response = campaign_service.mutate_campaigns(customer_id, [campaign_operation])
    print(f"Created Dynamic Search Ads campaign with resource name: {response.results[0].resource_name}")

create_dynamic_search_ads_campaign_with_exclusions(client, "INSERT_CUSTOMER_ID_HERE")

Conclusió

Els Anuncis de Cerca Dinàmics són una eina valuosa per a qualsevol campanya SEM, ja que permeten generar anuncis rellevants de manera automàtica basant-se en el contingut del teu lloc web. Implementar DSA pot estalviar temps i augmentar la cobertura de paraules clau, millorant així el rendiment de les teves campanyes. Assegura't de seguir les millors pràctiques per optimitzar els teus resultats i mantenir un control constant sobre el rendiment dels teus anuncis.
