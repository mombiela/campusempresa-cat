Comandes de Xarxa

Següent ►
►

En aquest tema, aprendrem a utilitzar diverses comandes de xarxa en Linux. Aquestes comandes són essencials per a la gestió i el diagnòstic de xarxes, així com per a la configuració de connexions de xarxa. A continuació, es presenten les comandes més importants, juntament amb exemples pràctics i exercicis per reforçar els conceptes apresos.

  1. ifconfig

Descripció

ifconfig és una comanda utilitzada per configurar interfícies de xarxa. També es pot utilitzar per veure la configuració de les interfícies de xarxa.

Sintaxi

ifconfig [interfície] [opcions]

Exemples

  1. Veure la configuració de totes les interfícies de xarxa:

    ifconfig

  2. Veure la configuració d'una interfície específica (per exemple, eth0):

    ifconfig eth0

  3. Assignar una adreça IP a una interfície:

    sudo ifconfig eth0 192.168.1.100

Exercici

  1. Utilitza ifconfig per veure la configuració de la teva interfície de xarxa principal.
  2. Assigna una nova adreça IP a la interfície eth0 i comprova el canvi amb ifconfig.

  1. ping

Descripció

ping és una comanda utilitzada per comprovar la connectivitat de xarxa entre el teu ordinador i un altre dispositiu de xarxa.

Sintaxi

ping [adreça IP o nom de domini]

Exemples

  1. Comprovar la connectivitat amb google.com:

    ping google.com

  2. Comprovar la connectivitat amb una adreça IP específica:

    ping 8.8.8.8

Exercici

  1. Utilitza ping per comprovar la connectivitat amb google.com.
  2. Utilitza ping per comprovar la connectivitat amb una adreça IP de la teva xarxa local.

  1. netstat

Descripció

netstat és una comanda utilitzada per mostrar connexions de xarxa, taules de ruta, estadístiques d'interfícies i molt més.

Sintaxi

netstat [opcions]

Exemples

  1. Veure totes les connexions de xarxa actives:

    netstat

  2. Veure les connexions de xarxa actives amb informació del programa:

    netstat -p

  3. Veure les estadístiques de les interfícies de xarxa:

    netstat -i

Exercici

  1. Utilitza netstat per veure totes les connexions de xarxa actives.
  2. Utilitza netstat -i per veure les estadístiques de les interfícies de xarxa.

  1. traceroute

Descripció

traceroute és una comanda utilitzada per traçar la ruta que segueixen els paquets per arribar a una destinació específica.

Sintaxi

traceroute [adreça IP o nom de domini]

Exemples

  1. Traçar la ruta fins a google.com:

    traceroute google.com

  2. Traçar la ruta fins a una adreça IP específica:

    traceroute 8.8.8.8

Exercici

  1. Utilitza traceroute per traçar la ruta fins a google.com.
  2. Utilitza traceroute per traçar la ruta fins a una adreça IP de la teva xarxa local.

  1. nslookup

Descripció

nslookup és una comanda utilitzada per obtenir informació del sistema de noms de domini (DNS) sobre un nom de domini o una adreça IP.

Sintaxi

nslookup [nom de domini o adreça IP]

Exemples

  1. Obtenir informació DNS sobre google.com:

    nslookup google.com

  2. Obtenir informació DNS sobre una adreça IP específica:

    nslookup 8.8.8.8

Exercici

  1. Utilitza nslookup per obtenir informació DNS sobre google.com.
  2. Utilitza nslookup per obtenir informació DNS sobre una adreça IP de la teva xarxa local.

Resum

En aquesta secció, hem après a utilitzar diverses comandes de xarxa en Linux, incloent ifconfig, ping, netstat, traceroute i nslookup. Aquestes comandes són essencials per a la gestió i el diagnòstic de xarxes. Practica amb els exercicis proporcionats per reforçar els teus coneixements i estar preparat per a situacions reals de gestió de xarxes.
