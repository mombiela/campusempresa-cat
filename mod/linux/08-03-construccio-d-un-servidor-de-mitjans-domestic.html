Construcció d'un Servidor de Mitjans Domèstic
Següent ►

En aquest tema, aprendrem a configurar un servidor de mitjans domèstic utilitzant Linux. Un servidor de mitjans domèstic permet emmagatzemar, organitzar i transmetre contingut multimèdia (com ara vídeos, música i fotos) a diversos dispositius dins de la teva xarxa domèstica.

Objectius

  • Entendre els requisits i la configuració bàsica d'un servidor de mitjans.
  • Instal·lar i configurar Plex Media Server.
  • Configurar l'accés remot i la seguretat del servidor de mitjans.

Requisits Previs

  • Coneixements bàsics de la línia de comandes de Linux.
  • Un ordinador amb una distribució de Linux instal·lada (preferiblement Ubuntu o Debian).
  • Connexió a Internet.

1. Introducció al Servidor de Mitjans

Què és un Servidor de Mitjans?

Un servidor de mitjans és un sistema que emmagatzema i distribueix contingut multimèdia a altres dispositius de la xarxa. Això permet que dispositius com televisors intel·ligents, ordinadors, tauletes i telèfons mòbils accedeixin i reprodueixin aquest contingut.

Beneficis d'un Servidor de Mitjans Domèstic

  • Centralització del contingut: Tots els teus fitxers multimèdia es troben en un sol lloc.
  • Accés remot: Pots accedir al teu contingut des de qualsevol lloc amb connexió a Internet.
  • Transmissió a múltiples dispositius: Reprodueix contingut en diversos dispositius simultàniament.

2. Instal·lació de Plex Media Server

Pas 1: Actualitzar el Sistema

Abans d'instal·lar Plex, assegura't que el teu sistema estigui actualitzat.

sudo apt update
sudo apt upgrade

Pas 2: Descarregar i Instal·lar Plex

Descarrega el paquet d'instal·lació de Plex Media Server des del lloc web oficial.

wget https://downloads.plex.tv/plex-media-server-new/1.23.3.4707-8a2f9d1b2/debian/plexmediaserver_1.23.3.4707-8a2f9d1b2_amd64.deb

Instal·la el paquet descarregat.

sudo dpkg -i plexmediaserver_1.23.3.4707-8a2f9d1b2_amd64.deb

Pas 3: Verificar la Instal·lació

Assegura't que Plex Media Server s'està executant correctament.

sudo systemctl status plexmediaserver

Si el servei no està actiu, pots iniciar-lo amb:

sudo systemctl start plexmediaserver

3. Configuració de Plex Media Server

Pas 1: Accedir a la Interfície Web de Plex

Obre un navegador web i accedeix a la interfície de configuració de Plex:

http://localhost:32400/web

Pas 2: Crear un Compte Plex

Si no tens un compte Plex, crea'n un. Si ja en tens, inicia sessió.

Pas 3: Configurar les Biblioteques de Mitjans

Segueix les instruccions per afegir biblioteques de mitjans. Pots afegir carpetes que continguin vídeos, música i fotos.

Pas 4: Configurar l'Accés Remot

Per accedir al teu servidor de mitjans des de fora de la teva xarxa domèstica, activa l'accés remot a la configuració de Plex.

4. Seguretat del Servidor de Mitjans

Pas 1: Configurar un Tallafocs

Assegura't que el teu servidor de mitjans estigui protegit per un tallafocs. Pots utilitzar ufw per configurar-lo.

sudo ufw allow 32400/tcp
sudo ufw enable

Pas 2: Utilitzar HTTPS

Per protegir les teves dades durant la transmissió, utilitza HTTPS. Plex ofereix suport per a certificats SSL.

5. Exercicis Pràctics

Exercici 1: Afegir una Biblioteca de Vídeos

  1. Crea una carpeta anomenada Videos al teu directori d'inici.
  2. Mou alguns fitxers de vídeo a aquesta carpeta.
  3. Afegeix aquesta carpeta com a biblioteca de vídeos a Plex.

Exercici 2: Configurar l'Accés Remot

  1. Activa l'accés remot a la configuració de Plex.
  2. Verifica que pots accedir al teu servidor de mitjans des d'un dispositiu fora de la teva xarxa domèstica.

Solucions als Exercicis

Solució a l'Exercici 1

mkdir ~/Videos
mv /path/to/your/videos/* ~/Videos/

A la interfície web de Plex, afegeix la carpeta ~/Videos com a biblioteca de vídeos.

Solució a l'Exercici 2

A la configuració de Plex, activa l'opció d'accés remot i segueix les instruccions per verificar la connexió.

Conclusió

En aquesta secció, hem après a configurar un servidor de mitjans domèstic utilitzant Plex Media Server. Hem cobert la instal·lació, configuració i seguretat bàsica del servidor. Amb aquest coneixement, pots centralitzar i transmetre el teu contingut multimèdia a diversos dispositius dins i fora de la teva xarxa domèstica.
