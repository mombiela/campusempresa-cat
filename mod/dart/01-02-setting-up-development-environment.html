Configuració de l'Entorn de Desenvolupament
Següent ►

En aquesta secció, aprendrem com configurar l'entorn de desenvolupament per començar a programar en Dart. Seguir aquests passos t'assegurarà que tens tot el necessari per escriure, executar i depurar el teu codi Dart.

  1. Instal·lació del SDK de Dart

El primer pas és instal·lar el Software Development Kit (SDK) de Dart. L'SDK inclou totes les eines necessàries per desenvolupar aplicacions en Dart.

Passos per a la instal·lació:

  1. Descarrega l'SDK de Dart:

    • Visita la pàgina oficial de Dart: https://dart.dev/get-dart
    • Selecciona el sistema operatiu que estàs utilitzant (Windows, macOS, Linux) i segueix les instruccions per descarregar l'instal·lador.

  2. Instal·la l'SDK:

    • Windows: Executa l'instal·lador descarregat i segueix les instruccions de la pantalla.
    • macOS: Obre el terminal i executa el següent comandament: 
      brew tap dart-lang/dart
brew install dart
    • Linux: Obre el terminal i executa els següents comandaments: 
      sudo apt update
sudo apt install apt-transport-https
sudo sh -c 'wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -'
sudo sh -c 'wget -qO- https://storage.googleapis.com/download.dartlang.org/linux/debian/dart_stable.list > /etc/apt/sources.list.d/dart_stable.list'
sudo apt update
sudo apt install dart

  3. Verifica la instal·lació:

    • Obre una terminal o línia de comandes i executa: 
      dart --version
    • Hauries de veure la versió de Dart instal·lada.

  1. Instal·lació d'un Editor de Codi

Per escriure codi Dart, necessitaràs un editor de codi. Hi ha diverses opcions disponibles, però Visual Studio Code (VS Code) és una de les més populars i recomanades.

Instal·lació de Visual Studio Code:

  1. Descarrega Visual Studio Code:

    • Visita la pàgina oficial de VS Code: https://code.visualstudio.com/
    • Descarrega l'instal·lador per al teu sistema operatiu i segueix les instruccions per instal·lar-lo.

  2. Instal·la l'extensió de Dart:

    • Obre VS Code.
    • Ves a la secció d'extensions (icona de quadrat amb quatre quadrats petits a la barra lateral esquerra).
    • Cerca "Dart" i instal·la l'extensió oficial de Dart.

  1. Configuració de l'Entorn de Desenvolupament

Un cop tens l'SDK de Dart i l'editor de codi instal·lats, és hora de configurar l'entorn per començar a programar.

Configuració de VS Code per a Dart:

  1. Crea un nou projecte Dart:

    • Obre VS Code.
    • Obre la terminal integrada (pots fer-ho amb Ctrl + o des del menú Terminal > New Terminal).
    • Executa el següent comandament per crear un nou projecte: 
      dart create my_first_dart_project
    • Això crearà una nova carpeta amb el nom my_first_dart_project que conté l'estructura bàsica d'un projecte Dart.

  2. Obre el projecte en VS Code:

    • Ves a File > Open Folder i selecciona la carpeta my_first_dart_project.

  3. Executa el teu primer programa:

    • Obre el fitxer bin/main.dart dins de la carpeta del projecte.
    • Hauries de veure un codi similar a aquest: 
      void main() {
  print('Hello, World!');
}
    • Executa el programa des de la terminal integrada amb el següent comandament: 
      dart run
    • Hauries de veure el missatge "Hello, World!" a la terminal.

  1. Exercici Pràctic

Exercici 1: Configura el teu entorn de desenvolupament

  1. Segueix els passos anteriors per instal·lar l'SDK de Dart.
  2. Instal·la Visual Studio Code i l'extensió de Dart.
  3. Crea un nou projecte Dart anomenat my_first_dart_project.
  4. Executa el programa per assegurar-te que tot està configurat correctament.

Solució:

Si has seguit tots els passos correctament, hauries de veure el missatge "Hello, World!" a la terminal després d'executar el programa. Si tens algun problema, revisa els passos anteriors per assegurar-te que no t'has saltat cap detall.

Conclusió

Ara que tens l'entorn de desenvolupament configurat, estàs llest per començar a programar en Dart. En la següent secció, escriuràs el teu primer programa en Dart i aprendràs sobre la sintaxi i l'estructura bàsica del llenguatge.
