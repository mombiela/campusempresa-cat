El Teu Primer Programa en Dart

Següent ►
►

En aquesta secció, aprendrem a crear el nostre primer programa en Dart. Aquest és un pas fonamental per familiaritzar-nos amb la sintaxi i l'estructura bàsica del llenguatge. Començarem amb un simple programa "Hola, món!" i després explorarem alguns conceptes bàsics.

Objectius

  • Escriure i executar el primer programa en Dart.
  • Entendre la sintaxi bàsica de Dart.
  • Familiaritzar-se amb l'ús de l'entorn de desenvolupament.

  1. Configuració de l'Entorn de Desenvolupament

Abans de començar a escriure el nostre primer programa, assegurem-nos que tenim l'entorn de desenvolupament configurat correctament. Si encara no ho has fet, revisa la secció anterior sobre la configuració de l'entorn de desenvolupament.

  1. El Teu Primer Programa: "Hola, món!"

Pas 1: Crear un Nou Fitxer Dart

  1. Obre el teu editor de codi preferit (per exemple, Visual Studio Code).
  2. Crea un nou fitxer i anomena'l main.dart.

Pas 2: Escriure el Codi

A continuació, escriurem el codi per al nostre primer programa en Dart.

void main() {
  print('Hola, món!');
}

Explicació del Codi

  • void main() { ... }: Aquesta és la funció principal del programa. En Dart, el punt d'entrada d'un programa és la funció main.
  • print('Hola, món!');: Aquesta línia imprimeix el text "Hola, món!" a la consola. La funció print és una funció integrada en Dart que s'utilitza per mostrar missatges a la consola.

Pas 3: Executar el Programa

Per executar el programa, obre una terminal i navega fins al directori on es troba el fitxer main.dart. A continuació, executa la següent comanda:

dart run main.dart

Si tot està configurat correctament, hauràs de veure el següent resultat a la consola:

Hola, món!

  1. Conceptes Bàsics

Funció main

La funció main és el punt d'entrada de qualsevol aplicació Dart. És aquí on comença l'execució del programa. La seva signatura és void main(), on void indica que la funció no retorna cap valor.

Funció print

La funció print s'utilitza per mostrar informació a la consola. És molt útil per a la depuració i per a mostrar resultats als usuaris.

Comentaris

Els comentaris són línies de text que no s'executen com a codi. S'utilitzen per a documentar el codi i fer-lo més llegible. En Dart, hi ha dos tipus de comentaris:

  • Comentaris d'una sola línia: Comencen amb //.
  • Comentaris de múltiples línies: Comencen amb /* i acaben amb */.

Exemple:

// Aquest és un comentari d'una sola línia.

/*
Aquest és un comentari
de múltiples línies.
*/

Exercicis Pràctics

Exercici 1: Modificar el Missatge

Modifica el programa per imprimir un missatge diferent, per exemple, "Benvingut a Dart!".

Solució

void main() {
  print('Benvingut a Dart!');
}

Exercici 2: Afegir Comentaris

Afegeix comentaris al teu programa per descriure què fa cada línia de codi.

Solució

void main() {
  // Imprimeix un missatge de benvinguda a la consola.
  print('Benvingut a Dart!');
}

Errors Comuns

  • Error de Sintaxi: Si oblides el punt i coma (;) al final d'una línia, obtindràs un error de sintaxi.
  • Nom de Fitxer Incorrecte: Assegura't que el fitxer tingui l'extensió .dart.

Resum

En aquesta secció, hem creat i executat el nostre primer programa en Dart. Hem après sobre la funció main, la funció print i com utilitzar comentaris en el codi. A més, hem practicat amb alguns exercicis per reforçar els conceptes apresos. En la següent secció, explorarem la sintaxi i l'estructura bàsica de Dart en més detall.
