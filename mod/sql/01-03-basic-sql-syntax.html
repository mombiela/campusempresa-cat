Sintaxi bàsica de SQL
Següent ►

Introducció

En aquest tema, aprendrem la sintaxi bàsica de SQL, que és fonamental per a qualsevol operació amb bases de dades. Ens centrarem en les instruccions més comunes i com estructurar-les correctament.

Conceptes Clau

  1. Instruccions SQL: Ordres que s'envien a la base de dades per realitzar operacions.
  2. Clàusules: Parts d'una instrucció SQL que especifiquen condicions o modificacions.
  3. Operadors: S'utilitzen per realitzar operacions en els valors de les columnes.

Estructura Bàsica d'una Instrucció SQL

Una instrucció SQL típica segueix aquesta estructura:

SELECT column1, column2, ...
FROM table_name
WHERE condition;

Components de l'Estructura

  • SELECT: Especifica les columnes que es volen recuperar.
  • FROM: Indica la taula de la qual es volen obtenir les dades.
  • WHERE: Filtra els registres basant-se en una condició.

Exemples Pràctics

Exemple 1: Seleccionar Totes les Columnes

SELECT * FROM employees;

Aquest exemple selecciona totes les columnes de la taula employees.

Exemple 2: Seleccionar Columnes Específiques

SELECT first_name, last_name FROM employees;

Aquest exemple selecciona només les columnes first_name i last_name de la taula employees.

Exemple 3: Filtrar Resultats amb WHERE

SELECT first_name, last_name FROM employees
WHERE department = 'Sales';

Aquest exemple selecciona els noms i cognoms dels empleats que treballen al departament de vendes.

Exercicis Pràctics

Exercici 1: Seleccionar Totes les Columnes

Escriu una instrucció SQL per seleccionar totes les columnes de la taula products.

-- Solució
SELECT * FROM products;

Exercici 2: Seleccionar Columnes Específiques

Escriu una instrucció SQL per seleccionar les columnes product_name i price de la taula products.

-- Solució
SELECT product_name, price FROM products;

Exercici 3: Filtrar Resultats

Escriu una instrucció SQL per seleccionar els noms i preus dels productes que costen més de 50 unitats monetàries.

-- Solució
SELECT product_name, price FROM products
WHERE price > 50;

Errors Comuns i Consells

  1. Oblidar el punt i coma (;) al final de la instrucció: Això pot causar errors en l'execució de la consulta.
  2. Utilitzar noms de columnes o taules incorrectes: Assegura't que els noms coincideixin exactament amb els de la base de dades.
  3. No utilitzar cometes simples per a valors de text: Els valors de text en la clàusula WHERE han d'estar entre cometes simples.

Resum

En aquest tema, hem après la sintaxi bàsica de SQL, incloent les instruccions SELECT, FROM i WHERE. Hem vist com seleccionar totes les columnes, columnes específiques i com filtrar resultats. També hem practicat amb alguns exercicis per reforçar els conceptes apresos.

En el següent tema, explorarem com configurar el teu entorn SQL per començar a treballar amb bases de dades.
