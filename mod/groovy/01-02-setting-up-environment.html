Configuració de l'Entorn
Següent ►

En aquesta secció, aprendrem a configurar l'entorn necessari per començar a programar en Groovy. Aquest procés inclou la instal·lació del JDK (Java Development Kit), la descàrrega i instal·lació de Groovy, i la configuració d'un editor de codi o IDE (Integrated Development Environment) per facilitar el desenvolupament.

  1. Instal·lació del JDK

Groovy s'executa sobre la màquina virtual de Java (JVM), per tant, necessitem tenir instal·lat el JDK. A continuació, es mostren els passos per instal·lar el JDK:

Pas 1: Descàrrega del JDK

  1. Visita la pàgina oficial d'Oracle: Oracle JDK Downloads.
  2. Selecciona la versió més recent del JDK.
  3. Descarrega l'instal·lador adequat per al teu sistema operatiu (Windows, macOS, Linux).

Pas 2: Instal·lació del JDK

  1. Executa l'instal·lador descarregat.
  2. Segueix les instruccions de l'assistent d'instal·lació.
  3. Un cop finalitzada la instal·lació, assegura't que la variable d'entorn JAVA_HOME està configurada correctament.

Pas 3: Verificació de la Instal·lació

  1. Obre una terminal o línia de comandes.
  2. Executa la següent comanda per verificar la instal·lació del JDK:
java -version

Hauries de veure una sortida similar a aquesta:

java version "17.0.1" 2021-10-19 LTS
Java(TM) SE Runtime Environment (build 17.0.1+12-LTS-39)
Java HotSpot(TM) 64-Bit Server VM (build 17.0.1+12-LTS-39, mixed mode, sharing)

  1. Instal·lació de Groovy

Un cop tenim el JDK instal·lat, podem procedir a instal·lar Groovy.

Pas 1: Descàrrega de Groovy

  1. Visita la pàgina oficial de Groovy: Groovy Downloads.
  2. Descarrega l'última versió estable de Groovy.

Pas 2: Instal·lació de Groovy

  1. Descomprimeix l'arxiu descarregat en una ubicació de la teva elecció.
  2. Configura la variable d'entorn GROOVY_HOME apuntant al directori on has descomprimit Groovy.
  3. Afegeix el directori bin de Groovy al teu PATH.

Pas 3: Verificació de la Instal·lació

  1. Obre una terminal o línia de comandes.
  2. Executa la següent comanda per verificar la instal·lació de Groovy:
groovy -version

Hauries de veure una sortida similar a aquesta:

Groovy Version: 3.0.9 JVM: 17.0.1 Vendor: Oracle Corporation OS: Windows 10

  1. Configuració de l'Editor de Codi o IDE

Per facilitar el desenvolupament en Groovy, és recomanable utilitzar un editor de codi o IDE. Alguns dels més populars són:

IntelliJ IDEA

  1. Descarrega IntelliJ IDEA des de la seva pàgina oficial.
  2. Instal·la IntelliJ IDEA seguint les instruccions de l'assistent d'instal·lació.
  3. Un cop instal·lat, obre IntelliJ IDEA i crea un nou projecte Groovy.

Visual Studio Code

  1. Descarrega Visual Studio Code des de la seva pàgina oficial.
  2. Instal·la Visual Studio Code seguint les instruccions de l'assistent d'instal·lació.
  3. Afegeix l'extensió "Groovy" des del marketplace de Visual Studio Code.

Eclipse

  1. Descarrega Eclipse des de la seva pàgina oficial.
  2. Instal·la Eclipse seguint les instruccions de l'assistent d'instal·lació.
  3. Afegeix el plugin Groovy-Eclipse des del marketplace d'Eclipse.

Resum

En aquesta secció hem après a:

  • Instal·lar el JDK necessari per executar Groovy.
  • Descarregar i instal·lar Groovy.
  • Configurar un editor de codi o IDE per facilitar el desenvolupament en Groovy.

Ara que tenim l'entorn configurat, estem preparats per començar a escriure el nostre primer script en Groovy. En la següent secció, explorarem els conceptes bàsics de Groovy per familiaritzar-nos amb el llenguatge.
