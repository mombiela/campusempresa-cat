Configuració del teu entorn de desenvolupament
Següent ►

En aquest tema, aprendràs a configurar el teu entorn de desenvolupament per treballar amb Apache Cordova. Aquest procés inclou la instal·lació de les eines necessàries i la configuració del teu sistema per crear aplicacions mòbils multiplataforma.

Requisits previs

Abans de començar, assegura't de tenir els següents requisits:

  1. Sistema operatiu: Windows, macOS o Linux.
  2. Node.js: Cordova requereix Node.js per funcionar. Pots descarregar-lo des de nodejs.org.
  3. Git: És recomanable tenir Git instal·lat per gestionar el codi font. Pots descarregar-lo des de git-scm.com.

Passos per configurar l'entorn

  1. Instal·lació de Node.js i npm

Node.js inclou npm (Node Package Manager), que és necessari per instal·lar Cordova. Segueix aquests passos per instal·lar Node.js:

  1. Ves a nodejs.org i descarrega la versió recomanada per al teu sistema operatiu.

  2. Executa l'instal·lador i segueix les instruccions per completar la instal·lació.

  3. Verifica la instal·lació obrint una terminal i executant les següents comandes:

    node -v
npm -v

    Hauries de veure les versions instal·lades de Node.js i npm.

  1. Instal·lació de Cordova

Un cop tinguis Node.js i npm instal·lats, pots instal·lar Cordova globalment al teu sistema:

npm install -g cordova

Verifica la instal·lació de Cordova executant:

cordova -v

  1. Instal·lació de les eines de plataforma

Cordova permet desenvolupar aplicacions per a diverses plataformes mòbils. A continuació, es detallen els passos per configurar les eines necessàries per a Android i iOS.

Android

  1. Instal·la Android Studio: Descarrega i instal·la Android Studio des de developer.android.com/studio.

  2. Configura les variables d'entorn: Afegeix les rutes de les eines d'Android a les variables d'entorn del teu sistema.

    • Windows:

      1. Obre el "Panell de control" i ves a "Sistema i seguretat" > "Sistema" > "Configuració avançada del sistema".

      2. Fes clic a "Variables d'entorn" i afegeix les següents variables:

        ANDROID_HOME = C:\Users\<el_teu_usuari>\AppData\Local\Android\Sdk
PATH = %PATH%;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools

    • macOS/Linux:

      1. Obre el fitxer .bash_profile, .zshrc o .bashrc (segons el teu shell) i afegeix les següents línies:

        export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/tools:$ANDROID_HOME/platform-tools

      2. Guarda el fitxer i executa source ~/.bash_profile (o el fitxer corresponent) per aplicar els canvis.

iOS

  1. Instal·la Xcode: Descarrega i instal·la Xcode des de l'App Store.

  2. Instal·la les eines de línia de comandes de Xcode: Obre una terminal i executa:

    xcode-select --install

  3. Configura CocoaPods: CocoaPods és un gestor de dependències per a projectes iOS. Instal·la'l executant:

    sudo gem install cocoapods

  1. Verificació de la configuració

Per assegurar-te que tot està configurat correctament, crea un nou projecte Cordova i compila'l per a una plataforma específica.

  1. Crea un nou projecte:

    cordova create myApp com.example.myapp MyApp
cd myApp

  2. Afegeix una plataforma (per exemple, Android):

    cordova platform add android

  3. Compila el projecte:

    cordova build android

Si la compilació es completa sense errors, la teva configuració és correcta.

Resum

En aquesta secció, has après a configurar el teu entorn de desenvolupament per treballar amb Apache Cordova. Has instal·lat Node.js, npm i Cordova, i has configurat les eines necessàries per desenvolupar aplicacions per a Android i iOS. Ara estàs preparat per començar a crear aplicacions mòbils multiplataforma amb Cordova.
