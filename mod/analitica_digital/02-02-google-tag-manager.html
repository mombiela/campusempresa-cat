Google Tag Manager: Implementació Bàsica
Següent ►

Introducció

Google Tag Manager (GTM) és una eina gratuïta que permet gestionar i desplegar etiquetes de màrqueting (tags) en el teu lloc web o aplicació mòbil sense haver de modificar el codi. Amb GTM, pots afegir i actualitzar etiquetes de seguiment, píxels de conversió, i altres fragments de codi de manera fàcil i ràpida.

Conceptes Bàsics de Google Tag Manager

Què és una Etiqueta (Tag)?

Una etiqueta és un fragment de codi que s'utilitza per recollir dades i enviar-les a una plataforma de tercers, com Google Analytics, Facebook Pixel, o altres eines de màrqueting.

Què és un Activador (Trigger)?

Un activador és una condició que determina quan i on s'ha d'executar una etiqueta. Per exemple, pots configurar un activador perquè una etiqueta s'executi quan es carrega una pàgina específica o quan un usuari fa clic en un botó.

Què és una Variable?

Les variables són elements que emmagatzemen informació que pot ser utilitzada per les etiquetes i els activadors. Per exemple, una variable pot emmagatzemar l'URL de la pàgina actual o el valor d'una compra.

Configuració Inicial de Google Tag Manager

Pas 1: Crear un Compte de Google Tag Manager

  1. Visita el lloc web de Google Tag Manager.
  2. Inicia sessió amb el teu compte de Google.
  3. Fes clic a "Crear compte".
  4. Omple la informació del compte:
    • Nom del compte
    • País
    • Nom del contenidor (normalment el nom del teu lloc web)
    • Plataforma de destinació (Web, iOS, Android, AMP)
  5. Fes clic a "Crear".

Pas 2: Instal·lar el Fragment de Codi de Google Tag Manager

  1. Després de crear el compte, se't proporcionarà un fragment de codi.
  2. Copia el fragment de codi i enganxa'l en totes les pàgines del teu lloc web, just després de l'etiqueta <head> d'obertura.
  3. Afegeix el segon fragment de codi immediatament després de l'etiqueta <body> d'obertura.
<!-- Google Tag Manager -->

Pas 3: Crear una Etiqueta de Google Analytics

  1. A la interfície de Google Tag Manager, fes clic a "Etiquetes" al menú de l'esquerra.
  2. Fes clic a "Nou" i després a "Configuració de l'etiqueta".
  3. Selecciona "Google Analytics: Universal Analytics".
  4. Configura la teva etiqueta:
    • Tipus de seguiment: "Pàgina vista".
    • ID de seguiment: Introduïu el vostre ID de seguiment de Google Analytics (format UA-XXXXXX-Y).
  5. Fes clic a "Activadors" i selecciona "All Pages" (Totes les pàgines).
  6. Guarda la teva etiqueta.

Pas 4: Publicar el Contenidor

  1. Després de configurar la teva etiqueta, fes clic a "Enviar" a la part superior dreta de la interfície de Google Tag Manager.
  2. Omple la informació de la versió (nom i descripció).
  3. Fes clic a "Publicar".

Exercici Pràctic

Objectiu

Configurar una etiqueta de seguiment de clics en un botó específic del teu lloc web utilitzant Google Tag Manager.

Passos

  1. Crear una Etiqueta de Seguiment de Clics:

    • A la interfície de Google Tag Manager, fes clic a "Etiquetes" i després a "Nou".
    • Selecciona "Google Analytics: Universal Analytics".
    • Configura la teva etiqueta:
      • Tipus de seguiment: "Esdeveniment".
      • Categoria: "Botó".
      • Acció: "Clic".
      • Etiqueta: "Botó de Subscripció".
      • ID de seguiment: Introduïu el vostre ID de seguiment de Google Analytics.

  2. Crear un Activador de Clics:

    • Fes clic a "Activadors" i després a "Nou".
    • Selecciona "Clic - Tots els elements".
    • Configura el teu activador:
      • Tipus d'activador: "Clic - Tots els elements".
      • Selecciona "Alguns clics".
      • Configura la condició: "Click Text" conté "Subscripció".
    • Guarda el teu activador.

  3. Assignar l'Activador a l'Etiqueta:

    • Torna a la configuració de la teva etiqueta i assigna l'activador de clics que acabes de crear.
    • Guarda la teva etiqueta.

  4. Publicar el Contenidor:

    • Fes clic a "Enviar" a la part superior dreta de la interfície de Google Tag Manager.
    • Omple la informació de la versió (nom i descripció).
    • Fes clic a "Publicar".

Conclusió

Google Tag Manager és una eina poderosa que simplifica la gestió de les etiquetes de màrqueting al teu lloc web. Amb una configuració inicial senzilla, pots començar a recollir dades valuoses sense necessitat de modificar constantment el codi del teu lloc web. En aquest mòdul, hem après a crear un compte de GTM, instal·lar el fragment de codi, crear etiquetes i activadors, i publicar el contenidor. En el proper mòdul, explorarem com configurar objectius i esdeveniments per mesurar accions específiques dels usuaris.
