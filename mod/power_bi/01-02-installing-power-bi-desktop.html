Instal·lant Power BI Desktop
Següent ►

Introducció

Power BI Desktop és una eina essencial per a la creació de visualitzacions de dades i informes interactius. En aquesta secció, aprendrem com instal·lar Power BI Desktop al nostre ordinador. Aquest procés és senzill i només requereix uns quants passos.

Requisits del sistema

Abans de començar la instal·lació, assegura't que el teu sistema compleixi els següents requisits mínims:

  • Sistema operatiu: Windows 7 / Windows Server 2008 R2, o versions posteriors.
  • Memòria RAM: 2 GB (4 GB o més recomanats).
  • Espai en disc: Almenys 1 GB d'espai disponible.
  • Resolució de pantalla: 1440 x 900 o superior.

Passos per a la instal·lació

  1. Descarregar Power BI Desktop

  1. Visita el lloc web oficial de Power BI:

    • Power BI Desktop Download

  2. Selecciona l'opció de descàrrega:

    • Fes clic al botó "Descarregar gratis".

  3. Tria la versió adequada:

    • Selecciona la versió que correspongui al teu sistema operatiu (32 bits o 64 bits).

  1. Instal·lar Power BI Desktop

  1. Executa l'instal·lador:

    • Un cop descarregat el fitxer, fes doble clic per executar l'instal·lador.

  2. Segueix les instruccions de l'assistent d'instal·lació:

    • Accepta els termes i condicions.
    • Tria la ubicació d'instal·lació (pots deixar la ubicació per defecte).

  3. Completa la instal·lació:

    • Fes clic a "Instal·lar" i espera que el procés finalitzi.
    • Un cop completat, fes clic a "Finalitzar".

  1. Iniciar Power BI Desktop

  1. Obre Power BI Desktop:

    • Després de la instal·lació, pots trobar Power BI Desktop al menú d'inici o a l'escriptori.

  2. Configura les opcions inicials:

    • Quan obris Power BI Desktop per primera vegada, se't demanarà que iniciïs sessió amb el teu compte de Microsoft. Si no tens un compte, pots crear-ne un de nou.

  3. Explora la interfície:

    • Un cop iniciat, dedica uns minuts a explorar la interfície per familiaritzar-te amb les diferents opcions i eines disponibles.

Exercici pràctic

Objectiu

Instal·lar Power BI Desktop al teu ordinador i obrir-lo per primera vegada.

Passos

  1. Descarrega Power BI Desktop des del lloc web oficial.
  2. Executa l'instal·lador i segueix les instruccions per completar la instal·lació.
  3. Obre Power BI Desktop i inicia sessió amb el teu compte de Microsoft.

Solució

  1. Descarregar Power BI Desktop:

    • Visita Power BI Desktop Download i fes clic a "Descarregar gratis".

  2. Instal·lar Power BI Desktop:

    • Executa el fitxer descarregat, accepta els termes i condicions, i fes clic a "Instal·lar".

  3. Iniciar Power BI Desktop:

    • Obre Power BI Desktop des del menú d'inici o l'escriptori i inicia sessió amb el teu compte de Microsoft.

Errors comuns i consells

  • Error de descàrrega: Si tens problemes per descarregar el fitxer, assegura't que la teva connexió a internet és estable i que tens suficient espai en disc.
  • Problemes d'instal·lació: Si l'instal·lador no s'executa correctament, comprova que estàs utilitzant la versió adequada (32 bits o 64 bits) per al teu sistema operatiu.
  • Inici de sessió: Si no pots iniciar sessió, verifica que les teves credencials de Microsoft són correctes o crea un compte nou si no en tens.

Conclusió

Ara que has instal·lat Power BI Desktop, estàs preparat per començar a explorar les seves capacitats. En el següent tema, farem una visió general de la interfície de Power BI per ajudar-te a familiaritzar-te amb les seves funcionalitats bàsiques.
