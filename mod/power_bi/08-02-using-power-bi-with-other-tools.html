Utilitzant Power BI amb altres eines
Següent ►

En aquest tema, explorarem com Power BI es pot integrar amb altres eines per millorar la funcionalitat i l'eficiència de les vostres anàlisis de dades. Aprendrem a connectar Power BI amb aplicacions com Excel, SharePoint, Microsoft Teams, i altres serveis de dades. També veurem com utilitzar Power BI amb serveis de núvol com Azure i com integrar-lo amb altres eines de BI i anàlisi.

1. Integració amb Excel

1.1 Importació de dades des d'Excel

Power BI permet importar dades directament des de fitxers Excel. Això és útil quan ja teniu dades estructurades en fulls de càlcul.

Passos per importar dades des d'Excel:

  1. Obriu Power BI Desktop.
  2. Feu clic a "Obtenir dades" a la barra d'eines.
  3. Seleccioneu "Excel" de la llista de fonts de dades.
  4. Navegueu fins al fitxer Excel que voleu importar i feu clic a "Obrir".
  5. Seleccioneu les taules o fulls que voleu importar i feu clic a "Carregar".
# Exemple de codi M per importar dades des d'Excel
let
    Source = Excel.Workbook(File.Contents("C:\Ruta\Al\Fitxer.xlsx"), null, true),
    Taula1 = Source{[Item="Full1",Kind="Sheet"]}[Data]
in
    Taula1

1.2 Exportació de dades a Excel

També podeu exportar dades de Power BI a Excel per a una anàlisi més detallada o per compartir-les amb altres usuaris.

Passos per exportar dades a Excel:

  1. Seleccioneu la visualització o taula que voleu exportar.
  2. Feu clic a "Exportar dades" al menú de la visualització.
  3. Trieu el format d'exportació (CSV o XLSX) i feu clic a "Exportar".

2. Integració amb SharePoint

2.1 Connectant-se a llistes de SharePoint

Power BI pot connectar-se a llistes de SharePoint per importar dades directament des d'aquestes.

Passos per connectar-se a llistes de SharePoint:

  1. Feu clic a "Obtenir dades" a la barra d'eines.
  2. Seleccioneu "Llista de SharePoint en línia".
  3. Introduïu l'URL del lloc de SharePoint i feu clic a "Connectar".
  4. Seleccioneu la llista que voleu importar i feu clic a "Carregar".

2.2 Publicació de informes a SharePoint

Podeu publicar informes de Power BI directament a SharePoint per compartir-los amb el vostre equip.

Passos per publicar informes a SharePoint:

  1. Creeu o obriu un informe a Power BI Desktop.
  2. Feu clic a "Publicar" a la barra d'eines.
  3. Seleccioneu l'espai de treball de Power BI on voleu publicar l'informe.
  4. A SharePoint, afegiu una web part de Power BI i seleccioneu l'informe publicat.

3. Integració amb Microsoft Teams

3.1 Afegint informes de Power BI a Teams

Podeu afegir informes de Power BI directament a canals de Microsoft Teams per facilitar la col·laboració.

Passos per afegir informes a Teams:

  1. A Microsoft Teams, aneu al canal on voleu afegir l'informe.
  2. Feu clic a "Afegeix una pestanya" (+).
  3. Seleccioneu "Power BI" de la llista d'aplicacions.
  4. Trieu l'informe que voleu afegir i feu clic a "Desa".

4. Integració amb Azure

4.1 Connectant-se a Azure SQL Database

Power BI pot connectar-se a bases de dades SQL a Azure per importar dades.

Passos per connectar-se a Azure SQL Database:

  1. Feu clic a "Obtenir dades" a la barra d'eines.
  2. Seleccioneu "Azure" i després "Azure SQL Database".
  3. Introduïu el nom del servidor i la base de dades, i feu clic a "Connectar".
  4. Introduïu les credencials d'accés i seleccioneu les taules que voleu importar.

4.2 Utilitzant Azure Data Lake

Power BI també pot connectar-se a Azure Data Lake per accedir a grans volums de dades.

Passos per connectar-se a Azure Data Lake:

  1. Feu clic a "Obtenir dades" a la barra d'eines.
  2. Seleccioneu "Azure" i després "Azure Data Lake Storage".
  3. Introduïu l'URL del compte de Data Lake i feu clic a "Connectar".
  4. Navegueu fins a les dades que voleu importar i feu clic a "Carregar".

5. Integració amb altres eines de BI

5.1 Connectant-se a serveis de BI de tercers

Power BI pot connectar-se a altres serveis de BI com Tableau o Qlik per importar dades o compartir informes.

Passos per connectar-se a serveis de BI de tercers:

  1. Feu clic a "Obtenir dades" a la barra d'eines.
  2. Seleccioneu "Altres" i després el servei de BI al qual voleu connectar-vos.
  3. Introduïu les credencials d'accés i seleccioneu les dades que voleu importar.

Exercicis pràctics

Exercici 1: Importar dades des d'Excel

  1. Creeu un fitxer Excel amb dades de vendes.
  2. Importeu aquestes dades a Power BI.
  3. Creeu una visualització bàsica amb les dades importades.

Exercici 2: Publicar un informe a SharePoint

  1. Creeu un informe a Power BI Desktop.
  2. Publiqueu l'informe a Power BI Service.
  3. Afegiu l'informe a una pàgina de SharePoint.

Exercici 3: Afegir un informe a Microsoft Teams

  1. Publiqueu un informe a Power BI Service.
  2. Afegiu l'informe a un canal de Microsoft Teams.

Solucions

Solució a l'Exercici 1

  1. Creeu un fitxer Excel amb les següents dades:
Producte Vendes Data
A 100 01/01/2023
B 150 02/01/2023
C 200 03/01/2023
  1. Importeu les dades a Power BI seguint els passos descrits a la secció 1.1.
  2. Creeu una visualització de gràfic de barres amb les dades de vendes per producte.

Solució a l'Exercici 2

  1. Creeu un informe amb les dades importades a l'exercici 1.
  2. Publiqueu l'informe a Power BI Service seguint els passos descrits a la secció 2.2.
  3. A SharePoint, afegiu una web part de Power BI i seleccioneu l'informe publicat.

Solució a l'Exercici 3

  1. Publiqueu l'informe creat a l'exercici 2 a Power BI Service.
  2. A Microsoft Teams, afegiu una pestanya de Power BI al canal desitjat i seleccioneu l'informe publicat.

Conclusió

En aquest tema, hem après com integrar Power BI amb altres eines com Excel, SharePoint, Microsoft Teams, i serveis de núvol com Azure. Aquestes integracions permeten una major flexibilitat i col·laboració en l'anàlisi de dades, millorant l'eficiència i l'eficàcia dels vostres informes i anàlisis. A la propera secció, explorarem tècniques avançades de visualització per portar les vostres habilitats de Power BI al següent nivell.
