Components Principals d'una Infraestructura TI
Següent ►

En aquesta secció, explorarem els components essencials que constitueixen una infraestructura TI. Aquests components són fonamentals per garantir el funcionament eficient i segur de les operacions tecnològiques d'una empresa.

1. Servidors

Definició

Els servidors són equips informàtics que proporcionen serveis a altres ordinadors (clients) dins d'una xarxa. Poden ser físics o virtuals.

Tipus de Servidors

  • Servidors de fitxers: Emmagatzemen i gestionen fitxers.
  • Servidors de bases de dades: Gestionen bases de dades.
  • Servidors web: Allotgen llocs web.
  • Servidors d'aplicacions: Executen aplicacions empresarials.
  • Servidors de correu: Gestionen el correu electrònic.

Exemples

# Exemple de configuració bàsica d'un servidor web Apache en Linux
sudo apt update
sudo apt install apache2
sudo systemctl start apache2
sudo systemctl enable apache2

2. Xarxes

Definició

Les xarxes permeten la comunicació entre diferents dispositius dins d'una organització. Inclouen tant el maquinari com el programari necessaris per a la transmissió de dades.

Components de Xarxes

  • Routers: Dirigeixen el tràfic de xarxa.
  • Switches: Connecten dispositius dins d'una xarxa local.
  • Firewalls: Protegeixen la xarxa de tràfic no desitjat.
  • Punts d'accés: Permeten la connexió sense fils.

Exemples

# Exemple de configuració bàsica d'un router Cisco
enable
configure terminal
hostname Router1
interface GigabitEthernet0/0
ip address 192.168.1.1 255.255.255.0
no shutdown
exit

3. Emmagatzematge

Definició

L'emmagatzematge és el component que permet guardar i recuperar dades. Pot ser local o remot.

Tipus d'Emmagatzematge

  • Discos durs (HDD): Emmagatzematge magnètic tradicional.
  • Unitats d'estat sòlid (SSD): Emmagatzematge més ràpid basat en memòria flash.
  • Emmagatzematge en xarxa (NAS): Emmagatzematge compartit a través de la xarxa.
  • Emmagatzematge d'àrea de xarxa (SAN): Xarxa especialitzada per a emmagatzematge.

Exemples

# Exemple de muntatge d'un dispositiu d'emmagatzematge en Linux
sudo mkdir /mnt/storage
sudo mount /dev/sdb1 /mnt/storage

4. Programari de Gestió

Definició

El programari de gestió ajuda a administrar i monitoritzar els diferents components de la infraestructura TI.

Tipus de Programari de Gestió

  • Sistemes operatius: Windows Server, Linux, etc.
  • Plataformes de virtualització: VMware, Hyper-V.
  • Eines de monitoratge: Nagios, Zabbix.
  • Eines de gestió de configuració: Ansible, Puppet.

Exemples

# Exemple d'instal·lació de Nagios en Linux
sudo apt update
sudo apt install nagios4
sudo systemctl start nagios
sudo systemctl enable nagios

5. Seguretat

Definició

La seguretat és un component crític que protegeix la infraestructura TI contra amenaces internes i externes.

Components de Seguretat

  • Antivirus i antimalware: Protegeixen contra programari maliciós.
  • Sistemes de detecció d'intrusions (IDS): Detecten activitats sospitoses.
  • Sistemes de prevenció d'intrusions (IPS): Bloquegen activitats malicioses.
  • Xifratge: Protegeix les dades en trànsit i en repòs.

Exemples

# Exemple de configuració bàsica d'un tallafoc UFW en Linux
sudo apt install ufw
sudo ufw enable
sudo ufw allow ssh
sudo ufw allow http
sudo ufw allow https

6. Altres Components

Definició

A més dels components principals, hi ha altres elements que poden ser part d'una infraestructura TI.

Components Addicionals

  • Sistemes de còpia de seguretat: Protegeixen les dades amb còpies de seguretat regulars.
  • Sistemes d'alimentació ininterrompuda (SAI): Proporcionen energia en cas de fallada elèctrica.
  • Sistemes de climatització: Mantenen la temperatura adequada per als equips.

Exemples

# Exemple de configuració d'una còpia de seguretat amb rsync en Linux
rsync -avz /source/directory /backup/directory

Conclusió

Els components principals d'una infraestructura TI inclouen servidors, xarxes, emmagatzematge, programari de gestió, seguretat i altres elements addicionals. Cada component té un paper crucial en garantir que les operacions tecnològiques de l'empresa siguin eficients, segures i fiables. En els següents mòduls, aprofundirem en cadascun d'aquests components per comprendre millor com gestionar-los i mantenir-los.
