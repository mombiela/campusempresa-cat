4-3: Diccionaris

Introducció als Diccionaris

Els diccionaris són una estructura de dades fonamental en Python que permet emmagatzemar dades en parelles clau-valor. A diferència de les llistes, els diccionaris no estan ordenats i les seves claus han de ser úniques.

Creació de Diccionaris

Pots crear un diccionari utilitzant claus {} o la funció dict().

# Creació d'un diccionari buit
diccionari_buit = {}

# Creació d'un diccionari amb dades
diccionari = {
    "nom": "Joan",
    "edat": 25,
    "ciutat": "Barcelona"
}

# Utilitzant la funció dict()
diccionari_alternatiu = dict(nom="Joan", edat=25, ciutat="Barcelona")

Accés als Elements

Pots accedir als valors d'un diccionari utilitzant les seves claus.

print(diccionari["nom"])  # Output: Joan
print(diccionari["edat"])  # Output: 25

Modificació d'Elements

Pots modificar el valor associat a una clau existent o afegir una nova parella clau-valor.

# Modificar un valor existent
diccionari["edat"] = 26

# Afegir una nova parella clau-valor
diccionari["professió"] = "Enginyer"

Eliminació d'Elements

Pots eliminar elements d'un diccionari utilitzant la paraula clau del o el mètode pop().

# Utilitzant del
del diccionari["ciutat"]

# Utilitzant pop()
professio = diccionari.pop("professió")

Iteració sobre Diccionaris

Pots iterar sobre les claus, els valors o les parelles clau-valor d'un diccionari.

# Iterar sobre les claus
for clau in diccionari:
    print(clau)

# Iterar sobre els valors
for valor in diccionari.values():
    print(valor)

# Iterar sobre les parelles clau-valor
for clau, valor in diccionari.items():
    print(f"{clau}: {valor}")

Mètodes Útils

Els diccionaris tenen diversos mètodes útils per treballar amb les seves dades.

Mètode Descripció
clear() Elimina tots els elements del diccionari.
copy() Retorna una còpia superficial del diccionari.
fromkeys(seq) Crea un nou diccionari amb les claus de seq i valors establerts a None.
get(clau) Retorna el valor de la clau si existeix, sinó retorna None.
items() Retorna una vista de les parelles clau-valor.
keys() Retorna una vista de les claus del diccionari.
pop(clau) Elimina la clau i retorna el seu valor.
popitem() Elimina i retorna l'última parella clau-valor afegida.
setdefault(clau, valor) Retorna el valor de la clau si existeix, sinó afegeix la clau amb el valor donat.
update(dic) Actualitza el diccionari amb les parelles clau-valor d'un altre diccionari.
values() Retorna una vista dels valors del diccionari.

Exemple Pràctic

A continuació, es mostra un exemple pràctic que utilitza diverses operacions amb diccionaris.

# Creació d'un diccionari
contactes = {
    "Joan": "joan@example.com",
    "Maria": "maria@example.com",
    "Pere": "pere@example.com"
}

# Afegir un nou contacte
contactes["Anna"] = "anna@example.com"

# Modificar un contacte existent
contactes["Joan"] = "joan.nou@example.com"

# Eliminar un contacte
del contactes["Pere"]

# Iterar sobre els contactes
for nom, email in contactes.items():
    print(f"Nom: {nom}, Email: {email}")

# Utilitzar el mètode get
email_maria = contactes.get("Maria", "No trobat")
print(f"Email de Maria: {email_maria}")

# Utilitzar el mètode update
nous_contactes = {
    "Laura": "laura@example.com",
    "Marc": "marc@example.com"
}
contactes.update(nous_contactes)

print(contactes)

Exercicis Pràctics

  1. Creació i Accés a Diccionaris

    • Crea un diccionari que contingui informació sobre un llibre (títol, autor, any de publicació, gènere).
    • Accedeix i imprimeix el títol i l'autor del llibre.

  2. Modificació i Eliminació d'Elements

    • Afegeix una nova clau-valor al diccionari del llibre per indicar el nombre de pàgines.
    • Modifica l'any de publicació del llibre.
    • Elimina la clau del gènere del diccionari.

  3. Iteració sobre Diccionaris

    • Itera sobre les claus del diccionari del llibre i imprimeix-les.
    • Itera sobre els valors del diccionari del llibre i imprimeix-los.
    • Itera sobre les parelles clau-valor del diccionari del llibre i imprimeix-les en format "clau: valor".

Solucions

  1. Creació i Accés a Diccionaris
# Creació del diccionari
llibre = {
    "títol": "El Quixot",
    "autor": "Miguel de Cervantes",
    "any_publicació": 1605,
    "gènere": "Novel·la"
}

# Accés als elements
print(f"Títol: {llibre['títol']}")
print(f"Autor: {llibre['autor']}")
  1. Modificació i Eliminació d'Elements
# Afegir una nova clau-valor
llibre["pàgines"] = 863

# Modificar l'any de publicació
llibre["any_publicació"] = 1615

# Eliminar la clau del gènere
del llibre["gènere"]

print(llibre)
  1. Iteració sobre Diccionaris
# Iterar sobre les claus
for clau in llibre:
    print(clau)

# Iterar sobre els valors
for valor in llibre.values():
    print(valor)

# Iterar sobre les parelles clau-valor
for clau, valor in llibre.items():
    print(f"{clau}: {valor}")

Conclusió

Els diccionaris són una eina poderosa i flexible per emmagatzemar i gestionar dades en Python. Comprendre com crear, accedir, modificar i iterar sobre diccionaris és essencial per a qualsevol programador que treballi amb dades estructurades. Amb la pràctica, podràs utilitzar diccionaris per resoldre una àmplia varietat de problemes de programació de manera eficient.
