4-5: Comprensions

Introducció a les Comprensions

Les comprensions són una característica poderosa i concisa de Python que permet crear noves estructures de dades (llistes, diccionaris, conjunts) de manera eficient i llegible. Les comprensions poden substituir bucles i funcions com map() i filter() per fer el codi més net i fàcil de llegir.

Comprensions de Llistes

Les comprensions de llistes són la forma més comuna de comprensions en Python. La seva sintaxi és:

[expressió for element in iterable if condició]

Exemple Bàsic

# Crear una llista de nombres al quadrat
quadrats = [x**2 for x in range(10)]
print(quadrats)  # Sortida: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Exemple amb Condició

# Crear una llista de nombres parells al quadrat
quadrats_parells = [x**2 for x in range(10) if x % 2 == 0]
print(quadrats_parells)  # Sortida: [0, 4, 16, 36, 64]

Comprensions de Diccionaris

Les comprensions de diccionaris permeten crear diccionaris de manera similar a les comprensions de llistes.

{clau: valor for element in iterable if condició}

Exemple Bàsic

# Crear un diccionari de nombres i els seus quadrats
quadrats_dict = {x: x**2 for x in range(10)}
print(quadrats_dict)  # Sortida: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25, 6: 36, 7: 49, 8: 64, 9: 81}

Exemple amb Condició

# Crear un diccionari de nombres parells i els seus quadrats
quadrats_parells_dict = {x: x**2 for x in range(10) if x % 2 == 0}
print(quadrats_parells_dict)  # Sortida: {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}

Comprensions de Conjunts

Les comprensions de conjunts permeten crear conjunts de manera similar a les comprensions de llistes.

{expressió for element in iterable if condició}

Exemple Bàsic

# Crear un conjunt de nombres al quadrat
quadrats_set = {x**2 for x in range(10)}
print(quadrats_set)  # Sortida: {0, 1, 4, 9, 16, 25, 36, 49, 64, 81}

Exemple amb Condició

# Crear un conjunt de nombres parells al quadrat
quadrats_parells_set = {x**2 for x in range(10) if x % 2 == 0}
print(quadrats_parells_set)  # Sortida: {0, 4, 16, 36, 64}

Exercicis Pràctics

Exercici 1

Crea una llista de les primeres 10 potències de 2 utilitzant una comprensió de llista.

# Solució
potencies_de_2 = [2**x for x in range(10)]
print(potencies_de_2)  # Sortida: [1, 2, 4, 8, 16, 32, 64, 128, 256, 512]

Exercici 2

Crea un diccionari que mapeja cada lletra de la paraula "Python" al seu codi ASCII utilitzant una comprensió de diccionari.

# Solució
ascii_map = {lletra: ord(lletra) for lletra in "Python"}
print(ascii_map)  # Sortida: {'P': 80, 'y': 121, 't': 116, 'h': 104, 'o': 111, 'n': 110}

Exercici 3

Crea un conjunt de les longituds de cada paraula en una llista de paraules utilitzant una comprensió de conjunt.

# Solució
paraules = ["Python", "és", "genial"]
longituds = {len(paraula) for paraula in paraules}
print(longituds)  # Sortida: {2, 6, 7}

Errors Comuns i Consells

  1. Oblidar la condició opcional: Recorda que la condició és opcional, però pot ser molt útil per filtrar elements.
  2. Confondre la sintaxi: Assegura’t de seguir la sintaxi correcta [expressió for element in iterable if condició].
  3. Utilitzar comprensions per a tasques complexes: Les comprensions són ideals per a tasques senzilles i llegibles. Per a tasques més complexes, considera utilitzar bucles tradicionals per mantenir la claredat del codi.

Resum

Les comprensions són una eina poderosa per crear estructures de dades de manera concisa i llegible. Hem explorat comprensions de llistes, diccionaris i conjunts, i hem vist com utilitzar condicions per filtrar elements. Practicar amb exercicis ajuda a consolidar aquests conceptes i a utilitzar les comprensions de manera efectiva en el teu codi Python.
