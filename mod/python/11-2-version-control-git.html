11-2: Control de Versions amb Git

Introducció

El control de versions és una part fonamental del desenvolupament de programari modern. Git és una de les eines de control de versions més populars i utilitzades en la indústria. En aquesta secció, aprendràs els conceptes bàsics de Git, com configurar-lo, i com utilitzar-lo per gestionar el teu codi de manera eficient.

1. Què és Git?

Git és un sistema de control de versions distribuït que permet als desenvolupadors seguir els canvis en el codi, col·laborar amb altres desenvolupadors i gestionar diferents versions del projecte de manera eficient.

2. Instal·lació i Configuració Inicial

Instal·lació

Windows:

  1. Descarrega l'instal·lador des de git-scm.com.
  2. Executa l'instal·lador i segueix les instruccions.

macOS:

  1. Utilitza Homebrew: 
    brew install git

Linux:

  1. Utilitza el gestor de paquets de la teva distribució: 
    sudo apt-get install git  # Debian/Ubuntu
sudo yum install git      # CentOS/RHEL

Configuració Inicial

Després d'instal·lar Git, configura el teu nom d'usuari i correu electrònic:

git config --global user.name "El teu Nom"
git config --global user.email "el.teu.email@example.com"

3. Conceptes Bàsics de Git

  • Repository (Repo): Un directori que conté el teu projecte i un subdirectori .git amb tots els fitxers de control de versions.
  • Commit: Un instantani del teu projecte en un moment determinat.
  • Branch: Una línia de desenvolupament independent.
  • Remote: Un repositori allotjat en un servidor remot.

4. Operacions Fonamentals

Crear un Nou Repositori

git init nom-del-repositori
cd nom-del-repositori

Clonar un Repositori Existent

git clone https://github.com/usuari/nom-del-repositori.git

Afegir Fitxers al Repositori

git add fitxer.txt

Fer un Commit

git commit -m "Missatge del commit"

Veure l'Estat del Repositori

git status

Veure l'Historial de Commits

git log

5. Treballant amb Branques

Crear una Nova Branca

git branch nom-de-la-branca

Canviar de Branca

git checkout nom-de-la-branca

Crear i Canviar de Branca

git checkout -b nom-de-la-branca

6. Fusionant Branques

Fusionar una Branca a la Branca Actual

git merge nom-de-la-branca

7. Resolució de Conflictes

Quan hi ha conflictes durant una fusió, Git marca els fitxers conflictius. Has de resoldre els conflictes manualment i després fer un commit.

8. Treballant amb Repositoris Remots

Afegir un Repositori Remot

git remote add origin https://github.com/usuari/nom-del-repositori.git

Pujar Canvis al Repositori Remot

git push origin master

Baixar Canvis del Repositori Remot

git pull origin master

9. Exercicis Pràctics

Exercici 1: Crear i Gestionar un Repositori Local

  1. Crea un nou repositori local.
  2. Afegeix un fitxer README.md amb una descripció del projecte.
  3. Fes un commit del fitxer README.md.

Exercici 2: Treballar amb Branques

  1. Crea una nova branca feature-x.
  2. Canvia a la branca feature-x i crea un fitxer feature.txt.
  3. Fes un commit del fitxer feature.txt.
  4. Torna a la branca master i fusiona la branca feature-x.

Exercici 3: Treballar amb un Repositori Remot

  1. Clona un repositori remot (pots utilitzar un repositori de prova de GitHub).
  2. Crea una nova branca bugfix.
  3. Fes alguns canvis i puja'ls al repositori remot.

Conclusió

En aquesta secció, has après els conceptes bàsics de Git i com utilitzar-lo per gestionar el teu codi. Has après a crear i clonar repositoris, treballar amb branques, fusionar canvis i gestionar repositoris remots. Aquests coneixements són fonamentals per a qualsevol desenvolupador de programari i et permetran col·laborar de manera eficient amb altres desenvolupadors.
