Què és una Base de Dades?

Una base de dades és una col·lecció organitzada de dades, generalment emmagatzemada i accedida electrònicament des d'un sistema informàtic. Les bases de dades s'utilitzen per emmagatzemar, recuperar i gestionar dades de manera eficient.

Conceptes Clau

  • Dades: Fets i xifres en brut.
  • Sistema de Gestió de Bases de Dades (SGBD): Programari que interactua amb els usuaris finals, aplicacions i la pròpia base de dades per capturar i analitzar dades.
  • SQL (Llenguatge de Consulta Estructurat): Un llenguatge estàndard per gestionar i manipular bases de dades.

Tipus de Bases de Dades

  • Bases de Dades Relacionals: Utilitzen taules per emmagatzemar dades. Exemples inclouen MySQL, PostgreSQL i SQLite.
  • Bases de Dades NoSQL: Dissenyades per a models de dades específics i tenen esquemes flexibles. Exemples inclouen MongoDB, Cassandra i Redis.

Bases de Dades Relacionals vs. NoSQL

Característica Bases de Dades Relacionals Bases de Dades NoSQL
Estructura de Dades Taules Documents, Clau-Valor, Grafs, etc.
Esquema Fix Flexible
Llenguatge de Consulta SQL Varia (per exemple, MongoDB utilitza MQL)
Transaccions Compliment ACID Compliment BASE
Escalabilitat Vertical Horitzontal

Configuració d'una Base de Dades en Python

Python proporciona diverses biblioteques per interactuar amb bases de dades. Per a bases de dades relacionals, SQLite és un bon punt de partida perquè és lleuger i ve integrat amb Python.

Exemple de SQLite

  1. Connexió a una Base de Dades: Utilitza la biblioteca sqlite3 per connectar-se a una base de dades SQLite.
  2. Creació d'una Taula: Defineix una taula per emmagatzemar dades.
  3. Inserció de Dades: Afegeix dades a la taula.
  4. Consulta de Dades: Recupera dades de la taula.
import sqlite3

# Connectar-se a la base de dades SQLite (o crear-la si no existeix)
conn = sqlite3.connect('example.db')

# Crear un objecte cursor per interactuar amb la base de dades
cursor = conn.cursor()

# Crear una taula
cursor.execute('''
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    age INTEGER NOT NULL
)
''')

# Inserir dades a la taula
cursor.execute('''
INSERT INTO users (name, age) VALUES ('Alice', 30)
''')

# Confirmar la transacció
conn.commit()

# Consultar dades de la taula
cursor.execute('SELECT * FROM users')
rows = cursor.fetchall()

for row in rows:
    print(row)

# Tancar la connexió
conn.close()

Explicació

  • Connexió a la Base de Dades: sqlite3.connect('example.db') es connecta a un fitxer de base de dades anomenat example.db. Si el fitxer no existeix, es crearà.
  • Creació d'un Cursor: conn.cursor() crea un objecte cursor per executar comandes SQL.
  • Creació d'una Taula: cursor.execute('CREATE TABLE IF NOT EXISTS users (...)') crea una taula anomenada users amb les columnes id, name i age.
  • Inserció de Dades: cursor.execute('INSERT INTO users (name, age) VALUES (...)') insereix un nou registre a la taula users.
  • Confirmació de la Transacció: conn.commit() guarda els canvis a la base de dades.
  • Consulta de Dades: cursor.execute('SELECT * FROM users') recupera tots els registres de la taula users.
  • Obtenció de Resultats: cursor.fetchall() obté totes les files de la consulta executada.
  • Tancament de la Connexió: conn.close() tanca la connexió a la base de dades.

Operacions Avançades de Bases de Dades

A mesura que et familiaritzis amb les operacions bàsiques de bases de dades, pots explorar temes més avançats com:

  • Unions: Combinació de files de dues o més taules basades en una columna relacionada.
  • Índexs: Millora de la velocitat de les operacions de recuperació de dades.
  • Transaccions: Assegurament de la integritat de les dades amb operacions de confirmació i desfer.
  • ORM (Mapeig Objecte-Relacional): Utilització de biblioteques com SQLAlchemy per interactuar amb bases de dades utilitzant objectes de Python.

Exemple d'Ús de SQLAlchemy

from sqlalchemy import create_engine, Column, Integer, String
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker

# Crear un motor
engine = create_engine('sqlite:///example.db')

# Crear una classe base
Base = declarative_base()

# Definir una classe User
class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, primary_key=True)
    name = Column(String)
    age = Column(Integer)

# Crear la taula users
Base.metadata.create_all(engine)

# Crear una sessió
Session = sessionmaker(bind=engine)
session = Session()

# Afegir un nou usuari
new_user = User(name='Bob', age=25)
session.add(new_user)
session.commit()

# Consultar la base de dades
users = session.query(User).all()
for user in users:
    print(user.name, user.age)

# Tancar la sessió
session.close()

Explicació

  • Creació d'un Motor: create_engine('sqlite:///example.db') configura la connexió a la base de dades SQLite.
  • Definició d'una Classe Base: declarative_base() crea una classe base per a definicions de classes declaratives.
  • Definició d'una Classe User: La classe User representa la taula users.
  • Creació de la Taula: Base.metadata.create_all(engine) crea la taula a la base de dades.
  • Creació d'una Sessió: sessionmaker(bind=engine) crea una fàbrica de sessions.
  • Afegir un Nou Usuari: session.add(new_user) afegeix un nou usuari a la sessió.
  • Confirmació de la Transacció: session.commit() guarda els canvis a la base de dades.
  • Consulta de la Base de Dades: session.query(User).all() recupera tots els usuaris de la base de dades.
  • Tancament de la Sessió: session.close() tanca la sessió.

Conclusió

Les bases de dades són essencials per emmagatzemar i gestionar dades en aplicacions. Python proporciona biblioteques potents com sqlite3 i SQLAlchemy per interactuar amb bases de dades de manera eficient. Entenent els conceptes bàsics de les operacions de bases de dades i explorant temes avançats, pots construir aplicacions robustes i escalables.
