9-5: Integració de Bases de Dades

Introducció

En aquesta secció, aprendrem com integrar una base de dades amb una aplicació web desenvolupada amb Flask. Utilitzarem SQLite, una base de dades lleugera i fàcil d'utilitzar, juntament amb SQLAlchemy, un ORM (Object-Relational Mapping) que facilita la interacció amb la base de dades.

Objectius

  • Entendre els conceptes bàsics de bases de dades relacionals.
  • Configurar SQLAlchemy en una aplicació Flask.
  • Crear models de dades amb SQLAlchemy.
  • Realitzar operacions bàsiques de CRUD (Crear, Llegir, Actualitzar, Esborrar).

Conceptes Clau

  1. Bases de Dades Relacionals: Emmagatzemen dades en taules amb files i columnes. Les taules poden estar relacionades entre si.
  2. SQLAlchemy: Una biblioteca de Python que proporciona una interfície d'alt nivell per treballar amb bases de dades relacionals.
  3. ORM (Object-Relational Mapping): Una tècnica que permet manipular dades de bases de dades utilitzant objectes de programació.

Configuració de SQLAlchemy

Instal·lació

Primer, instal·lem SQLAlchemy i Flask-SQLAlchemy:

pip install sqlalchemy flask-sqlalchemy

Configuració Bàsica

Afegim la configuració necessària al nostre fitxer principal de Flask (app.py):

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///mydatabase.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False

db = SQLAlchemy(app)

Creació de Models

Els models representen les taules de la base de dades. Creem un model per a una taula User:

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'

Explicació del Codi

  • id: Clau primària de la taula.
  • username: Columna de tipus string amb una longitud màxima de 80 caràcters, única i no nul·la.
  • email: Columna de tipus string amb una longitud màxima de 120 caràcters, única i no nul·la.

Creació de la Base de Dades

Per crear la base de dades i les taules definides pels models, executem:

db.create_all()

Operacions CRUD

Crear

Afegim un nou usuari a la base de dades:

new_user = User(username='john_doe', email='john@example.com')
db.session.add(new_user)
db.session.commit()

Llegir

Obtenim tots els usuaris de la base de dades:

users = User.query.all()
print(users)

Obtenim un usuari per username:

user = User.query.filter_by(username='john_doe').first()
print(user)

Actualitzar

Actualitzem l'email d'un usuari:

user = User.query.filter_by(username='john_doe').first()
user.email = 'john_new@example.com'
db.session.commit()

Esborrar

Eliminem un usuari de la base de dades:

user = User.query.filter_by(username='john_doe').first()
db.session.delete(user)
db.session.commit()

Exercici Pràctic

Exercici: Crea un model per a una taula Post amb les següents columnes: id, title, content, user_id. Implementa les operacions CRUD per a aquesta taula.

Solució:

class Post(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(120), nullable=False)
    content = db.Column(db.Text, nullable=False)
    user_id = db.Column(db.Integer, db.ForeignKey('user.id'), nullable=False)

    def __repr__(self):
        return f'<Post {self.title}>'

# Crear la taula
db.create_all()

# Crear un nou post
new_post = Post(title='My First Post', content='This is the content of my first post', user_id=1)
db.session.add(new_post)
db.session.commit()

# Llegir tots els posts
posts = Post.query.all()
print(posts)

# Actualitzar un post
post = Post.query.filter_by(title='My First Post').first()
post.content = 'Updated content'
db.session.commit()

# Esborrar un post
post = Post.query.filter_by(title='My First Post').first()
db.session.delete(post)
db.session.commit()

Conclusió

En aquesta secció, hem après com integrar una base de dades amb una aplicació Flask utilitzant SQLAlchemy. Hem vist com crear models de dades, realitzar operacions CRUD i configurar correctament la base de dades. Aquestes habilitats són fonamentals per desenvolupar aplicacions web dinàmiques i interactives.

Resum

  • Hem configurat SQLAlchemy en una aplicació Flask.
  • Hem creat models de dades per representar taules de la base de dades.
  • Hem realitzat operacions bàsiques de CRUD.
  • Hem practicat amb un exercici pràctic per reforçar els conceptes apresos.
