10-2: Tècniques de Depuració

La depuració és una part essencial del desenvolupament de programari. Ens ajuda a identificar i corregir errors en el nostre codi. En aquesta secció, explorarem diverses tècniques de depuració que et permetran trobar i solucionar problemes de manera eficient.

Objectius d'Aprenentatge

  • Comprendre la importància de la depuració.
  • Aprendre a utilitzar tècniques bàsiques de depuració.
  • Familiaritzar-se amb les eines de depuració disponibles en Python.
  • Aplicar tècniques de depuració en exemples pràctics.

1. Importància de la Depuració

La depuració és crucial per assegurar que el codi funcioni correctament i sigui lliure d'errors. Els errors poden ser de diversos tipus:

  • Errors de sintaxi: Errors en l'escriptura del codi que impedeixen que el programa s'executi.
  • Errors lògics: Errors en la lògica del programa que fan que no es comporti com s'esperava.
  • Errors d'execució: Errors que es produeixen durant l'execució del programa, com ara divisions per zero o accés a variables no definides.

2. Tècniques Bàsiques de Depuració

2.1. Impressió de Missatges

Una de les tècniques més senzilles és utilitzar la funció print() per mostrar el valor de les variables en diferents punts del programa.

def suma(a, b):
    print(f"a: {a}, b: {b}")  # Imprimir els valors de les variables
    return a + b

resultat = suma(3, 4)
print(f"Resultat: {resultat}")

2.2. Assert

L'ús de l'assert és una manera de verificar que una condició sigui certa durant l'execució del programa. Si la condició no es compleix, el programa llança una excepció.

def dividir(a, b):
    assert b != 0, "El divisor no pot ser zero"
    return a / b

resultat = dividir(10, 2)
print(f"Resultat: {resultat}")

3. Eines de Depuració en Python

3.1. pdb: Python Debugger

pdb és el depurador integrat de Python. Permet executar el codi pas a pas, inspeccionar variables i establir punts de ruptura.

3.1.1. Iniciar pdb

Per iniciar pdb, pots afegir la següent línia al teu codi:

import pdb; pdb.set_trace()
3.1.2. Comandes bàsiques de pdb
Comanda Descripció
n Executa la següent línia de codi
c Continua l'execució fins al següent punt de ruptura
l Mostra el codi al voltant de la línia actual
p Imprimeix el valor d'una variable
q Surt del depurador
Exemple d'ús de pdb
def multiplicar(a, b):
    import pdb; pdb.set_trace()
    resultat = a * b
    return resultat

resultat = multiplicar(5, 6)
print(f"Resultat: {resultat}")

4. Exercicis Pràctics

Exercici 1: Depuració amb print()

  1. Escriu una funció que calculi la mitjana d'una llista de nombres.
  2. Utilitza print() per depurar i assegurar-te que la funció funciona correctament.
def calcular_mitjana(llista):
    suma = 0
    for num in llista:
        suma += num
    mitjana = suma / len(llista)
    return mitjana

# Prova la funció
llista = [10, 20, 30, 40, 50]
print(f"Mitjana: {calcular_mitjana(llista)}")

Exercici 2: Depuració amb pdb

  1. Escriu una funció que calculi el factorial d'un nombre.
  2. Utilitza pdb per depurar la funció i assegurar-te que funciona correctament.
def factorial(n):
    import pdb; pdb.set_trace()
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

# Prova la funció
print(f"Factorial de 5: {factorial(5)}")

Solucions

Solució Exercici 1

def calcular_mitjana(llista):
    suma = 0
    for num in llista:
        print(f"Afegint {num} a la suma")
        suma += num
    mitjana = suma / len(llista)
    print(f"Suma: {suma}, Longitud: {len(llista)}, Mitjana: {mitjana}")
    return mitjana

# Prova la funció
llista = [10, 20, 30, 40, 50]
print(f"Mitjana: {calcular_mitjana(llista)}")

Solució Exercici 2

def factorial(n):
    import pdb; pdb.set_trace()
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

# Prova la funció
print(f"Factorial de 5: {factorial(5)}")

Conclusió

En aquesta secció, hem après la importància de la depuració i hem explorat diverses tècniques per depurar el codi en Python. Hem vist com utilitzar print(), assert i el depurador integrat pdb per trobar i corregir errors. La pràctica d'aquestes tècniques t'ajudarà a convertir-te en un programador més eficient i efectiu.
