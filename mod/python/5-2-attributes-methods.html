5-2: Atributs i Mètodes

Introducció

En aquest tema, explorarem els atributs i els mètodes en les classes de Python. Els atributs són variables que pertanyen a una classe o a una instància d'una classe, mentre que els mètodes són funcions definides dins d'una classe que poden operar sobre els seus atributs.

Atributs

Els atributs es poden dividir en dos tipus principals:

  1. Atributs d'instància: Pertanyen a una instància específica de la classe.
  2. Atributs de classe: Són compartits per totes les instàncies de la classe.

Atributs d'Instància

Els atributs d'instància es defineixen dins del mètode __init__ i són prefixats amb self.

class Cotxe:
    def __init__(self, marca, model):
        self.marca = marca
        self.model = model

# Crear una instància de la classe Cotxe
cotxe1 = Cotxe("Toyota", "Corolla")
cotxe2 = Cotxe("Honda", "Civic")

print(cotxe1.marca)  # Sortida: Toyota
print(cotxe2.model)  # Sortida: Civic

Atributs de Classe

Els atributs de classe es defineixen fora de qualsevol mètode i són accessibles per totes les instàncies de la classe.

class Cotxe:
    rodes = 4  # Atribut de classe

    def __init__(self, marca, model):
        self.marca = marca
        self.model = model

print(Cotxe.rodes)  # Sortida: 4

cotxe1 = Cotxe("Toyota", "Corolla")
print(cotxe1.rodes)  # Sortida: 4

Mètodes

Els mètodes són funcions definides dins d'una classe que poden operar sobre els atributs de la classe o de la instància.

Mètodes d'Instància

Els mètodes d'instància són els més comuns i sempre tenen self com a primer paràmetre.

class Cotxe:
    def __init__(self, marca, model):
        self.marca = marca
        self.model = model

    def descriu(self):
        return f"{self.marca} {self.model}"

cotxe1 = Cotxe("Toyota", "Corolla")
print(cotxe1.descriu())  # Sortida: Toyota Corolla

Mètodes de Classe

Els mètodes de classe utilitzen el decorador @classmethod i tenen cls com a primer paràmetre.

class Cotxe:
    rodes = 4

    def __init__(self, marca, model):
        self.marca = marca
        self.model = model

    @classmethod
    def descriu_rodes(cls):
        return f"Tots els cotxes tenen {cls.rodes} rodes"

print(Cotxe.descriu_rodes())  # Sortida: Tots els cotxes tenen 4 rodes

Mètodes Estàtics

Els mètodes estàtics utilitzen el decorador @staticmethod i no tenen self ni cls com a paràmetre.

class Cotxe:
    def __init__(self, marca, model):
        self.marca = marca
        self.model = model

    @staticmethod
    def salutacio():
        return "Hola, sóc un cotxe!"

print(Cotxe.salutacio())  # Sortida: Hola, sóc un cotxe!

Exercicis Pràctics

Exercici 1

Crea una classe Persona amb atributs d'instància nom i edat, i un mètode d'instància saluda que imprimeixi un missatge de salutació.

class Persona:
    def __init__(self, nom, edat):
        self.nom = nom
        self.edat = edat

    def saluda(self):
        print(f"Hola, em dic {self.nom} i tinc {self.edat} anys.")

# Solució
persona1 = Persona("Joan", 30)
persona1.saluda()  # Sortida: Hola, em dic Joan i tinc 30 anys.

Exercici 2

Afegeix un atribut de classe espècie a la classe Persona i un mètode de classe descriu_espècie que imprimeixi el valor de espècie.

class Persona:
    espècie = "Humà"

    def __init__(self, nom, edat):
        self.nom = nom
        self.edat = edat

    def saluda(self):
        print(f"Hola, em dic {self.nom} i tinc {self.edat} anys.")

    @classmethod
    def descriu_espècie(cls):
        print(f"Tots som de l'espècie {cls.espècie}")

# Solució
Persona.descriu_espècie()  # Sortida: Tots som de l'espècie Humà
