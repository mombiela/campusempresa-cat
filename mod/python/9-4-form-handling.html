9-4: Gestió de Formularis

Introducció

En aquesta secció, aprendrem com gestionar formularis en una aplicació web desenvolupada amb Flask. Els formularis són una part essencial de moltes aplicacions web, ja que permeten als usuaris enviar dades al servidor. Aprendrem a crear formularis, validar les dades i processar-les de manera segura.

Conceptes Clau

  • Creació de Formularis HTML: Com estructurar formularis en HTML.
  • Gestió de Peticions POST: Com rebre i processar dades enviades a través de formularis.
  • Validació de Dades: Com assegurar-se que les dades enviades són vàlides abans de processar-les.
  • Flask-WTF: Una extensió de Flask que facilita la creació i validació de formularis.

Creació de Formularis HTML

Els formularis HTML són la manera més bàsica de permetre als usuaris enviar dades al servidor. Aquí tens un exemple senzill d'un formulari HTML:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Formulari de Contacte</title>
</head>
<body>
    <form action="/submit" method="post">
        <label for="name">Nom:</label>
        <input type="text" id="name" name="name" required>
        <br>
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
        <br>
        <label for="message">Missatge:</label>
        <textarea id="message" name="message" required></textarea>
        <br>
        <button type="submit">Enviar</button>
    </form>
</body>
</html>

Gestió de Peticions POST

Quan l'usuari envia el formulari, les dades es poden processar en el servidor utilitzant Flask. Aquí tens un exemple de com gestionar una petició POST en Flask:

from flask import Flask, request, render_template

app = Flask(__name__)

@app.route('/submit', methods=['POST'])
def submit():
    name = request.form['name']
    email = request.form['email']
    message = request.form['message']
    # Processar les dades aquí
    return f'Gràcies, {name}! Hem rebut el teu missatge.'

@app.route('/')
def index():
    return render_template('form.html')

if __name__ == '__main__':
    app.run(debug=True)

Validació de Dades

És important validar les dades enviades pels usuaris per assegurar-se que són correctes i segures. Flask-WTF és una extensió que facilita aquesta tasca.

Instal·lació de Flask-WTF

pip install flask-wtf

Creació d'un Formulari amb Flask-WTF

from flask import Flask, render_template, request
from flask_wtf import FlaskForm
from wtforms import StringField, TextAreaField, SubmitField
from wtforms.validators import DataRequired, Email

app = Flask(__name__)
app.secret_key = 's3cr3t'

class ContactForm(FlaskForm):
    name = StringField('Nom', validators=[DataRequired()])
    email = StringField('Email', validators=[DataRequired(), Email()])
    message = TextAreaField('Missatge', validators=[DataRequired()])
    submit = SubmitField('Enviar')

@app.route('/', methods=['GET', 'POST'])
def index():
    form = ContactForm()
    if form.validate_on_submit():
        name = form.name.data
        email = form.email.data
        message = form.message.data
        # Processar les dades aquí
        return f'Gràcies, {name}! Hem rebut el teu missatge.'
    return render_template('form.html', form=form)

if __name__ == '__main__':
    app.run(debug=True)

Plantilla HTML amb Flask-WTF

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Formulari de Contacte</title>
</head>
<body>
    <form method="post">
        {{ form.hidden_tag() }}
        <p>
            {{ form.name.label }}<br>
            {{ form.name(size=32) }}<br>
            {% for error in form.name.errors %}
                <span style="color: red;">[{{ error }}]</span>
            {% endfor %}
        </p>
        <p>
            {{ form.email.label }}<br>
            {{ form.email(size=32) }}<br>
            {% for error in form.email.errors %}
                <span style="color: red;">[{{ error }}]</span>
            {% endfor %}
        </p>
        <p>
            {{ form.message.label }}<br>
            {{ form.message(rows=4, cols=32) }}<br>
            {% for error in form.message.errors %}
                <span style="color: red;">[{{ error }}]</span>
            {% endfor %}
        </p>
        <p>{{ form.submit() }}</p>
    </form>
</body>
</html>

Exercici Pràctic

Objectiu: Crear un formulari de registre d'usuari que demani el nom d'usuari, la contrasenya i la confirmació de la contrasenya. Validar que les contrasenyes coincideixin.

Pas 1: Crear el Formulari

from flask import Flask, render_template, request
from flask_wtf import FlaskForm
from wtforms import StringField, PasswordField, SubmitField
from wtforms.validators import DataRequired, EqualTo

app = Flask(__name__)
app.secret_key = 's3cr3t'

class RegistrationForm(FlaskForm):
    username = StringField('Nom d\'usuari', validators=[DataRequired()])
    password = PasswordField('Contrasenya', validators=[DataRequired()])
    confirm_password = PasswordField('Confirma la Contrasenya', validators=[DataRequired(), EqualTo('password')])
    submit = SubmitField('Registrar-se')

@app.route('/register', methods=['GET', 'POST'])
def register():
    form = RegistrationForm()
    if form.validate_on_submit():
        username = form.username.data
        password = form.password.data
        # Processar les dades aquí
        return f'Gràcies per registrar-te, {username}!'
    return render_template('register.html', form=form)

if __name__ == '__main__':
    app.run(debug=True)
