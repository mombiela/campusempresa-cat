Tema 1.3: Sintaxi Bàsica i Hola Món

Objectius

En aquest tema, aprendràs:

  • La sintaxi bàsica de Python.
  • Com escriure i executar el teu primer programa en Python: "Hola Món".
  • Conceptes fonamentals com comentaris, indentació i estructura de codi.

1. Sintaxi Bàsica de Python

Python és conegut per la seva sintaxi clara i llegible. A diferència d'altres llenguatges de programació, Python utilitza l'indentació per definir blocs de codi.

Comentaris

Els comentaris són línies de codi que no s'executen. S'utilitzen per explicar el codi i fer-lo més llegible.

  • Comentaris d'una sola línia:
# Això és un comentari d'una sola línia
  • Comentaris de múltiples línies:
"""
Això és un comentari
de múltiples línies
"""

Indentació

Python utilitza l'indentació per definir blocs de codi. És important utilitzar la mateixa quantitat d'espais o tabuladors per a cada nivell d'indentació.

Exemple:

if True:
    print("Això està dins del bloc if")
    if True:
        print("Això està dins del bloc if dins d'un altre if")

2. El teu Primer Programa: "Hola Món"

El primer programa que escriuràs en qualsevol llenguatge de programació és el clàssic "Hola Món". En Python, és molt senzill.

Exemple de Codi

print("Hola Món")

Explicació del Codi

  • print(): És una funció integrada en Python que s'utilitza per mostrar missatges a la pantalla.
  • "Hola Món": És una cadena de text (string) que es passarà com a argument a la funció print().

3. Executant el teu Primer Programa

Pas 1: Crear un Fitxer Python

Crea un fitxer nou amb l'extensió .py, per exemple, hola_mon.py.

Pas 2: Escriure el Codi

Obre el fitxer amb el teu editor de text preferit i escriu el codi següent:

print("Hola Món")

Pas 3: Executar el Programa

Obre una terminal o línia de comandes i navega fins al directori on has guardat el fitxer hola_mon.py. Executa el següent comandament:

python hola_mon.py

4. Exercicis Pràctics

Exercici 1: Comentaris i Indentació

Escriu un programa que contingui comentaris d'una sola línia i de múltiples línies. Utilitza la indentació per crear un bloc de codi dins d'una condició if.

# Comentari d'una sola línia

"""
Comentari de múltiples línies
que explica el següent bloc de codi
"""

if True:
    print("Això està dins del bloc if")

Exercici 2: Hola Món Personalitzat

Modifica el programa "Hola Món" per imprimir el teu nom.

print("Hola, [El teu Nom]")

5. Solucions als Exercicis

Solució a l'Exercici 1

# Comentari d'una sola línia

"""
Comentari de múltiples línies
que explica el següent bloc de codi
"""

if True:
    print("Això està dins del bloc if")

Solució a l'Exercici 2

print("Hola, Joan")

Resum

En aquest tema, has après:

  • La sintaxi bàsica de Python, incloent comentaris i indentació.
  • Com escriure i executar el teu primer programa en Python: "Hola Món".
  • Has practicat amb exercicis per reforçar els conceptes apresos.

En el següent tema, explorarem les variables i els tipus de dades en Python.
