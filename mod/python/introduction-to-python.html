Què és Python?

Python és un llenguatge de programació d'alt nivell i interpretat conegut per la seva simplicitat i llegibilitat. És àmpliament utilitzat en diversos dominis com el desenvolupament web, l'anàlisi de dades, la intel·ligència artificial, la computació científica i més.

  • D'alt nivell: Python abstrau la major part dels detalls complexos del maquinari de l'ordinador.
  • Interpretat: El codi Python s'executa línia per línia, cosa que facilita la depuració.
  • Llegibilitat: La sintaxi de Python està dissenyada per ser fàcil de llegir i escriure.

Per què aprendre Python?

  • Facilitat d'aprenentatge: Python té una sintaxi simple que imita el llenguatge natural.
  • Versatilitat: Python es pot utilitzar per al desenvolupament web, la ciència de dades, l'automatització i més.
  • Suport de la comunitat: Python té una comunitat gran i activa, la qual cosa significa que hi ha molts recursos i biblioteques disponibles.

Configuració de Python

Instal·lació de Python

  1. Descarregar Python: Ves al lloc web oficial de Python i descarrega l'última versió.
  2. Instal·lar Python: Segueix les instruccions d'instal·lació per al teu sistema operatiu.

Configuració d'un IDE

Un Entorn de Desenvolupament Integrat (IDE) pot fer que programar en Python sigui més fàcil. Algunes opcions populars inclouen:

  • PyCharm
  • VS Code
  • Jupyter Notebook

Escriure el teu primer programa en Python

Comencem amb un programa senzill que imprimeix "Hola, món!" a la pantalla.

print("Hola, món!")
  • Explicació: La funció print mostra text a la consola. El text que es vol imprimir està entre cometes.

Sintaxi bàsica

Variables i tipus de dades

Python admet diversos tipus de dades, incloent-hi enters, flotants, cadenes de text i booleans.

# Enter
edat = 25

# Flotant
alçada = 5.9

# Cadena de text
nom = "Alice"

# Booleà
és_estudiant = True
  • Explicació: Les variables s'utilitzen per emmagatzemar dades. El tipus de dada d'una variable es determina pel valor assignat.

Operadors bàsics

Python admet operadors aritmètics, de comparació i lògics.

# Operadors aritmètics
suma = 5 + 3
resta = 5 - 3
multiplicació = 5 * 3
divisió = 5 / 3

# Operadors de comparació
igual = (5 == 3)
diferent = (5 != 3)
més_gran_que = (5 > 3)
menys_que = (5 < 3)

# Operadors lògics
operador_i = (5 > 3) i (3 > 1)
operador_o = (5 > 3) o (3 < 1)
operador_no = no(5 > 3)
  • Explicació: Els operadors s'utilitzen per realitzar operacions sobre variables i valors.

Estructures de control

Sentències If

edat = 18
if edat >= 18:
    print("Ets un adult.")
else:
    print("Ets menor d'edat.")
  • Explicació: La sentència if permet executar codi basat en una condició.

Bucles

Bucle For
for i in range(5):
    print(i)
  • Explicació: El bucle for itera sobre una seqüència (en aquest cas, nombres del 0 al 4).
Bucle While
compte = 0
while compte < 5:
    print(compte)
    compte += 1
  • Explicació: El bucle while continua executant-se mentre la condició sigui certa.

Funcions

Les funcions són blocs de codi reutilitzables que realitzen una tasca específica.

def saluda(nom):
    print(f"Hola, {nom}!")

saluda("Alice")
  • Explicació: La paraula clau def s'utilitza per definir una funció. La funció saluda pren un paràmetre nom i imprimeix una salutació.

Conclusió

En aquesta introducció, hem cobert els conceptes bàsics de Python, incloent-hi la seva sintaxi, variables, operadors, estructures de control i funcions. La simplicitat i llegibilitat de Python el converteixen en una excel·lent opció per a principiants. A mesura que avancem en el curs, aprofundirem en temes més avançats i aplicacions de Python.
