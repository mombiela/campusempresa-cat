Tema 2-3: Flux de Control: `break`, `continue` i `pass`

En aquest tema, explorarem tres instruccions de control molt útils en Python: break, continue i pass. Aquestes instruccions permeten controlar el flux d'execució dels bucles i altres estructures de control de manera més precisa.

`break`

La instrucció break s'utilitza per sortir d'un bucle abans que aquest hagi completat totes les seves iteracions. Quan s'executa break, el bucle es trenca immediatament i el control del programa passa a la següent instrucció després del bucle.

Exemple:

for i in range(10):
    if i == 5:
        break
    print(i)

Explicació:

  • Aquest codi imprimeix els números del 0 al 4.
  • Quan i és igual a 5, la instrucció break es troba i el bucle es trenca.

`continue`

La instrucció continue s'utilitza per saltar l'iteració actual d'un bucle i passar a la següent iteració. Quan s'executa continue, el codi restant dins del bucle per a l'iteració actual es salta i el bucle continua amb la següent iteració.

Exemple:

for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

Explicació:

  • Aquest codi imprimeix els números imparells del 0 al 9.
  • Quan i és parell, la instrucció continue es troba i el bucle salta a la següent iteració.

`pass`

La instrucció pass s'utilitza com a marcador de posició. No fa res quan s'executa. És útil en llocs on es requereix una instrucció sintàcticament, però no es necessita cap acció o codi específic.

Exemple:

for i in range(10):
    if i % 2 == 0:
        pass
    else:
        print(i)

Explicació:

  • Aquest codi imprimeix els números imparells del 0 al 9.
  • Quan i és parell, la instrucció pass es troba, però no fa res, i el bucle continua amb la següent iteració.

Comparació de `break`, `continue` i `pass`

Instrucció Funció Exemple d'Ús
break Trenca el bucle i passa a la següent instrucció fora del bucle Sortir d'un bucle quan es compleix una condició específica
continue Salta l'iteració actual i passa a la següent iteració del bucle Saltar iteracions en un bucle quan es compleix una condició específica
pass No fa res, s'utilitza com a marcador de posició Crear estructures de codi que encara no estan implementades

Exercicis Pràctics

Exercici 1: Utilitzant break

Escriu un programa que llegeixi números del 1 al 10 i es trenqui quan trobi el número 7.

for i in range(1, 11):
    if i == 7:
        break
    print(i)

Exercici 2: Utilitzant continue

Escriu un programa que imprimeixi tots els números del 1 al 10, excepte els múltiples de 3.

for i in range(1, 11):
    if i % 3 == 0:
        continue
    print(i)

Exercici 3: Utilitzant pass

Escriu un programa que recorri els números del 1 al 10 i utilitzi pass per als números parells, imprimint només els números imparells.

for i in range(1, 11):
    if i % 2 == 0:
        pass
    else:
        print(i)

Solucions als Exercicis

Solució Exercici 1

for i in range(1, 11):
    if i == 7:
        break
    print(i)

Solució Exercici 2

for i in range(1, 11):
    if i % 3 == 0:
        continue
    print(i)

Solució Exercici 3

for i in range(1, 11):
    if i % 2 == 0:
        pass
    else:
        print(i)

Resum

En aquest tema, hem après com utilitzar les instruccions break, continue i pass per controlar el flux d'execució dels bucles en Python. Hem vist exemples pràctics i hem practicat amb exercicis per reforçar els conceptes apresos. Aquestes instruccions són eines poderoses que ens permeten escriure codi més flexible i eficient.
