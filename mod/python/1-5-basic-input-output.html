1.5 Entrada i Sortida Bàsiques

En aquest tema, aprendrem com gestionar l'entrada i la sortida de dades en Python. Això inclou com llegir dades de l'usuari i com mostrar informació a la pantalla.

Objectius d'Aprenentatge

  • Comprendre com utilitzar la funció input() per obtenir dades de l'usuari.
  • Aprendre a utilitzar la funció print() per mostrar informació a la pantalla.
  • Manipular cadenes de text per formatar la sortida.

Funció `input()`

La funció input() s'utilitza per obtenir dades de l'usuari. Aquesta funció sempre retorna una cadena de text.

Sintaxi

variable = input("Missatge per a l'usuari: ")

Exemple

nom = input("Quin és el teu nom? ")
print("Hola, " + nom + "!")

Funció `print()`

La funció print() s'utilitza per mostrar informació a la pantalla. Pot acceptar múltiples arguments i té diverses opcions per formatar la sortida.

Sintaxi

print(valors, sep=' ', end='\n', file=sys.stdout, flush=False)
  • valors: Els valors que es volen imprimir.
  • sep: El separador entre els valors (per defecte és un espai).
  • end: El caràcter que es mostra al final (per defecte és un salt de línia).
  • file: L'objecte de fitxer on s'escriu la sortida (per defecte és la consola).
  • flush: Si es buida el buffer de sortida (per defecte és False).

Exemple

print("Hola", "món", sep=", ", end="!\n")

Formatació de Cadenes

Python ofereix diverses maneres de formatar cadenes de text per a la sortida.

Operador %

nom = "Anna"
edat = 25
print("El seu nom és %s i té %d anys." % (nom, edat))

Mètode str.format()

nom = "Anna"
edat = 25
print("El seu nom és {} i té {} anys.".format(nom, edat))

F-strings (a partir de Python 3.6)

nom = "Anna"
edat = 25
print(f"El seu nom és {nom} i té {edat} anys.")

Exercicis Pràctics

Exercici 1: Entrada i Sortida Bàsica

Escriu un programa que demani a l'usuari el seu nom i edat, i després mostri un missatge amb aquesta informació.

# Solució
nom = input("Quin és el teu nom? ")
edat = input("Quants anys tens? ")
print(f"Hola, {nom}! Tens {edat} anys.")

Exercici 2: Formatació de Cadenes

Escriu un programa que demani a l'usuari el seu nom, edat i ciutat, i després mostri un missatge formatat amb aquesta informació.

# Solució
nom = input("Quin és el teu nom? ")
edat = input("Quants anys tens? ")
ciutat = input("On vius? ")
print(f"El teu nom és {nom}, tens {edat} anys i vius a {ciutat}.")

Errors Comuns

  • No convertir les dades d'entrada: Recorda que input() sempre retorna una cadena. Si necessites un altre tipus de dada (com un enter), has de convertir-lo explícitament.

    edat = int(input("Quants anys tens? "))

  • Oblidar el format de la cadena: Quan utilitzes formatació de cadenes, assegura't de seguir la sintaxi correcta per evitar errors.

Resum

En aquest tema, hem après com utilitzar les funcions input() i print() per gestionar l'entrada i la sortida de dades en Python. També hem vist diverses maneres de formatar cadenes de text per mostrar informació de manera clara i llegible. Amb aquests coneixements, estàs preparat per crear programes interactius que puguin comunicar-se amb l'usuari.
