8-3: Matplotlib per a la Visualització de Dades

Introducció a Matplotlib

Matplotlib és una llibreria de Python per a la generació de gràfics i visualitzacions de dades. És molt utilitzada en la comunitat científica i d'anàlisi de dades per la seva capacitat de crear gràfics de qualitat professional de manera senzilla.

Instal·lació de Matplotlib

Abans de començar a utilitzar Matplotlib, necessitem instal·lar la llibreria. Això es pot fer fàcilment utilitzant pip:

pip install matplotlib

Creació d'un Gràfic Simple

Comencem amb un exemple bàsic de com crear un gràfic de línies.

import matplotlib.pyplot as plt

# Dades per al gràfic
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Crear el gràfic
plt.plot(x, y)

# Afegir títol i etiquetes
plt.title('Gràfic de Línies Simple')
plt.xlabel('X')
plt.ylabel('Y')

# Mostrar el gràfic
plt.show()

Explicació del Codi

  1. Importació de Matplotlib: import matplotlib.pyplot as plt
  2. Dades: Definim les llistes x i y amb les dades que volem representar.
  3. Creació del Gràfic: Utilitzem plt.plot(x, y) per crear el gràfic de línies.
  4. Títol i Etiquetes: Afegim un títol i etiquetes als eixos amb plt.title(), plt.xlabel(), i plt.ylabel().
  5. Mostrar el Gràfic: Finalment, utilitzem plt.show() per mostrar el gràfic.

Tipus de Gràfics

Matplotlib permet crear diversos tipus de gràfics. A continuació, veurem alguns dels més comuns.

Gràfic de Barres

import matplotlib.pyplot as plt

# Dades per al gràfic
categories = ['A', 'B', 'C', 'D']
values = [4, 7, 1, 8]

# Crear el gràfic de barres
plt.bar(categories, values)

# Afegir títol i etiquetes
plt.title('Gràfic de Barres')
plt.xlabel('Categories')
plt.ylabel('Valors')

# Mostrar el gràfic
plt.show()

Gràfic de Dispersió

import matplotlib.pyplot as plt

# Dades per al gràfic
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Crear el gràfic de dispersió
plt.scatter(x, y)

# Afegir títol i etiquetes
plt.title('Gràfic de Dispersió')
plt.xlabel('X')
plt.ylabel('Y')

# Mostrar el gràfic
plt.show()

Gràfic de Pastís

import matplotlib.pyplot as plt

# Dades per al gràfic
labels = ['A', 'B', 'C', 'D']
sizes = [15, 30, 45, 10]

# Crear el gràfic de pastís
plt.pie(sizes, labels=labels, autopct='%1.1f%%')

# Afegir títol
plt.title('Gràfic de Pastís')

# Mostrar el gràfic
plt.show()

Personalització de Gràfics

Matplotlib ofereix moltes opcions per personalitzar els gràfics. A continuació, veurem algunes de les més útils.

Canviar Colors i Estils de Línia

import matplotlib.pyplot as plt

# Dades per al gràfic
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Crear el gràfic amb color i estil de línia personalitzats
plt.plot(x, y, color='green', linestyle='--', marker='o')

# Afegir títol i etiquetes
plt.title('Gràfic Personalitzat')
plt.xlabel('X')
plt.ylabel('Y')

# Mostrar el gràfic
plt.show()

Afegir Llegenda

import matplotlib.pyplot as plt

# Dades per al gràfic
x = [1, 2, 3, 4, 5]
y1 = [2, 3, 5, 7, 11]
y2 = [1, 4, 6, 8, 10]

# Crear els gràfics
plt.plot(x, y1, label='Sèrie 1')
plt.plot(x, y2, label='Sèrie 2')

# Afegir títol, etiquetes i llegenda
plt.title('Gràfic amb Llegenda')
plt.xlabel('X')
plt.ylabel('Y')
plt.legend()

# Mostrar el gràfic
plt.show()

Exercicis Pràctics

Exercici 1: Gràfic de Línies

Crea un gràfic de línies amb les següents dades:

  • x = [0, 1, 2, 3, 4, 5]
  • y = [0, 1, 4, 9, 16, 25]

Solució:

import matplotlib.pyplot as plt

x = [0, 1, 2, 3, 4, 5]
y = [0, 1, 4, 9, 16, 25]

plt.plot(x, y)
plt.title('Gràfic de Línies')
plt.xlabel('X')
plt.ylabel('Y')
plt.show()

Exercici 2: Gràfic de Barres

Crea un gràfic de barres amb les següents dades:

  • categories = ['E', 'F', 'G', 'H']
  • values = [5, 6, 2, 9]

Solució:

import matplotlib.pyplot as plt

categories = ['E', 'F', 'G', 'H']
values = [5, 6, 2, 9]

plt.bar(categories, values)
plt.title('Gràfic de Barres')
plt.xlabel('Categories')
plt.ylabel('Valors')
plt.show()
