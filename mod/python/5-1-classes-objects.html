5-1: Classes i Objectes

Introducció

La Programació Orientada a Objectes (POO) és un paradigma de programació que utilitza "objectes" i "classes" per organitzar el codi. Aquest mòdul t'introduirà als conceptes fonamentals de la POO en Python, començant per les classes i els objectes.

Conceptes Clau

  1. Classe: Una plantilla per crear objectes. Defineix un conjunt d'atributs i mètodes que els objectes creats a partir de la classe tindran.
  2. Objecte: Una instància d'una classe. Cada objecte pot tenir diferents valors per als atributs definits en la seva classe.
  3. Atribut: Una variable que pertany a una classe o a un objecte.
  4. Mètode: Una funció que pertany a una classe o a un objecte.

Creació d'una Classe

Per crear una classe en Python, utilitzem la paraula clau class seguida del nom de la classe i dos punts. Dins de la classe, podem definir atributs i mètodes.

class Persona:
    def __init__(self, nom, edat):
        self.nom = nom
        self.edat = edat

    def saludar(self):
        print(f"Hola, em dic {self.nom} i tinc {self.edat} anys.")

Explicació del Codi

  • class Persona:: Defineix una nova classe anomenada Persona.
  • def __init__(self, nom, edat):: El mètode __init__ és un constructor que s'executa quan es crea una nova instància de la classe. self és una referència a l'objecte actual.
  • self.nom = nom: Assigna el valor del paràmetre nom a l'atribut nom de l'objecte.
  • self.edat = edat: Assigna el valor del paràmetre edat a l'atribut edat de l'objecte.
  • def saludar(self):: Defineix un mètode anomenat saludar que imprimeix un missatge de salutació.

Creació d'Objectes

Per crear un objecte d'una classe, simplement cridem la classe com si fos una funció.

persona1 = Persona("Joan", 30)
persona2 = Persona("Anna", 25)

Utilització d'Objectes

Podem accedir als atributs i mètodes d'un objecte utilitzant la notació de punt.

print(persona1.nom)  # Sortida: Joan
print(persona2.edat)  # Sortida: 25

persona1.saludar()  # Sortida: Hola, em dic Joan i tinc 30 anys.
persona2.saludar()  # Sortida: Hola, em dic Anna i tinc 25 anys.

Exercicis Pràctics

Exercici 1

Crea una classe anomenada Cotxe amb els següents atributs: marca, model i any. Afegeix un mètode anomenat descripcio que imprimeixi una descripció del cotxe.

class Cotxe:
    def __init__(self, marca, model, any):
        self.marca = marca
        self.model = model
        self.any = any

    def descripcio(self):
        print(f"Aquest cotxe és un {self.marca} {self.model} de l'any {self.any}.")

Solució

cotxe1 = Cotxe("Toyota", "Corolla", 2020)
cotxe2 = Cotxe("Honda", "Civic", 2018)

cotxe1.descripcio()  # Sortida: Aquest cotxe és un Toyota Corolla de l'any 2020.
cotxe2.descripcio()  # Sortida: Aquest cotxe és un Honda Civic de l'any 2018.

Errors Comuns

  1. Oblidar el paràmetre self: Tots els mètodes dins d'una classe han de tenir self com a primer paràmetre.
  2. No inicialitzar els atributs: Assegura't d'inicialitzar tots els atributs dins del mètode __init__.

Consells Addicionals

  • Utilitza noms de classes en majúscula per seguir les convencions de Python.
  • Mantén els mètodes curts i enfocats a una sola tasca per millorar la llegibilitat del codi.

Resum

En aquesta secció, hem après els conceptes bàsics de la Programació Orientada a Objectes en Python, incloent la creació de classes i objectes, així com la definició d'atributs i mètodes. Aquests conceptes són fonamentals per a la programació en Python i seran la base per a temes més avançats en els següents mòduls.
