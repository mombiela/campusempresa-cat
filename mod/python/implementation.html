Introducció a la Implementació

En aquesta secció, cobrirem com implementar codi Python de manera efectiva. La implementació implica escriure i organitzar codi per resoldre problemes o realitzar tasques. Començarem amb conceptes bàsics i gradualment passarem a temes més avançats.

Escriure el teu Primer Programa en Python

  • Programa Hello World: El programa més senzill que imprimeix "Hello, World!" a la consola.
print("Hello, World!")

Explicació: La funció print mostra la cadena "Hello, World!" a la consola.

Variables i Tipus de Dades

  • Variables: Contenidors per emmagatzemar valors de dades.
  • Tipus de Dades: Diferents tipus de dades que es poden emmagatzemar en variables, com ara enters, floats, cadenes i booleans.
# Assignació de variables
x = 10        # Enter
y = 3.14      # Float
nom = "Alice" # Cadena
es_estudiant = True # Booleà

print(x, y, nom, es_estudiant)

Explicació: Les variables x, y, nom i es_estudiant s'assignen a diferents tipus de dades. La funció print mostra els seus valors.

Estructures de Control

  • Declaracions Condicionals: Declaracions if, elif i else per controlar el flux del programa basat en condicions.
edat = 18
if edat >= 18:
    print("Ets un adult.")
else:
    print("Ets un menor.")

Explicació: El programa comprova si edat és 18 o més i imprimeix un missatge en conseqüència.

  • Bucles: Bucles for i while per executar un bloc de codi múltiples vegades.
# Bucle For
for i in range(5):
    print(i)

# Bucle While
compte = 0
while compte < 5:
    print(compte)
    compte += 1

Explicació: El bucle for itera de 0 a 4, i el bucle while continua fins que compte és menor que 5.

Funcions

  • Definir Funcions: Utilitzant la paraula clau def per crear blocs de codi reutilitzables.
def saluda(nom):
    return f"Hola, {nom}!"

print(saluda("Alice"))

Explicació: La funció saluda pren un paràmetre nom i retorna una cadena de salutació.

Estructures de Dades

  • Llistes: Col·leccions ordenades d'elements.
fruites = ["poma", "plàtan", "cirera"]
print(fruites[0])  # Sortida: poma

Explicació: fruites és una llista, i fruites[0] accedeix al primer element.

  • Diccionaris: Col·leccions de parelles clau-valor.
estudiant = {"nom": "Alice", "edat": 21}
print(estudiant["nom"])  # Sortida: Alice

Explicació: estudiant és un diccionari, i estudiant["nom"] accedeix al valor associat amb la clau "nom".

Programació Orientada a Objectes (POO)

  • Classes i Objectes: Crear classes i objectes per modelar entitats del món real.
class Gos:
    def __init__(self, nom, edat):
        self.nom = nom
        self.edat = edat

    def bordar(self):
        return "Bup!"

el_meu_gos = Gos("Buddy", 3)
print(el_meu_gos.nom)  # Sortida: Buddy
print(el_meu_gos.bordar())  # Sortida: Bup!

Explicació: La classe Gos té un inicialitzador i un mètode. el_meu_gos és una instància de la classe Gos.

Temes Avançats

  • Comprensions de Llistes: Manera concisa de crear llistes.
quadrats = [x**2 per x en range(10)]
print(quadrats)

Explicació: La comprensió de llista genera una llista de quadrats de nombres de 0 a 9.

  • Generadors: Funcions que produeixen elements un a un.
def comptar_fins_a(max):
    compte = 1
    while compte <= max:
        yield compte
        compte += 1

for numero en comptar_fins_a(5):
    print(numero)

Explicació: El generador comptar_fins_a produeix nombres de l'1 a max.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics d'implementar codi Python, incloent escriure programes simples, utilitzar variables i tipus de dades, estructures de control, funcions, estructures de dades i programació orientada a objectes. També hem tocat alguns temes avançats com les comprensions de llistes i els generadors. Entendre aquests conceptes és crucial per escriure codi Python eficient i efectiu.
