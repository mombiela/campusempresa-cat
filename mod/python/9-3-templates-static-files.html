9-3: Plantilles i Fitxers Estàtics

Introducció

En aquesta secció, aprendrem com utilitzar plantilles i fitxers estàtics en Flask per crear aplicacions web més dinàmiques i atractives. Les plantilles ens permeten separar la lògica de la presentació, mentre que els fitxers estàtics (com ara imatges, CSS i JavaScript) ens ajuden a millorar l'experiència de l'usuari.

Conceptes Clau

  1. Plantilles: Són arxius HTML que poden contenir marcadors de posició per a dades dinàmiques.
  2. Jinja2: És el motor de plantilles per defecte de Flask.
  3. Fitxers Estàtics: Inclouen recursos com CSS, JavaScript i imatges que no canvien dinàmicament.

Utilitzant Plantilles

Creació d'una Plantilla

  1. Directori de Plantilles: Flask busca plantilles dins del directori templates per defecte.
  2. Sintaxi de Jinja2: Utilitza {{ }} per a expressions i {% %} per a sentències de control.

Exemple de Plantilla (templates/index.html):

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">
</head>
<body>
    <h1>Hola, {{ name }}!</h1>
    <p>Aquesta és una pàgina generada amb una plantilla.</p>
</body>
</html>

Renderitzant Plantilles

Exemple de Codi Flask:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html', title='Pàgina Principal', name='Estudiant')

if __name__ == '__main__':
    app.run(debug=True)

Utilitzant Fitxers Estàtics

Directori de Fitxers Estàtics

  1. Directori Estàtic: Flask busca fitxers estàtics dins del directori static per defecte.
  2. Referenciant Fitxers Estàtics: Utilitza la funció url_for('static', filename='path/to/file').

Exemple de Fitxer CSS (static/styles.css):

body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

Incloent Fitxers Estàtics en Plantilles

Exemple d'Incorporació en HTML:

<link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">

Exercicis Pràctics

Exercici 1: Crear una Plantilla amb Dades Dinàmiques

  1. Objectiu: Crear una plantilla que mostri una llista de noms.
  2. Passos:
    • Crear un fitxer templates/names.html.
    • Renderitzar la plantilla des d'una ruta Flask.

Solució:

templates/names.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Llista de Noms</title>
</head>
<body>
    <h1>Llista de Noms</h1>
    <ul>
        {% for name in names %}
            <li>{{ name }}</li>
        {% endfor %}
    </ul>
</body>
</html>

app.py:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/names')
def names():
    name_list = ['Anna', 'Bernat', 'Carla', 'David']
    return render_template('names.html', names=name_list)

if __name__ == '__main__':
    app.run(debug=True)

Exercici 2: Afegir Estils a la Pàgina

  1. Objectiu: Crear un fitxer CSS i aplicar estils a la plantilla.
  2. Passos:
    • Crear un fitxer static/styles.css.
    • Incloure el fitxer CSS en la plantilla.

Solució:

static/styles.css:

body {
    font-family: Arial, sans-serif;
    background-color: #e0f7fa;
    color: #00796b;
}
h1 {
    color: #004d40;
}
ul {
    list-style-type: none;
}
li {
    padding: 5px 0;
}

templates/names.html (actualitzat):

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Llista de Noms</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='styles.css') }}">
</head>
<body>
    <h1>Llista de Noms</h1>
    <ul>
        {% for name in names %}
            <li>{{ name }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Errors Comuns i Consells

  1. Error: TemplateNotFound: Assegura't que el fitxer de plantilla es troba en el directori templates i que el nom del fitxer és correcte.
  2. Error: Static File Not Found: Verifica que el fitxer estàtic es troba en el directori static i que la ruta és correcta.
  3. Consell: Utilitza url_for per generar rutes dinàmiques per a fitxers estàtics i plantilles, ja que això ajuda a evitar errors de ruta.

Resum

En aquesta secció, hem après com utilitzar plantilles i fitxers estàtics en Flask per crear aplicacions web més dinàmiques i atractives. Hem vist com crear i renderitzar plantilles amb Jinja2, així com com incloure fitxers CSS per estilitzar les nostres pàgines web. A més, hem practicat aquests conceptes amb exercicis pràctics.
