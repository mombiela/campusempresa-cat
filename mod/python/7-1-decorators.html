7-1: Decoradors

Introducció als Decoradors

Els decoradors són una característica poderosa i flexible de Python que permeten modificar el comportament de funcions o mètodes. Són molt útils per afegir funcionalitats addicionals de manera neta i reutilitzable.

Conceptes Clau

  • Funcions d'Ordre Superior: Funcions que poden acceptar altres funcions com a arguments o retornar funcions.
  • Closures: Funcions internes que recorden l'entorn en què van ser creades.

Sintaxi Bàsica

Un decorador és una funció que pren una altra funció com a argument i retorna una nova funció amb funcionalitats afegides.

def decorador(func):
    def func_decorada():
        print("Abans de la funció")
        func()
        print("Després de la funció")
    return func_decorada

@decorador
def funcio_original():
    print("Funció original")

funcio_original()

Explicació del Codi

  1. Definició del Decorador:

    def decorador(func):
    def func_decorada():
        print("Abans de la funció")
        func()
        print("Després de la funció")
    return func_decorada
    • decorador és una funció que pren una funció func com a argument.
    • func_decorada és una funció interna que afegeix comportament abans i després de cridar func.
    • decorador retorna func_decorada.

  2. Aplicació del Decorador:

    @decorador
def funcio_original():
    print("Funció original")
    • @decorador aplica el decorador a funcio_original, modificant el seu comportament.

  3. Crida a la Funció Decorada:

    funcio_original()
    • Quan es crida funcio_original, s'executa func_decorada, mostrant: 
      Abans de la funció
Funció original
Després de la funció

Decoradors amb Arguments

Els decoradors també poden acceptar arguments. Això es fa mitjançant una funció que retorna un decorador.

def decorador_amb_args(prefix):
    def decorador(func):
        def func_decorada():
            print(f"{prefix} Abans de la funció")
            func()
            print(f"{prefix} Després de la funció")
        return func_decorada
    return decorador

@decorador_amb_args("LOG:")
def funcio_original():
    print("Funció original")

funcio_original()

Exercicis Pràctics

Exercici 1: Decorador Simple

Crea un decorador que imprimeixi "Inici de la funció" abans de cridar la funció original i "Fi de la funció" després.

def decorador_simple(func):
    def func_decorada():
        print("Inici de la funció")
        func()
        print("Fi de la funció")
    return func_decorada

@decorador_simple
def funcio_exemple():
    print("Funció exemple")

funcio_exemple()

Solució

def decorador_simple(func):
    def func_decorada():
        print("Inici de la funció")
        func()
        print("Fi de la funció")
    return func_decorada

@decorador_simple

Errors Comuns i Consells

  • Oblidar retornar la funció decorada: Assegura't que el decorador sempre retorni la funció decorada.
  • No utilitzar functools.wraps: Quan es crea un decorador, utilitza functools.wraps per preservar el nom i la docstring de la funció original.
from functools import wraps

def decorador(func):
    @wraps(func)
    def func_decorada(*args, **kwargs):
        print("Abans de la funció")
        result = func(*args, **kwargs)
        print("Després de la funció")
        return result
    return func_decorada

Conclusió

Els decoradors són una eina poderosa per modificar el comportament de les funcions de manera neta i reutilitzable. Amb la pràctica, es poden utilitzar per simplificar i organitzar el codi de manera eficient. En el proper tema, explorarem els generadors, una altra característica avançada de Python.
