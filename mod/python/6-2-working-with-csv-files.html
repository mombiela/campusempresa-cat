6-2: Treballant amb Fitxers CSV

Introducció

Els fitxers CSV (Comma-Separated Values) són un format comú per emmagatzemar dades tabulars en text pla. Cada línia del fitxer representa un registre, i cada camp dins del registre està separat per una coma. Python proporciona diverses maneres de treballar amb fitxers CSV, sent la més comuna el mòdul csv.

Objectius

  1. Comprendre el format dels fitxers CSV.
  2. Aprendre a llegir fitxers CSV utilitzant el mòdul csv.
  3. Aprendre a escriure fitxers CSV utilitzant el mòdul csv.
  4. Realitzar operacions bàsiques de manipulació de dades en fitxers CSV.

Contingut

1. Format dels Fitxers CSV

Un fitxer CSV típic pot semblar així:

nom,edat,ciutat
Joan,25,Barcelona
Maria,30,Madrid
Pere,22,València

Cada línia representa un registre, i cada camp dins del registre està separat per una coma.

2. Llegir Fitxers CSV

Per llegir fitxers CSV en Python, utilitzarem el mòdul csv i la classe csv.reader.

import csv

with open('dades.csv', mode='r', newline='') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Explicació del Codi:

  • import csv: Importa el mòdul csv.
  • open('dades.csv', mode='r', newline=''): Obre el fitxer dades.csv en mode lectura (r).
  • csv.reader(file): Crea un objecte reader que iterarà sobre les línies del fitxer.
  • for row in reader: Itera sobre cada fila del fitxer CSV.
  • print(row): Imprimeix cada fila.

3. Escriure Fitxers CSV

Per escriure fitxers CSV, utilitzarem la classe csv.writer.

import csv

dades = [
    ['nom', 'edat', 'ciutat'],
    ['Joan', 25, 'Barcelona'],
    ['Maria', 30, 'Madrid'],
    ['Pere', 22, 'València']
]

with open('dades.csv', mode='w', newline='') as file:
    writer = csv.writer(file)
    writer.writerows(dades)

Explicació del Codi:

  • import csv: Importa el mòdul csv.
  • dades: Una llista de llistes que conté les dades a escriure.
  • open('dades.csv', mode='w', newline=''): Obre el fitxer dades.csv en mode escriptura (w).
  • csv.writer(file): Crea un objecte writer que escriurà les dades al fitxer.
  • writer.writerows(dades): Escriu totes les files de la llista dades al fitxer CSV.

4. Manipulació de Dades en Fitxers CSV

Podem realitzar diverses operacions de manipulació de dades, com ara filtrar o transformar dades mentre les llegim o escrivim.

Exemple: Filtrar Dades

import csv

with open('dades.csv', mode='r', newline='') as file:
    reader = csv.reader(file)
    filtered_data = [row for row in reader if int(row[1]) > 23]

print(filtered_data)

Explicació del Codi:

  • filtered_data = [row for row in reader if int(row[1]) > 23]: Filtra les files on l'edat (row[1]) és superior a 23.

Exercicis Pràctics

  1. Llegir i Imprimir Fitxer CSV:

    • Llegeix un fitxer CSV i imprimeix cada fila.

  2. Escriure Fitxer CSV:

    • Escriu un conjunt de dades en un fitxer CSV.

  3. Filtrar Dades:

    • Llegeix un fitxer CSV i filtra les files segons un criteri donat (per exemple, edat superior a 25).

Solucions

  1. Llegir i Imprimir Fitxer CSV:
import csv

with open('dades.csv', mode='r', newline='') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)
  1. Escriure Fitxer CSV:
import csv

dades = [
    ['nom', 'edat', 'ciutat'],
    ['Joan', 25, 'Barcelona'],
    ['Maria', 30, 'Madrid'],
    ['Pere', 22, 'València']
]

with open('dades.csv', mode='w', newline='') as file:
    writer = csv.writer(file)
    writer.writerows(dades)
  1. Filtrar Dades:
import csv

with open('dades.csv', mode='r', newline='') as file:
    reader = csv.reader(file)
    filtered_data = [row for row in reader if int(row[1]) > 25]

print(filtered_data)

Errors Comuns i Consells

  • Error Comú: No especificar newline='' en obrir el fitxer.

    • Solució: Sempre utilitza newline='' per evitar problemes amb les noves línies en diferents sistemes operatius.

  • Error Comú: Manipular directament les dades sense convertir-les al tipus adequat.

    • Solució: Assegura't de convertir les dades al tipus adequat (per exemple, int(row[1]) per convertir l'edat a enter).

Conclusió

Aquestes habilitats són fonamentals per treballar amb dades tabulars en molts projectes de programació.
