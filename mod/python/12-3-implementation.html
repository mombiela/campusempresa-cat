12-3: Implementació

Objectius de la Secció

En aquesta secció, implementarem el projecte final basant-nos en la planificació i el disseny realitzats en les seccions anteriors. L'objectiu és aplicar tots els conceptes apresos al llarg del curs per crear una aplicació funcional.

Passos per a la Implementació

  1. Configuració de l'Entorn de Desenvolupament

    • Assegura't que tens instal·lades totes les llibreries necessàries.
    • Crea un entorn virtual per al projecte.
    • Instal·la les dependències utilitzant un fitxer requirements.txt.
    python -m venv env
source env/bin/activate  # O env\Scripts\activate en Windows
pip install -r requirements.txt

  2. Estructura del Projecte

    • Organitza el teu projecte seguint una estructura clara i lògica.
    • Exemple d'estructura de directoris:
    my_project/
├── app/
│   ├── __init__.py
│   ├── models.py
│   ├── routes.py
│   ├── templates/
│   │   ├── base.html
│   │   ├── index.html
│   └── static/
│       ├── css/
│       ├── js/
├── tests/
│   ├── test_models.py
│   ├── test_routes.py
├── requirements.txt
├── config.py
├── run.py
└── README.md

  3. Creació de la Base de Dades

    • Defineix els models de dades utilitzant SQLAlchemy (o l'ORM que hagis triat).
    • Exemple de model de dades:
    from app import db

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'

  4. Desenvolupament de les Rutes

    • Defineix les rutes de la teva aplicació web.
    • Exemple de ruta en Flask:
    from flask import Flask, render_template
app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

  5. Creació de Plantilles

    • Utilitza Jinja2 per crear plantilles HTML reutilitzables.
    • Exemple de plantilla base:
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Project</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/style.css') }}">
</head>
<body>
    <header>
        <h1>Welcome to My Project</h1>
    </header>
    <main>
        {% block content %}{% endblock %}
    </main>
</body>
</html>

  6. Implementació de Funcionalitats Clau

    • Desenvolupa les funcionalitats principals del projecte segons els requisits.
    • Exemple de formulari de registre:
    from flask import Flask, render_template, request, redirect, url_for
from app import app, db
from app.models import User

@app.route('/register', methods=['GET', 'POST'])
def register():
    if request.method == 'POST':
        username = request.form['username']
        email = request.form['email']
        new_user = User(username=username, email=email)
        db.session.add(new_user)
        db.session.commit()
        return redirect(url_for('index'))
    return render_template('register.html')

  7. Proves i Depuració Inicial

    • Realitza proves inicials per assegurar-te que les funcionalitats bàsiques funcionen correctament.
    • Utilitza unittest o pytest per crear proves unitàries.
    import unittest
from app import app, db
from app.models import User

class UserModelCase(unittest.TestCase):
    def setUp(self):
        app.config['TESTING'] = True
        app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite://'
        self.app = app.test_client()
        db.create_all()

    def tearDown(self):
        db.session.remove()
        db.drop_all()

    def test_user_creation(self):
        u = User(username='testuser', email='test@example.com')
        db.session.add(u)
        db.session.commit()
        self.assertEqual(User.query.count(), 1)

if __name__ == '__main__':
    unittest.main()

Exercicis Pràctics

  1. Implementa una Ruta de Login

    • Crea una ruta /login que permeti als usuaris iniciar sessió.
    • Afegeix validació per assegurar-te que l'usuari existeix a la base de dades.

  2. Afegeix Estils CSS

    • Crea un fitxer CSS per millorar l'aspecte visual de la teva aplicació.
    • Aplica estils a les plantilles HTML.

  3. Proves Addicionals

    • Escriu proves unitàries per a les noves funcionalitats que has implementat.
    • Assegura't que totes les proves passen correctament.

Resum

En aquesta secció, hem implementat el projecte final seguint una estructura clara i organitzada. Hem creat la base de dades, desenvolupat les rutes, creat plantilles HTML i implementat funcionalitats clau. També hem realitzat proves inicials per assegurar-nos que tot funciona correctament. En la següent secció, ens centrarem en les proves i la depuració del projecte per assegurar-nos que està llest per al desplegament.
