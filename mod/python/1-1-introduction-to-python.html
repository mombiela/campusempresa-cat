1.1 Introducció a Python

Objectius del Tema

  • Entendre què és Python i per què és un llenguatge de programació popular.
  • Conèixer les característiques principals de Python.
  • Aprendre sobre les aplicacions i usos comuns de Python.

Què és Python?

Python és un llenguatge de programació d'alt nivell, interpretat i de propòsit general. Va ser creat per Guido van Rossum i la seva primera versió va ser llançada el 1991. Python és conegut per la seva sintaxi clara i llegible, que facilita l'aprenentatge i l'ús.

Característiques Principals de Python

  1. Sintaxi Clara i Llegible:

    • Python utilitza una sintaxi que és fàcil de llegir i escriure, la qual cosa redueix la corba d'aprenentatge per als nous programadors.
    • Exemple: 
      print("Hola, món!")

  2. Interpretat:

    • Python és un llenguatge interpretat, el que significa que el codi es pot executar directament sense necessitat de compilar-lo prèviament.
    • Això permet una ràpida iteració i depuració.

  3. Tipatge Dinàmic:

    • Python utilitza tipatge dinàmic, el que significa que no cal declarar el tipus de variable explícitament.
    • Exemple: 
      x = 10  # Enter
x = "Hola"  # Ara és una cadena

  4. Alt Nivell:

    • Python proporciona abstraccions de baix nivell, el que permet als programadors centrar-se en la resolució de problemes en lloc de detalls tècnics.

  5. Gran Biblioteca Estàndard:

    • Python ve amb una extensa biblioteca estàndard que cobreix moltes tasques comunes, com ara manipulació de fitxers, treball amb dades, comunicacions de xarxa, etc.

  6. Multiparadigma:

    • Python suporta múltiples paradigmes de programació, incloent programació orientada a objectes, programació funcional i programació imperativa.

Aplicacions i Usos de Python

Python és un llenguatge de propòsit general que s'utilitza en una àmplia varietat d'aplicacions:

  1. Desenvolupament Web:

    • Frameworks com Django i Flask permeten crear aplicacions web robustes i escalables.
    • Exemple: 
      from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return "Hola, món!"

if __name__ == '__main__':
    app.run(debug=True)

  2. Anàlisi de Dades:

    • Llibreries com Pandas, NumPy i Matplotlib són àmpliament utilitzades per a l'anàlisi i visualització de dades.
    • Exemple: 
      import pandas as pd

dades = {'Nom': ['Anna', 'Joan', 'Maria'], 'Edat': [28, 34, 29]}
df = pd.DataFrame(dades)
print(df)

  3. Intel·ligència Artificial i Aprenentatge Automàtic:

    • Llibreries com TensorFlow, Keras i Scikit-learn són utilitzades per desenvolupar models d'aprenentatge automàtic i intel·ligència artificial.
    • Exemple: 
      from sklearn.linear_model import LinearRegression

model = LinearRegression()

  4. Automatització i Scripts:

    • Python és ideal per a l'automatització de tasques repetitives i la creació de scripts.
    • Exemple: 
      import os

for fitxer in os.listdir('.'):
    if fitxer.endswith('.txt'):
        print(fitxer)

  5. Desenvolupament de Jocs:

    • Llibreries com Pygame permeten crear jocs senzills i interactius.
    • Exemple: 
      import pygame

pygame.init()
pantalla = pygame.display.set_mode((800, 600))

Conclusió

Python és un llenguatge de programació versàtil i potent que és fàcil d'aprendre i utilitzar. La seva sintaxi clara, la seva gran biblioteca estàndard i la seva comunitat activa el fan una excel·lent elecció per a una àmplia varietat d'aplicacions, des del desenvolupament web fins a l'anàlisi de dades i l'automatització.

En el proper tema, aprendrem com configurar l'entorn de desenvolupament per començar a programar en Python.

Exercicis Pràctics

  1. Instal·la Python:

    • Visita la pàgina oficial de Python (https://www.python.org/) i descarrega la versió més recent de Python per al teu sistema operatiu.
    • Segueix les instruccions per instal·lar Python al teu ordinador.

  2. Executa el teu Primer Programa:

    • Obre un editor de text o un entorn de desenvolupament integrat (IDE) com VS Code, PyCharm o IDLE.
    • Escriu el següent codi i guarda'l en un fitxer anomenat hola_mon.py: 
      print("Hola, món!")
    • Executa el fitxer des de la línia de comandes o des de l'IDE.

  3. Explora la Documentació de Python:

    • Visita la documentació oficial de Python (https://docs.python.org/3/) i explora les seccions introductòries per familiaritzar-te amb els recursos disponibles.

Solucions

  1. Instal·lació de Python:

    • Segueix les instruccions detallades a la pàgina oficial de Python per descarregar i instal·lar Python al teu sistema operatiu.

  2. Executa el teu Primer Programa:

    • Des de la línia de comandes, navega fins al directori on has guardat el fitxer hola_mon.py i executa: 
      python hola_mon.py
    • Hauries de veure la sortida: 
      Hola, món!

  3. Explora la Documentació de Python:

    • La documentació oficial de Python és una excel·lent font d'informació per a qualsevol dubte o consulta que puguis tenir mentre aprens Python.
