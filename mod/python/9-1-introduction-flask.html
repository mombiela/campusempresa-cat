9-1: Introducció a Flask

Objectius del Tema

  • Entendre què és Flask i per què és útil per al desenvolupament web.
  • Configurar un entorn de desenvolupament per a una aplicació Flask.
  • Crear una aplicació web bàsica amb Flask.

Què és Flask?

Flask és un microframework per al desenvolupament web en Python. És lleuger i flexible, permetent als desenvolupadors crear aplicacions web ràpidament i amb facilitat. Flask és conegut per la seva simplicitat i extensibilitat, ja que proporciona les eines bàsiques necessàries per construir una aplicació web, però permet afegir funcionalitats addicionals mitjançant extensions.

Característiques de Flask

  • Lleuger i Micro: Flask és un microframework, el que significa que és lleuger i no inclou moltes funcionalitats per defecte. Això permet als desenvolupadors afegir només les funcionalitats que necessiten.
  • Extensible: Flask té una gran varietat d'extensions que es poden utilitzar per afegir funcionalitats com l'autenticació, la gestió de bases de dades, etc.
  • Flexible: Flask no imposa cap estructura específica per a les aplicacions, permetent als desenvolupadors organitzar el codi com considerin més adequat.
  • Documentació Completa: Flask té una documentació molt completa i una comunitat activa, el que facilita l'aprenentatge i la resolució de problemes.

Configuració de l'Entorn de Desenvolupament

Instal·lació de Flask

Per començar a treballar amb Flask, primer necessitem instal·lar-lo. Utilitzarem pip, el gestor de paquets de Python.

pip install Flask

Creació d'un Entorn Virtual

És recomanable utilitzar un entorn virtual per aïllar les dependències de la nostra aplicació. Això ens permetrà evitar conflictes amb altres projectes.

python -m venv env
source env/bin/activate  # Per a Unix/Mac
env\Scripts\activate     # Per a Windows

Creació d'una Aplicació Web Bàsica amb Flask

Estructura del Projecte

Crearem una estructura de projecte senzilla per a la nostra aplicació Flask.

my_flask_app/
│
├── app.py
└── templates/
    └── index.html

El Fitxer app.py

Aquest fitxer contindrà el codi principal de la nostra aplicació Flask.

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

Explicació del Codi:

  • Importació de Flask: Importem la classe Flask del paquet flask.
  • Creació de l'Aplicació: Creem una instància de l'aplicació Flask.
  • Definició de Rutes: Definim una ruta (/) que retorna una plantilla HTML (index.html).
  • Execució de l'Aplicació: Utilitzem app.run(debug=True) per executar l'aplicació en mode de depuració.

La Plantilla index.html

Crearem una plantilla HTML senzilla per a la nostra pàgina d'inici.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Flask App</title>
</head>
<body>
    <h1>Welcome to My Flask App!</h1>
</body>
</html>

Executar l'Aplicació

Per executar l'aplicació, simplement executem el fitxer app.py.

python app.py

Ara, si obrim un navegador web i anem a http://127.0.0.1:5000/, hauríem de veure la nostra pàgina d'inici amb el missatge "Welcome to My Flask App!".

Exercici Pràctic

Exercici:

  1. Modifica l'aplicació per afegir una nova ruta /about que mostri una pàgina amb informació sobre l'aplicació.
  2. Crea una nova plantilla about.html per a la pàgina /about.

Solució:

Modificació de app.py:

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/about')
def about():
    return render_template('about.html')

if __name__ == '__main__':
    app.run(debug=True)

Creació de about.html:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>About My Flask App</title>
</head>
<body>
    <h1>About My Flask App</h1>
    <p>This is a simple Flask application.</p>
</body>
</html>

També hem practicat afegint una nova ruta i una nova plantilla a la nostra aplicació. En la següent secció, aprofundirem en la creació d'una aplicació web més complexa amb Flask.
