Tema 1.4: Variables i Tipus de Dades

Introducció

En aquest tema, aprendrem sobre les variables i els tipus de dades en Python. Les variables són essencials per emmagatzemar informació que pot ser utilitzada i manipulada al llarg del programa. Els tipus de dades defineixen el tipus de valor que una variable pot emmagatzemar.

Continguts

  1. Què és una Variable?
  2. Tipus de Dades Bàsics
  3. Assignació de Valors a Variables
  4. Operacions amb Variables
  5. Conversió de Tipus de Dades
  6. Exemples Pràctics
  7. Exercicis Pràctics

1. Què és una Variable?

Una variable és un nom que fa referència a un valor emmagatzemat a la memòria. En Python, no cal declarar el tipus de variable abans d'assignar-li un valor.

# Exemple de variable
nom = "Anna"
edat = 25

2. Tipus de Dades Bàsics

Python té diversos tipus de dades bàsics:

  • int: Nombres enters
  • float: Nombres decimals
  • str: Cadenes de text
  • bool: Valors booleans (True o False)
Tipus de Dada Exemple
int 10
float 10.5
str "Hola, món!"
bool True, False

3. Assignació de Valors a Variables

Assignar valors a variables és senzill en Python. Utilitzem l'operador =.

# Assignació de valors
nom = "Anna"
edat = 25
altura = 1.68
es_estudiant = True

4. Operacions amb Variables

Podem realitzar diverses operacions amb variables, depenent del seu tipus de dada.

# Operacions amb nombres
a = 10
b = 5
suma = a + b
resta = a - b
producte = a * b
divisio = a / b

# Operacions amb cadenes de text
nom_complet = "Anna" + " " + "Martínez"

# Operacions amb booleans
es_major = edat > 18

5. Conversió de Tipus de Dades

És possible convertir una variable d'un tipus de dada a un altre utilitzant funcions de conversió.

# Conversió de tipus de dades
edat = "25"
edat = int(edat)  # Converteix la cadena "25" a l'enter 25

altura = 1.68
altura = str(altura)  # Converteix el float 1.68 a la cadena "1.68"

6. Exemples Pràctics

# Exemple complet
nom = "Anna"
edat = 25
altura = 1.68
es_estudiant = True

# Imprimir informació
print("Nom:", nom)
print("Edat:", edat)
print("Altura:", altura)
print("És estudiant?", es_estudiant)

# Operacions
any_naixement = 2023 - edat
print("Any de naixement:", any_naixement)

# Conversió de tipus
edat_str = str(edat)
print("Edat (com a cadena):", edat_str)

7. Exercicis Pràctics

Exercici 1

Crea un programa que demani a l'usuari el seu nom, edat i altura. Després, imprimeix aquesta informació en format de frase.

# Solució
nom = input("Introdueix el teu nom: ")
edat = int(input("Introdueix la teva edat: "))
altura = float(input("Introdueix la teva altura en metres: "))

print(f"El teu nom és {nom}, tens {edat} anys i fas {altura} metres d'altura.")

Exercici 2

Escriu un programa que calculi l'any de naixement d'una persona a partir de la seva edat i l'any actual.

# Solució
edat = int(input("Introdueix la teva edat: "))
any_actual = 2023
any_naixement = any_actual - edat

print(f"Vas néixer l'any {any_naixement}.")

Conclusió

En aquest tema, hem après què són les variables i els tipus de dades en Python, com assignar valors a variables, realitzar operacions amb elles i convertir tipus de dades. Aquestes habilitats són fonamentals per a qualsevol programador i seran la base per a temes més avançats en el futur.
