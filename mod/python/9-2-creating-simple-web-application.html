9-2: Creació d'una Aplicació Web Simple

En aquesta secció, aprendrem a crear una aplicació web simple utilitzant Flask, un microframework per a Python que facilita la creació d'aplicacions web. Aquest tema inclou la configuració inicial, la creació de rutes bàsiques i la renderització de plantilles HTML.

Objectius

  1. Configurar una aplicació Flask.
  2. Crear rutes bàsiques.
  3. Renderitzar plantilles HTML.

Contingut

  1. Configuració Inicial
  2. Creació de Rutes Bàsiques
  3. Renderització de Plantilles HTML
  4. Exercicis Pràctics

1. Configuració Inicial

Instal·lació de Flask

Abans de començar, assegura't de tenir Flask instal·lat. Pots instal·lar-lo utilitzant pip:

pip install flask

Estructura del Projecte

Crea una estructura de directoris per al teu projecte:

my_flask_app/
│
├── app.py
├── templates/
│   └── index.html
└── static/

2. Creació de Rutes Bàsiques

Una ruta en Flask és una URL específica que l'aplicació pot gestionar. Comencem amb una ruta bàsica que mostra un missatge de benvinguda.

Exemple de Codi

# app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Benvingut a la meva primera aplicació Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Explicació

  • from flask import Flask: Importa la classe Flask.
  • app = Flask(__name__): Crea una instància de l'aplicació Flask.
  • @app.route('/'): Defineix una ruta per a la URL arrel ('/').
  • def home(): Defineix la funció que es cridarà quan es visiti la ruta arrel.
  • app.run(debug=True): Executa l'aplicació en mode de depuració.

3. Renderització de Plantilles HTML

Flask utilitza el motor de plantilles Jinja2 per renderitzar HTML. Crearem una plantilla HTML i la renderitzarem des de la nostra aplicació Flask.

Exemple de Codi

# app.py
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

Plantilla HTML

Crea un fitxer index.html dins del directori templates:

<!-- templates/index.html -->
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Aplicació Flask</title>
</head>
<body>
    <h1>Benvingut a la meva primera aplicació Flask!</h1>
</body>
</html>

Explicació

  • from flask import render_template: Importa la funció render_template.
  • return render_template('index.html'): Renderitza la plantilla index.html i la retorna com a resposta.

4. Exercicis Pràctics

Exercici 1: Ruta Addicional

Crea una nova ruta /about que mostri una pàgina amb informació sobre l'aplicació.

Solució:

# app.py
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/about')
def about():
    return render_template('about.html')

if __name__ == '__main__':
    app.run(debug=True)

Crea una nova plantilla about.html dins del directori templates:

<!-- templates/about.html -->
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Sobre l'Aplicació</title>
</head>
<body>
    <h1>Sobre l'Aplicació</h1>
    <p>Aquesta és una aplicació web simple creada amb Flask.</p>
</body>
</html>

Exercici 2: Passar Dades a la Plantilla

Modifica la ruta arrel (/) per passar una variable a la plantilla index.html i mostrar-la.

Solució:

# app.py
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    message = "Benvingut a la meva primera aplicació Flask!"
    return render_template('index.html', message=message)

if __name__ == '__main__':
    app.run(debug=True)

Modifica la plantilla index.html per mostrar la variable message:

<!-- templates/index.html -->
<!DOCTYPE html>
<html lang="ca">
<head>
    <meta charset="UTF-8">
    <title>Aplicació Flask</title>
</head>
<body>
    <h1>{{ message }}</h1>
</body>
</html>

Conclusió

En aquesta secció, hem après a configurar una aplicació Flask, crear rutes bàsiques i renderitzar plantilles HTML. Ara estem preparats per avançar cap a temes més complexos com la gestió de formularis i la integració de bases de dades.
