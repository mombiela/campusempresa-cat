3-4: Mòduls i Paquets

Introducció

En Python, els mòduls i els paquets són eines fonamentals per organitzar i reutilitzar el codi. Un mòdul és simplement un fitxer que conté codi Python, mentre que un paquet és una col·lecció de mòduls organitzats en directoris. Aquest tema t'ensenyarà com crear i utilitzar mòduls i paquets per estructurar millor les teves aplicacions.

Conceptes Clau

  1. Mòduls:

    • Un mòdul és un fitxer Python amb l'extensió .py que pot contenir funcions, classes i variables.
    • Els mòduls permeten dividir el codi en parts més petites i manejables.

  2. Paquets:

    • Un paquet és un directori que conté múltiples mòduls i un fitxer especial __init__.py.
    • Els paquets permeten organitzar mòduls en una estructura jeràrquica.

Creació i Ús de Mòduls

Creació d'un Mòdul

  1. Crea un fitxer Python amb l'extensió .py. Per exemple, mimodul.py.
  2. Afegeix funcions, classes o variables al fitxer.
# mimodul.py

def saluda(nom):
    return f"Hola, {nom}!"

class Persona:
    def __init__(self, nom, edat):
        self.nom = nom
        self.edat = edat

    def info(self):
        return f"Nom: {self.nom}, Edat: {self.edat}"

Ús d'un Mòdul

  1. Importa el mòdul al teu script o a un altre mòdul utilitzant la paraula clau import.
# script.py

import mimodul

print(mimodul.saluda("Anna"))

persona = mimodul.Persona("Joan", 30)
print(persona.info())
  1. També pots importar només parts específiques d'un mòdul.
# script.py

from mimodul import saluda, Persona

print(saluda("Anna"))

persona = Persona("Joan", 30)
print(persona.info())

Creació i Ús de Paquets

Creació d'un Paquet

  1. Crea un directori per al teu paquet. Per exemple, mipaquet.
  2. Afegeix un fitxer __init__.py al directori. Aquest fitxer pot estar buit o contenir codi d'inicialització per al paquet.
  3. Afegeix mòduls al directori del paquet.
mipaquet/
    __init__.py
    modul1.py
    modul2.py

Contingut dels Mòduls

# mipaquet/modul1.py

def funcio1():
    return "Funció 1 del Mòdul 1"

# mipaquet/modul2.py

def funcio2():
    return "Funció 2 del Mòdul 2"

Ús d'un Paquet

  1. Importa el paquet o els seus mòduls al teu script.
# script.py

import mipaquet.modul1
import mipaquet.modul2

print(mipaquet.modul1.funcio1())
print(mipaquet.modul2.funcio2())
  1. També pots importar funcions específiques dels mòduls del paquet.
# script.py

from mipaquet.modul1 import funcio1
from mipaquet.modul2 import funcio2

print(funcio1())
print(funcio2())

Exercicis Pràctics

Exercici 1: Crear i Utilitzar un Mòdul

  1. Crea un mòdul anomenat matematica.py que contingui les següents funcions:

    • suma(a, b): retorna la suma de a i b.
    • resta(a, b): retorna la resta de a i b.

  2. Escriu un script que importi el mòdul matematica i utilitzi les seves funcions per sumar i restar dos nombres.

Solució

# matematica.py

def suma(a, b):
    return a + b

def resta(a, b):
    return a - b
# script.py

import matematica

print(matematica.suma(5, 3))  # Sortida: 8
print(matematica.resta(5, 3))  # Sortida: 2

Exercici 2: Crear i Utilitzar un Paquet

  1. Crea un paquet anomenat geometria amb els següents mòduls:

    • circul.py que contingui una funció area(radi) que calculi l'àrea d'un cercle.
    • rectangle.py que contingui una funció area(base, altura) que calculi l'àrea d'un rectangle.

  2. Escriu un script que importi el paquet geometria i utilitzi les seves funcions per calcular l'àrea d'un cercle i d'un rectangle.

Solució

# geometria/circul.py

import math

def area(radi):
    return math.pi * radi ** 2
# geometria/rectangle.py

def area(base, altura):
    return base * altura
# script.py

from geometria.circul import area as area_circul
from geometria.rectangle import area as area_rectangle

print(area_circul(5))  # Sortida: 78.53981633974483
print(area_rectangle(4, 6))  # Sortida: 24

Resum

En aquesta secció, has après a crear i utilitzar mòduls i paquets en Python. Els mòduls et permeten organitzar el codi en fitxers separats, mentre que els paquets et permeten agrupar mòduls en una estructura jeràrquica. Aquestes eines són essencials per mantenir el codi net, organitzat i fàcil de gestionar a mesura que els teus projectes creixen en complexitat.
