Tema 2-1: Sentències Condicionals

Introducció

Les sentències condicionals permeten que el teu programa prengui decisions basades en certes condicions. En Python, les sentències condicionals es gestionen principalment amb les instruccions if, elif i else.

Conceptes Clau

  1. Sentència if: Executa un bloc de codi si una condició és certa.
  2. Sentència elif: Executa un bloc de codi si la condició anterior no és certa i aquesta condició és certa.
  3. Sentència else: Executa un bloc de codi si cap de les condicions anteriors és certa.
  4. Condicions: Expressions que es poden avaluar com a vertaderes (True) o falses (False).

Sintaxi Bàsica

if condició:
    # Bloc de codi si la condició és certa
elif altra_condició:
    # Bloc de codi si la condició anterior no és certa i aquesta és certa
else:
    # Bloc de codi si cap de les condicions anteriors és certa

Exemples Pràctics

Exemple 1: Sentència if

x = 10
if x > 5:
    print("x és més gran que 5")

Explicació: Aquest codi imprimeix "x és més gran que 5" perquè la condició x > 5 és certa.

Exemple 2: Sentència if-else

x = 3
if x > 5:
    print("x és més gran que 5")
else:
    print("x no és més gran que 5")

Explicació: Aquest codi imprimeix "x no és més gran que 5" perquè la condició x > 5 és falsa.

Exemple 3: Sentència if-elif-else

x = 7
if x > 10:
    print("x és més gran que 10")
elif x > 5:
    print("x és més gran que 5 però no més gran que 10")
else:
    print("x no és més gran que 5")

Explicació: Aquest codi imprimeix "x és més gran que 5 però no més gran que 10" perquè la condició x > 5 és certa i x > 10 és falsa.

Exercicis Pràctics

Exercici 1: Verificar si un número és positiu, negatiu o zero

Escriu un programa que demani a l'usuari un número i imprimeixi si el número és positiu, negatiu o zero.

# Solució
num = float(input("Introdueix un número: "))

if num > 0:
    print("El número és positiu")
elif num < 0:
    print("El número és negatiu")
else:
    print("El número és zero")

Exercici 2: Classificar una nota

Escriu un programa que demani a l'usuari una nota (0-100) i imprimeixi la qualificació corresponent (A, B, C, D, F).

# Solució
nota = int(input("Introdueix la teva nota (0-100): "))

if nota >= 90:
    print("Qualificació: A")
elif nota >= 80:
    print("Qualificació: B")
elif nota >= 70:
    print("Qualificació: C")
elif nota >= 60:
    print("Qualificació: D")
else:
    print("Qualificació: F")

Errors Comuns i Consells

  1. Indentació Incorrecta: Python utilitza l'indentació per definir blocs de codi. Assegura't que tots els blocs de codi dins de les sentències condicionals estiguin correctament indentats.
  2. Ús Incorrecte d'Operadors de Comparació: Recorda utilitzar els operadors de comparació correctes (==, !=, >, <, >=, <=).
  3. Condicions Complexes: Per a condicions complexes, utilitza parèntesis per assegurar-te que les expressions es processen en l'ordre correcte.

Resum

Hem vist com utilitzar aquestes instruccions per prendre decisions basades en condicions específiques i hem practicat amb exemples i exercicis.
