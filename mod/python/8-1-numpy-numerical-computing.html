8-1: NumPy per al Càlcul Numèric

Introducció a NumPy

NumPy (Numerical Python) és una llibreria fonamental per a la computació científica en Python. Proporciona una estructura de dades poderosa, l'array multidimensional, així com funcions per a operar amb aquests arrays de manera eficient.

Objectius de la Secció

  • Entendre què és NumPy i per què és útil.
  • Aprendre a instal·lar i importar NumPy.
  • Familiaritzar-se amb els arrays de NumPy.
  • Realitzar operacions bàsiques amb arrays.
  • Utilitzar funcions matemàtiques i estadístiques de NumPy.

Instal·lació de NumPy

Per instal·lar NumPy, pots utilitzar pip:

pip install numpy

Importació de NumPy

Per utilitzar NumPy en el teu codi, primer has d'importar-lo. És una pràctica comuna importar NumPy amb l'àlies np:

import numpy as np

Creació d'Arrays

Els arrays són la peça central de NumPy. Pots crear arrays de diverses maneres:

Des d'una llista

import numpy as np

llista = [1, 2, 3, 4, 5]
array = np.array(llista)
print(array)

Arrays de zeros i uns

zeros = np.zeros(5)
print(zeros)

uns = np.ones(5)
print(uns)

Arrays amb valors aleatoris

aleatoris = np.random.rand(5)
print(aleatoris)

Propietats dels Arrays

Els arrays de NumPy tenen diverses propietats útils:

array = np.array([[1, 2, 3], [4, 5, 6]])

print("Forma de l'array:", array.shape)
print("Nombre de dimensions:", array.ndim)
print("Nombre total d'elements:", array.size)
print("Tipus de dades dels elements:", array.dtype)

Operacions Bàsiques amb Arrays

Operacions aritmètiques

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print("Suma:", a + b)
print("Resta:", a - b)
print("Multiplicació:", a * b)
print("Divisió:", a / b)

Funcions matemàtiques

array = np.array([1, 2, 3, 4, 5])

print("Suma de tots els elements:", np.sum(array))
print("Producte de tots els elements:", np.prod(array))
print("Mitjana:", np.mean(array))
print("Desviació estàndard:", np.std(array))

Indexació i Tallat d'Arrays

Indexació

array = np.array([1, 2, 3, 4, 5])
print("Element a la posició 0:", array[0])
print("Element a la posició 4:", array[4])

Tallat

array = np.array([1, 2, 3, 4, 5])
print("Elements del 0 al 2:", array[0:3])
print("Elements del 2 al final:", array[2:])
print("Elements del principi al 3:", array[:4])

Exercicis Pràctics

  1. Creació d'Arrays

    • Crea un array de NumPy amb els valors del 1 al 10.
    • Crea un array de zeros de longitud 10.
    • Crea un array de valors aleatoris de longitud 10.

  2. Operacions amb Arrays

    • Crea dos arrays de NumPy amb els valors [1, 2, 3] i [4, 5, 6].
    • Realitza operacions de suma, resta, multiplicació i divisió amb aquests arrays.

  3. Funcions Matemàtiques

    • Crea un array de NumPy amb els valors [1, 2, 3, 4, 5].
    • Calcula la suma, producte, mitjana i desviació estàndard dels elements de l'array.

Solucions

  1. Creació d'Arrays
# Array amb valors del 1 al 10
array1 = np.arange(1, 11)
print(array1)

# Array de zeros de longitud 10
array2 = np.zeros(10)
print(array2)

# Array de valors aleatoris de longitud 10
array3 = np.random.rand(10)
print(array3)
  1. Operacions amb Arrays
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print("Suma:", a + b)
print("Resta:", a - b)
print("Multiplicació:", a * b)
print("Divisió:", a / b)
  1. Funcions Matemàtiques
array = np.array([1, 2, 3, 4, 5])

print("Suma de tots els elements:", np.sum(array))
print("Producte de tots els elements:", np.prod(array))
print("Mitjana:", np.mean(array))
print("Desviació estàndard:", np.std(array))

Conclusió

En aquesta secció, hem après els conceptes bàsics de NumPy, incloent la creació d'arrays, operacions bàsiques i funcions matemàtiques. NumPy és una eina poderosa per al càlcul numèric i és essencial per a qualsevol treball científic o d'anàlisi de dades en Python. En les següents seccions, explorarem altres llibreries útils com Pandas i Matplotlib.
