8-2: Pandas per a l'Anàlisi de Dades

Introducció a Pandas

Pandas és una llibreria de Python que proporciona estructures de dades i eines d'anàlisi de dades d'alt rendiment i fàcil d'usar. És especialment útil per treballar amb dades tabulars, com les que es troben en fulls de càlcul o bases de dades.

Instal·lació de Pandas

Abans de començar, assegura't de tenir Pandas instal·lat. Pots instal·lar-lo utilitzant pip:

pip install pandas

Conceptes Clau

  1. Series: Una estructura de dades unidimensional similar a una llista o un array.
  2. DataFrame: Una estructura de dades bidimensional amb etiquetes d'eix (files i columnes), similar a una taula en una base de dades o un full de càlcul.

Creació de Series i DataFrames

Series

Una Series és essencialment una columna. Pots crear una Series a partir d'una llista, array o fins i tot un diccionari.

import pandas as pd

# Creació d'una Series a partir d'una llista
s = pd.Series([1, 3, 5, 7, 9])
print(s)

DataFrame

Un DataFrame és una col·lecció de Series i és l'estructura de dades més utilitzada en Pandas.

# Creació d'un DataFrame a partir d'un diccionari
data = {
    'Nom': ['Anna', 'Bernat', 'Carla'],
    'Edat': [28, 34, 29],
    'Ciutat': ['Barcelona', 'Madrid', 'València']
}

df = pd.DataFrame(data)
print(df)

Operacions Bàsiques amb DataFrames

Lectura de Fitxers

Pandas pot llegir dades de diversos formats de fitxers, com CSV, Excel, SQL, etc.

# Lectura d'un fitxer CSV
df = pd.read_csv('dades.csv')
print(df.head())  # Mostra les primeres 5 files

Selecció de Dades

Pots seleccionar columnes específiques d'un DataFrame de diverses maneres.

# Selecció d'una columna
print(df['Nom'])

# Selecció de múltiples columnes
print(df[['Nom', 'Edat']])

Filtratge de Dades

Pots filtrar les dades basant-te en condicions.

# Filtratge de files on l'edat és major de 30
print(df[df['Edat'] > 30])

Afegeix i Elimina Columnes

Pots afegir noves columnes o eliminar-ne d'existents.

# Afegeix una nova columna
df['Salari'] = [50000, 60000, 55000]
print(df)

# Elimina una columna
df = df.drop('Salari', axis=1)
print(df)

Operacions Avançades

Agrupació i Agregació

Pandas permet agrupar dades i aplicar funcions d'agregació com sum, mean, count, etc.

# Agrupació per ciutat i càlcul de l'edat mitjana
grouped = df.groupby('Ciutat')['Edat'].mean()
print(grouped)

Fusió i Unió de DataFrames

Pots combinar múltiples DataFrames utilitzant funcions com merge i concat.

# Fusió de dos DataFrames
df1 = pd.DataFrame({'Clau': ['A', 'B', 'C'], 'Valor1': [1, 2, 3]})
df2 = pd.DataFrame({'Clau': ['A', 'B', 'D'], 'Valor2': [4, 5, 6]})

merged_df = pd.merge(df1, df2, on='Clau', how='inner')
print(merged_df)

Exercicis Pràctics

Exercici 1: Lectura i Selecció de Dades

  1. Llegeix un fitxer CSV anomenat empleats.csv que conté les columnes Nom, Edat, Departament i Salari.
  2. Selecciona només les columnes Nom i Salari.
# Solució
df = pd.read_csv('empleats.csv')
result = df[['Nom', 'Salari']]
print(result)

Exercici 2: Filtratge i Agrupació

  1. Filtra les files on el Departament és 'IT'.
  2. Agrupa les dades filtrades per Departament i calcula el salari mitjà.
# Solució
filtered_df = df[df['Departament'] == 'IT']
grouped = filtered_df.groupby('Departament')['Salari'].mean()
print(grouped)

Errors Comuns i Consells

  • Error: KeyError en seleccionar columnes.
    • Solució: Assegura't que el nom de la columna és correcte i existeix en el DataFrame.
  • Error: FileNotFoundError en llegir fitxers.
    • Solució: Verifica que el fitxer existeix i que el camí és correcte.

Conclusió

En aquesta secció, hem après els conceptes bàsics de Pandas, incloent la creació de Series i DataFrames, operacions bàsiques i avançades, i hem practicat amb alguns exercicis. Pandas és una eina poderosa per a l'anàlisi de dades i és essencial per a qualsevol treball relacionat amb dades en Python.
