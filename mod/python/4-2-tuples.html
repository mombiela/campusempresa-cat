4-2: Tuples

Introducció a les Tuples

Les tuples són una estructura de dades fonamental en Python, similar a les llistes, però amb una diferència clau: les tuples són immutables. Això significa que, un cop creada una tupla, no es poden modificar els seus elements. Aquesta característica les fa útils per a dades que no han de canviar durant l'execució del programa.

Creació de Tuples

Les tuples es poden crear de diverses maneres:

  1. Utilitzant parèntesis:

    my_tuple = (1, 2, 3)

  2. Sense parèntesis (separats per comes):

    my_tuple = 1, 2, 3

  3. Tupla buida:

    empty_tuple = ()

  4. Tupla amb un sol element (cal incloure una coma):

    single_element_tuple = (1,)

Accés als Elements de les Tuples

Els elements de les tuples es poden accedir utilitzant índexs, de la mateixa manera que amb les llistes:

my_tuple = (1, 2, 3, 4, 5)
print(my_tuple[0])  # Sortida: 1
print(my_tuple[3])  # Sortida: 4

Desempaquetament de Tuples

El desempaquetament de tuples permet assignar els valors de la tupla a variables individuals:

my_tuple = (1, 2, 3)
a, b, c = my_tuple
print(a)  # Sortida: 1
print(b)  # Sortida: 2
print(c)  # Sortida: 3

Operacions amb Tuples

Les tuples suporten diverses operacions:

  1. Concatenació:

    tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
result = tuple1 + tuple2
print(result)  # Sortida: (1, 2, 3, 4, 5, 6)

  2. Repetició:

    my_tuple = (1, 2, 3)
result = my_tuple * 2
print(result)  # Sortida: (1, 2, 3, 1, 2, 3)

  3. Pertinença:

    my_tuple = (1, 2, 3)
print(2 in my_tuple)  # Sortida: True
print(4 in my_tuple)  # Sortida: False

Funcions i Mètodes per a Tuples

Python proporciona diverses funcions i mètodes útils per treballar amb tuples:

  1. len(): Retorna la longitud de la tupla.

    my_tuple = (1, 2, 3)
print(len(my_tuple))  # Sortida: 3

  2. max() i min(): Retornen el valor màxim i mínim de la tupla, respectivament.

    my_tuple = (1, 2, 3)
print(max(my_tuple))  # Sortida: 3
print(min(my_tuple))  # Sortida: 1

  3. tuple(): Converteix una llista o un altre iterable en una tupla.

    my_list = [1, 2, 3]
my_tuple = tuple(my_list)
print(my_tuple)  # Sortida: (1, 2, 3)

Exercicis Pràctics

  1. Exercici 1: Crea una tupla amb els elements "Python", "Java", "C++" i "Ruby". Desempaqueta la tupla en quatre variables i imprimeix-les.

    languages = ("Python", "Java", "C++", "Ruby")
lang1, lang2, lang3, lang4 = languages
print(lang1)  # Sortida: Python
print(lang2)  # Sortida: Java
print(lang3)  # Sortida: C++
print(lang4)  # Sortida: Ruby

  2. Exercici 2: Crea una tupla amb els números de l'1 al 5. Utilitza una funció per trobar el valor màxim i mínim de la tupla.

    numbers = (1, 2, 3, 4, 5)
print(max(numbers))  # Sortida: 5
print(min(numbers))  # Sortida: 1

  3. Exercici 3: Converteix la llista [10, 20, 30, 40, 50] en una tupla i imprimeix-la.

    my_list = [10, 20, 30, 40, 50]
my_tuple = tuple(my_list)
print(my_tuple)  # Sortida: (10, 20, 30, 40, 50)

Errors Comuns i Consells

  1. Oblidar la coma en tuples d'un sol element:

    single_element_tuple = (1)  # Això és un enter, no una tupla
single_element_tuple = (1,)  # Això és una tupla

  2. Intentar modificar una tupla:

    my_tuple = (1, 2, 3)
# my_tuple[0] = 10  # Això produirà un error, ja que les tuples són immutables

Conclusió

Les tuples són una estructura de dades essencial en Python, especialment útils quan es necessita una col·lecció de dades que no canviï. La seva immutabilitat les fa ideals per a l'ús com a claus en diccionaris o per a emmagatzemar dades constants. Amb aquesta comprensió de les tuples, estàs preparat per explorar altres estructures de dades en Python.
