4-4: Conjunts (Sets)

Introducció als Conjunts

Els conjunts (sets) en Python són una col·lecció no ordenada d'elements únics. Això significa que no poden contenir elements duplicats i no mantenen cap ordre específic dels elements.

Característiques dels Conjunts

  • No ordenats: Els elements no tenen una posició fixa.
  • Únics: No permeten elements duplicats.
  • Mutables: Els conjunts poden ser modificats després de la seva creació, però els seus elements han de ser immutables.

Creació de Conjunts

Pots crear conjunts utilitzant la funció set() o utilitzant claus {}.

# Creació d'un conjunt buit
conjunt_buit = set()

# Creació d'un conjunt amb elements
conjunt = {1, 2, 3, 4, 5}

Operacions Bàsiques amb Conjunts

Afegir Elements

Pots afegir elements a un conjunt utilitzant el mètode add().

conjunt = {1, 2, 3}
conjunt.add(4)
print(conjunt)  # Sortida: {1, 2, 3, 4}

Eliminar Elements

Pots eliminar elements utilitzant els mètodes remove() o discard().

conjunt = {1, 2, 3, 4}
conjunt.remove(3)
print(conjunt)  # Sortida: {1, 2, 4}

conjunt.discard(2)
print(conjunt)  # Sortida: {1, 4}

Nota: remove() llança una excepció si l'element no existeix, mentre que discard() no.

Comprovar Existència d'un Element

Pots comprovar si un element està present en un conjunt utilitzant l'operador in.

conjunt = {1, 2, 3, 4}
print(3 in conjunt)  # Sortida: True
print(5 in conjunt)  # Sortida: False

Operacions de Conjunts

Unió

La unió de dos conjunts retorna un nou conjunt amb tots els elements dels dos conjunts.

conjunt1 = {1, 2, 3}
conjunt2 = {3, 4, 5}
unio = conjunt1 | conjunt2
print(unio)  # Sortida: {1, 2, 3, 4, 5}

Intersecció

La intersecció de dos conjunts retorna un nou conjunt amb els elements comuns als dos conjunts.

conjunt1 = {1, 2, 3}
conjunt2 = {3, 4, 5}
interseccio = conjunt1 & conjunt2
print(interseccio)  # Sortida: {3}

Diferència

La diferència de dos conjunts retorna un nou conjunt amb els elements que estan en el primer conjunt però no en el segon.

conjunt1 = {1, 2, 3}
conjunt2 = {3, 4, 5}
diferencia = conjunt1 - conjunt2
print(diferencia)  # Sortida: {1, 2}

Diferència Simètrica

La diferència simètrica de dos conjunts retorna un nou conjunt amb els elements que estan en un conjunt o en l'altre, però no en tots dos.

conjunt1 = {1, 2, 3}
conjunt2 = {3, 4, 5}
diferencia_simetrica = conjunt1 ^ conjunt2
print(diferencia_simetrica)  # Sortida: {1, 2, 4, 5}

Exercicis Pràctics

Exercici 1: Creació i Manipulació de Conjunts

  1. Crea un conjunt amb els números de l'1 al 5.
  2. Afegeix el número 6 al conjunt.
  3. Elimina el número 3 del conjunt.
  4. Comprova si el número 4 està en el conjunt.
# Solució
conjunt = {1, 2, 3, 4, 5}
conjunt.add(6)
conjunt.remove(3)
print(4 in conjunt)  # Sortida: True

Exercici 2: Operacions amb Conjunts

  1. Crea dos conjunts: {1, 2, 3} i {3, 4, 5}.
  2. Troba la unió dels dos conjunts.
  3. Troba la intersecció dels dos conjunts.
  4. Troba la diferència entre el primer i el segon conjunt.
  5. Troba la diferència simètrica entre els dos conjunts.
# Solució
conjunt1 = {1, 2, 3}
conjunt2 = {3, 4, 5}

unio = conjunt1 | conjunt2
interseccio = conjunt1 & conjunt2
diferencia = conjunt1 - conjunt2
diferencia_simetrica = conjunt1 ^ conjunt2

print(unio)  # Sortida: {1, 2, 3, 4, 5}
print(interseccio)  # Sortida: {3}
print(diferencia)  # Sortida: {1, 2}
print(diferencia_simetrica)  # Sortida: {1, 2, 4, 5}

Resum

En aquesta secció, hem après sobre els conjunts en Python, incloent com crear-los, afegir i eliminar elements, i realitzar operacions com unió, intersecció, diferència i diferència simètrica. Els conjunts són una eina poderosa per treballar amb col·leccions d'elements únics i poden ser molt útils en diverses aplicacions de programació.
