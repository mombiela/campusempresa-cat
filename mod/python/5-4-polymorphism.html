5-4: Polimorfisme

Introducció al Polimorfisme

El polimorfisme és un concepte fonamental en la programació orientada a objectes (POO) que permet que objectes de diferents classes siguin tractats com objectes del mateix tipus de classe base. Això és possible gràcies a la capacitat de les classes derivades de redefinir els mètodes de la classe base.

Conceptes Clau

  1. Polimorfisme de Sobrecàrrega: Permet que múltiples mètodes amb el mateix nom coexisteixin en una classe, diferenciant-se pel tipus o nombre de paràmetres.
  2. Polimorfisme d'Inclusió: Permet que una referència de la classe base apunti a objectes de les seves subclasses.

Exemple Pràctic

Definició de Classes

class Animal:
    def make_sound(self):
        raise NotImplementedError("Subclass must implement abstract method")

class Dog(Animal):
    def make_sound(self):
        return "Woof!"

class Cat(Animal):
    def make_sound(self):
        return "Meow!"

En aquest exemple, Animal és la classe base amb un mètode abstracte make_sound. Les classes Dog i Cat hereten d'Animal i implementen el mètode make_sound.

Ús del Polimorfisme

def animal_sound(animals):
    for animal in animals:
        print(animal.make_sound())

animals = [Dog(), Cat()]
animal_sound(animals)

Sortida:

Woof!
Meow!

En aquest cas, la funció animal_sound pot treballar amb qualsevol objecte que sigui una instància de Animal o de les seves subclasses, gràcies al polimorfisme.

Exercicis Pràctics

Exercici 1: Implementació de Polimorfisme

Descripció: Crea una classe base Shape amb un mètode area i dues subclasses Circle i Rectangle que implementin el mètode area.

import math

class Shape:
    def area(self):
        raise NotImplementedError("Subclass must implement abstract method")

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return math.pi * self.radius ** 2

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

shapes = [Circle(5), Rectangle(4, 6)]
for shape in shapes:
    print(f"The area is: {shape.area()}")

Sortida esperada:

The area is: 78.53981633974483
The area is: 24

Exercici 2: Polimorfisme amb Mètodes de Classe

Descripció: Crea una classe base Vehicle amb un mètode move i dues subclasses Car i Boat que implementin el mètode move.

class Vehicle:
    def move(self):
        raise NotImplementedError("Subclass must implement abstract method")

class Car(Vehicle):
    def move(self):
        return "Driving on the road"

class Boat(Vehicle):
    def move(self):
        return "Sailing on the water"

vehicles = [Car(), Boat()]
for vehicle in vehicles:
    print(vehicle.move())

Sortida esperada:

Driving on the road
Sailing on the water

Errors Comuns i Consells

  1. No Implementar Mètodes Abstractes: Si una subclasse no implementa un mètode abstracte de la classe base, es produirà un error. Assegura't que totes les subclasses implementin els mètodes abstractes.
  2. Confusió amb Sobrecàrrega: Python no suporta la sobrecàrrega de mètodes de la mateixa manera que altres llenguatges com Java o C++. Utilitza paràmetres per defecte o *args i **kwargs per gestionar diferents nombres d'arguments.

Resum

El polimorfisme permet que objectes de diferents classes siguin tractats de manera uniforme, sempre que aquestes classes comparteixin una classe base comuna. Això facilita la creació de codi flexible i reutilitzable. En aquest tema, hem vist com definir i utilitzar el polimorfisme a través d'exemples pràctics i exercicis.
