Tema 3-3: Funcions Lambda

Introducció a les Funcions Lambda

Les funcions lambda, també conegudes com a funcions anònimes, són funcions petites i sense nom que es defineixen utilitzant la paraula clau lambda. Aquestes funcions són especialment útils per a operacions senzilles que es poden expressar en una sola línia de codi.

Sintaxi de les Funcions Lambda

La sintaxi bàsica d'una funció lambda és la següent:

lambda arguments: expressió
  • arguments: Els arguments que la funció lambda accepta.
  • expressió: Una única expressió que es calcula i es retorna.

Exemple Bàsic

Vegem un exemple senzill d'una funció lambda que suma dos números:

sumar = lambda x, y: x + y
resultat = sumar(5, 3)
print(resultat)  # Sortida: 8

Comparació amb Funcions Definides amb `def`

Les funcions lambda són útils per a operacions simples i ràpides, però tenen limitacions en comparació amb les funcions definides amb def. Aquí hi ha una comparació:

Funció Lambda Funció Definida amb def
Sintaxi concisa i en una sola línia Pot tenir múltiples línies de codi
No té nom propi (anònima) Té un nom propi
Limitada a una sola expressió Pot contenir múltiples expressions i codi complex
No pot contenir docstrings Pot contenir docstrings per documentació

Ús de Funcions Lambda amb Funcions d'Ordinació

Les funcions lambda són sovint utilitzades amb funcions d'ordenació com sorted(), filter(), i map(). Vegem alguns exemples:

Exemple amb sorted()

llista = [(1, 2), (3, 1), (5, 0)]
llista_ordenada = sorted(llista, key=lambda x: x[1])
print(llista_ordenada)  # Sortida: [(5, 0), (3, 1), (1, 2)]

Exemple amb filter()

numeros = [1, 2, 3, 4, 5, 6]
parells = list(filter(lambda x: x % 2 == 0, numeros))
print(parells)  # Sortida: [2, 4, 6]

Exemple amb map()

numeros = [1, 2, 3, 4, 5]
quadrats = list(map(lambda x: x ** 2, numeros))
print(quadrats)  # Sortida: [1, 4, 9, 16, 25]

Exercicis Pràctics

Exercici 1: Filtrar Nombres Positius

Escriu una funció lambda que filtri els nombres positius d'una llista.

numeros = [-2, -1, 0, 1, 2]
positius = list(filter(lambda x: x > 0, numeros))
print(positius)  # Sortida esperada: [1, 2]

Exercici 2: Ordenar per Segon Element

Utilitza una funció lambda per ordenar una llista de tuples pel segon element.

llista = [(2, 3), (1, 2), (3, 1)]
llista_ordenada = sorted(llista, key=lambda x: x[1])
print(llista_ordenada)  # Sortida esperada: [(3, 1), (1, 2), (2, 3)]

Exercici 3: Transformar Nombres

Utilitza una funció lambda amb map() per transformar una llista de nombres multiplicant-los per 10.

numeros = [1, 2, 3, 4, 5]
transformats = list(map(lambda x: x * 10, numeros))
print(transformats)  # Sortida esperada: [10, 20, 30, 40, 50]

Solucions als Exercicis

Solució Exercici 1

numeros = [-2, -1, 0, 1, 2]
positius = list(filter(lambda x: x > 0, numeros))
print(positius)  # Sortida: [1, 2]

Solució Exercici 2

llista = [(2, 3), (1, 2), (3, 1)]
llista_ordenada = sorted(llista, key=lambda x: x[1])
print(llista_ordenada)  # Sortida: [(3, 1), (1, 2), (2, 3)]

Solució Exercici 3

numeros = [1, 2, 3, 4, 5]
transformats = list(map(lambda x: x * 10, numeros))
print(transformats)  # Sortida: [10, 20, 30, 40, 50]

Conclusió

Les funcions lambda són una eina poderosa i concisa per a operacions simples en Python. Tot i que tenen limitacions en comparació amb les funcions definides amb def, són molt útils en combinació amb funcions com sorted(), filter(), i map(). Practicar amb aquestes funcions t'ajudarà a comprendre millor com i quan utilitzar-les de manera efectiva.
