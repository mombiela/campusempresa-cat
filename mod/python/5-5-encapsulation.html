5-5: Encapsulació

Introducció a l'Encapsulació

L'encapsulació és un dels quatre pilars de la programació orientada a objectes (POO), juntament amb l'herència, el polimorfisme i l'abstracció. L'encapsulació consisteix en restringir l'accés directe a alguns components d'un objecte, de manera que es pugui protegir la integritat de les dades i evitar que es modifiquin de manera inapropiada. Això es fa utilitzant modificadors d'accés per controlar la visibilitat dels atributs i mètodes.

Conceptes Clau

  • Atributs Privats: Atributs que no poden ser accedits directament des de fora de la classe.
  • Mètodes Accessors (Getters): Mètodes que permeten accedir als valors dels atributs privats.
  • Mètodes Modificadors (Setters): Mètodes que permeten modificar els valors dels atributs privats de manera controlada.

Exemples Pràctics

Definició d'Atributs Privats

En Python, els atributs privats es defineixen prefixant el nom de l'atribut amb dos guions baixos (__).

class Persona:
    def __init__(self, nom, edat):
        self.__nom = nom
        self.__edat = edat

    def mostrar_informacio(self):
        print(f"Nom: {self.__nom}, Edat: {self.__edat}")

# Crear una instància de la classe Persona
persona = Persona("Anna", 30)
persona.mostrar_informacio()

# Intentar accedir als atributs privats directament (això generarà un error)
# print(persona.__nom)  # AttributeError: 'Persona' object has no attribute '__nom'

Utilització de Getters i Setters

Els getters i setters permeten accedir i modificar els atributs privats de manera controlada.

class Persona:
    def __init__(self, nom, edat):
        self.__nom = nom
        self.__edat = edat

    # Getter per a l'atribut nom
    def get_nom(self):
        return self.__nom

    # Setter per a l'atribut nom
    def set_nom(self, nom):
        self.__nom = nom

    # Getter per a l'atribut edat
    def get_edat(self):
        return self.__edat

    # Setter per a l'atribut edat
    def set_edat(self, edat):
        if edat > 0:
            self.__edat = edat
        else:
            print("L'edat ha de ser un valor positiu")

# Crear una instància de la classe Persona
persona = Persona("Anna", 30)

# Utilitzar getters per accedir als atributs privats
print(persona.get_nom())  # Anna
print(persona.get_edat())  # 30

# Utilitzar setters per modificar els atributs privats
persona.set_nom("Maria")
persona.set_edat(25)

print(persona.get_nom())  # Maria
print(persona.get_edat())  # 25

# Intentar establir una edat negativa (això generarà un missatge d'error)
persona.set_edat(-5)  # L'edat ha de ser un valor positiu

Exercicis Pràctics

Exercici 1: Crear una Classe amb Encapsulació

Crea una classe CompteBancari amb els atributs privats __saldo i __titular. Implementa els getters i setters per aquests atributs, assegurant-te que el saldo no pugui ser negatiu.

class CompteBancari:
    def __init__(self, titular, saldo):
        self.__titular = titular
        self.__saldo = saldo

    def get_titular(self):
        return self.__titular

    def set_titular(self, titular):
        self.__titular = titular

    def get_saldo(self):
        return self.__saldo

    def set_saldo(self, saldo):
        if saldo >= 0:
            self.__saldo = saldo
        else:
            print("El saldo no pot ser negatiu")

# Crear una instància de la classe CompteBancari
compte = CompteBancari("Joan", 1000)

# Utilitzar getters per accedir als atributs privats
print(compte.get_titular())  # Joan
print(compte.get_saldo())  # 1000

# Utilitzar setters per modificar els atributs privats
compte.set_titular("Laura")
compte.set_saldo(1500)

print(compte.get_titular())  # Laura
print(compte.get_saldo())  # 1500

# Intentar establir un saldo negatiu (això generarà un missatge d'error)
compte.set_saldo(-500)  # El saldo no pot ser negatiu

Errors Comuns i Consells

  • Accés Directe als Atributs Privats: Intentar accedir directament als atributs privats des de fora de la classe generarà un error. Utilitza sempre els getters i setters.
  • Validació en Setters: Assegura't de validar els valors dins dels setters per mantenir la integritat de les dades.

Resum

En aquesta secció, hem après sobre l'encapsulació en Python, com definir atributs privats i com utilitzar getters i setters per accedir i modificar aquests atributs de manera controlada. L'encapsulació ajuda a protegir la integritat de les dades i a evitar modificacions inapropiades. Hem vist exemples pràctics i hem realitzat exercicis per reforçar els conceptes apresos.
