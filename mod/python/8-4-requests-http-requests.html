8-4: Requests per a Peticions HTTP

Introducció

La llibreria requests és una de les més populars i utilitzades en Python per fer peticions HTTP. És coneguda per la seva simplicitat i facilitat d'ús. Amb requests, pots enviar peticions HTTP/1.1 amb mètodes com GET, POST, PUT, DELETE, etc., de manera senzilla i intuïtiva.

Instal·lació

Abans de començar a utilitzar requests, necessitem instal·lar-la. Pots fer-ho utilitzant pip:

pip install requests

Enviant una Petició GET

Una de les operacions més comunes és enviar una petició GET per obtenir dades d'un servidor. Aquí tens un exemple bàsic:

import requests

response = requests.get('https://api.github.com')

print(response.status_code)  # Mostra el codi d'estat de la resposta
print(response.text)         # Mostra el contingut de la resposta

Explicació:

  • requests.get(url) envia una petició GET a l'URL especificat.
  • response.status_code retorna el codi d'estat HTTP de la resposta.
  • response.text retorna el contingut de la resposta com a cadena de text.

Enviant una Petició POST

Les peticions POST s'utilitzen per enviar dades al servidor. Aquí tens un exemple:

import requests

url = 'https://httpbin.org/post'
data = {'key': 'value'}

response = requests.post(url, data=data)

print(response.status_code)
print(response.json())  # Mostra el contingut de la resposta en format JSON

Explicació:

  • requests.post(url, data=data) envia una petició POST a l'URL especificat amb les dades proporcionades.
  • response.json() retorna el contingut de la resposta en format JSON.

Enviant Headers i Paràmetres

Pots enviar headers i paràmetres en les teves peticions per personalitzar-les:

import requests

url = 'https://api.github.com/search/repositories'
params = {'q': 'requests+language:python'}
headers = {'User-Agent': 'my-app'}

response = requests.get(url, params=params, headers=headers)

print(response.status_code)
print(response.json())

Explicació:

  • params és un diccionari de paràmetres que s'afegeixen a l'URL.
  • headers és un diccionari de headers HTTP que s'afegeixen a la petició.

Gestió d'Errors

És important gestionar els errors quan treballes amb peticions HTTP. Aquí tens un exemple de com fer-ho:

import requests

try:
    response = requests.get('https://api.github.com/invalid-endpoint')
    response.raise_for_status()  # Genera una excepció per codis d'estat d'error
except requests.exceptions.HTTPError as err:
    print(f'HTTP error occurred: {err}')
except Exception as err:
    print(f'Other error occurred: {err}')
else:
    print('Success!')

Explicació:

  • response.raise_for_status() genera una excepció si el codi d'estat indica un error.
  • requests.exceptions.HTTPError captura errors HTTP específics.
  • Exception captura altres tipus d'errors.

Exercicis Pràctics

  1. Exercici 1: Petició GET

    • Envia una petició GET a https://jsonplaceholder.typicode.com/posts i mostra el codi d'estat i el contingut de la resposta.
    import requests

response = requests.get('https://jsonplaceholder.typicode.com/posts')
print(response.status_code)
print(response.text)

  2. Exercici 2: Petició POST

    • Envia una petició POST a https://jsonplaceholder.typicode.com/posts amb les dades {'title': 'foo', 'body': 'bar', 'userId': 1} i mostra el codi d'estat i el contingut de la resposta.
    import requests

url = 'https://jsonplaceholder.typicode.com/posts'
data = {'title': 'foo', 'body': 'bar', 'userId': 1}

response = requests.post(url, data=data)
print(response.status_code)
print(response.json())

  3. Exercici 3: Gestió d'Errors

    • Envia una petició GET a un endpoint invàlid https://jsonplaceholder.typicode.com/invalid i gestiona l'error adequadament.
    import requests

try:
    response = requests.get('https://jsonplaceholder.typicode.com/invalid')
    response.raise_for_status()
except requests.exceptions.HTTPError as err:
    print(f'HTTP error occurred: {err}')
except Exception as err:
    print(f'Other error occurred: {err}')
else:
    print('Success!')

Conclusió

La llibreria requests és una eina poderosa i fàcil d'utilitzar per fer peticions HTTP en Python. Hem vist com enviar peticions GET i POST, com afegir headers i paràmetres, i com gestionar errors. Amb aquests coneixements, estaràs preparat per interactuar amb APIs i serveis web de manera eficient.
