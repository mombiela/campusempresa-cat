3-1: Definició i Crida de Funcions

Introducció

Les funcions són blocs de codi reutilitzables que permeten estructurar el codi de manera més clara i modular. En aquest tema, aprendrem com definir i cridar funcions en Python.

Objectius

  • Entendre la sintaxi per definir funcions en Python.
  • Aprendre a cridar funcions.
  • Comprendre la importància de les funcions per a la modularitat i la reutilització del codi.

Definició de Funcions

Per definir una funció en Python, utilitzem la paraula clau def seguida del nom de la funció, parèntesis i dos punts. El codi que forma part de la funció s'ha d'indentar.

Sintaxi

def nom_de_la_funció(paràmetres):
    # Bloc de codi
    return valor_de_retorn

Exemple

def saludar(nom):
    missatge = f"Hola, {nom}!"
    return missatge

En aquest exemple, hem definit una funció anomenada saludar que pren un paràmetre nom i retorna un missatge de salutació.

Crida de Funcions

Per cridar una funció, simplement utilitzem el nom de la funció seguit de parèntesis, passant els arguments necessaris.

Exemple

resultat = saludar("Anna")
print(resultat)  # Sortida: Hola, Anna!

Exemples Pràctics

Exemple 1: Funció sense Paràmetres

def dir_hola():
    print("Hola, món!")

# Crida de la funció
dir_hola()

Exemple 2: Funció amb Paràmetres

def sumar(a, b):
    return a + b

# Crida de la funció
resultat = sumar(3, 5)
print(resultat)  # Sortida: 8

Exemple 3: Funció amb Valor de Retorn

def multiplicar(a, b):
    producte = a * b
    return producte

# Crida de la funció
resultat = multiplicar(4, 7)
print(resultat)  # Sortida: 28

Exercicis Pràctics

Exercici 1: Funció de Salutació

Defineix una funció anomenada salutacio que prengui un nom com a paràmetre i imprimeixi un missatge de salutació.

def salutacio(nom):
    print(f"Hola, {nom}!")

# Crida de la funció
salutacio("Pere")

Exercici 2: Funció de Càlcul de l'Àrea d'un Rectangle

Defineix una funció anomenada area_rectangle que prengui l'amplada i l'alçada d'un rectangle com a paràmetres i retorni l'àrea.

def area_rectangle(amplada, alçada):
    return amplada * alçada

# Crida de la funció
area = area_rectangle(5, 10)
print(area)  # Sortida: 50

Errors Comuns

  1. Oblidar els Parèntesis en la Crida de la Funció:

    dir_hola  # Incorrecte
dir_hola()  # Correcte

  2. No Indentar el Bloc de Codi dins de la Funció:

    def dir_hola():
print("Hola, món!")  # Incorrecte

Consells Addicionals

  • Utilitza noms de funció descriptius per millorar la llegibilitat del codi.
  • Mantén les funcions curtes i enfocades a una sola tasca.
  • Documenta les funcions amb comentaris o docstrings per explicar què fan i quins paràmetres esperen.

Resum

En aquesta secció, hem après com definir i cridar funcions en Python. Les funcions són una eina poderosa per organitzar i reutilitzar el codi. Hem vist exemples pràctics i hem practicat amb exercicis per reforçar els conceptes apresos.

Preparació per al Següent Tema

Ara que entenem com definir i cridar funcions, estem preparats per explorar els arguments de funció i els valors de retorn en més detall en el següent tema.
