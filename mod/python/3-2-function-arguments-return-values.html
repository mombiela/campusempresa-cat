3-2: Arguments de Funció i Valors de Retorn

Introducció

En aquest tema, explorarem com passar arguments a les funcions i com retornar valors des de les funcions en Python. Aquest concepte és fonamental per escriure codi modular i reutilitzable.

Arguments de Funció

Els arguments són valors que es passen a una funció quan aquesta és cridada. Python permet diversos tipus d'arguments:

  1. Arguments posicionals: Els arguments es passen a la funció segons la seva posició.
  2. Arguments per nom (keyword arguments): Els arguments es passen a la funció especificant el nom del paràmetre.
  3. Arguments per defecte: Els arguments tenen valors per defecte que s'utilitzen si no es passen valors.
  4. Arguments variables: Permeten passar un nombre variable d'arguments a una funció.

Arguments posicionals

def suma(a, b):
    return a + b

resultat = suma(3, 4)
print(resultat)  # Sortida: 7

Arguments per nom

def salutacio(nom, missatge):
    return f"{missatge}, {nom}!"

resultat = salutacio(missatge="Hola", nom="Anna")
print(resultat)  # Sortida: Hola, Anna!

Arguments per defecte

def salutacio(nom, missatge="Hola"):
    return f"{missatge}, {nom}!"

resultat = salutacio("Anna")
print(resultat)  # Sortida: Hola, Anna!

Arguments variables

  • Arguments posicionals variables (*args)
def suma(*args):
    return sum(args)

resultat = suma(1, 2, 3, 4)
print(resultat)  # Sortida: 10
  • Arguments per nom variables (**kwargs)
def mostrar_info(**kwargs):
    for clau, valor in kwargs.items():
        print(f"{clau}: {valor}")

mostrar_info(nom="Anna", edat=25, ciutat="Barcelona")
# Sortida:
# nom: Anna
# edat: 25
# ciutat: Barcelona

Valors de Retorn

Les funcions poden retornar valors utilitzant la paraula clau return. Una funció pot retornar múltiples valors com a tuple.

Retornant un sol valor

def quadrat(x):
    return x * x

resultat = quadrat(5)
print(resultat)  # Sortida: 25

Retornant múltiples valors

def operacions(a, b):
    suma = a + b
    resta = a - b
    return suma, resta

resultat = operacions(10, 5)
print(resultat)  # Sortida: (15, 5)

Exercicis Pràctics

  1. Exercici 1:

    Escriu una funció multiplica que accepti dos arguments i retorni el seu producte.

    def multiplica(a, b):
    return a * b

# Prova la funció
resultat = multiplica(3, 4)
print(resultat)  # Sortida esperada: 12

  2. Exercici 2:

    Escriu una funció info_persona que accepti arguments per nom, edat i ciutat, i retorni una cadena amb aquesta informació.

    def info_persona(nom, edat, ciutat):
    return f"{nom} té {edat} anys i viu a {ciutat}."

# Prova la funció
resultat = info_persona("Anna", 25, "Barcelona")
print(resultat)  # Sortida esperada: Anna té 25 anys i viu a Barcelona.

  3. Exercici 3:

    Escriu una funció suma_total que accepti un nombre variable d'arguments i retorni la seva suma.

    def suma_total(*args):
    return sum(args)

# Prova la funció
resultat = suma_total(1, 2, 3, 4, 5)
print(resultat)  # Sortida esperada: 15

Errors Comuns i Consells

  • Oblidar els valors per defecte: Si una funció té arguments amb valors per defecte, assegura't de col·locar-los després dels arguments obligatoris.

    # Incorrecte
def funcio(a=1, b):
    pass

# Correcte
def funcio(a, b=1):
    pass

  • Arguments per nom i posicionals: Quan es barregen arguments posicionals i per nom en una crida de funció, els arguments posicionals han d'anar primer.

    # Incorrecte
funcio(a=1, 2)

# Correcte
funcio(1, b=2)

Resum

En aquesta secció, hem après com passar arguments a les funcions i com retornar valors. Hem vist diferents tipus d'arguments, incloent arguments posicionals, per nom, per defecte i variables. També hem explorat com retornar un o múltiples valors des de les funcions. Aquests conceptes són essencials per escriure funcions flexibles i reutilitzables en Python.

En el següent tema, explorarem les funcions lambda, que són funcions anònimes i concises en Python.
