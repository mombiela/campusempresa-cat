Introducció a Python
Següent ►

Què és Python?

Python és un llenguatge de programació d'alt nivell, interpretat i de propòsit general. Va ser creat per Guido van Rossum i va ser llançat per primera vegada el 1991. Python és conegut per la seva sintaxi clara i llegible, la qual cosa el fa ideal per a principiants, però també és prou potent per a ser utilitzat en aplicacions complexes.

Característiques Clau de Python

  • Simplicitat: La sintaxi de Python és senzilla i fàcil d'aprendre, la qual cosa permet als programadors centrar-se en la resolució de problemes en lloc de preocupar-se per la complexitat del llenguatge.
  • Interpretat: Python és un llenguatge interpretat, el que significa que el codi es pot executar directament sense necessitat de compilar-lo prèviament.
  • Tipatge Dinàmic: No cal declarar el tipus de variable abans d'utilitzar-la. Python determina el tipus en temps d'execució.
  • Alt Nivell: Python gestiona automàticament la memòria i altres recursos, el que permet als programadors centrar-se en la lògica de l'aplicació.
  • Extensible: Python es pot estendre amb mòduls escrits en altres llenguatges com C o C++.
  • Gran Comunitat i Biblioteca Estàndard: Python té una comunitat molt activa i una àmplia biblioteca estàndard que cobreix moltes funcionalitats comunes.

Aplicacions de Python

Python s'utilitza en una àmplia varietat de camps, incloent:

  • Desenvolupament Web: Frameworks com Django i Flask permeten construir aplicacions web robustes.
  • Ciència de Dades: Llibreries com NumPy, Pandas i Matplotlib són àmpliament utilitzades per a l'anàlisi de dades.
  • Machine Learning: Llibreries com scikit-learn, TensorFlow i PyTorch permeten desenvolupar models de machine learning.
  • Automatització i Scripts: Python és ideal per a l'automatització de tasques repetitives i l'escriptura de scripts.
  • Desenvolupament de Jocs: Llibreries com Pygame permeten desenvolupar jocs senzills.
  • Aplicacions de Xarxa: Python es pot utilitzar per desenvolupar aplicacions de xarxa i serveis web.

Història de Python

Python va ser creat per Guido van Rossum a la fi dels anys 80 com a successor del llenguatge ABC. La primera versió pública, Python 0.9.0, va ser llançada el 1991. Des de llavors, Python ha evolucionat significativament, amb la versió 2.0 llançada el 2000 i la versió 3.0 llançada el 2008. Python 3 va introduir molts canvis que no són compatibles amb Python 2, però la comunitat ha adoptat àmpliament Python 3.

Instal·lació de Python

Pas 1: Descarregar Python

  1. Visita el lloc web oficial de Python: python.org.
  2. Fes clic a "Downloads" i selecciona la versió adequada per al teu sistema operatiu (Windows, macOS, Linux).

Pas 2: Instal·lar Python

Windows

  1. Executa l'instal·lador descarregat.
  2. Assegura't de marcar l'opció "Add Python to PATH".
  3. Fes clic a "Install Now".

macOS

  1. Obre el paquet .pkg descarregat.
  2. Segueix les instruccions de l'instal·lador.

Linux

  1. Obre una terminal.
  2. Utilitza el gestor de paquets del teu sistema per instal·lar Python. Per exemple, en Ubuntu: 
    sudo apt update
sudo apt install python3

Verificar la Instal·lació

Per verificar que Python s'ha instal·lat correctament, obre una terminal o el símbol del sistema i executa:

python --version

o

python3 --version

Hauries de veure la versió de Python instal·lada.

Primer Programa en Python

Ara que tens Python instal·lat, és hora d'escriure el teu primer programa. Tradicionalment, el primer programa que s'escriu en qualsevol llenguatge de programació és "Hello, World!".

Exemple de Codi

Obre un editor de text (com Visual Studio Code, Sublime Text o fins i tot el bloc de notes) i escriu el següent codi:

print("Hello, World!")

Executar el Programa

Guarda el fitxer amb l'extensió .py, per exemple, hello.py. Després, obre una terminal o el símbol del sistema, navega fins al directori on has guardat el fitxer i executa:

python hello.py

Hauries de veure la sortida:

Hello, World!

Conclusió

En aquesta lliçó, hem après què és Python, les seves característiques clau, les seves aplicacions, una mica de la seva història i com instal·lar-lo. També hem escrit i executat el nostre primer programa en Python. En la propera lliçó, configurarem l'entorn de desenvolupament per treballar de manera més eficient amb Python.
