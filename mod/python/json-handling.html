Introducció al JSON

  • JSON (JavaScript Object Notation) és un format lleuger d'intercanvi de dades.
  • És fàcil de llegir i escriure per als humans, i fàcil de parsejar i generar per a les màquines.
  • JSON s'utilitza sovint per transmetre dades entre un servidor i una aplicació web.

Conceptes bàsics del JSON

  • Les dades JSON s'escriuen com a parells clau/valor.
  • Les claus són cadenes de text, i els valors poden ser cadenes de text, números, objectes, arrays, true, false, o null.

Exemple de dades JSON

{
  "name": "John",
  "age": 30,
  "isStudent": false,
  "courses": ["Math", "Science"],
  "address": {
    "street": "123 Main St",
    "city": "Anytown"
  }
}

Treballar amb JSON en Python

  • Python proporciona un mòdul integrat anomenat json per gestionar dades JSON.

Importar el mòdul JSON

import json

Llegir dades JSON

  • Les dades JSON es poden llegir des d'una cadena de text o un fitxer.

Llegir JSON des d'una cadena de text

import json

json_string = '{"name": "John", "age": 30, "isStudent": false}'
data = json.loads(json_string)

print(data)
# Sortida: {'name': 'John', 'age': 30, 'isStudent': False}

Llegir JSON des d'un fitxer

import json

with open('data.json', 'r') as file:
    data = json.load(file)

print(data)
# Sortida: {'name': 'John', 'age': 30, 'isStudent': False}

Escriure dades JSON

  • Les dades JSON es poden escriure en una cadena de text o un fitxer.

Escriure JSON en una cadena de text

import json

data = {
    "name": "John",
    "age": 30,
    "isStudent": False
}

json_string = json.dumps(data)
print(json_string)
# Sortida: {"name": "John", "age": 30, "isStudent": false}

Escriure JSON en un fitxer

import json

data = {
    "name": "John",
    "age": 30,
    "isStudent": False
}

with open('data.json', 'w') as file:
    json.dump(data, file)

Gestió avançada de JSON

  • Personalitzar la codificació i descodificació de JSON.
  • Gestionar tipus de dades complexes.

Personalitzar la codificació de JSON

  • Utilitza el paràmetre default en json.dumps() per gestionar objectes no serialitzables.
import json
from datetime import datetime

def datetime_handler(x):
    if isinstance(x, datetime):
        return x.isoformat()
    raise TypeError("Unknown type")

data = {
    "name": "John",
    "joined": datetime(2021, 5, 17)
}

json_string = json.dumps(data, default=datetime_handler)
print(json_string)
# Sortida: {"name": "John", "joined": "2021-05-17T00:00:00"}

Personalitzar la descodificació de JSON

  • Utilitza el paràmetre object_hook en json.loads() per gestionar la descodificació personalitzada.
import json
from datetime import datetime

def datetime_parser(dct):
    for k, v in dct.items():
        if isinstance(v, str) and len(v) == 10:
            try:
                dct[k] = datetime.strptime(v, '%Y-%m-%d')
            except ValueError:
                pass
    return dct

json_string = '{"name": "John", "joined": "2021-05-17"}'
data = json.loads(json_string, object_hook=datetime_parser)
print(data)
# Sortida: {'name': 'John', 'joined': datetime.datetime(2021, 5, 17, 0, 0)}

Conclusió

  • JSON és un format versàtil per a l'intercanvi de dades.
  • El mòdul json de Python proporciona mètodes senzills per llegir i escriure dades JSON.
  • La personalització avançada permet gestionar tipus de dades complexes i codificació/descodificació personalitzada.
  • Dominar la gestió de JSON en Python és essencial per treballar amb APIs i serveis web.
