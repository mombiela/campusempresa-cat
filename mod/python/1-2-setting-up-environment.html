1-2: Configuració de l'Entorn de Desenvolupament

Objectius

En aquesta secció, aprendràs a:

  • Instal·lar Python al teu sistema.
  • Configurar un entorn de desenvolupament integrat (IDE).
  • Verificar la instal·lació de Python.
  • Instal·lar i utilitzar un gestor de paquets.

1. Instal·lació de Python

Pas 1: Descarregar Python

  1. Visita la pàgina oficial de Python: python.org.
  2. Fes clic a "Downloads" i selecciona la versió adequada per al teu sistema operatiu (Windows, macOS, Linux).

Pas 2: Instal·lar Python

  • Windows:

    1. Executa l'instal·lador descarregat.
    2. Assegura't de marcar l'opció "Add Python to PATH".
    3. Fes clic a "Install Now".

  • macOS:

    1. Obre el fitxer .pkg descarregat.
    2. Segueix les instruccions de l'instal·lador.

  • Linux:

    1. Obre una terminal.
    2. Executa la comanda següent (pot requerir permisos d'administrador): 
      sudo apt-get install python3

2. Verificar la Instal·lació de Python

Després d'instal·lar Python, és important verificar que la instal·lació s'ha realitzat correctament.

Pas 1: Obrir una Terminal o Símbol del Sistema

  • Windows: Cerca "cmd" o "PowerShell" i obre'l.
  • macOS/Linux: Obre l'aplicació "Terminal".

Pas 2: Executar la Comanda de Verificació

Escriu la següent comanda i prem Enter:

python --version

o

python3 --version

Hauries de veure un missatge que indica la versió de Python instal·lada, per exemple:

Python 3.10.0

3. Configuració d'un Entorn de Desenvolupament Integrat (IDE)

Un IDE facilita l'escriptura de codi amb funcionalitats com l'autocompletat, el ressaltat de sintaxi i la depuració.

Opcions Recomanades d'IDE

  • Visual Studio Code (VS Code):

    1. Descarrega'l des de code.visualstudio.com.
    2. Instal·la l'extensió de Python des del Marketplace de VS Code.

  • PyCharm:

    1. Descarrega'l des de jetbrains.com/pycharm.
    2. Instal·la la versió Community (gratuïta) o Professional (de pagament).

Configuració de VS Code per a Python

  1. Obre VS Code.
  2. Instal·la l'extensió de Python:
    • Fes clic a la icona d'extensions a la barra lateral esquerra.
    • Cerca "Python" i fes clic a "Install" a l'extensió desenvolupada per Microsoft.
  3. Configura l'intèrpret de Python:
    • Obre la paleta de comandes (Ctrl+Shift+P o Cmd+Shift+P).
    • Escriu "Python: Select Interpreter" i selecciona la versió de Python instal·lada.

4. Instal·lació i Ús d'un Gestor de Paquets

Python utilitza pip com a gestor de paquets per instal·lar biblioteques i dependències.

Verificar la Instal·lació de pip

Obre una terminal i executa:

pip --version

Hauries de veure un missatge que indica la versió de pip instal·lada, per exemple:

pip 21.2.4 from /usr/local/lib/python3.10/site-packages/pip (python 3.10)

Instal·lar un Paquet amb pip

Per exemple, per instal·lar la biblioteca requests:

pip install requests

Exercici Pràctic

  1. Verifica la Instal·lació de Python:

    • Obre una terminal.
    • Executa python --version o python3 --version.
    • Assegura't que es mostra la versió de Python.

  2. Instal·la i Configura VS Code:

    • Descarrega i instal·la VS Code.
    • Instal·la l'extensió de Python.
    • Configura l'intèrpret de Python.

  3. Instal·la un Paquet amb pip:

    • Obre una terminal.
    • Executa pip install requests.
    • Verifica la instal·lació executant pip list i cercant requests a la llista de paquets instal·lats.

Conclusió

En aquesta secció, has après a instal·lar Python, configurar un IDE com VS Code, verificar la instal·lació de Python i utilitzar pip per gestionar paquets. Ara estàs preparat per començar a escriure i executar codi Python en un entorn de desenvolupament adequat. En la següent secció, explorarem la sintaxi bàsica de Python i escriurem el nostre primer programa "Hola Món".
