11-3: Estil de Codi i PEP 8

Introducció a PEP 8

PEP 8 és la guia d'estil per a Python, que proporciona recomanacions sobre com escriure codi Python de manera clara i coherent. Seguir PEP 8 ajuda a mantenir el codi llegible i facilita la col·laboració entre desenvolupadors.

Conceptes Clau de PEP 8

  1. Indentació

    • Utilitza 4 espais per a la indentació. No utilitzis tabuladors.
    def funció_exemple():
    if True:
        print("Hola, món!")

  2. Longitud de Línia

    • Limita la longitud de les línies a 79 caràcters.
    • Per a docstrings i comentaris, limita la longitud a 72 caràcters.

  3. Línies en Blanc

    • Utilitza línies en blanc per separar funcions i classes, així com blocs de codi dins de funcions.
    class ExempleClasse:
    def __init__(self):
        pass

    def mètode(self):
        pass

  4. Imports

    • Col·loca tots els imports al començament del fitxer.
    • Utilitza una línia per import.
    import os
import sys

  5. Espais en Blanc en Expressions i Sentències

    • Evita espais en blanc innecessaris.
    # Correcte:
x = 1
y = x + 2

# Incorrecte:
x= 1
y =x+ 2

  6. Comentaris

    • Utilitza comentaris per explicar el codi complex.
    • Els comentaris han de ser concisos i clars.
    # Aquest és un comentari d'una sola línia

"""
Aquest és un comentari de múltiples línies
o docstring.
"""

  7. Nomenclatura

    • Utilitza noms descriptius per a variables, funcions, classes, etc.
    • Segueix les convencions de nomenclatura:
      • Funcions i variables: nom_variable
      • Classes: NomClasse
      • Constants: NOM_CONSTANT

Exemples Pràctics

Exemple 1: Funció ben escrita segons PEP 8

def calcular_suma(a, b):
    """
    Calcula la suma de dos números.

    Arguments:
    a -- primer número
    b -- segon número

    Retorna:
    La suma de a i b.
    """
    suma = a + b
    return suma

Exemple 2: Classe ben escrita segons PEP 8

class Cotxe:
    def __init__(self, marca, model):
        self.marca = marca
        self.model = model

    def descripció(self):
        """
        Retorna una descripció del cotxe.
        """
        return f"{self.marca} {self.model}"

Exercicis Pràctics

Exercici 1: Correcció d'Estil

Corregiu el següent codi per complir amb PEP 8:

def suma(a,b): return a+b
class persona: def __init__(self,nom,edat): self.nom=nom; self.edat=edat

Solució:

def suma(a, b):
    return a + b

class Persona:
    def __init__(self, nom, edat):
        self.nom = nom
        self.edat = edat

Exercici 2: Escriure una Funció Seguint PEP 8

Escriviu una funció multiplicar que multipliqui dos números i seguiu les guies de PEP 8.

Solució:

def multiplicar(a, b):
    """
    Multiplica dos números.

    Arguments:
    a -- primer número
    b -- segon número

    Retorna:
    El producte de a i b.
    """
    producte = a * b
    return producte

Resum

En aquesta secció, hem après sobre la importància de seguir PEP 8 per mantenir el codi Python llegible i coherent. Hem revisat les principals recomanacions de PEP 8, incloent la indentació, la longitud de línia, els imports, els espais en blanc, els comentaris i la nomenclatura. També hem vist exemples pràctics i hem realitzat exercicis per reforçar els conceptes apresos. Seguir aquestes guies ajudarà a escriure codi de qualitat que sigui fàcil de mantenir i col·laborar.
