Conceptes Bàsics de Mapeig UV

Següent ► ►

El mapeig UV és una tècnica essencial en el procés de texturització 3D que permet aplicar textures 2D a models 3D. En aquest tema, aprendrem els conceptes bàsics del mapeig UV, com crear i editar mapes UV, i com aplicar textures als nostres models.

  1. Què és el Mapeig UV?

El mapeig UV és el procés de desplegar un model 3D en un espai 2D per aplicar-hi textures. Les coordenades UV defineixen com es mapeja la textura 2D sobre la superfície del model 3D.

  • U i V: Són les coordenades en l'espai 2D, equivalents a X i Y en un pla 2D.
  • Mapa UV: És una representació 2D del model 3D desplegat.

  1. Per què és Important el Mapeig UV?

El mapeig UV és crucial per a la texturització precisa dels models 3D. Sense un bon mapa UV, les textures poden aparèixer distorsionades o mal alineades.

  1. Creació d'un Mapa UV a Blender

Pas 1: Selecció del Model

  1. Obre Blender i carrega el model 3D al qual vols aplicar el mapeig UV.
  2. Selecciona el model fent clic amb el botó dret del ratolí.

Pas 2: Entrar en Mode d'Edició

  1. Amb el model seleccionat, prem la tecla Tab per entrar en el Mode d'Edició.
  2. Assegura't que tens seleccionats tots els vèrtexs, arestes o cares del model (prem A per seleccionar-ho tot).

Pas 3: Desplegar el Model

  1. Amb el model seleccionat, prem U per obrir el menú de desplegament UV.
  2. Selecciona l'opció Unwrap per desplegar el model automàticament.

Pas 4: Editar el Mapa UV

  1. Canvia a la vista UV/Image Editor des del menú de la finestra.
  2. Aquí podràs veure el mapa UV generat. Pots moure, escalar i rotar les coordenades UV per ajustar la textura.

  1. Aplicació de Textures

Pas 1: Assignar una Imatge de Textura

  1. Amb el model seleccionat, ves a la pestanya Shading a la part superior de la interfície de Blender.
  2. Afegeix un nou material al model i assigna-li una imatge de textura.

Pas 2: Ajustar la Textura

  1. A la finestra UV/Image Editor, carrega la imatge de textura que vols aplicar.
  2. Ajusta les coordenades UV per assegurar-te que la textura s'aplica correctament al model.
# Exemple de codi per assignar una textura a un material
import bpy

# Crear un nou material
material = bpy.data.materials.new(name="Material_Textura")
material.use_nodes = True
bsdf = material.node_tree.nodes["Principled BSDF"]

# Afegir un node d'imatge de textura
texture_node = material.node_tree.nodes.new('ShaderNodeTexImage')
texture_node.image = bpy.data.images.load("ruta/a/la/teva/imatge.png")

# Connectar el node d'imatge de textura al BSDF
material.node_tree.links.new(bsdf.inputs['Base Color'], texture_node.outputs['Color'])

# Assignar el material al model
bpy.context.object.data.materials.append(material)

  1. Exercici Pràctic

Objectiu

Crea un mapa UV per a un cub i aplica una textura de fusta.

Passos

  1. Crea un cub a Blender (Shift + A > Mesh > Cube).
  2. Entra en Mode d'Edició (Tab).
  3. Selecciona totes les cares (A).
  4. Desplega el cub (U > Unwrap).
  5. Canvia a la vista UV/Image Editor.
  6. Carrega una imatge de textura de fusta.
  7. Ajusta les coordenades UV per assegurar-te que la textura s'aplica correctament.

Solució

  1. Crea el cub i entra en Mode d'Edició.
  2. Selecciona totes les cares i desplega el cub.
  3. Canvia a la vista UV/Image Editor i carrega la imatge de textura.
  4. Ajusta les coordenades UV per assegurar-te que la textura s'aplica correctament.

  1. Errors Comuns i Consells

  • Distorsió de la Textura: Si la textura apareix distorsionada, revisa les coordenades UV i ajusta-les.
  • Superposició de Coordenades UV: Evita que les coordenades UV es superposin, ja que això pot causar problemes en la texturització.
  • Escala de la Textura: Assegura't que la textura té la resolució adequada per al model. Una textura de baixa resolució pot aparèixer pixelada.

Conclusió

El mapeig UV és una habilitat fonamental per a qualsevol artista 3D. Amb els conceptes bàsics apresos en aquest tema, ara estàs preparat per crear i editar mapes UV i aplicar textures als teus models 3D. En el següent tema, explorarem com aplicar textures més complexes i utilitzar l'Editor de Shaders per crear materials realistes.
