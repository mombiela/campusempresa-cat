Utilitzant Modificadors
Següent ►

Els modificadors a Blender són eines poderoses que permeten aplicar canvis no destructius als objectes 3D. Aquests canvis poden ser de diferents tipus, com ara deformacions, subdivisions, miralls, entre d'altres. En aquesta secció, aprendrem a utilitzar alguns dels modificadors més comuns i útils.

  1. Què són els Modificadors?

Els modificadors són operacions que es poden aplicar a un objecte per alterar la seva geometria de manera no destructiva. Això significa que els canvis es poden ajustar o eliminar en qualsevol moment sense afectar la geometria original de l'objecte.

Tipus de Modificadors

Alguns dels modificadors més comuns inclouen:

  • Subdivision Surface: Suavitza la geometria de l'objecte subdividint les seves cares.
  • Mirror: Reflecteix la geometria de l'objecte al llarg d'un eix.
  • Array: Duplica l'objecte en una sèrie lineal o circular.
  • Boolean: Permet combinar o restar geometries d'objectes diferents.
  • Solidify: Afegeix espessor a una superfície.

  1. Com Afegir un Modificador

Per afegir un modificador a un objecte, segueix aquests passos:

  1. Selecciona l'objecte al qual vols aplicar el modificador.
  2. A la barra de propietats, ves a la pestanya de modificadors (icona de clau anglesa).
  3. Fes clic a "Add Modifier" i selecciona el modificador que vols aplicar.

  1. Exemples Pràctics

3.1. Subdivision Surface

El modificador Subdivision Surface s'utilitza per suavitzar la geometria d'un objecte. A continuació, es mostra com aplicar-lo:

1. Selecciona l'objecte.
2. Afegeix el modificador "Subdivision Surface".
3. Ajusta els nivells de subdivisions per a la vista i el render.

Exemple de Codi:

import bpy

# Crear un cub
bpy.ops.mesh.primitive_cube_add(size=2)

# Seleccionar el cub
obj = bpy.context.active_object

# Afegir el modificador Subdivision Surface
subsurf = obj.modifiers.new(name="Subdivision Surface", type='SUBSURF')

# Ajustar els nivells de subdivisions
subsurf.levels = 2
subsurf.render_levels = 3

3.2. Mirror

El modificador Mirror reflecteix la geometria de l'objecte al llarg d'un eix. És útil per crear objectes simètrics.

1. Selecciona l'objecte.
2. Afegeix el modificador "Mirror".
3. Selecciona l'eix al llarg del qual vols reflectir l'objecte.

Exemple de Codi:

import bpy

# Crear un cub
bpy.ops.mesh.primitive_cube_add(size=2)

# Seleccionar el cub
obj = bpy.context.active_object

# Afegir el modificador Mirror
mirror = obj.modifiers.new(name="Mirror", type='MIRROR')

# Seleccionar l'eix de reflexió
mirror.use_axis[0] = True  # Reflexió al llarg de l'eix X

3.3. Array

El modificador Array duplica l'objecte en una sèrie lineal o circular.

1. Selecciona l'objecte.
2. Afegeix el modificador "Array".
3. Ajusta el nombre de còpies i la distància entre elles.

Exemple de Codi:

import bpy

# Crear un cub
bpy.ops.mesh.primitive_cube_add(size=2)

# Seleccionar el cub
obj = bpy.context.active_object

# Afegir el modificador Array
array = obj.modifiers.new(name="Array", type='ARRAY')

# Ajustar el nombre de còpies i la distància
array.count = 5
array.relative_offset_displace[0] = 1.5  # Distància en l'eix X

  1. Exercicis Pràctics

Exercici 1: Aplicar un Modificador Subdivision Surface

  1. Crea un objecte primitiu (per exemple, un cub).
  2. Aplica el modificador Subdivision Surface.
  3. Ajusta els nivells de subdivisions per veure com afecta la geometria de l'objecte.

Exercici 2: Crear una Sèrie d'Objectes amb el Modificador Array

  1. Crea un objecte primitiu (per exemple, un cilindre).
  2. Aplica el modificador Array.
  3. Ajusta el nombre de còpies i la distància entre elles per crear una sèrie d'objectes.

  1. Conclusió

Els modificadors són eines essencials a Blender que permeten aplicar canvis no destructius als objectes 3D. Hem vist com afegir i utilitzar alguns dels modificadors més comuns, com ara Subdivision Surface, Mirror i Array. Practicar amb aquests modificadors t'ajudarà a comprendre millor com funcionen i com poden millorar el teu flux de treball en Blender.

En el següent mòdul, explorarem tècniques avançades de modelatge, incloent el modelatge de superfície de subdivisió i l'escultura.
