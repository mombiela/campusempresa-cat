Utilitzant el Compositor
Següent ►

El compositor de Blender és una eina poderosa que permet combinar diferents elements visuals, aplicar efectes especials, i realitzar correccions de color i altres ajustos postprocessament. En aquesta secció, aprendrem a utilitzar el compositor per millorar les nostres escenes i donar-los un acabat professional.

1. Introducció al Compositor

Què és el Compositor?

El compositor és una eina de postprocessament que permet combinar diferents imatges i aplicar efectes per crear una imatge final. És especialment útil per a la correcció de color, l'addició d'efectes especials, i la composició de múltiples capes d'imatges.

Per què utilitzar el Compositor?

  • Correcció de color: Ajustar els colors de la teva escena per aconseguir l'ambient desitjat.
  • Efectes especials: Afegir efectes com el desenfocament, la brillantor, i altres.
  • Composició de capes: Combinar diferents elements visuals en una sola imatge.

2. Configuració del Compositor

Activació del Compositor

  1. Obre Blender i carrega el teu projecte.
  2. Canvia a la pestanya de Compositing a la part superior de la interfície.
  3. Marca la casella Use Nodes per activar el compositor.

Interfície del Compositor

  • Node Editor: Aquí és on crearàs i connectaràs els nodes per aplicar els efectes.
  • Render Layers: Representa les capes de la teva escena renderitzada.
  • Viewer Node: Permet veure el resultat de la composició en temps real.

3. Nodes Bàsics del Compositor

Tipus de Nodes

  • Input Nodes: Com el node de Render Layers, que proporciona la imatge renderitzada.
  • Output Nodes: Com el node de Composite, que és la sortida final de la composició.
  • Color Nodes: Per ajustar els colors, com el node de RGB Curves.
  • Filter Nodes: Per aplicar efectes com el desenfocament, com el node de Blur.

Exemple de Nodes Bàsics

Render Layers -> RGB Curves -> Composite

Exemple Pràctic: Ajust de Color

  1. Afegeix un node de Render Layers (Shift + A > Input > Render Layers).
  2. Afegeix un node de RGB Curves (Shift + A > Color > RGB Curves).
  3. Connecta la sortida del node de Render Layers a l'entrada del node de RGB Curves.
  4. Afegeix un node de Composite (Shift + A > Output > Composite).
  5. Connecta la sortida del node de RGB Curves a l'entrada del node de Composite.
Render Layers -> RGB Curves -> Composite
  1. Ajusta les corbes de color al node de RGB Curves per modificar els colors de la teva imatge.

4. Aplicació d'Efectes

Desenfocament (Blur)

  1. Afegeix un node de Blur (Shift + A > Filter > Blur).
  2. Connecta la sortida del node de RGB Curves a l'entrada del node de Blur.
  3. Connecta la sortida del node de Blur a l'entrada del node de Composite.
Render Layers -> RGB Curves -> Blur -> Composite
  1. Ajusta els paràmetres del node de Blur per aplicar el desenfocament desitjat.

Exemple de Codi

Render Layers -> RGB Curves -> Blur -> Composite

5. Visualització del Resultat

Utilitzant el Viewer Node

  1. Afegeix un node de Viewer (Shift + A > Output > Viewer).
  2. Connecta la sortida del node de Blur a l'entrada del node de Viewer.
Render Layers -> RGB Curves -> Blur -> Composite
                                      -> Viewer
  1. Això et permetrà veure el resultat de la composició en temps real a la finestra de visualització.

6. Exercicis Pràctics

Exercici 1: Correcció de Color

  1. Carrega una escena renderitzada.
  2. Utilitza el node de RGB Curves per ajustar els colors de la imatge.
  3. Visualitza el resultat utilitzant el node de Viewer.

Exercici 2: Aplicació de Desenfocament

  1. Afegeix un node de Blur a la teva composició.
  2. Ajusta els paràmetres per aplicar un desenfocament suau.
  3. Visualitza el resultat utilitzant el node de Viewer.

7. Resum

En aquesta secció, hem après a utilitzar el compositor de Blender per aplicar correccions de color i efectes especials a les nostres imatges renderitzades. Hem vist com configurar el compositor, utilitzar nodes bàsics, i aplicar efectes com el desenfocament. Amb aquestes tècniques, pots millorar significativament la qualitat visual de les teves escenes.

En la següent secció, explorarem la Correcció de Color i Gradació per donar un acabat professional a les teves imatges.
