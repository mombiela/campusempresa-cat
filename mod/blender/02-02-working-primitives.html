Treballant amb Primitives
Següent ►

Introducció

En aquest tema, aprendrem a treballar amb les primitives a Blender. Les primitives són els objectes bàsics que es poden utilitzar com a punt de partida per a la majoria dels projectes de modelatge 3D. Aquests inclouen cubs, esferes, cilindres, cons, i altres formes geomètriques bàsiques.

Conceptes Clau

  1. Primitives: Objectes bàsics predefinits que es poden utilitzar com a base per a la creació de models més complexos.
  2. Tipus de Primitives: Cub, Esfera UV, Esfera Icosfèrica, Cilindre, Con, Torus, Pla, Cercle.
  3. Propietats de les Primitives: Paràmetres que es poden ajustar en el moment de la creació, com ara segments, radis, alçades, etc.

Afegint Primitives

Pas 1: Accedir al Menú d'Afegir

  1. Obre Blender i crea un nou projecte o carrega un projecte existent.
  2. Assegura't que estàs en el mode Objecte.
  3. Premeu Shift + A per obrir el menú d'afegir.

Pas 2: Seleccionar una Primitiva

  1. Des del menú d'afegir, selecciona Mesh per veure les opcions de primitives disponibles.
  2. Tria una de les primitives, com ara Cube, UV Sphere, Cylinder, etc.

Pas 3: Ajustar les Propietats de la Primitiva

  1. Després d'afegir una primitiva, apareixerà un panell a la part inferior esquerra de la pantalla amb les propietats de la primitiva.
  2. Ajusta els paràmetres com ara segments, radis, alçades, etc., segons les teves necessitats.

Exemples Pràctics

Exemple 1: Afegir un Cub

1. Premeu `Shift + A` per obrir el menú d'afegir.
2. Selecciona `Mesh` -> `Cube`.
3. Ajusta les dimensions del cub des del panell de propietats si cal.

Exemple 2: Afegir una Esfera UV

1. Premeu `Shift + A` per obrir el menú d'afegir.
2. Selecciona `Mesh` -> `UV Sphere`.
3. Ajusta els segments i els anells des del panell de propietats per obtenir la resolució desitjada.

Exemple 3: Afegir un Cilindre

1. Premeu `Shift + A` per obrir el menú d'afegir.
2. Selecciona `Mesh` -> `Cylinder`.
3. Ajusta el nombre de vèrtexs, el radi i l'alçada des del panell de propietats.

Exercicis Pràctics

Exercici 1: Crear una Taula Simple

  1. Afegiu un Cube per a la part superior de la taula.
  2. Escaleu el cub per fer-lo més prim i ample.
  3. Afegiu quatre Cylinders per a les potes de la taula.
  4. Escaleu els cilindres per fer-los més prims i llargs.
  5. Col·loqueu els cilindres a les quatre cantonades de la part superior de la taula.

Exercici 2: Crear una Pila de Llibres

  1. Afegiu un Cube per al primer llibre.
  2. Escaleu el cub per fer-lo més prim i ample.
  3. Dupliqueu el cub diverses vegades (Shift + D) per crear més llibres.
  4. Escaleu i col·loqueu els llibres de manera que semblin una pila.

Solucions als Exercicis

Solució 1: Crear una Taula Simple

1. Afegiu un `Cube` (`Shift + A` -> `Mesh` -> `Cube`).
2. Escaleu el cub (`S` -> `Z` -> 0.1, `S` -> `X` -> 2, `S` -> `Y` -> 2).
3. Afegiu un `Cylinder` (`Shift + A` -> `Mesh` -> `Cylinder`).
4. Escaleu el cilindre (`S` -> `Shift + Z` -> 0.1, `S` -> `Z` -> 2).
5. Col·loqueu el cilindre a una cantonada de la part superior de la taula (`G` -> `X` -> moure, `G` -> `Y` -> moure).
6. Dupliqueu el cilindre tres vegades (`Shift + D`) i col·loqueu-los a les altres tres cantonades.

Solució 2: Crear una Pila de Llibres

1. Afegiu un `Cube` (`Shift + A` -> `Mesh` -> `Cube`).
2. Escaleu el cub (`S` -> `Z` -> 0.1, `S` -> `X` -> 1.5, `S` -> `Y` -> 1).
3. Dupliqueu el cub (`Shift + D`) i col·loqueu-lo a sobre del primer llibre (`G` -> `Z` -> moure).
4. Repetiu el procés per crear més llibres i col·loqueu-los en una pila.

Conclusió

En aquest tema, hem après a treballar amb les primitives a Blender. Hem vist com afegir diferents tipus de primitives, ajustar les seves propietats i utilitzar-les per crear objectes més complexos. Practicar amb aquestes tècniques és essencial per a qualsevol projecte de modelatge 3D. En el proper tema, explorarem les transformacions bàsiques com moure, rotar i escalar objectes.
