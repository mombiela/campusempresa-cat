Introducció a la Composició
Següent ►

La composició és una part fonamental del procés de producció en Blender, ja que permet combinar diferents elements visuals per crear una imatge final cohesiva. En aquest mòdul, aprendrem els conceptes bàsics de la composició, com utilitzar el compositor de Blender i com aplicar correccions de color i efectes per millorar les nostres escenes.

Objectius del Mòdul

  • Entendre què és la composició i la seva importància en la producció visual.
  • Familiaritzar-se amb la interfície del compositor de Blender.
  • Aprendre a utilitzar nodes per combinar elements visuals.
  • Aplicar correccions de color i efectes bàsics.

  1. Què és la Composició?

La composició és el procés de combinar múltiples imatges o elements visuals en una sola imatge final. Això pot incloure la integració de diferents capes de renderització, l'aplicació d'efectes visuals, la correcció de color i molt més. La composició és essencial per aconseguir una imatge final polida i professional.

Conceptes Clau

  • Capas: Diferents elements visuals que es combinen en la imatge final.
  • Nodes: Blocs funcionals que es connecten per processar i combinar imatges.
  • Render Passes: Diferents components d'una imatge renderitzada (com la llum, les ombres, els reflexos) que es poden combinar en la composició.

  1. La Interfície del Compositor de Blender

El compositor de Blender utilitza una interfície basada en nodes que permet una gran flexibilitat i control sobre el procés de composició. A continuació, es mostra una breu descripció dels elements principals de la interfície del compositor.

Elements Principals

  • Editor de Nodes: L'àrea on es creen i es connecten els nodes.
  • Vista de Resultat: Mostra el resultat de la composició en temps real.
  • Propietats del Node: Panell on es poden ajustar les propietats dels nodes seleccionats.

Configuració Inicial

  1. Activar el Compositor: Aneu a la pestanya "Compositing" a la part superior de la interfície de Blender.
  2. Utilitzar Nodes: Assegureu-vos que la casella "Use Nodes" estigui marcada.
  3. Vista de Resultat: Afegiu un node "Viewer" per veure el resultat de la composició.

  1. Utilitzant Nodes per a la Composició

Els nodes són la base del compositor de Blender. Cada node realitza una funció específica, com combinar imatges, aplicar efectes o ajustar colors. A continuació, es mostra un exemple bàsic de com utilitzar nodes per combinar dues imatges.

Exemple Pràctic: Combinació de Dues Imatges

1. Afegiu dos nodes "Image" i carregueu les imatges que voleu combinar.
2. Afegiu un node "Alpha Over" per combinar les dues imatges.
3. Connecteu les sortides de les imatges a les entrades del node "Alpha Over".
4. Connecteu la sortida del node "Alpha Over" al node "Composite" i al node "Viewer".
# Exemple de nodes en codi Python (pseudo-codi)
image1 = bpy.data.images.load('path/to/image1.png')
image2 = bpy.data.images.load('path/to/image2.png')

alpha_over_node = nodes.new(type='CompositorNodeAlphaOver')
alpha_over_node.inputs[1].default_value = image1
alpha_over_node.inputs[2].default_value = image2

composite_node = nodes.new(type='CompositorNodeComposite')
viewer_node = nodes.new(type='CompositorNodeViewer')

alpha_over_node.outputs[0].connect(composite_node.inputs[0])
alpha_over_node.outputs[0].connect(viewer_node.inputs[0])

  1. Correcció de Color i Efectes Bàsics

La correcció de color i l'aplicació d'efectes són passos crucials en la composició per millorar l'aparença de la imatge final.

Correcció de Color

  • Node "Color Balance": Permet ajustar els tons de la imatge.
  • Node "Hue Saturation Value": Permet ajustar la saturació i el to de la imatge.

Aplicació d'Efectes

  • Node "Blur": Aplica un efecte de desenfocament.
  • Node "Glare": Afegeix efectes de resplendor a les parts brillants de la imatge.

Exemple Pràctic: Aplicació de Correcció de Color

1. Afegiu un node "Color Balance" entre el node "Alpha Over" i el node "Composite".
2. Ajusteu els controls del node "Color Balance" per modificar els tons de la imatge.
# Exemple de nodes en codi Python (pseudo-codi)
color_balance_node = nodes.new(type='CompositorNodeColorBalance')
alpha_over_node.outputs[0].connect(color_balance_node.inputs[0])
color_balance_node.outputs[0].connect(composite_node.inputs[0])
color_balance_node.outputs[0].connect(viewer_node.inputs[0])

Exercici Pràctic

Exercici: Combinació i Correcció de Color

  1. Carregueu dues imatges diferents en el compositor.
  2. Combineu-les utilitzant un node "Alpha Over".
  3. Afegiu un node "Color Balance" i ajusteu els tons de la imatge combinada.
  4. Visualitzeu el resultat en el node "Viewer".

Solució

1. Afegiu dos nodes "Image" i carregueu les imatges.
2. Afegiu un node "Alpha Over" i connecteu les imatges.
3. Afegiu un node "Color Balance" i connecteu-lo després del node "Alpha Over".
4. Connecteu el node "Color Balance" al node "Viewer" i al node "Composite".
# Solució en codi Python (pseudo-codi)
image1 = bpy.data.images.load('path/to/image1.png')
image2 = bpy.data.images.load('path/to/image2.png')

alpha_over_node = nodes.new(type='CompositorNodeAlphaOver')
alpha_over_node.inputs[1].default_value = image1
alpha_over_node.inputs[2].default_value = image2

color_balance_node = nodes.new(type='CompositorNodeColorBalance')
alpha_over_node.outputs[0].connect(color_balance_node.inputs[0])

composite_node = nodes.new(type='CompositorNodeComposite')
viewer_node = nodes.new(type='CompositorNodeViewer')

color_balance_node.outputs[0].connect(composite_node.inputs[0])
color_balance_node.outputs[0].connect(viewer_node.inputs[0])

Conclusió

En aquesta secció, hem après els conceptes bàsics de la composició en Blender, com utilitzar la interfície del compositor i com aplicar correccions de color i efectes bàsics. La composició és una eina poderosa que permet millorar significativament la qualitat de les nostres imatges finals. En el proper mòdul, aprofundirem en l'ús avançat del compositor per crear efectes més complexos i professionals.
