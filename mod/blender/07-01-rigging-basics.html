Conceptes Bàsics de Rigging
Següent ►

El rigging és una tècnica essencial en l'animació 3D que permet donar moviment als models. En aquest mòdul, aprendrem els conceptes bàsics del rigging a Blender, incloent la creació d'ossos (bones), la configuració de jerarquies i la vinculació dels ossos amb els models.

  1. Introducció al Rigging

Què és el Rigging?

El rigging és el procés de crear un esquelet (rig) per a un model 3D que permeti animar-lo. Aquest esquelet està format per ossos (bones) que es poden moure, rotar i escalar per deformar el model associat.

Per què és important?

El rigging és crucial per a l'animació de personatges i objectes complexos. Permet controlar el moviment de manera precisa i realista, facilitant la creació d'animacions fluides i naturals.

  1. Creació d'Ossos (Bones)

Pas 1: Afegir un Esquelet Bàsic

  1. Selecciona el model al qual vols afegir un esquelet.
  2. Canvia al mode Objecte (Object Mode) si no hi ets.
  3. Afegeix un os (Add > Armature > Single Bone).
Shift + A > Armature > Single Bone

Pas 2: Editar l'Esquelet

  1. Selecciona l'os i canvia al mode Edició (Edit Mode).
  2. Mou, escala i rota l'os per ajustar-lo a la forma del model.
Tab (per canviar al mode Edició)
G (moure), S (escalar), R (rotar)

Pas 3: Afegir Més Ossos

  1. Selecciona l'extrem de l'os existent.
  2. Extrudeix un nou os (Extrude).
E (extruir)

Exemple de Codi

# Afegir un os
import bpy

# Crear un nou esquelet
bpy.ops.object.armature_add()

# Canviar al mode Edició
bpy.ops.object.mode_set(mode='EDIT')

# Afegir un os nou
bpy.ops.armature.extrude_move(TRANSFORM_OT_translate={"value":(0, 1, 0)})

  1. Configuració de Jerarquies

Pas 1: Crear una Jerarquia

  1. Selecciona l'os fill.
  2. Shift + Selecciona l'os pare.
  3. Estableix la relació de parent (Ctrl + P > Keep Offset).
Ctrl + P > Keep Offset

Pas 2: Verificar la Jerarquia

  1. Canvia al mode Objecte.
  2. Mou l'os pare i observa com els ossos fills es mouen amb ell.

  1. Vinculació dels Ossos amb el Model

Pas 1: Vincular el Model a l'Esquelet

  1. Selecciona el model.
  2. Shift + Selecciona l'esquelet.
  3. Estableix la relació de parent (Ctrl + P > With Automatic Weights).
Ctrl + P > With Automatic Weights

Pas 2: Ajustar els Pesos

  1. Selecciona el model.
  2. Canvia al mode Pesos (Weight Paint Mode).
  3. Pinta els pesos per ajustar la influència dels ossos sobre el model.
Tab (per canviar al mode Pesos)

Exercici Pràctic

Objectiu

Crear un esquelet bàsic per a un model de personatge i vincular-lo correctament.

Passos

  1. Importa o crea un model de personatge.
  2. Afegeix un esquelet bàsic seguint els passos anteriors.
  3. Configura la jerarquia dels ossos.
  4. Vincula el model a l'esquelet amb pesos automàtics.
  5. Ajusta els pesos per assegurar-te que el model es deforma correctament.

Solució

# Importar el model
import bpy

# Afegir un esquelet
bpy.ops.object.armature_add()

# Canviar al mode Edició
bpy.ops.object.mode_set(mode='EDIT')

# Afegir ossos addicionals
bpy.ops.armature.extrude_move(TRANSFORM_OT_translate={"value":(0, 1, 0)})
bpy.ops.armature.extrude_move(TRANSFORM_OT_translate={"value":(0, 1, 0)})

# Canviar al mode Objecte
bpy.ops.object.mode_set(mode='OBJECT')

# Seleccionar el model i l'esquelet
bpy.ops.object.select_all(action='DESELECT')
bpy.data.objects['Model'].select_set(True)
bpy.data.objects['Armature'].select_set(True)

# Establir la relació de parent
bpy.ops.object.parent_set(type='ARMATURE_AUTO')

Conclusió

En aquest tema, hem après els conceptes bàsics del rigging a Blender, incloent la creació d'ossos, la configuració de jerarquies i la vinculació dels ossos amb els models. Aquestes habilitats són fonamentals per a l'animació de personatges i objectes complexos. En el següent tema, explorarem tècniques més avançades d'animació de personatges.
