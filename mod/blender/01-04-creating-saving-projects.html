Creació i Guardat de Projectes
Següent ►

En aquest tema, aprendrem com crear un nou projecte a Blender, com guardar-lo correctament i com gestionar els fitxers del projecte. Aquestes habilitats són fonamentals per assegurar que el teu treball es mantingui segur i organitzat.

  1. Creació d'un Nou Projecte

1.1. Obrir Blender

Quan obres Blender per primera vegada, veuràs la pantalla de benvinguda. Aquí pots seleccionar diverses opcions, però per crear un nou projecte, simplement fes clic a "General" per començar amb una escena buida.

1.2. Entenent l'Escena per Defecte

Quan obres un nou projecte, Blender carrega una escena per defecte que inclou:

  • Un cub
  • Una càmera
  • Una llum

Aquesta configuració bàsica és suficient per començar a treballar immediatament.

1.3. Configuració Inicial

Abans de començar a modelar, és una bona pràctica configurar algunes preferències inicials:

  • Unitats de Mesura: Pots configurar les unitats de mesura a "Scene Properties" > "Units".
  • Guardat Automàtic: Activa el guardat automàtic a "Edit" > "Preferences" > "Save & Load" > "Auto Save".

  1. Guardat del Projecte

2.1. Guardar per Primera Vegada

Per guardar el teu projecte per primera vegada:

  1. Ves a "File" > "Save As".
  2. Selecciona la ubicació on vols guardar el fitxer.
  3. Assigna un nom al teu projecte.
  4. Fes clic a "Save As Blender File".

2.2. Guardar Canvis

Després de guardar el projecte per primera vegada, pots guardar els canvis simplement prement Ctrl + S o anant a "File" > "Save".

2.3. Guardar Versions

És una bona pràctica guardar versions del teu projecte per poder tornar a estats anteriors si cal:

  1. Ves a "File" > "Save As".
  2. Canvia lleugerament el nom del fitxer (per exemple, afegint un número de versió).
  3. Fes clic a "Save As Blender File".

  1. Gestió de Fitxers del Projecte

3.1. Organització de Fitxers

Per mantenir el teu projecte organitzat, és recomanable crear una estructura de carpetes clara. Per exemple:

/Projecte
    /Models
    /Textures
    /Renders
    /References

3.2. Enllaçar Fitxers Externs

Blender permet enllaçar fitxers externs com imatges de textures o altres models. Per enllaçar un fitxer:

  1. Ves a "File" > "Link" o "Append".
  2. Selecciona el fitxer que vols enllaçar.
  3. Selecciona l'objecte o material que vols importar.

3.3. Empaquetar Dades

Per assegurar que tots els recursos externs es mantenen amb el teu projecte, pots empaquetar les dades:

  1. Ves a "File" > "External Data" > "Pack All Into .blend".
  2. Això inclourà totes les textures i altres fitxers externs dins del fitxer .blend.

Exercicis Pràctics

Exercici 1: Crear i Guardar un Nou Projecte

  1. Obre Blender i crea un nou projecte.
  2. Configura les unitats de mesura a metres.
  3. Guarda el projecte amb el nom "PrimerProjecte.blend".

Exercici 2: Guardar Versions

  1. Fes alguns canvis a l'escena (per exemple, mou el cub).
  2. Guarda el projecte com "PrimerProjecte_v1.blend".
  3. Fes més canvis (per exemple, afegeix una esfera).
  4. Guarda el projecte com "PrimerProjecte_v2.blend".

Exercici 3: Organitzar Fitxers

  1. Crea una estructura de carpetes per al teu projecte.
  2. Guarda el fitxer .blend a la carpeta principal.
  3. Afegeix una textura a l'escena i guarda-la a la carpeta "Textures".
  4. Empaqueta totes les dades externes dins del fitxer .blend.

Resum

En aquesta secció, hem après com crear un nou projecte a Blender, com guardar-lo correctament i com gestionar els fitxers del projecte. Aquestes habilitats són essencials per mantenir el teu treball segur i organitzat. En el proper mòdul, explorarem les tècniques bàsiques de modelatge per començar a donar forma a les teves idees en 3D.
