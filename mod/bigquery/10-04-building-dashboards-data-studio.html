Creació de quadres de comandament amb Data Studio

Introducció

Google Data Studio és una eina de visualització de dades que permet crear informes i quadres de comandament interactius i personalitzats. En aquest tema, aprendrem a connectar BigQuery amb Data Studio per crear visualitzacions que ens ajudin a analitzar i interpretar les dades de manera efectiva.

Objectius

  • Comprendre com connectar BigQuery amb Data Studio.
  • Aprendre a crear informes i quadres de comandament.
  • Explorar diferents tipus de visualitzacions disponibles a Data Studio.
  • Aprendre a compartir i col·laborar en informes de Data Studio.

1. Connexió de BigQuery amb Data Studio

Passos per connectar BigQuery amb Data Studio

  1. Accedir a Google Data Studio:

    • Visita Google Data Studio i inicia sessió amb el teu compte de Google.

  2. Crear una nova font de dades:

    • Fes clic a "Create" i selecciona "Data Source".
    • Cerca i selecciona "BigQuery" com a connector.

  3. Autoritzar l'accés:

    • Autoritza Data Studio per accedir al teu compte de BigQuery.

  4. Seleccionar el projecte i la taula:

    • Tria el projecte de BigQuery, el conjunt de dades i la taula que vols utilitzar.

  5. Configurar la font de dades:

    • Revisa i ajusta els camps de la taula si és necessari.
    • Fes clic a "Add to Report" per afegir la font de dades al teu informe.

2. Creació d'un informe

Passos per crear un informe

  1. Iniciar un nou informe:

    • Fes clic a "Create" i selecciona "Report".

  2. Afegir la font de dades:

    • Selecciona la font de dades de BigQuery que has creat anteriorment.

  3. Afegir visualitzacions:

    • Utilitza el menú de la barra lateral per afegir diferents tipus de visualitzacions com taules, gràfics de barres, gràfics de línies, mapes, etc.

Tipus de visualitzacions

Tipus de Visualització Descripció
Taula Mostra les dades en format de taula.
Gràfic de barres Visualitza dades categòriques amb barres.
Gràfic de línies Mostra tendències al llarg del temps.
Mapa Visualitza dades geogràfiques.
Gràfic circular Mostra la distribució percentual de les dades.

Exemple de codi SQL per a una visualització

SELECT
  product_category,
  SUM(sales) as total_sales
FROM
  `project.dataset.sales_data`
GROUP BY
  product_category
ORDER BY
  total_sales DESC

Aquest codi SQL agrega les vendes per categoria de producte i ordena els resultats de major a menor. Pots utilitzar aquest resultat per crear un gràfic de barres a Data Studio.

3. Personalització de l'informe

Personalitzar les visualitzacions

  • Canviar els colors i estils: Pots ajustar els colors, les fonts i els estils de les visualitzacions per fer-les més atractives i coherents amb la teva marca.
  • Afegir filtres i controls: Pots afegir filtres i controls interactius com menús desplegables i rangs de dates per permetre als usuaris explorar les dades de manera dinàmica.

Exemple de personalització

  1. Afegir un filtre de data:

    • Fes clic a "Add a control" i selecciona "Date range control".
    • Col·loca el control a l'informe i ajusta les opcions segons les teves necessitats.

  2. Canviar el color d'un gràfic:

    • Selecciona el gràfic que vols personalitzar.
    • A la barra lateral, ajusta els colors de les barres, línies o altres elements visuals.

4. Compartir i col·laborar

Compartir l'informe

  • Compartir amb altres usuaris: Pots compartir l'informe amb altres usuaris de Google mitjançant el botó "Share". Pots ajustar els permisos per permetre la visualització o l'edició.
  • Publicar l'informe: Pots publicar l'informe i obtenir un enllaç públic que pots compartir amb qualsevol persona.

Col·laboració en temps real

  • Edició col·laborativa: Diversos usuaris poden editar l'informe simultàniament, similar a Google Docs.
  • Comentaris: Pots afegir comentaris a l'informe per facilitar la comunicació i la col·laboració amb el teu equip.

Exercici pràctic

Exercici

  1. Crear una connexió amb BigQuery:

    • Connecta't a una taula de BigQuery que contingui dades de vendes.

  2. Crear un informe:

    • Crea un informe nou a Data Studio i afegeix la font de dades de BigQuery.

  3. Afegir visualitzacions:

    • Afegeix un gràfic de barres que mostri les vendes totals per categoria de producte.
    • Afegeix un gràfic de línies que mostri les vendes al llarg del temps.

  4. Personalitzar l'informe:

    • Canvia els colors dels gràfics per fer-los més atractius.
    • Afegeix un filtre de data per permetre als usuaris seleccionar un rang de dates.

  5. Compartir l'informe:

    • Comparteix l'informe amb un company de classe o col·lega i permet-li editar-lo.

Solució

  1. Connexió amb BigQuery:

    • Segueix els passos descrits a la secció "Connexió de BigQuery amb Data Studio".

  2. Creació de l'informe:

    • Segueix els passos descrits a la secció "Creació d'un informe".

  3. Afegir visualitzacions:

    • Utilitza el codi SQL proporcionat a la secció "Tipus de visualitzacions" per crear els gràfics de barres i línies.

  4. Personalització:

    • Ajusta els colors i afegeix el filtre de data seguint les instruccions de la secció "Personalització de l'informe".

  5. Compartir:

    • Utilitza el botó "Share" per compartir l'informe i ajusta els permisos segons sigui necessari.

Conclusió

En aquesta secció, hem après a connectar BigQuery amb Data Studio, crear informes i quadres de comandament, personalitzar visualitzacions i compartir els nostres informes amb altres usuaris. Aquestes habilitats són essencials per analitzar i comunicar dades de manera efectiva, i et preparen per utilitzar BigQuery i Data Studio en projectes del món real.
