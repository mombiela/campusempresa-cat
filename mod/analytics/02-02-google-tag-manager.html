Google Tag Manager: Implementació i Gestió d'Etiquetes

Següent ►
►

Introducció

Google Tag Manager (GTM) és una eina gratuïta que permet gestionar i desplegar etiquetes de màrqueting (fragments de codi o píxels de seguiment) al vostre lloc web o aplicació mòbil sense haver de modificar el codi directament. Això facilita la implementació de diverses eines d'analítica i seguiment, com Google Analytics, Facebook Pixel, entre d'altres.

Conceptes Bàsics de Google Tag Manager

Components Principals

  1. Etiquetes (Tags): Fragments de codi que s'executen en el vostre lloc web o aplicació. Poden ser per a Google Analytics, píxels de seguiment, etc.
  2. Activadors (Triggers): Condicions que determinen quan i on s'han d'executar les etiquetes.
  3. Variables: Valors que poden ser utilitzats tant per les etiquetes com pels activadors. Poden ser constants, valors de pàgina, etc.

Beneficis de Google Tag Manager

  • Facilitat d'ús: No cal modificar el codi del lloc web cada vegada que es vol afegir o modificar una etiqueta.
  • Gestió centralitzada: Totes les etiquetes es gestionen des d'una única interfície.
  • Seguretat: Permet controlar qui pot afegir o modificar etiquetes.
  • Velocitat: Les etiquetes es carreguen de manera més eficient, millorant el rendiment del lloc web.

Configuració Inicial de Google Tag Manager

Pas 1: Crear un Compte de Google Tag Manager

  1. Aneu a Google Tag Manager.
  2. Feu clic a "Iniciar sessió" i utilitzeu el vostre compte de Google.
  3. Feu clic a "Crear compte".
  4. Introduïu el nom del compte, el nom del contenidor (normalment el nom del lloc web) i seleccioneu la plataforma (Web, iOS, Android, AMP).

Pas 2: Instal·lar el Fragment de Codi de Google Tag Manager

  1. Després de crear el contenidor, Google Tag Manager us proporcionarà dos fragments de codi.
  2. Inseriu el primer fragment de codi immediatament després de l'etiqueta <head> del vostre lloc web.
  3. Inseriu el segon fragment de codi immediatament després de l'etiqueta <body> del vostre lloc web.

Pas 3: Verificar la Instal·lació

  1. Utilitzeu l'eina de previsualització de Google Tag Manager per assegurar-vos que el codi s'ha instal·lat correctament.
  2. Aneu a l'interfície de Google Tag Manager i feu clic a "Previsualitzar".
  3. Introduïu l'URL del vostre lloc web i feu clic a "Connectar".
  4. Navegueu pel vostre lloc web i assegureu-vos que les etiquetes s'estan activant correctament.

Creació i Gestió d'Etiquetes

Exemple: Configuració d'una Etiqueta de Google Analytics

  1. Aneu a l'interfície de Google Tag Manager.
  2. Feu clic a "Etiquetes" i després a "Nou".
  3. Feu clic a "Configuració de l'etiqueta" i seleccioneu "Google Analytics: Universal Analytics".
  4. Seleccioneu "Tipus de seguiment" com "Pàgina vista".
  5. Introduïu l'ID de seguiment de Google Analytics.
  6. Feu clic a "Activadors" i seleccioneu "All Pages".
  7. Feu clic a "Desar" i doneu un nom a l'etiqueta, per exemple, "GA - Pàgina vista".

Exemple: Configuració d'un Activador Personalitzat

  1. Aneu a l'interfície de Google Tag Manager.
  2. Feu clic a "Activadors" i després a "Nou".
  3. Feu clic a "Configuració de l'activador" i seleccioneu "Click - Just Links".
  4. Configureu les condicions segons les vostres necessitats, per exemple, "Click URL" conté "example.com".
  5. Feu clic a "Desar" i doneu un nom a l'activador, per exemple, "Enllaços a Example.com".

Exercici Pràctic

Implementació d'una Etiqueta de Seguiment de Conversions

  1. Creeu una nova etiqueta a Google Tag Manager.
  2. Seleccioneu "HTML personalitzat" com a tipus d'etiqueta.
  3. Introduïu el codi de seguiment de conversions proporcionat per la vostra plataforma de publicitat.
  4. Configureu un activador per a la pàgina de gràcies o confirmació de la conversió.
  5. Desar i publicar els canvis.
<!-- Exemple de codi de seguiment de conversions -->
<script>
  gtag('event', 'conversion', {
      'send_to': 'AW-XXXX/XXXX',
      'transaction_id': ''
  });
</script>

Resum

Google Tag Manager és una eina poderosa que simplifica la gestió d'etiquetes de màrqueting i seguiment. Amb una configuració inicial senzilla i una interfície intuïtiva, permet als professionals del màrqueting i analítica implementar i gestionar etiquetes sense necessitat de modificar el codi del lloc web constantment. En aquest mòdul, hem après a configurar Google Tag Manager, crear etiquetes i activadors, i implementar una etiqueta de seguiment de conversions. Amb aquestes habilitats, estareu preparats per gestionar les vostres etiquetes de manera eficient i efectiva.
