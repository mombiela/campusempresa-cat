Mètodes de Recopilació de Dades: Enquestes, Formularis, Cookies
Següent ►

Introducció

La recopilació de dades és una etapa fonamental en el procés d'analítica, ja que proporciona la informació necessària per a l'anàlisi i la presa de decisions. En aquest tema, explorarem diferents mètodes de recopilació de dades, incloent enquestes, formularis i cookies. Cada mètode té els seus avantatges i desavantatges, i la seva elecció dependrà dels objectius específics de la vostra anàlisi.

Enquestes

Què són les enquestes?

Les enquestes són eines de recopilació de dades que permeten obtenir informació directa dels usuaris mitjançant preguntes estructurades. Poden ser distribuïdes en línia, per correu electrònic, o fins i tot en persona.

Avantatges de les enquestes

  • Dades Directes: Proporcionen informació directa dels usuaris.
  • Flexibilitat: Poden ser adaptades per a diferents tipus de preguntes (obertes, tancades, escala Likert, etc.).
  • Ampli Abast: Poden arribar a un gran nombre de persones de manera eficient.

Desavantatges de les enquestes

  • Taxa de Resposta: Pot ser baixa si els usuaris no estan motivats a respondre.
  • Biaix de Resposta: Les respostes poden estar influenciades per la manera en què es formulen les preguntes.
  • Temps i Cost: Pot requerir temps i recursos per dissenyar, distribuir i analitzar les enquestes.

Exemple d'Enquesta

1. Quina és la teva edat?
   a) Menys de 18 anys
   b) 18-24 anys
   c) 25-34 anys
   d) 35-44 anys
   e) 45-54 anys
   f) 55-64 anys
   g) 65 anys o més

2. Com valores la teva experiència amb el nostre lloc web?
   a) Molt satisfactòria
   b) Satisfactòria
   c) Neutral
   d) Insatisfactòria
   e) Molt insatisfactòria

3. Quines funcionalitats t'agradaria veure en el futur? (Resposta oberta)

Formularis

Què són els formularis?

Els formularis són eines de recopilació de dades que permeten als usuaris introduir informació en camps específics. Són àmpliament utilitzats en llocs web per recopilar dades com ara registres, subscripcions, comentaris, etc.

Avantatges dels formularis

  • Personalització: Poden ser dissenyats per recopilar informació específica.
  • Integració: Es poden integrar fàcilment en llocs web i aplicacions.
  • Automatització: Les dades es poden processar automàticament per a l'anàlisi.

Desavantatges dels formularis

  • Abandonament: Els usuaris poden abandonar el formulari si és massa llarg o complicat.
  • Dades Incompletes: Els usuaris poden ometre camps obligatoris o proporcionar informació incorrecta.
  • Seguretat: Cal garantir la seguretat de les dades recopilades per evitar filtracions.

Exemple de Formulari

<form action="/submit" method="post">
  <label for="name">Nom:</label><br>
  <input type="text" id="name" name="name"><br>
  <label for="email">Correu electrònic:</label><br>
  <input type="email" id="email" name="email"><br>
  <label for="feedback">Comentaris:</label><br>
  <textarea id="feedback" name="feedback"></textarea><br>
  <input type="submit" value="Enviar">
</form>

Cookies

Què són les cookies?

Les cookies són petits fitxers de text que es guarden en el navegador de l'usuari quan visita un lloc web. S'utilitzen per recopilar dades sobre el comportament de l'usuari, com ara les pàgines visitades, el temps de permanència, i les preferències.

Avantatges de les cookies

  • Seguiment: Permeten fer un seguiment del comportament de l'usuari al llarg del temps.
  • Personalització: Ajuden a personalitzar l'experiència de l'usuari basant-se en les seves preferències.
  • Persistència: Les dades es poden emmagatzemar durant un període prolongat.

Desavantatges de les cookies

  • Privacitat: Poden plantejar problemes de privacitat si no s'utilitzen de manera transparent.
  • Bloqueig: Els usuaris poden bloquejar o esborrar les cookies, limitant la recopilació de dades.
  • Limitacions: No poden emmagatzemar grans quantitats de dades.

Exemple de Cookie

// Crear una cookie
document.cookie = "username=JohnDoe; expires=Fri, 31 Dec 2023 23:59:59 GMT; path=/";

// Llegir una cookie
let cookies = document.cookie.split(';');
for (let i = 0; i < cookies.length; i++) {
  let cookie = cookies[i].trim();
  if (cookie.startsWith("username=")) {
    let username = cookie.substring("username=".length);
    console.log("Username: " + username);
  }
}

Exercici Pràctic

Objectiu

Crear un formulari en HTML per recopilar informació bàsica dels usuaris i implementar una cookie per recordar el nom de l'usuari.

Instruccions

  1. Crea un formulari en HTML amb els següents camps:

    • Nom
    • Correu electrònic
    • Comentaris

  2. Implementa una cookie en JavaScript per recordar el nom de l'usuari quan torni a visitar el lloc web.

Solució

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Formulari de Recopilació de Dades</title>
  <script>
    function setCookie(name, value, days) {
      let expires = "";
      if (days) {
        let date = new Date();
        date.setTime(date.getTime() + (days * 24 * 60 * 60 * 1000));
        expires = "; expires=" + date.toUTCString();
      }
      document.cookie = name + "=" + (value || "") + expires + "; path=/";
    }

    function getCookie(name) {
      let nameEQ = name + "=";
      let ca = document.cookie.split(';');
      for (let i = 0; i < ca.length; i++) {
        let c = ca[i];
        while (c.charAt(0) == ' ') c = c.substring(1, c.length);
        if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length, c.length);
      }
      return null;
    }

    function checkCookie() {
      let username = getCookie("username");
      if (username != null) {
        alert("Benvingut de nou, " + username);
      }
    }

    function submitForm() {
      let name = document.getElementById("name").value;
      setCookie("username", name, 7);
      alert("Formulari enviat!");
    }
  </script>
</head>
<body onload="checkCookie()">
  <form onsubmit="submitForm(); return false;">
    <label for="name">Nom:</label><br>
    <input type="text" id="name" name="name"><br>
    <label for="email">Correu electrònic:</label><br>
    <input type="email" id="email" name="email"><br>
    <label for="feedback">Comentaris:</label><br>
    <textarea id="feedback" name="feedback"></textarea><br>
    <input type="submit" value="Enviar">
  </form>
</body>
</html>
