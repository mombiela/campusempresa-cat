Programa Hola, Món!

Següent ► ►

En aquesta secció, aprendrem a crear el nostre primer programa en Rust: el clàssic "Hola, Món!". Aquest exercici ens ajudarà a familiaritzar-nos amb la configuració bàsica i la sintaxi del llenguatge.

Objectius

  • Crear un projecte nou en Rust.
  • Escriure i executar un programa senzill que imprimeixi "Hola, Món!" a la consola.
  • Comprendre la sintaxi bàsica i l'estructura d'un programa en Rust.

Passos per Crear el Programa "Hola, Món!"

  1. Crear un Nou Projecte

Rust utilitza una eina anomenada cargo per gestionar projectes. cargo és una eina de línia de comandes que ajuda a crear, compilar i gestionar projectes Rust.

  1. Obre una terminal.

  2. Executa el següent comandament per crear un nou projecte anomenat hola_mon:

    cargo new hola_mon

    Això crearà una nova carpeta anomenada hola_mon amb l'estructura bàsica d'un projecte Rust.

  3. Navega a la carpeta del projecte:

    cd hola_mon

  1. Escriure el Codi

El fitxer principal del projecte es troba a src/main.rs. Obre aquest fitxer amb el teu editor de text preferit i escriu el següent codi:

fn main() {
    println!("Hola, Món!");
}

Explicació del Codi

  • fn main() { ... }: Aquesta línia defineix una funció anomenada main. En Rust, el punt d'entrada d'un programa és la funció main.
  • println!("Hola, Món!");: Aquesta línia imprimeix el text "Hola, Món!" a la consola. println! és una macro que s'utilitza per imprimir text.

  1. Compilar i Executar el Programa

  1. Torna a la terminal.

  2. Assegura't que estàs dins la carpeta del projecte (hola_mon).

  3. Executa el següent comandament per compilar i executar el programa:

    cargo run

    Hauries de veure la següent sortida a la consola:

    Compiling hola_mon v0.1.0 (file:///path/to/hola_mon)
Finished dev [unoptimized + debuginfo] target(s) in 1.23s
Running `target/debug/hola_mon`
Hola, Món!

Exercici Pràctic

Exercici 1: Modificar el Missatge

Modifica el programa per imprimir un missatge diferent, com ara "Benvingut a Rust!".

Solució

Obre src/main.rs i canvia el codi a:

fn main() {
    println!("Benvingut a Rust!");
}

Després, executa cargo run per veure el nou missatge a la consola.

Exercici 2: Afegir una Segona Línia

Modifica el programa per imprimir dues línies de text. Per exemple, "Hola, Món!" i "Aprenent Rust és divertit!".

Solució

Obre src/main.rs i canvia el codi a:

fn main() {
    println!("Hola, Món!");
    println!("Aprenent Rust és divertit!");
}

Després, executa cargo run per veure les dues línies de text a la consola.

Resum

En aquesta secció, hem après a crear un projecte nou en Rust, escriure un programa senzill que imprimeix "Hola, Món!" a la consola, i executar-lo. També hem practicat modificant el missatge i afegint línies addicionals. Ara estem preparats per explorar més sobre la sintaxi i l'estructura bàsica de Rust en la següent secció.
