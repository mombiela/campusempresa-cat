Introducció a les Comandes de Configuració del Sistema

Les comandes de configuració del sistema en CL permeten als administradors ajustar diversos paràmetres del sistema per optimitzar el rendiment, la seguretat i la funcionalitat. Aquestes comandes poden ser utilitzades per configurar aspectes com la memòria, els dispositius d'emmagatzematge, els usuaris i els permisos, entre d'altres.

Comandes de Configuració del Sistema

A continuació, es presenten algunes de les comandes més utilitzades per a la configuració del sistema:

  1. CHGSYSVAL (Change System Value)

Aquesta comanda s'utilitza per canviar els valors del sistema. Els valors del sistema són paràmetres que controlen el comportament del sistema.

Sintaxi:

CHGSYSVAL SYSVAL(nom_del_valor) VALUE(nou_valor)

Exemple:

CHGSYSVAL SYSVAL(QTIME) VALUE('120000')

Aquest exemple canvia el valor del sistema QTIME a 12:00:00.

  1. WRKSYSVAL (Work with System Values)

Aquesta comanda permet visualitzar i treballar amb els valors del sistema.

Sintaxi:

WRKSYSVAL SYSVAL(nom_del_valor)

Exemple:

WRKSYSVAL SYSVAL(*ALL)

Aquest exemple mostra tots els valors del sistema.

  1. CHGUSRPRF (Change User Profile)

Aquesta comanda s'utilitza per canviar els perfils d'usuari. Els perfils d'usuari defineixen els permisos i les configuracions específiques per a cada usuari.

Sintaxi:

CHGUSRPRF USRPRF(nom_usuari) PASSWORD(nova_contrasenya)

Exemple:

CHGUSRPRF USRPRF(USER1) PASSWORD('newpassword')

Aquest exemple canvia la contrasenya de l'usuari USER1 a newpassword.

  1. CHGJOB (Change Job)

Aquesta comanda s'utilitza per canviar els paràmetres d'un treball en execució.

Sintaxi:

CHGJOB JOB(nom_treball) RUNPTY(nova_prioritat)

Exemple:

CHGJOB JOB(123456/QUSER/TESTJOB) RUNPTY(20)

Aquest exemple canvia la prioritat d'execució del treball TESTJOB a 20.

  1. CHGNETA (Change Network Attributes)

Aquesta comanda s'utilitza per canviar els atributs de la xarxa del sistema.

Sintaxi:

CHGNETA LCLCPNAME(nom_local) LCLNETID(id_xarxa)

Exemple:

CHGNETA LCLCPNAME(SYSTEM1) LCLNETID(NETID1)

Aquest exemple canvia el nom local del sistema a SYSTEM1 i l'ID de la xarxa a NETID1.

Exemples Pràctics

Exemple 1: Canviar el Valor del Sistema per Defecte de la Llengua

CHGSYSVAL SYSVAL(QLANGID) VALUE('ENU')

Aquest exemple canvia el valor del sistema QLANGID a ENU (anglès).

Exemple 2: Canviar la Prioritat d'Execució d'un Treball

CHGJOB JOB(987654/QUSER/DAILYJOB) RUNPTY(10)

Aquest exemple canvia la prioritat d'execució del treball DAILYJOB a 10.

Exercicis Pràctics

Exercici 1: Canviar el Valor del Sistema per Defecte de la Data

Canvia el valor del sistema QDATE a 2023-12-31.

Solució:

CHGSYSVAL SYSVAL(QDATE) VALUE('2023-12-31')

Exercici 2: Canviar la Contrasenya d'un Usuari

Canvia la contrasenya de l'usuari ADMIN a securepassword.

Solució:

CHGUSRPRF USRPRF(ADMIN) PASSWORD('securepassword')

Conclusió

En aquest mòdul, hem après sobre les comandes de configuració del sistema en CL. Aquestes comandes són essencials per a l'administració del sistema i permeten ajustar diversos paràmetres per optimitzar el rendiment i la seguretat del sistema. Hem vist com utilitzar comandes com CHGSYSVAL, WRKSYSVAL, CHGUSRPRF, CHGJOB i CHGNETA per configurar diferents aspectes del sistema. A més, hem practicat amb exemples i exercicis per reforçar els conceptes apresos.

En el següent mòdul, explorarem les comandes de seguretat i permisos, que són crucials per protegir el sistema i les dades que conté.
