Visió general de la consola de GCP
Següent ►

Introducció

La consola de Google Cloud Platform (GCP) és una interfície web que permet als usuaris gestionar els seus recursos de GCP. Aquesta consola proporciona una manera visual i intuïtiva per interactuar amb els serveis de GCP, configurar projectes, gestionar la facturació i molt més.

Objectius d'aprenentatge

En aquesta secció, aprendràs a:

  • Navegar per la consola de GCP.
  • Comprendre les diferents seccions i funcionalitats de la consola.
  • Utilitzar les eines bàsiques per gestionar els teus recursos de GCP.

Navegació per la consola de GCP

Accés a la consola

  1. Inicia sessió a GCP: Visita Google Cloud Console i inicia sessió amb el teu compte de Google.
  2. Selecciona un projecte: Un cop iniciada la sessió, selecciona un projecte existent o crea'n un de nou.

Estructura de la consola

La consola de GCP està organitzada en diverses seccions clau:

  1. Barra de navegació superior:

    • Menú de navegació: Accés ràpid a tots els serveis de GCP.
    • Cerca: Troba recursos, serveis i documentació.
    • Notificacions: Informació sobre esdeveniments importants i alertes.
    • Compte d'usuari: Configuració del compte i opcions de sessió.

  2. Panell de navegació lateral:

    • Projectes: Selecciona i gestiona els teus projectes.
    • Facturació: Accés a la informació de facturació i costos.
    • API i serveis: Gestiona les API i serveis habilitats per al teu projecte.
    • IAM i administració: Configura permisos i rols d'usuari.
    • Serveis de GCP: Accés a tots els serveis disponibles, com Compute Engine, Cloud Storage, BigQuery, etc.

  3. Panell principal:

    • Tauler de control del projecte: Vista general del projecte seleccionat, incloent-hi recursos, activitat recent i informació de facturació.
    • Detalls del servei: Informació específica sobre el servei seleccionat.

Exemple pràctic: Explorant la consola

A continuació, es mostra un exemple pràctic de com navegar per la consola de GCP per gestionar una instància de Compute Engine.

Pas 1: Accedir a Compute Engine

  1. A la barra de navegació lateral, fes clic a Compute Engine.
  2. Selecciona Instàncies de VM.

Pas 2: Crear una nova instància

  1. Fes clic a Crear instància.
  2. Omple els detalls necessaris, com el nom de la instància, la regió, el tipus de màquina, etc.
  3. Fes clic a Crear per llançar la nova instància.
# Exemple de com crear una instància de VM utilitzant la línia de comandes de gcloud
gcloud compute instances create my-instance \
    --zone=us-central1-a \
    --machine-type=n1-standard-1 \
    --image-family=debian-9 \
    --image-project=debian-cloud

Exercici pràctic

Objectiu: Crear i gestionar una instància de Compute Engine utilitzant la consola de GCP.

  1. Accedeix a la consola de GCP.

  2. Navega a Compute Engine > Instàncies de VM.

  3. Crea una nova instància amb els següents paràmetres:

    • Nom: test-instance
    • Regió: us-central1
    • Tipus de màquina: n1-standard-1
    • Imatge: Debian 9

  4. Un cop creada la instància, explora les opcions disponibles per gestionar-la (iniciar, aturar, suprimir, etc.).

Solució de l'exercici

  1. Accedeix a la consola de GCP i inicia sessió.
  2. Navega a Compute Engine > Instàncies de VM.
  3. Fes clic a Crear instància.
  4. Omple els detalls:
    • Nom: test-instance
    • Regió: us-central1
    • Tipus de màquina: n1-standard-1
    • Imatge: Debian 9
  5. Fes clic a Crear.
  6. Un cop creada la instància, utilitza les opcions disponibles per gestionar-la.

Resum

En aquesta secció, has après a navegar per la consola de GCP, comprendre les seves diferents seccions i utilitzar les eines bàsiques per gestionar els teus recursos. Aquesta habilitat és fonamental per treballar eficientment amb Google Cloud Platform i aprofitar al màxim els seus serveis. En la següent secció, explorarem com comprendre els projectes i la facturació a GCP.
