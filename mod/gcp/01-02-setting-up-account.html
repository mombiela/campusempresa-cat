Configurar el teu compte de GCP
Següent ►

En aquesta secció, aprendrem com configurar un compte de Google Cloud Platform (GCP). Aquest és el primer pas per començar a utilitzar els serveis de GCP i desplegar les teves aplicacions al núvol.

  1. Crear un compte de Google

Si ja tens un compte de Google (per exemple, un compte de Gmail), pots saltar aquest pas. Si no, segueix aquests passos per crear-ne un:

  1. Ves a accounts.google.com.
  2. Fes clic a "Crea un compte".
  3. Omple el formulari amb la teva informació personal.
  4. Segueix les instruccions per verificar el teu compte.

  1. Registrar-se a Google Cloud Platform

  1. Ves a cloud.google.com.
  2. Fes clic a "Get started for free" (Comença de franc).
  3. Inicia sessió amb el teu compte de Google.
  4. Omple la informació requerida, incloent-hi la informació de facturació. Google ofereix un crèdit gratuït per als nous usuaris, així que no et preocupis per les despeses inicials.

  1. Crear un projecte nou

Els projectes són la unitat bàsica d'organització a GCP. Cada projecte té la seva pròpia configuració, permisos i facturació.

  1. Un cop hagis iniciat sessió a la consola de GCP, fes clic a la icona del menú a l'extrem superior esquerre.
  2. Selecciona "Home" i després "Select a project" (Selecciona un projecte).
  3. Fes clic a "New Project" (Nou projecte).
  4. Omple el formulari amb el nom del projecte i selecciona una organització si és necessari.
  5. Fes clic a "Create" (Crear).

  1. Configurar la facturació

Per utilitzar els serveis de GCP, has de configurar la facturació. Això no vol dir que hagis de pagar immediatament, ja que pots utilitzar el crèdit gratuït inicial.

  1. Ves a la secció de facturació de la consola de GCP.
  2. Selecciona o crea un compte de facturació.
  3. Assigna el compte de facturació al teu projecte.

  1. Activar les API necessàries

Molts serveis de GCP requereixen que activis les API corresponents.

  1. Ves a la secció "API & Services" de la consola de GCP.
  2. Selecciona "Library" (Biblioteca).
  3. Cerca les API que necessites i fes clic a "Enable" (Activar).

  1. Configurar permisos i IAM

La gestió d'identitat i accés (IAM) et permet controlar qui té accés als teus recursos de GCP.

  1. Ves a la secció "IAM & Admin" de la consola de GCP.
  2. Afegeix usuaris i assigna'ls rols adequats segons les seves necessitats.

Exemple pràctic: Crear un projecte i activar l'API de Compute Engine

# Inicia sessió a la consola de GCP
gcloud auth login

# Configura el projecte
gcloud projects create my-sample-project --name="My Sample Project"

# Selecciona el projecte
gcloud config set project my-sample-project

# Activa l'API de Compute Engine
gcloud services enable compute.googleapis.com

Exercici pràctic

  1. Crea un nou compte de GCP si encara no en tens un.
  2. Crea un projecte nou anomenat "My First GCP Project".
  3. Activa l'API de Cloud Storage per al teu projecte.
  4. Configura la facturació per al teu projecte.
  5. Afegeix un usuari amb el rol de "Viewer" al teu projecte.

Solució de l'exercici

  1. Segueix els passos descrits anteriorment per crear un compte de GCP.
  2. Per crear un projecte nou:
    • Ves a la consola de GCP.
    • Fes clic a "Select a project" i després a "New Project".
    • Omple el formulari amb el nom "My First GCP Project" i fes clic a "Create".
  3. Per activar l'API de Cloud Storage:
    • Ves a "API & Services" > "Library".
    • Cerca "Cloud Storage" i fes clic a "Enable".
  4. Per configurar la facturació:
    • Ves a "Billing" i segueix les instruccions per configurar un compte de facturació.
  5. Per afegir un usuari amb el rol de "Viewer":
    • Ves a "IAM & Admin".
    • Fes clic a "Add" i introdueix l'adreça de correu electrònic de l'usuari.
    • Assigna el rol de "Viewer" i fes clic a "Save".

Conclusió

Ara que has configurat el teu compte de GCP, creat un projecte i activat les API necessàries, estàs preparat per començar a explorar els serveis de Google Cloud Platform. En el següent mòdul, aprendrem sobre els serveis bàsics de GCP, començant per Compute Engine.
