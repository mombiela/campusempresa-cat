Cursos i Tutorials en Línia
Següent ►

En aquesta secció, proporcionarem una llista de recursos en línia que poden ajudar-te a aprofundir en els patrons de disseny de programari. Aquests recursos inclouen cursos, tutorials, vídeos i plataformes d'aprenentatge que cobreixen una àmplia gamma de temes relacionats amb els patrons de disseny. Aquests recursos són útils tant per a principiants com per a professionals experimentats que volen millorar les seves habilitats.

Plataformes d'Aprenentatge en Línia

  1. Coursera

Curs Recomanat:

  • Design Patterns per la Universitat de Alberta
    • Descripció: Aquest curs cobreix els patrons de disseny més comuns i com aplicar-los en el desenvolupament de programari.
    • Enllaç: Coursera - Design Patterns

  1. Udemy

Cursos Recomanats:

  • Java Design Patterns & Architecture per Tim Buchalka

    • Descripció: Un curs complet que cobreix els patrons de disseny en Java, amb exemples pràctics.
    • Enllaç: Udemy - Java Design Patterns & Architecture

  • Design Patterns in Modern C++ per Dmitri Nesteruk

    • Descripció: Aquest curs se centra en l'aplicació dels patrons de disseny en C++.
    • Enllaç: Udemy - Design Patterns in Modern C++

  1. edX

Curs Recomanat:

  • Software Construction: Object-Oriented Design per el MIT
    • Descripció: Aquest curs cobreix els fonaments del disseny orientat a objectes i els patrons de disseny.
    • Enllaç: edX - Software Construction: Object-Oriented Design

  1. Pluralsight

Curs Recomanat:

  • Design Patterns Library per diversos autors
    • Descripció: Una biblioteca de cursos que cobreixen una àmplia gamma de patrons de disseny en diferents llenguatges de programació.
    • Enllaç: Pluralsight - Design Patterns Library

Conclusió

Aquests recursos en línia proporcionen una àmplia gamma d'opcions per aprendre sobre els patrons de disseny de programari. Tant si prefereixes cursos estructurats, tutorials en vídeo, articles detallats o plataformes interactives, hi ha una opció per a cada estil d'aprenentatge. Utilitza aquests recursos per aprofundir en els teus coneixements i aplicar els patrons de disseny de manera efectiva en els teus projectes de programari.
