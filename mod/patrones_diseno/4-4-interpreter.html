4-4: Interpreter

Què és el Patró Interpreter?

El patró Interpreter és un patró de disseny de comportament que proporciona una manera d'avaluar sentències en un llenguatge. Aquest patró és utilitzat per interpretar expressions en llenguatges de domini específic (DSLs). El patró defineix una representació gramatical per a un llenguatge i utilitza aquesta representació per interpretar sentències en el llenguatge.

Components del Patró Interpreter

  1. Context: Conté informació global que és utilitzada per l'intèrpret.
  2. Abstract Expression: Declara una interfície per interpretar el context.
  3. Terminal Expression: Implementa una operació d'interpretació associada amb símbols terminals del llenguatge.
  4. Nonterminal Expression: Implementa una operació d'interpretació per a símbols no terminals del llenguatge.

Exemple de Codi

Suposem que volem interpretar expressions matemàtiques simples com "3 + 5" o "10 - 2".

class Context:
    def __init__(self, expression):
        self.expression = expression
        self.index = 0

class AbstractExpression:
    def interpret(self, context):
        pass

class NumberExpression(AbstractExpression):
    def interpret(self, context):
        start = context.index
        while context.index < len(context.expression) and context.expression[context.index].isdigit():
            context.index += 1
        return int(context.expression[start:context.index])

class PlusExpression(AbstractExpression):
    def interpret(self, context):
        context.index += 1  # Skip '+'
        left = NumberExpression().interpret(context)
        right = NumberExpression().interpret(context)
        return left + right

class MinusExpression(AbstractExpression):
    def interpret(self, context):
        context.index += 1  # Skip '-'
        left = NumberExpression().interpret(context)
        right = NumberExpression().interpret(context)
        return left - right

# Client code
expression = "3+5"
context = Context(expression)
result = PlusExpression().interpret(context)
print(f"Result of '{expression}' is {result}")

expression = "10-2"
context = Context(expression)
result = MinusExpression().interpret(context)
print(f"Result of '{expression}' is {result}")

Explicació del Codi

  1. Context: Manté la informació de l'expressió i l'índex actual.
  2. AbstractExpression: Defineix la interfície interpret.
  3. NumberExpression: Interpreta números en l'expressió.
  4. PlusExpression: Interpreta l'operació de suma.
  5. MinusExpression: Interpreta l'operació de resta.

Avantatges del Patró Interpreter

  • Simplicitat: Facilita la implementació de llenguatges de domini específic.
  • Extensibilitat: És fàcil afegir noves regles gramaticals.

Desavantatges del Patró Interpreter

  • Rendiment: Pot ser ineficient per a llenguatges complexos o expressions llargues.
  • Mantenibilitat: Pot resultar en un gran nombre de classes per a llenguatges complexos.

Conclusió

El patró Interpreter és útil per a la interpretació de llenguatges de domini específic i expressions simples. Tot i que és fàcil d'implementar i estendre, pot no ser adequat per a llenguatges complexos a causa de problemes de rendiment i mantenibilitat. És important considerar aquests factors abans de decidir utilitzar aquest patró en un projecte.
