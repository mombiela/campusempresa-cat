Instal·lant Redis
Següent ►

En aquesta secció, aprendrem com instal·lar Redis al vostre sistema. Redis és una base de dades en memòria que es pot instal·lar en diverses plataformes. Aquí, ens centrarem en la instal·lació en sistemes operatius comuns com Linux, macOS i Windows.

  1. Instal·lació a Linux

1.1. Utilitzant el Gestor de Paquets (APT per a Debian/Ubuntu)

  1. Actualitza la llista de paquets:

    sudo apt update

  2. Instal·la Redis:

    sudo apt install redis-server

  3. Verifica la instal·lació:

    redis-server --version

1.2. Utilitzant el Gestor de Paquets (YUM per a CentOS/RHEL)

  1. Afegeix el repositori EPEL:

    sudo yum install epel-release

  2. Instal·la Redis:

    sudo yum install redis

  3. Inicia el servei Redis:

    sudo systemctl start redis

  4. Verifica la instal·lació:

    redis-server --version

  1. Instal·lació a macOS

2.1. Utilitzant Homebrew

  1. Instal·la Homebrew (si no el tens instal·lat):

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  2. Actualitza Homebrew:

    brew update

  3. Instal·la Redis:

    brew install redis

  4. Inicia Redis:

    brew services start redis

  5. Verifica la instal·lació:

    redis-server --version

  1. Instal·lació a Windows

3.1. Utilitzant WSL (Windows Subsystem for Linux)

  1. Instal·la WSL (si no el tens instal·lat):

    wsl --install

  2. **Instal·la una distribució de Linux (per exemple, Ubuntu) des de la Microsoft Store.

  3. Obre la terminal de WSL i segueix els passos per a la instal·lació a Linux (Debian/Ubuntu):

    sudo apt update
sudo apt install redis-server
redis-server --version

3.2. Utilitzant Redis per a Windows

  1. Descarrega Redis per a Windows des del repositori oficial: https://github.com/microsoftarchive/redis/releases

  2. Descomprimeix el fitxer descarregat i executa redis-server.exe.

  3. Verifica la instal·lació:

    redis-server --version

  1. Configuració Bàsica

Després d'instal·lar Redis, és important configurar-lo correctament per assegurar un rendiment òptim i seguretat. Aquí tens alguns passos bàsics:

  1. Edita el fitxer de configuració:

    sudo nano /etc/redis/redis.conf

  2. Configura la seguretat:

    • Estableix una contrasenya: 
      requirepass yourpassword

  3. Configura la persistència:

    • Habilita les instantànies (RDB): 
      save 900 1
save 300 10
save 60 10000

  4. Reinicia el servei Redis per aplicar els canvis:

    sudo systemctl restart redis

  1. Verificació de la Instal·lació

Per assegurar-te que Redis està funcionant correctament, pots utilitzar la CLI de Redis:

  1. Inicia la CLI de Redis:

    redis-cli

  2. Executa una comanda de prova:

    ping

    Hauries de veure la resposta:

    PONG

Conclusió

Ara que has instal·lat Redis al teu sistema, estàs preparat per començar a explorar les seves capacitats. En el següent mòdul, aprendrem sobre els tipus de dades bàsics de Redis i com utilitzar-los.
