Desenvolupador Fullstack

Combina habilitats de frontend i backend per crear aplicacions completes, treballant tant en el costat del client com en el del servidor.

Llenguatges

CSS
Llenguatge d'estil utilitzat per definir l'aparença i el disseny de les pàgines web.
SQL
Llenguatge de consulta estructurat utilitzat per gestionar i manipular bases de dades relacionals.
C#
Llenguatge de programació orientat a objectes desenvolupat per Microsoft per a la creació d'aplicacions en múltiples plataformes.
Java
Llenguatge de programació àmpliament utilitzat per al desenvolupament d'aplicacions empresarials i mòbils.
HTML
Llenguatge de marcatge utilitzat per a l'estructura i contingut de les pàgines web.
MongoDB
Base de dades NoSQL orientada a documents, utilitzada per emmagatzemar dades de manera flexible i escalable.
Python
Llenguatge de programació multipropòsit popular per la seva senzillesa i gran ecosistema per a ciència de dades, backend i més.
JavaScript
Llenguatge de programació utilitzat per crear interactivitat i dinamisme en les pàgines web.
Bash
Shell d'Unix utilitzat per a l'automatització de tasques i l'execució de comandes en sistemes operatius basats en Unix.

Frameworks

React
Biblioteca de JavaScript per a la construcció d'interfícies d'usuari interactives i dinàmiques.
Node.js
Entorn d'execució per a JavaScript en el costat del servidor, utilitzat per construir aplicacions escalables.
Spring Boot
Framework de Java per a la creació ràpida d'aplicacions empresarials, facilitant la configuració i desplegament.
Webpack
Eina d'empaquetat de mòduls utilitzada per compilar i optimitzar arxius JavaScript, CSS i altres recursos web.
Django
Framework d'alt nivell per al desenvolupament d'aplicacions web en Python, conegut per la seva facilitat d'ús i robustesa.
Vue.js
Framework progressiu de JavaScript per construir interfícies d'usuari i aplicacions web interactives.
Flask
Framework lleuger de Python per al desenvolupament d'aplicacions web.
CI/CD
Integració i desplegament continu (CI/CD), pràctiques per automatitzar la construcció, proves i entrega de programari.

Eines

Git
Sistema de control de versions distribuït utilitzat per al seguiment de canvis en el codi.

Competències

Adaptabilitat
Capacitat per ajustar-se ràpidament a noves circumstàncies, tecnologies o mètodes de treball.
Comunicació efectiva
Capacitat per expressar idees de manera clara i comprensible tant en forma oral com escrita.
Gestió del temps
Habilitat per organitzar i planificar el temps de manera eficient, complint amb terminis establerts.

Fonaments

Estructures de Dades
Introducció a les estructures de dades comunes com llistes, piles, cues, arbres i grafs, essencials per a programadors.
Algorismes
Curs sobre anàlisi i disseny d'algorismes, que cobreix l'eficiència dels algorismes i com millorar el rendiment del codi.
OWASP
Conjunt de directrius i estàndards per a la seguretat en aplicacions web, creat per la Open Web Application Security Project.
Patrons de Disseny
Curs sobre patrons de disseny de programari que proporcionen solucions reutilitzables a problemes comuns de desenvolupament.
REST API
Principis de disseny i desenvolupament d'APIs RESTful, utilitzades per a la interacció entre aplicacions.
Fonaments de Bases de Dades
Introducció als principis de bases de dades relacionals i no relacionals, disseny d'esquemes i normalització.
Fonaments de la Programació
Curs introductori sobre conceptes bàsics de la programació, com variables, estructures de control, i algorismes.
Microserveis
Arquitectura de programari per construir aplicacions en petits serveis desacoblats que interactuen a través d'APIs.
Fonaments de Seguretat Informàtica
Curs sobre principis bàsics de seguretat, ciberseguretat, i criptografia, cobrint riscos i mesures de protecció.
