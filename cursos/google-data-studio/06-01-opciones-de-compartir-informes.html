Opcions de compartir informes

Següent ►
►

En aquest tema, explorarem les diferents opcions que Google Data Studio ofereix per compartir els informes que has creat. Compartir informes és una part essencial del procés, ja que permet col·laborar amb altres usuaris, presentar dades a clients o equips, i assegurar que la informació sigui accessible per a les persones adequades. A continuació, desglossarem les opcions disponibles i com utilitzar-les.

  1. Compartir amb individus o grups

1.1. Compartir amb correus electrònics específics

Google Data Studio permet compartir informes amb usuaris específics mitjançant les seves adreces de correu electrònic. Això és útil quan vols controlar qui té accés al teu informe.

Passos per compartir amb correus electrònics específics:

  1. Obre l'informe que vols compartir.
  2. Fes clic al botó "Compartir" a la part superior dreta de la pantalla.
  3. Introdueix les adreces de correu electrònic dels usuaris amb qui vols compartir l'informe.
  4. Selecciona el nivell d'accés (visualització o edició).
  5. Fes clic a "Enviar".

1.2. Compartir amb grups de Google

Si treballes en una organització que utilitza Google Workspace, pots compartir informes amb grups de Google. Això facilita la gestió d'accés per a equips grans.

Passos per compartir amb grups de Google:

  1. Segueix els mateixos passos que per compartir amb correus electrònics específics.
  2. En lloc d'introduir adreces de correu individuals, introdueix l'adreça del grup de Google.

  1. Enllaços compartits

2.1. Crear un enllaç compartit

Google Data Studio permet generar un enllaç compartit que pots enviar a qualsevol persona. Pots controlar si els usuaris que reben l'enllaç poden veure o editar l'informe.

Passos per crear un enllaç compartit:

  1. Obre l'informe que vols compartir.
  2. Fes clic al botó "Compartir" a la part superior dreta de la pantalla.
  3. Fes clic a "Obtenir enllaç compartit".
  4. Selecciona el nivell d'accés (visualització o edició).
  5. Copia l'enllaç i comparteix-lo amb qui vulguis.

2.2. Configuració de permisos d'enllaç

Pots ajustar els permisos de l'enllaç compartit per assegurar-te que només les persones adequades tinguin accés.

Opcions de permisos:

  • Qualsevol persona amb l'enllaç: Permet que qualsevol persona amb l'enllaç pugui veure o editar l'informe.
  • Només usuaris específics: Restringeix l'accés a usuaris específics que has convidat.

  1. Publicar informes

3.1. Publicar a la web

Google Data Studio permet publicar informes a la web, fent-los accessibles a qualsevol persona amb l'enllaç públic. Aquesta opció és útil per a informes que necessiten ser accessibles públicament.

Passos per publicar a la web:

  1. Obre l'informe que vols publicar.
  2. Fes clic al botó "Compartir" a la part superior dreta de la pantalla.
  3. Fes clic a "Publicar a la web".
  4. Selecciona les opcions de publicació (per exemple, si vols que l'informe sigui indexat pels motors de cerca).
  5. Fes clic a "Publicar".

3.2. Embedir informes en llocs web

Pots embedir informes de Google Data Studio en llocs web o blogs utilitzant codi HTML.

Passos per embedir un informe:

  1. Obre l'informe que vols embedir.
  2. Fes clic al botó "Compartir" a la part superior dreta de la pantalla.
  3. Fes clic a "Obtenir codi d'inserció".
  4. Copia el codi HTML proporcionat.
  5. Insereix el codi HTML en el lloc web o blog on vols que aparegui l'informe.

  1. Control de versions i historial

4.1. Historial de versions

Google Data Studio manté un historial de versions dels informes, permetent-te veure i restaurar versions anteriors si cal.

Passos per accedir a l'historial de versions:

  1. Obre l'informe.
  2. Fes clic a "Fitxer" al menú superior.
  3. Selecciona "Historial de versions".
  4. Revisa les versions anteriors i, si cal, restaura una versió anterior.

  1. Exercici pràctic

Exercici:

  1. Crea un informe de prova a Google Data Studio.
  2. Comparteix l'informe amb un company mitjançant la seva adreça de correu electrònic.
  3. Genera un enllaç compartit per a l'informe i envia'l a un altre company.
  4. Publica l'informe a la web i embedeix-lo en un lloc web de prova.

Solució: Segueix els passos descrits en les seccions anteriors per completar l'exercici. Assegura't de verificar els permisos d'accés i comprova que els teus companys poden accedir a l'informe segons els permisos que has configurat.

Conclusió

En aquesta secció, hem explorat les diferents opcions per compartir informes a Google Data Studio. Hem après a compartir informes amb individus i grups, crear enllaços compartits, publicar informes a la web i embedir-los en llocs web. També hem vist com accedir a l'historial de versions per gestionar els canvis en els informes. Aquestes habilitats són essencials per col·laborar eficaçment i assegurar que els teus informes siguin accessibles per a les persones adequades.
