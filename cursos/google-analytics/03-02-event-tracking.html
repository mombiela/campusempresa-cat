Seguiment d'esdeveniments

Següent ►
►

El seguiment d'esdeveniments a Google Analytics és una funcionalitat poderosa que permet als usuaris monitoritzar les interaccions específiques dels usuaris amb el seu lloc web o aplicació. Aquestes interaccions poden incloure clics en botons, descàrregues de fitxers, reproduccions de vídeos, entre altres accions que no es registren automàticament com a pàgines vistes.

Conceptes Clau

Abans de començar amb la configuració del seguiment d'esdeveniments, és important entendre alguns conceptes bàsics:

  • Categoria: Un nom que agrupa objectes que volen ser seguides. Per exemple, "Vídeos".
  • Acció: El tipus d'interacció que es vol seguir. Per exemple, "Reproduir".
  • Etiqueta: Informació addicional sobre l'esdeveniment. Per exemple, el títol del vídeo.
  • Valor: Un valor numèric associat a l'esdeveniment. Per exemple, la durada del vídeo en segons.

Configuració del Seguiment d'Esdeveniments

  1. Afegir el Codi de Seguiment

Per configurar el seguiment d'esdeveniments, cal afegir codi JavaScript al lloc web. A continuació es mostra un exemple bàsic de codi per seguir un clic en un botó:

<button id="playButton">Reproduir Vídeo</button>

<script>
  document.getElementById('playButton').addEventListener('click', function() {
    gtag('event', 'Reproduir', {
      'event_category': 'Vídeos',
      'event_label': 'Vídeo Promocional',
      'value': 30
    });
  });
</script>

Explicació del Codi

  • document.getElementById('playButton').addEventListener('click', function() { ... });: Afegeix un esdeveniment de clic al botó amb l'ID playButton.
  • gtag('event', 'Reproduir', { ... });: Envia un esdeveniment a Google Analytics amb l'acció "Reproduir".
  • 'event_category': 'Vídeos': Defineix la categoria de l'esdeveniment com "Vídeos".
  • 'event_label': 'Vídeo Promocional': Afegeix una etiqueta a l'esdeveniment amb el valor "Vídeo Promocional".
  • 'value': 30: Assigna un valor numèric de 30 a l'esdeveniment.

  1. Verificar el Seguiment d'Esdeveniments

Després d'afegir el codi, és important verificar que els esdeveniments es registren correctament a Google Analytics. Això es pot fer utilitzant els informes en temps real:

  1. Accedeix a Google Analytics.
  2. Navega a Temps Real > Esdeveniments.
  3. Interactua amb l'element del lloc web que has configurat per seguir.
  4. Verifica que l'esdeveniment apareix a l'informe en temps real.

Exercici Pràctic

Objectiu

Configurar el seguiment d'un esdeveniment per a un enllaç de descàrrega de fitxer.

Instruccions

  1. Afegeix un enllaç de descàrrega al teu lloc web:

    <a href="document.pdf" id="downloadLink">Descarregar Document</a>

  2. Afegeix el següent codi JavaScript per seguir els clics en l'enllaç de descàrrega:

    <script>
  document.getElementById('downloadLink').addEventListener('click', function() {
    gtag('event', 'Descàrrega', {
      'event_category': 'Documents',
      'event_label': 'Document PDF',
      'value': 1
    });
  });
</script>

  3. Verifica que l'esdeveniment es registra correctament a Google Analytics utilitzant els informes en temps real.

Solució

El codi complet hauria de semblar-se a això:

<!DOCTYPE html>
<html lang="ca">
<head>
  <meta charset="UTF-8">
  <title>Seguiment d'Esdeveniments</title>
  <!-- Inclou el codi de seguiment de Google Analytics -->
  <script async src="https://www.googletagmanager.com/gtag/js?id=UA-XXXXX-Y"></script>
  <script>
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());
    gtag('config', 'UA-XXXXX-Y');
  </script>
</head>
<body>
  <a href="document.pdf" id="downloadLink">Descarregar Document</a>

  <script>
    document.getElementById('downloadLink').addEventListener('click', function() {
      gtag('event', 'Descàrrega', {
        'event_category': 'Documents',
        'event_label': 'Document PDF',
        'value': 1
      });
    });
  </script>
</body>
</html>

Errors Comuns i Consells

  • Error: No es registren esdeveniments: Assegura't que el codi de seguiment de Google Analytics està correctament implementat i que no hi ha errors de JavaScript a la consola del navegador.
  • Consell: Utilitza Google Tag Manager: Per a una gestió més fàcil dels esdeveniments, considera utilitzar Google Tag Manager, que permet configurar esdeveniments sense necessitat de modificar el codi del lloc web.

Conclusió

El seguiment d'esdeveniments a Google Analytics és una eina essencial per comprendre com els usuaris interactuen amb el teu lloc web. Amb aquesta funcionalitat, pots obtenir informació detallada sobre accions específiques dels usuaris, la qual cosa et permetrà prendre decisions informades per millorar l'experiència de l'usuari i optimitzar el rendiment del teu lloc web.
