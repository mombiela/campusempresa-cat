Instal·lació i Configuració de Servidors

Següent ►
►

Introducció

La instal·lació i configuració de servidors és una tasca fonamental per a qualsevol professional de TI. Un servidor ben configurat pot proporcionar serveis fiables i segurs a una organització, mentre que una configuració inadequada pot resultar en problemes de rendiment, seguretat i disponibilitat. En aquesta secció, explorarem els passos necessaris per instal·lar i configurar un servidor, des de la selecció del maquinari fins a la configuració del sistema operatiu i els serveis.

  1. Selecció del Maquinari

Factors a Considerar

  • CPU: Nombre de nuclis i velocitat de processament.
  • Memòria RAM: Quantitat necessària per suportar les aplicacions i serveis.
  • Emmagatzematge: Tipus (HDD vs. SSD) i capacitat.
  • Redundància: Fonts d'alimentació redundants, RAID per a emmagatzematge.
  • Compatibilitat: Assegurar que el maquinari sigui compatible amb el sistema operatiu i les aplicacions.

Exemples de Maquinari

Component Opció Bàsica Opció Avançada
CPU Intel Xeon E3 Intel Xeon E7
RAM 16 GB 64 GB
Emmagatzematge 1 TB HDD 1 TB SSD RAID 10
Xarxa 1 Gbps NIC 10 Gbps NIC

  1. Instal·lació del Sistema Operatiu

Passos per a la Instal·lació

  1. Preparació del Mitjà d'Instal·lació: USB, DVD o imatge ISO.
  2. Configuració del BIOS/UEFI: Assegurar que el dispositiu d'arrencada sigui correcte.
  3. Inici de la Instal·lació: Seguir les instruccions del sistema operatiu (Windows Server, Linux, etc.).
  4. Particionament del Disc: Configurar les particions segons les necessitats.
  5. Configuració Inicial: Nom del servidor, contrasenya de l'administrador, etc.

Exemple de Comandament per a Instal·lar Ubuntu Server

sudo apt-get update
sudo apt-get install tasksel
sudo tasksel install server

  1. Configuració Bàsica del Sistema Operatiu

Configuracions Inicials

  • Actualització del Sistema: Assegurar que totes les actualitzacions estiguin instal·lades.
  • Configuració del Nom del Host: Assignar un nom únic al servidor.
  • Configuració de la Zona Horària: Ajustar la zona horària correcta.

Exemple de Configuració en Linux

sudo apt-get update && sudo apt-get upgrade -y
sudo hostnamectl set-hostname nom-del-servidor
sudo timedatectl set-timezone Europe/Madrid

  1. Instal·lació de Serveis i Aplicacions

Serveis Comuns

  • Servidor Web: Apache, Nginx.
  • Base de Dades: MySQL, PostgreSQL.
  • Servidor de Fitxers: Samba, NFS.

Exemple d'Instal·lació d'Apache en Linux

sudo apt-get install apache2
sudo systemctl start apache2
sudo systemctl enable apache2

  1. Configuració de la Xarxa

Configuracions Necessàries

  • Assignació d'IP: Estàtica vs. Dinàmica (DHCP).
  • Configuració de DNS: Assignar servidors DNS.
  • Configuració de la Passarel·la: Definir la passarel·la per a la sortida a Internet.

Exemple de Configuració d'IP Estàtica en Linux

sudo nano /etc/netplan/01-netcfg.yaml
network:
  version: 2
  ethernets:
    eth0:
      dhcp4: no
      addresses: [192.168.1.100/24]
      gateway4: 192.168.1.1
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4]
sudo netplan apply

  1. Configuració de la Seguretat

Mesures de Seguretat

  • Firewall: Configurar regles de tallafocs.
  • Actualitzacions de Seguretat: Assegurar que el sistema estigui sempre actualitzat.
  • Accés SSH: Configurar l'accés segur via SSH.

Exemple de Configuració de Firewall en Linux

sudo ufw allow OpenSSH
sudo ufw enable

  1. Proves i Validació

Passos per Validar la Configuració

  • Proves de Connectivitat: Ping, traceroute.
  • Proves de Rendiment: Utilitzar eines com htop, iostat.
  • Proves de Seguretat: Escanejar ports, revisar logs.

Exemple de Comandaments de Prova

ping -c 4 8.8.8.8
traceroute google.com
htop

Conclusió

La instal·lació i configuració de servidors és una tasca crítica que requereix atenció als detalls i una comprensió profunda de les necessitats de l'organització. Seguint els passos descrits en aquesta secció, els professionals de TI poden assegurar-se que els seus servidors estiguin ben configurats, segurs i preparats per oferir serveis fiables.

Amb aquesta guia, esperem que tinguis una comprensió clara dels passos necessaris per instal·lar i configurar un servidor. En la següent secció, explorarem com monitoritzar i mantenir els servidors per garantir el seu rendiment i disponibilitat continuada.
