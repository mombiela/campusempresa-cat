Tipus de Servidors i els seus Usos

Següent ►
►

En aquest tema, explorarem els diferents tipus de servidors que es poden trobar en una infraestructura TI, així com els seus usos específics. Els servidors són components essencials en qualsevol entorn TI, ja que proporcionen serveis i recursos a altres dispositius i usuaris de la xarxa.

  1. Introducció als Servidors

Què és un servidor?

Un servidor és un sistema informàtic que proporciona serveis, dades o recursos a altres ordinadors, coneguts com a clients, a través d'una xarxa. Els servidors poden ser físics o virtuals i poden executar una àmplia varietat de serveis, des de l'allotjament de pàgines web fins a la gestió de bases de dades.

  1. Tipus de Servidors

2.1 Servidors de Fitxers

  • Funció: Emmagatzemar i gestionar fitxers per a múltiples usuaris.
  • Usos: Compartició de documents, còpies de seguretat, emmagatzematge centralitzat.
  • Exemple: Un servidor NAS (Network Attached Storage) que permet als empleats accedir a documents compartits.

2.2 Servidors de Bases de Dades

  • Funció: Emmagatzemar, gestionar i proporcionar accés a bases de dades.
  • Usos: Aplicacions empresarials, sistemes de gestió de continguts, comerç electrònic.
  • Exemple: Un servidor MySQL que gestiona les dades d'una aplicació web.

2.3 Servidors Web

  • Funció: Allotjar i servir pàgines web a través d'Internet o una intranet.
  • Usos: Allotjament de llocs web, aplicacions web, serveis web.
  • Exemple: Un servidor Apache que allotja el lloc web corporatiu d'una empresa.

2.4 Servidors de Correu Electrònic

  • Funció: Gestionar l'enviament, recepció i emmagatzematge de correus electrònics.
  • Usos: Comunicació empresarial, gestió de correu electrònic.
  • Exemple: Un servidor Microsoft Exchange que gestiona els correus electrònics dels empleats.

2.5 Servidors d'Aplicacions

  • Funció: Executar aplicacions empresarials i proporcionar serveis a altres aplicacions o usuaris.
  • Usos: ERP, CRM, aplicacions personalitzades.
  • Exemple: Un servidor Tomcat que executa una aplicació Java per a la gestió de recursos humans.

2.6 Servidors de Virtualització

  • Funció: Crear i gestionar màquines virtuals (VMs).
  • Usos: Consolidació de servidors, proves i desenvolupament, alta disponibilitat.
  • Exemple: Un servidor VMware ESXi que allotja diverses màquines virtuals per a diferents departaments.

2.7 Servidors de Jocs

  • Funció: Allotjar i gestionar jocs en línia.
  • Usos: Jocs multijugador en línia, plataformes de jocs.
  • Exemple: Un servidor Minecraft que permet als jugadors connectar-se i jugar junts.

2.8 Servidors de Proxy

  • Funció: Actuar com a intermediari entre els clients i altres servidors per millorar la seguretat i el rendiment.
  • Usos: Filtratge de contingut, balanç de càrrega, anonimat.
  • Exemple: Un servidor Squid que gestiona les sol·licituds de navegació web dels empleats.

2.9 Servidors DNS (Domain Name System)

  • Funció: Resoldre noms de domini en adreces IP.
  • Usos: Navegació web, serveis de xarxa.
  • Exemple: Un servidor BIND que gestiona les sol·licituds DNS d'una organització.

  1. Comparació de Tipus de Servidors

  1. Exercicis Pràctics

Exercici 1: Identificació de Tipus de Servidors

Descripció: Identifica el tipus de servidor que s'utilitzaria en els següents escenaris.

  1. Una empresa necessita un servidor per emmagatzemar i compartir documents entre els empleats.
  2. Un lloc web de comerç electrònic necessita gestionar les dades dels seus clients i comandes.
  3. Una organització vol allotjar el seu lloc web corporatiu.
  4. Una empresa vol gestionar els correus electrònics dels seus empleats.
  5. Un equip de desenvolupament necessita un entorn per provar diferents aplicacions en màquines virtuals.

Solucions:

  1. Servidor de Fitxers
  2. Servidor de Bases de Dades
  3. Servidor Web
  4. Servidor de Correu Electrònic
  5. Servidor de Virtualització

Exercici 2: Configuració Bàsica d'un Servidor Web

Descripció: Configura un servidor Apache en una màquina virtual Linux per allotjar una pàgina web simple.

Passos:

  1. Instal·la Apache: 
    sudo apt update
sudo apt install apache2
  2. Crea un fitxer HTML simple: 
    echo "<html><body><h1>Hola, món!</h1></body></html>" | sudo tee /var/www/html/index.html
  3. Inicia el servei Apache: 
    sudo systemctl start apache2
  4. Verifica que el servidor web està funcionant accedint a http://<IP_del_servidor> des del navegador.

Conclusió: En aquest tema, hem explorat els diferents tipus de servidors i els seus usos específics dins d'una infraestructura TI. Hem après a identificar quin tipus de servidor és adequat per a diferents necessitats empresarials i hem realitzat exercicis pràctics per reforçar els conceptes apresos. En el següent tema, ens endinsarem en la instal·lació i configuració de servidors per comprendre millor com posar en marxa aquests components essencials.
