Operacions Bàsiques en SQL

Següent ►
►

En aquest tema, aprendrem les operacions bàsiques que es poden realitzar amb SQL (Structured Query Language). Aquestes operacions són fonamentals per interactuar amb bases de dades relacionals. Ens centrarem en les següents operacions:

  1. SELECT: Recuperar dades de la base de dades.
  2. INSERT: Afegir noves dades a la base de dades.
  3. UPDATE: Actualitzar dades existents.
  4. DELETE: Eliminar dades de la base de dades.

  1. SELECT: Recuperar Dades

La instrucció SELECT s'utilitza per recuperar dades d'una o més taules. Aquí teniu la sintaxi bàsica:

SELECT columna1, columna2, ...
FROM taula
WHERE condició;

Exemple

Suposem que tenim una taula empleats amb les següents columnes: id, nom, càrrec, i salari.

SELECT nom, càrrec
FROM empleats
WHERE salari > 50000;

Aquest exemple recupera els noms i càrrecs dels empleats que tenen un salari superior a 50.000.

Exercici

Recupera tots els camps de la taula empleats on el càrrec sigui 'Enginyer'.

SELECT * 
FROM empleats 
WHERE càrrec = 'Enginyer';

  1. INSERT: Afegir Noves Dades

La instrucció INSERT s'utilitza per afegir noves files a una taula. Aquí teniu la sintaxi bàsica:

INSERT INTO taula (columna1, columna2, ...)
VALUES (valor1, valor2, ...);

Exemple

Afegim un nou empleat a la taula empleats:

INSERT INTO empleats (id, nom, càrrec, salari)
VALUES (4, 'Anna', 'Analista', 60000);

Exercici

Afegeix un nou empleat amb els següents detalls: id = 5, nom = 'Joan', càrrec = 'Desenvolupador', salari = 55000.

INSERT INTO empleats (id, nom, càrrec, salari)
VALUES (5, 'Joan', 'Desenvolupador', 55000);

  1. UPDATE: Actualitzar Dades Existents

La instrucció UPDATE s'utilitza per modificar dades existents en una taula. Aquí teniu la sintaxi bàsica:

UPDATE taula
SET columna1 = valor1, columna2 = valor2, ...
WHERE condició;

Exemple

Actualitzem el salari de l'empleat amb id 4:

UPDATE empleats
SET salari = 65000
WHERE id = 4;

Exercici

Actualitza el càrrec de l'empleat amb id 5 a 'Senior Desenvolupador'.

UPDATE empleats
SET càrrec = 'Senior Desenvolupador'
WHERE id = 5;

  1. DELETE: Eliminar Dades

La instrucció DELETE s'utilitza per eliminar files d'una taula. Aquí teniu la sintaxi bàsica:

DELETE FROM taula
WHERE condició;

Exemple

Eliminem l'empleat amb id 5:

DELETE FROM empleats
WHERE id = 5;

Exercici

Elimina tots els empleats amb un salari inferior a 40000.

DELETE FROM empleats
WHERE salari < 40000;

Resum

En aquesta secció, hem après les operacions bàsiques en SQL:

  • SELECT per recuperar dades.
  • INSERT per afegir noves dades.
  • UPDATE per actualitzar dades existents.
  • DELETE per eliminar dades.

Aquestes operacions són fonamentals per treballar amb bases de dades relacionals i són la base per a operacions més complexes que veurem en els següents mòduls. Practiqueu aquestes instruccions amb diferents condicions i taules per familiaritzar-vos amb el seu ús.
