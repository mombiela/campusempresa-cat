Cursos i Tutorials en Línia

Següent ►
►

En aquesta secció, proporcionarem una llista de recursos en línia que poden ajudar-te a aprofundir en els coneixements adquirits durant aquest curs. Aquests recursos inclouen cursos, tutorials, vídeos i altres materials educatius que cobreixen diversos aspectes de les bases de dades.

  1. Cursos en Línia

1.1. Coursera

  • Databases and SQL for Data Science

    • Descripció: Aquest curs ofereix una introducció a les bases de dades i SQL, amb un enfocament especial en l'ús de SQL per a l'anàlisi de dades.
    • Durada: 4 setmanes
    • Nivell: Principiant
    • Idioma: Anglès

  • Database Management Essentials

    • Descripció: Aquest curs cobreix els fonaments de la gestió de bases de dades, incloent-hi el model relacional, SQL i el disseny de bases de dades.
    • Durada: 7 setmanes
    • Nivell: Intermedi
    • Idioma: Anglès

1.2. edX

  • Introduction to Databases

    • Descripció: Aquest curs proporciona una introducció completa a les bases de dades, incloent-hi SQL, NoSQL, i el disseny de bases de dades.
    • Durada: 12 setmanes
    • Nivell: Principiant a Intermedi
    • Idioma: Anglès

  • Advanced Database Systems

    • Descripció: Aquest curs cobreix temes avançats en bases de dades, com ara l'optimització de consultes, la gestió de transaccions i les bases de dades distribuïdes.
    • Durada: 8 setmanes
    • Nivell: Avançat
    • Idioma: Anglès

1.3. Udemy

  • The Complete SQL Bootcamp

    • Descripció: Un curs complet que cobreix des dels fonaments de SQL fins a temes més avançats com les funcions agregades i les subconsultes.
    • Durada: 9.5 hores de vídeo
    • Nivell: Principiant a Intermedi
    • Idioma: Anglès

  • NoSQL Databases: Big Data and Hadoop

    • Descripció: Aquest curs introdueix les bases de dades NoSQL i la seva aplicació en el context del Big Data i Hadoop.
    • Durada: 6 hores de vídeo
    • Nivell: Intermedi
    • Idioma: Anglès

  1. Tutorials en Línia

2.1. W3Schools

  • SQL Tutorial
    • Descripció: Un tutorial interactiu que cobreix els fonaments de SQL, incloent-hi consultes bàsiques, funcions agregades i gestió de bases de dades.
    • Nivell: Principiant
    • Idioma: Anglès

2.2. TutorialsPoint

  • SQL Tutorial

    • Descripció: Un tutorial detallat que cobreix una àmplia gamma de temes relacionats amb SQL, des de conceptes bàsics fins a temes avançats.
    • Nivell: Principiant a Avançat
    • Idioma: Anglès

  • NoSQL Tutorial

    • Descripció: Un tutorial que proporciona una introducció completa a les bases de dades NoSQL, incloent-hi diferents tipus de NoSQL com ara document, clau-valor, columna i gràfic.
    • Nivell: Principiant a Intermedi
    • Idioma: Anglès

  1. Vídeos Educatius

3.1. YouTube

  • MySQL Tutorial for Beginners

    • Canal: Programming with Mosh
    • Descripció: Una sèrie de vídeos que cobreixen els fonaments de MySQL, des de la instal·lació fins a la creació de consultes complexes.
    • Nivell: Principiant
    • Idioma: Anglès

  • NoSQL Databases Explained

    • Canal: Academind
    • Descripció: Un vídeo que explica els conceptes bàsics de les bases de dades NoSQL i les seves diferències amb les bases de dades relacionals.
    • Nivell: Principiant
    • Idioma: Anglès

  1. Plataformes de Pràctica

4.1. LeetCode

  • Database Problems
    • Descripció: Una col·lecció de problemes de bases de dades que permeten practicar SQL en un entorn interactiu.
    • Nivell: Intermedi a Avançat
    • Idioma: Anglès

4.2. HackerRank

  • SQL Practice
    • Descripció: Una plataforma que ofereix una àmplia gamma de problemes de SQL per practicar i millorar les habilitats de consulta.
    • Nivell: Principiant a Avançat
    • Idioma: Anglès

Conclusió

Aquests recursos addicionals poden complementar el teu aprenentatge i proporcionar-te una comprensió més profunda dels conceptes de bases de dades. Recomanem explorar aquests cursos i tutorials per reforçar els coneixements adquirits i per mantenir-te actualitzat amb les últimes tendències i tecnologies en el camp de les bases de dades.
