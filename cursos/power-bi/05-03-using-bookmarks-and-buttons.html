Utilitzant marcadors i botons

Següent ►
►

Introducció

En aquest tema, aprendrem a utilitzar marcadors i botons a Power BI per crear informes més interactius i dinàmics. Els marcadors permeten capturar l'estat actual d'una pàgina de l'informe, incloent filtres, seleccions visuals i la posició de la pàgina. Els botons es poden utilitzar per navegar entre pàgines, aplicar filtres o activar marcadors.

Objectius

  • Entendre què són els marcadors i com es poden utilitzar.
  • Aprendre a crear i gestionar marcadors.
  • Utilitzar botons per millorar la interactivitat dels informes.
  • Aplicar marcadors a botons per crear experiències d'usuari dinàmiques.

Què són els marcadors?

Els marcadors a Power BI són una eina que permet capturar l'estat actual d'una pàgina de l'informe. Això inclou:

  • Filtres aplicats
  • Seleccions visuals
  • Posició de la pàgina
  • Estat de visibilitat dels objectes

Els marcadors es poden utilitzar per crear experiències d'usuari dinàmiques, com ara mostrar o amagar visuals, navegar entre pàgines o aplicar filtres específics.

Creant i gestionant marcadors

Passos per crear un marcador

  1. Configura la pàgina de l'informe: Aplica els filtres, seleccions visuals i ajusta la posició de la pàgina segons sigui necessari.
  2. Obre el panell de marcadors: A la cinta de Power BI Desktop, ves a la pestanya "Vista" i selecciona "Marcadors".
  3. Afegeix un nou marcador: Fes clic a "Afegeix" al panell de marcadors. Això capturarà l'estat actual de la pàgina.
  4. Configura el marcador: Pots canviar el nom del marcador i ajustar les opcions com ara "Dades", "Visuals" i "Mostra".

Exemple pràctic

1. Aplica un filtre a un visual de gràfic de barres per mostrar només les vendes del 2022.
2. Obre el panell de marcadors.
3. Fes clic a "Afegeix" per crear un nou marcador.
4. Canvia el nom del marcador a "Vendes 2022".
5. Desactiva l'opció "Dades" si no vols que el marcador recordi els filtres aplicats.

Utilitzant botons

Tipus de botons

Power BI ofereix diversos tipus de botons que es poden utilitzar per millorar la interactivitat dels informes:

  • Botons de navegació: Permeten als usuaris navegar entre pàgines de l'informe.
  • Botons d'acció: Permeten aplicar accions específiques com ara aplicar un marcador o restablir filtres.

Creant un botó

  1. Afegeix un botó: A la cinta de Power BI Desktop, ves a la pestanya "Inserir" i selecciona "Botó".
  2. Configura el botó: Selecciona el botó i ves al panell de format per ajustar les propietats com ara el text, la icona i l'estil.
  3. Assigna una acció al botó: Al panell de format, activa l'opció "Acció" i selecciona el tipus d'acció (per exemple, "Marcador").
  4. Selecciona el marcador: Si has seleccionat "Marcador" com a acció, tria el marcador que vols aplicar quan es faci clic al botó.

Exemple pràctic

1. Afegeix un botó a la pàgina de l'informe.
2. Canvia el text del botó a "Mostrar vendes 2022".
3. Activa l'opció "Acció" al panell de format.
4. Selecciona "Marcador" com a tipus d'acció.
5. Tria el marcador "Vendes 2022" creat anteriorment.

Exercici pràctic

Objectiu

Crea un informe amb dos botons que permetin als usuaris alternar entre les vendes del 2021 i les vendes del 2022.

Passos

  1. Crea dos marcadors:

    • Aplica un filtre per mostrar les vendes del 2021 i crea un marcador anomenat "Vendes 2021".
    • Aplica un filtre per mostrar les vendes del 2022 i crea un marcador anomenat "Vendes 2022".

  2. Afegeix dos botons:

    • Afegeix un botó amb el text "Mostrar vendes 2021" i assigna-li el marcador "Vendes 2021".
    • Afegeix un botó amb el text "Mostrar vendes 2022" i assigna-li el marcador "Vendes 2022".

Solució

1. Aplica un filtre per mostrar les vendes del 2021.
2. Crea un marcador anomenat "Vendes 2021".
3. Aplica un filtre per mostrar les vendes del 2022.
4. Crea un marcador anomenat "Vendes 2022".
5. Afegeix un botó amb el text "Mostrar vendes 2021".
6. Assigna el marcador "Vendes 2021" al botó "Mostrar vendes 2021".
7. Afegeix un botó amb el text "Mostrar vendes 2022".
8. Assigna el marcador "Vendes 2022" al botó "Mostrar vendes 2022".

Conclusió

Els marcadors i botons són eines poderoses a Power BI que permeten crear informes més interactius i dinàmics. Amb els marcadors, pots capturar l'estat actual d'una pàgina de l'informe i utilitzar botons per permetre als usuaris navegar entre diferents estats o pàgines. Practica utilitzant aquestes eines per millorar la interactivitat dels teus informes i proporcionar una millor experiència d'usuari.
