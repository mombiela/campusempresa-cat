Treballant amb Formes i Text

Següent ►
►

En aquest tema, aprendrem a treballar amb formes i text a Figma, una de les habilitats fonamentals per a qualsevol dissenyador. Aquestes eines són essencials per crear dissenys atractius i funcionals. A continuació, desglossarem els conceptes clau i proporcionarem exemples pràctics per ajudar-te a dominar aquestes tècniques.

Conceptes Clau

  1. Formes Bàsiques

    • Rectangles i Quadrats: Ideals per a botons, fons i contenidors.
    • Cercles i El·lipses: Útils per a icones, botons rodons i elements decoratius.
    • Línies i Fletxes: Per a separadors, indicadors de direcció i connexions visuals.

  2. Text

    • Tipus de Font: Selecció de fonts adequades per a llegibilitat i estil.
    • Estils de Text: Aplicació de mides, pesos i colors per a jerarquia visual.
    • Alineació i Espaiat: Com alinear i espaiar text per a un disseny net i organitzat.

Eines i Funcions

Creació de Formes

  • Eina de Forma: Selecciona l'eina de forma des de la barra d'eines o utilitza les dreceres de teclat (R per a rectangles, O per a cercles).
  • Propietats de Forma: Ajusta les dimensions, el color de fons, el contorn i el radi de les cantonades.

Exemple de Codi

1. Selecciona l'eina de rectangle (R).
2. Dibuixa un rectangle al llenç.
3. Ajusta el color de fons a #3498db (blau).
4. Estableix el radi de les cantonades a 10 píxels.

Treballant amb Text

  • Eina de Text: Selecciona l'eina de text (T) i fes clic al llenç per començar a escriure.
  • Propietats de Text: Modifica la font, la mida, el pes, el color i l'alineació.

Exemple de Codi

1. Selecciona l'eina de text (T).
2. Fes clic al llenç i escriu "Benvingut a Figma".
3. Canvia la font a 'Roboto', mida 24, pes 'Bold'.
4. Ajusta el color del text a #2c3e50 (gris fosc).

Exercicis Pràctics

Exercici 1: Creació d'un Botó

Instruccions:

  1. Crea un rectangle de 150x50 píxels.
  2. Estableix el color de fons a #e74c3c (vermell).
  3. Afegeix text al centre del rectangle que digui "Enviar".
  4. Utilitza la font 'Arial', mida 16, color blanc.

Solució:

1. Dibuixa un rectangle de 150x50 píxels.
2. Canvia el color de fons a #e74c3c.
3. Selecciona l'eina de text (T) i escriu "Enviar".
4. Canvia la font a 'Arial', mida 16, color blanc.
5. Centra el text dins del rectangle.

Exercici 2: Creació d'un Títol

Instruccions:

  1. Utilitza l'eina de text per crear un títol que digui "Disseny de Prototips".
  2. Estableix la font a 'Montserrat', mida 32, pes 'SemiBold'.
  3. Ajusta el color del text a #34495e (blau fosc).

Solució:

1. Selecciona l'eina de text (T) i escriu "Disseny de Prototips".
2. Canvia la font a 'Montserrat', mida 32, pes 'SemiBold'.
3. Ajusta el color del text a #34495e.

Errors Comuns i Consells

  • Error Comú: No alinear correctament el text dins de les formes.

    • Consell: Utilitza les guies de Figma per assegurar-te que el text està centrat.

  • Error Comú: Utilitzar massa fonts diferents en un sol disseny.

    • Consell: Limita't a 2-3 fonts per mantenir la coherència visual.

Conclusió

En aquesta secció, hem après a treballar amb formes i text a Figma, una habilitat essencial per a qualsevol dissenyador. Hem explorat com crear i personalitzar formes, així com com treballar amb text per crear dissenys atractius i funcionals. Amb aquests coneixements, estàs preparat per avançar al següent tema, on explorarem l'ús de colors i estils per millorar els teus dissenys.
