Exercici 2: Implementació de Google Tag Manager en un Lloc Web

Següent ►
►

Objectius de l'Exercici

En aquest exercici, aprendràs a:

  1. Crear un compte i un contenidor a Google Tag Manager.
  2. Implementar el codi de Google Tag Manager en un lloc web.
  3. Configurar etiquetes bàsiques per al seguiment de Google Analytics.
  4. Provar i verificar que les etiquetes funcionen correctament.

Pas 1: Crear un Compte i un Contenidor a Google Tag Manager

Instruccions:

  1. Accedeix a Google Tag Manager.
  2. Inicia sessió amb el teu compte de Google.
  3. Fes clic a "Crea compte".
  4. Omple la informació del compte:
    • Nom del compte: Escriu el nom del teu compte (per exemple, el nom de la teva empresa).
    • País: Selecciona el teu país.
  5. Omple la informació del contenidor:
    • Nom del contenidor: Escriu el nom del teu lloc web.
    • Plataforma de destinació: Selecciona "Web".
  6. Fes clic a "Crea".
  7. Accepta els termes del servei.

Resultat Esperat:

Hauràs creat un compte i un contenidor a Google Tag Manager.

Pas 2: Implementar el Codi de Google Tag Manager en un Lloc Web

Instruccions:

  1. Després de crear el contenidor, Google Tag Manager et proporcionarà dos fragments de codi.
  2. Copia el primer fragment de codi i enganxa'l immediatament després de l'etiqueta <head> del teu lloc web.
  3. Copia el segon fragment de codi i enganxa'l immediatament després de l'etiqueta <body> del teu lloc web.

Exemple de Codi:

<!DOCTYPE html>
<html>
<head>
  <!-- Google Tag Manager -->
  <script>(function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':
  new Date().getTime(),event:'gtm.js'});var f=d.getElementsByTagName(s)[0],
  j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;j.src=
  'https://www.googletagmanager.com/gtm.js?id='+i+dl;f.parentNode.insertBefore(j,f);
  })(window,document,'script','dataLayer','GTM-XXXX');</script>
  <!-- End Google Tag Manager -->
</head>
<body>
  <!-- Google Tag Manager (noscript) -->
  <noscript><iframe src="https://www.googletagmanager.com/ns.html?id=GTM-XXXX"
  height="0" width="0" style="display:none;visibility:hidden"></iframe></noscript>
  <!-- End Google Tag Manager (noscript) -->
</body>
</html>

Nota: Substitueix GTM-XXXX pel teu ID de contenidor de Google Tag Manager.

Resultat Esperat:

El codi de Google Tag Manager estarà implementat correctament en el teu lloc web.

Pas 3: Configurar Etiquetes Bàsiques per al Seguiment de Google Analytics

Instruccions:

  1. Accedeix al teu compte de Google Tag Manager.
  2. Fes clic a "Etiquetes" al menú de l'esquerra.
  3. Fes clic a "Nou" per crear una nova etiqueta.
  4. Fes clic a "Configuració de l'etiqueta" i selecciona "Google Analytics: Universal Analytics".
  5. Configura l'etiqueta:
    • Tipus de seguiment: Selecciona "Pàgina vista".
    • ID de seguiment: Introdueix el teu ID de seguiment de Google Analytics (per exemple, UA-XXXXXX-Y).
  6. Fes clic a "Desencadenadors" i selecciona "All Pages".
  7. Dona un nom a l'etiqueta (per exemple, "GA - Pàgina vista") i fes clic a "Desa".

Resultat Esperat:

Hauràs creat una etiqueta per al seguiment de pàgines vistes amb Google Analytics.

Pas 4: Provar i Verificar les Etiquetes

Instruccions:

  1. Fes clic a "Vista prèvia" al menú superior dret de Google Tag Manager.
  2. Introdueix l'URL del teu lloc web i fes clic a "Connect".
  3. Navega pel teu lloc web per verificar que les etiquetes es disparen correctament.
  4. A la consola de vista prèvia de Google Tag Manager, verifica que l'etiqueta "GA - Pàgina vista" es dispara en cada pàgina.

Resultat Esperat:

Podràs veure que l'etiqueta "GA - Pàgina vista" es dispara correctament en cada pàgina del teu lloc web.

Conclusió

En aquest exercici, has après a:

  • Crear un compte i un contenidor a Google Tag Manager.
  • Implementar el codi de Google Tag Manager en un lloc web.
  • Configurar una etiqueta bàsica per al seguiment de Google Analytics.
  • Provar i verificar que les etiquetes funcionen correctament.

Aquestes habilitats són fonamentals per gestionar i optimitzar el seguiment de dades en el teu lloc web, permetent-te prendre decisions informades basades en dades reals.
