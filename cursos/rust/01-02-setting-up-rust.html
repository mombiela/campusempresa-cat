Configuració de l'Entorn de Rust

Següent ►
►

En aquesta secció, aprendrem com configurar l'entorn de desenvolupament per començar a programar en Rust. Rust és un llenguatge de programació modern que requereix una configuració específica per funcionar correctament. Seguint aquests passos, podreu tenir el vostre entorn de Rust llest en poc temps.

  1. Instal·lació de Rust

1.1. Utilitzant rustup

rustup és l'eina recomanada per instal·lar Rust, ja que facilita la gestió de versions i eines associades. Per instal·lar rustup, seguiu aquests passos:

Pas 1: Obriu una terminal

Pas 2: Executeu el següent comandament:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Pas 3: Seguiu les instruccions de la pantalla

Aquest comandament descarregarà i instal·larà rustup al vostre sistema. Durant la instal·lació, se us demanarà que confirmeu algunes opcions. Accepteu les opcions per defecte.

1.2. Verificació de la Instal·lació

Després d'instal·lar rustup, podeu verificar que Rust s'ha instal·lat correctament executant els següents comandaments a la terminal:

rustc --version
cargo --version

Aquests comandaments haurien de mostrar les versions de rustc (el compilador de Rust) i cargo (l'eina de gestió de projectes de Rust).

  1. Configuració de l'Editor de Codi

2.1. Visual Studio Code (VS Code)

VS Code és un editor de codi popular que es pot configurar fàcilment per treballar amb Rust. A continuació, es mostren els passos per configurar VS Code:

Pas 1: Instal·leu VS Code

Podeu descarregar VS Code des del lloc web oficial.

Pas 2: Instal·leu l'extensió de Rust

Obriu VS Code i aneu a la secció d'extensions (icona de quadrat a la barra lateral esquerra). Cerqueu "Rust" i instal·leu l'extensió anomenada "Rust (rls)".

Pas 3: Instal·leu l'extensió de Language Server Protocol (LSP)

L'extensió de Rust utilitza el Language Server Protocol per proporcionar funcionalitats com l'autocompletat i la navegació de codi. Assegureu-vos que l'extensió "Rust Analyzer" també estigui instal·lada.

2.2. Altres Editors

Rust és compatible amb molts altres editors de codi, com ara:

  • IntelliJ IDEA: Amb el plugin de Rust.
  • Sublime Text: Amb el paquet de Rust.
  • Atom: Amb el paquet de Rust.

  1. Creació del Primer Projecte

3.1. Utilitzant cargo

cargo és l'eina de gestió de projectes de Rust. Per crear un nou projecte, seguiu aquests passos:

Pas 1: Obriu una terminal

Pas 2: Executeu el següent comandament:

cargo new hola_mon

Aquest comandament crearà un nou directori anomenat hola_mon amb l'estructura bàsica d'un projecte de Rust.

Pas 3: Navegueu al directori del projecte:

cd hola_mon

Pas 4: Compileu i executeu el projecte:

cargo run

Aquest comandament compilarà el projecte i executarà el programa, mostrant "Hello, world!" a la terminal.

  1. Resum

En aquesta secció, hem après com instal·lar Rust utilitzant rustup, com configurar l'editor de codi per treballar amb Rust i com crear i executar el nostre primer projecte utilitzant cargo. Amb l'entorn de Rust configurat, estem llestos per començar a explorar les capacitats d'aquest potent llenguatge de programació.

En la següent secció, veurem com escriure el nostre primer programa en Rust: "Hola, Món!".
