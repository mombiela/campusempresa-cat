Configuració de TCP/IP

Següent ►
►

En aquest tema, aprendrem a configurar TCP/IP a OpenVMS. TCP/IP és un conjunt de protocols de comunicació utilitzats per interconnectar dispositius en una xarxa. La configuració correcta de TCP/IP és essencial per garantir la comunicació efectiva entre els sistemes OpenVMS i altres dispositius de la xarxa.

Objectius

  • Entendre els conceptes bàsics de TCP/IP.
  • Aprendre a configurar TCP/IP a OpenVMS.
  • Gestionar les adreces IP, les màscares de subxarxa i les passarel·les.
  • Configurar serveis de xarxa bàsics com DNS i DHCP.

Conceptes Bàsics de TCP/IP

Components Principals

  1. Adreça IP: Identificador únic per a cada dispositiu en una xarxa.
  2. Màscara de Subxarxa: Defineix la xarxa i la part de l'host d'una adreça IP.
  3. Passarel·la (Gateway): Dispositiu que connecta diferents xarxes i permet la comunicació entre elles.
  4. DNS (Domain Name System): Sistema que tradueix noms de domini en adreces IP.
  5. DHCP (Dynamic Host Configuration Protocol): Protocol que assigna automàticament adreces IP als dispositius de la xarxa.

Configuració de TCP/IP a OpenVMS

Pas 1: Verificar la Instal·lació de TCP/IP

Abans de començar la configuració, assegura't que el paquet de TCP/IP estigui instal·lat al sistema OpenVMS. Pots verificar-ho amb la següent comanda:

$ @SYS$MANAGER:TCPIP$CONFIG

Pas 2: Configurar la Interfície de Xarxa

  1. Inicia la configuració de TCP/IP: 
    $ @SYS$MANAGER:TCPIP$CONFIG
  2. Selecciona l'opció "CORE CONFIGURATION".
  3. Selecciona "Interfaces".
  4. Selecciona la interfície de xarxa que vols configurar (per exemple, WE0).
  5. Assigna una adreça IP i una màscara de subxarxa: 
    IP address: 192.168.1.10
Subnet mask: 255.255.255.0

Pas 3: Configurar la Passarel·la (Gateway)

  1. Selecciona "CORE CONFIGURATION".
  2. Selecciona "Routing".
  3. Afegeix la passarel·la per defecte: 
    Default gateway: 192.168.1.1

Pas 4: Configurar el DNS

  1. Selecciona "CORE CONFIGURATION".
  2. Selecciona "BIND Resolver".
  3. Afegeix els servidors DNS: 
    Primary DNS: 8.8.8.8
Secondary DNS: 8.8.4.4

Pas 5: Configurar DHCP (Opcional)

Si vols que el sistema obtingui automàticament una adreça IP mitjançant DHCP:

  1. Selecciona "CORE CONFIGURATION".
  2. Selecciona "Interfaces".
  3. Selecciona la interfície de xarxa.
  4. Habilita DHCP.

Pas 6: Guardar i Aplicar la Configuració

Després de completar la configuració, guarda i aplica els canvis. Això es pot fer seleccionant l'opció corresponent en el menú de configuració.

Exercicis Pràctics

Exercici 1: Configurar una Adreça IP Estàtica

Configura una adreça IP estàtica per a la interfície de xarxa WE0 amb els següents paràmetres:

  • Adreça IP: 192.168.2.20
  • Màscara de Subxarxa: 255.255.255.0
  • Passarel·la: 192.168.2.1
  • DNS Primari: 8.8.8.8
  • DNS Secundari: 8.8.4.4

Solució:

$ @SYS$MANAGER:TCPIP$CONFIG
Enter configuration option: 1
Enter core environment configuration option: 1
Enter network interface option: 1
Enter IP address: 192.168.2.20
Enter subnet mask: 255.255.255.0
Enter core environment configuration option: 2
Enter default gateway: 192.168.2.1
Enter core environment configuration option: 3
Enter primary DNS: 8.8.8.8
Enter secondary DNS: 8.8.4.4
Enter core environment configuration option: 7

Exercici 2: Configurar DHCP

Configura la interfície de xarxa WE1 per obtenir una adreça IP automàticament mitjançant DHCP.

Solució:

$ @SYS$MANAGER:TCPIP$CONFIG
Enter configuration option: 1
Enter core environment configuration option: 1
Enter network interface option: 2
Enable DHCP: YES
Enter core environment configuration option: 7

Conclusió

En aquesta secció, hem après a configurar TCP/IP a OpenVMS, incloent-hi la configuració d'adreces IP, màscares de subxarxa, passarel·les, DNS i DHCP. La configuració correcta de TCP/IP és fonamental per garantir la comunicació efectiva en una xarxa. En el proper tema, explorarem la configuració de DECnet a OpenVMS.
