Introducció a DECnet

DECnet és una suite de protocols de xarxa desenvolupada per Digital Equipment Corporation (DEC) per a la comunicació entre ordinadors. És àmpliament utilitzada en entorns OpenVMS per la seva robustesa i capacitat de gestió de xarxes complexes.

Objectius d'aquest tema:

  1. Entendre els conceptes bàsics de DECnet.
  2. Aprendre a configurar DECnet en un sistema OpenVMS.
  3. Conèixer les comandes bàsiques per gestionar DECnet.
  4. Realitzar exercicis pràctics per consolidar els coneixements.

Conceptes Bàsics de DECnet

Components Clau de DECnet:

  • Nod: Un ordinador o dispositiu en la xarxa DECnet.
  • Adreça de Node: Identificador únic per a cada node en la xarxa.
  • Circuit: Connexió física o lògica entre nodes.
  • Objecte: Un servei o aplicació que pot ser accedit a través de DECnet.

Versions de DECnet:

  • DECnet Phase IV: Versió més antiga, utilitzada en molts sistemes legacy.
  • DECnet-Plus (Phase V): Versió més recent, amb suport per a protocols moderns com TCP/IP.

Configuració de DECnet en OpenVMS

Passos per Configurar DECnet:

  1. Verificar la Instal·lació de DECnet:

    $ @SYS$MANAGER:NET$CONFIGURE

    Aquesta comanda inicia l'assistent de configuració de DECnet.

  2. Configurar l'Adreça de Node: Durant la configuració, se us demanarà que introduïu l'adreça de node. Aquesta adreça ha de ser única dins de la xarxa.

  3. Definir el Nom del Node: Assignar un nom descriptiu al node per facilitar la seva identificació.

  4. Configurar Circuits: Seleccionar i configurar els circuits de xarxa que utilitzarà DECnet. Això pot incloure interfícies Ethernet, FDDI, etc.

  5. Configurar Objectes: Definir els objectes que estaran disponibles a través de DECnet, com ara serveis de fitxers, impressió, etc.

Exemple de Configuració:

$ @SYS$MANAGER:NET$CONFIGURE

DECnet Phase IV Configuration Procedure

Enter the node name: NODE1
Enter the node address: 1.1
Enter the network circuit: EWA-0

Comandes Bàsiques de DECnet:

  • Mostrar Informació del Node:

    $ SHOW NETWORK

  • Enviar un Missatge a un Alt Node:

    $ MCR NCP SEND NODE <node_name> MESSAGE "Hola, DECnet!"

  • Mostrar Estat dels Circuits:

    $ MCR NCP SHOW KNOWN CIRCUITS

  • Mostrar Objectes Actius:

    $ MCR NCP SHOW ACTIVE OBJECTS

Exercicis Pràctics

Exercici 1: Configurar un Node DECnet

  1. Executeu l'assistent de configuració de DECnet.
  2. Assignar el nom de node NODE2 i l'adreça 2.2.
  3. Configurar el circuit EWA-1.

Solució:

$ @SYS$MANAGER:NET$CONFIGURE

DECnet Phase IV Configuration Procedure

Enter the node name: NODE2
Enter the node address: 2.2
Enter the network circuit: EWA-1

Exercici 2: Enviar un Missatge a un Alt Node

  1. Utilitzeu la comanda SEND per enviar un missatge al node NODE1.

Solució:

$ MCR NCP SEND NODE NODE1 MESSAGE "Hola, NODE1!"

Errors Comuns i Consells

  • Error: Adreça de Node Duplicada:

    • Assegureu-vos que cada node té una adreça única dins de la xarxa.

  • Error: Circuit No Configurat Correctament:

    • Verifiqueu que el circuit seleccionat està correctament configurat i actiu.

  • Consell:

    • Utilitzeu noms de nodes descriptius per facilitar la gestió i identificació dels nodes en la xarxa.

Conclusió

En aquesta secció, hem après els conceptes bàsics de DECnet i com configurar-lo en un sistema OpenVMS. Hem vist com definir l'adreça de node, configurar circuits i objectes, i utilitzar comandes bàsiques per gestionar DECnet. Els exercicis pràctics proporcionats us ajudaran a consolidar aquests coneixements. En la següent secció, explorarem els serveis i protocols de xarxa disponibles en OpenVMS.
