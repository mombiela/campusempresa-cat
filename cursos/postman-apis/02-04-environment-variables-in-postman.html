Variables d'Entorn a Postman

Següent ►
►

Les variables d'entorn a Postman són una eina poderosa que permet als usuaris gestionar i reutilitzar valors dinàmics en les seves sol·licituds d'API. Aquestes variables poden ajudar a simplificar les proves, especialment quan es treballa amb diferents entorns (com desenvolupament, proves i producció).

Què són les Variables d'Entorn?

  • Definició: Les variables d'entorn són valors que es poden definir i utilitzar dins de les sol·licituds de Postman. Aquests valors poden canviar segons l'entorn en què s'estigui treballant.
  • Ús Comú: S'utilitzen per emmagatzemar informació com URL de base, claus d'API, tokens d'autenticació, etc.

Creació de Variables d'Entorn

  1. Accedir a la Configuració d'Entorn:

    • A la interfície de Postman, fes clic a la icona de l'engranatge a la part superior dreta i selecciona "Manage Environments".

  2. Crear un Nou Entorn:

    • Fes clic a "Add" per crear un nou entorn.
    • Assigna un nom al teu entorn (per exemple, "Desenvolupament").

  3. Definir Variables:

    • Afegeix noves variables amb els seus noms i valors corresponents.
    • Exemple:
      • baseUrl: https://api.dev.example.com
      • apiKey: 12345-abcde

  4. Guardar l'Entorn:

    • Un cop definides les variables, fes clic a "Save" per guardar l'entorn.

Ús de Variables d'Entorn en Sol·licituds

  • Sintaxi: Per utilitzar una variable d'entorn en una sol·licitud, utilitza la sintaxi {{variableName}}.
  • Exemple de Sol·licitud: 
    GET {{baseUrl}}/users
Headers:
  Authorization: Bearer {{apiKey}}

Exercici Pràctic

Objectiu: Crear i utilitzar variables d'entorn per a una sol·licitud d'API.

Passos:

  1. Crea un nou entorn anomenat "Proves".
  2. Defineix les següents variables:
    • baseUrl: https://api.test.example.com
    • token: test-token-123
  3. Crea una nova sol·licitud a Postman:
    • Mètode: GET
    • URL: {{baseUrl}}/products
    • Afegeix un header: Authorization: Bearer {{token}}
  4. Executa la sol·licitud i verifica que la resposta sigui correcta.

Solució:

  • Assegura't que l'entorn "Proves" estigui seleccionat abans d'executar la sol·licitud.
  • La sol·licitud hauria de substituir les variables {{baseUrl}} i {{token}} pels seus valors definits.

Errors Comuns i Consells

  • Error: Les variables no es substitueixen correctament.

    • Solució: Comprova que l'entorn correcte estigui seleccionat i que els noms de les variables coincideixin exactament amb els definits.

  • Consell: Utilitza variables globals per a valors que es mantenen constants a través de tots els entorns.

Conclusió

Les variables d'entorn a Postman són essencials per gestionar configuracions dinàmiques i simplificar les proves d'API. Amb la pràctica, es poden utilitzar per automatitzar i escalar les proves de manera eficient. En el següent tema, explorarem com enllaçar sol·licituds per crear fluxos de treball més complexos.
