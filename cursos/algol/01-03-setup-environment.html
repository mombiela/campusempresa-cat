Configuració de l'Entorn ALGOL

Següent ►
►

En aquesta secció, aprendrem com configurar l'entorn necessari per començar a programar en ALGOL. Aquest procés inclou la instal·lació de les eines necessàries i la configuració del vostre sistema per executar programes ALGOL.

  1. Requisits Previs

Abans de començar, assegureu-vos de tenir els següents requisits:

  • Un ordinador amb un sistema operatiu compatible (Windows, macOS, Linux).
  • Connexió a Internet per descarregar les eines necessàries.

  1. Instal·lació d'un Compilador ALGOL

Per executar programes ALGOL, necessitem un compilador. Un dels compiladors més utilitzats per ALGOL és el ALGOL 68 Genie. A continuació, es mostren els passos per instal·lar-lo en diferents sistemes operatius.

2.1. Instal·lació en Windows

  1. Descarregar el Compilador:

    • Visiteu el lloc web oficial de ALGOL 68 Genie.
    • Descarregueu l'instal·lador per a Windows.

  2. Instal·lar el Compilador:

    • Executeu l'instal·lador descarregat.
    • Seguiu les instruccions de l'assistent d'instal·lació.
    • Un cop finalitzada la instal·lació, assegureu-vos que el compilador s'ha instal·lat correctament obrint una finestra de terminal i executant algol68g.

2.2. Instal·lació en macOS

  1. Descarregar el Compilador:

    • Visiteu el lloc web oficial de ALGOL 68 Genie.
    • Descarregueu l'instal·lador per a macOS.

  2. Instal·lar el Compilador:

    • Obriu el fitxer descarregat i seguiu les instruccions per instal·lar el compilador.
    • Un cop finalitzada la instal·lació, obriu una terminal i executeu algol68g per verificar que el compilador s'ha instal·lat correctament.

2.3. Instal·lació en Linux

  1. Descarregar el Compilador:

    • Obriu una terminal.
    • Executeu la següent comanda per descarregar i instal·lar ALGOL 68 Genie: 
      sudo apt-get install algol68g

  2. Verificar la Instal·lació:

    • Un cop finalitzada la instal·lació, executeu algol68g en la terminal per assegurar-vos que el compilador s'ha instal·lat correctament.

  1. Configuració de l'Entorn de Desenvolupament

3.1. Editor de Text

Per escriure codi ALGOL, necessitem un editor de text. Podeu utilitzar qualsevol editor de text que preferiu, però alguns dels més populars són:

  • Visual Studio Code: Un editor de codi font gratuït i de codi obert amb suport per a molts llenguatges de programació.
  • Sublime Text: Un editor de text sofisticat per a codi, marcatge i prosa.
  • Atom: Un editor de text hackeable per al segle XXI.

3.2. Configuració de l'Editor

Per facilitar la programació en ALGOL, podeu configurar el vostre editor de text per reconèixer la sintaxi d'ALGOL. A continuació, es mostren els passos per configurar Visual Studio Code:

  1. Instal·lar Visual Studio Code:

    • Visiteu el lloc web oficial de Visual Studio Code.
    • Descarregueu i instal·leu l'editor.

  2. Instal·lar Extensions:

    • Obriu Visual Studio Code.
    • Aneu a la secció d'extensions (icona de quadrat a la barra lateral esquerra).
    • Cerqueu i instal·leu extensions per a ALGOL, com ara "ALGOL 68 Syntax".

3.3. Configuració de la Terminal

Per compilar i executar programes ALGOL, necessitem utilitzar la terminal. A continuació, es mostren els passos per configurar la terminal en Visual Studio Code:

  1. Obrir la Terminal:

    • Aneu a Visual Studio Code i obriu la terminal integrada (View > Terminal).

  2. Configurar la Terminal:

    • Assegureu-vos que la terminal està configurada per utilitzar el compilador ALGOL. Podeu fer-ho executant algol68g a la terminal per verificar que el compilador està disponible.

  1. Primer Programa en ALGOL

Ara que tenim l'entorn configurat, escrivim i executem el nostre primer programa en ALGOL.

4.1. Escriure el Programa

  1. Crear un Fitxer Nou:

    • Obriu el vostre editor de text (per exemple, Visual Studio Code).
    • Creeu un fitxer nou i anomeneu-lo primer_programa.algol.

  2. Escriure el Codi:

    • Introduïu el següent codi en el fitxer: 
      BEGIN
  print("Hola, món!");
END.

4.2. Compilar i Executar el Programa

  1. Obrir la Terminal:

    • Assegureu-vos que la terminal està oberta en el directori on heu desat el fitxer primer_programa.algol.

  2. Compilar el Programa:

    • Executeu la següent comanda per compilar el programa: 
      algol68g primer_programa.algol

  3. Executar el Programa:

    • Si la compilació és correcta, el programa s'executarà i veureu la sortida Hola, món! a la terminal.

Conclusió

En aquesta secció, hem après com configurar l'entorn necessari per programar en ALGOL, incloent la instal·lació del compilador, la configuració de l'editor de text i la terminal, i hem escrit i executat el nostre primer programa en ALGOL. Amb l'entorn configurat, estem preparats per continuar amb els següents mòduls del curs.
