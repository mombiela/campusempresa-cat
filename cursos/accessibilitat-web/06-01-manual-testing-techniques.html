Tècniques de Prova Manual

Següent ►
►

En aquest tema, explorarem les tècniques de prova manual per avaluar l'accessibilitat d'un lloc web. Les proves manuals són essencials per identificar problemes d'accessibilitat que les eines automatitzades poden passar per alt. Aquestes tècniques impliquen l'ús de la web tal com ho faria un usuari amb discapacitats, utilitzant tecnologies assistencials i altres mètodes per assegurar que el contingut sigui accessible per a tothom.

Objectius d'Aprenentatge

  • Comprendre la importància de les proves manuals en l'accessibilitat web.
  • Aprendre a realitzar proves manuals utilitzant diferents tècniques i eines.
  • Identificar problemes comuns d'accessibilitat que es poden detectar mitjançant proves manuals.

Conceptes Clau

  1. Importància de les Proves Manuals

  • Detecció de Problemes Subtils: Les proves manuals poden identificar problemes que les eines automatitzades no detecten, com ara la comprensibilitat del contingut o la facilitat d'ús de la navegació.
  • Experiència Real de l'Usuari: Proporcionen una visió de com els usuaris amb discapacitats experimenten el lloc web, permetent una avaluació més precisa de l'accessibilitat.

  1. Tècniques de Prova Manual

Navegació amb el Teclat

  • Objectiu: Assegurar que tots els elements interactius del lloc web siguin accessibles mitjançant el teclat.
  • Com Provar:
    • Utilitza la tecla Tab per navegar a través dels elements interactius.
    • Assegura't que l'ordre de navegació sigui lògic i seqüencial.
    • Verifica que els elements es puguin activar amb la tecla Enter o Espai.

Ús de Lectors de Pantalla

  • Objectiu: Comprovar que el contingut sigui comprensible i navegable per als usuaris de lectors de pantalla.
  • Com Provar:
    • Utilitza un lector de pantalla com NVDA o JAWS per navegar pel lloc web.
    • Assegura't que tots els elements importants tinguin etiquetes descriptives.
    • Verifica que les imatges tinguin textos alternatius adequats.

Revisió Visual

  • Objectiu: Identificar problemes visuals que puguin afectar l'accessibilitat.
  • Com Provar:
    • Comprova el contrast de color entre el text i el fons.
    • Assegura't que el text sigui redimensionable sense pèrdua de funcionalitat.
    • Verifica que els elements visuals no depenguin únicament del color per transmetre informació.

  1. Errors Comuns Detectats en Proves Manuals

  • Falta de Focus Visible: Els elements interactius no mostren un indicador de focus quan es navega amb el teclat.
  • Text Alternatiu Inadequat: Les imatges no tenen descripcions adequades o aquestes són massa vagues.
  • Ordre de Navegació Incorrecte: L'ordre de navegació no segueix una seqüència lògica, confonent els usuaris.

Exercici Pràctic

Objectiu: Realitzar una prova manual d'accessibilitat en una pàgina web de mostra.

Instruccions

  1. Tria una pàgina web de mostra.
  2. Realitza una navegació completa utilitzant només el teclat.
  3. Utilitza un lector de pantalla per navegar i escoltar el contingut de la pàgina.
  4. Revisa visualment la pàgina per a problemes de contrast i redimensionament de text.

Solució

  • Navegació amb el Teclat: Assegura't que tots els elements interactius es poden accedir i activar amb el teclat.
  • Ús de Lectors de Pantalla: Verifica que el lector de pantalla llegeixi correctament tots els elements i que les descripcions siguin clares.
  • Revisió Visual: Comprova que el contrast de color sigui adequat i que el text es pugui redimensionar sense problemes.

Conclusió

Les tècniques de prova manual són una part essencial de l'avaluació de l'accessibilitat web. Aquestes proves ajuden a garantir que el lloc web sigui accessible per a tots els usuaris, incloent aquells amb discapacitats. En el proper tema, explorarem les eines de prova automatitzades que complementen aquestes tècniques manuals.
