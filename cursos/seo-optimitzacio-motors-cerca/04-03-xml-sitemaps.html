Sitemaps XML

Següent ►
►

Introducció

Un Sitemap XML és un fitxer que llista totes les pàgines d'un lloc web que vols que els motors de cerca indexin. Aquest fitxer ajuda els motors de cerca a comprendre l'estructura del teu lloc web i a trobar totes les pàgines importants, especialment aquelles que poden no estar ben enllaçades internament.

Per què són importants els Sitemaps XML?

Els Sitemaps XML són crucials per a la SEO tècnica per diverses raons:

  • Millora de la indexació: Ajuda els motors de cerca a trobar i indexar totes les pàgines del teu lloc web.
  • Priorització de pàgines: Permet indicar la importància relativa de les pàgines del teu lloc web.
  • Actualitzacions: Informa els motors de cerca sobre les actualitzacions recents del teu lloc web.
  • Gestió de contingut gran: És especialment útil per a llocs web grans amb moltes pàgines o per a llocs web nous amb pocs enllaços externs.

Com crear un Sitemap XML

Eines per crear Sitemaps XML

Hi ha diverses eines que poden ajudar-te a crear un Sitemap XML:

  • Generadors en línia: Com XML-Sitemaps.com.
  • Plugins de CMS: Si utilitzes WordPress, plugins com Yoast SEO o All in One SEO Pack poden generar automàticament un Sitemap XML.
  • Eines de desenvolupadors: Eines com Screaming Frog SEO Spider poden generar Sitemaps XML per a llocs web més grans.

Estructura d'un Sitemap XML

Un Sitemap XML típic té l'estructura següent:

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>http://www.example.com/</loc>
    <lastmod>2023-10-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>http://www.example.com/about</loc>
    <lastmod>2023-09-15</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <!-- Més URLs aquí -->
</urlset>

Explicació dels elements

  • : Conté tots els elements del Sitemap.
  • : Representa una URL individual del lloc web.
  • : La URL de la pàgina.
  • : La data de l'última modificació de la pàgina.
  • : La freqüència amb què s'espera que la pàgina canviï (per exemple, "daily", "weekly", "monthly").
  • : La prioritat de la pàgina en relació amb altres pàgines del lloc web (valor entre 0.0 i 1.0).

Com enviar el teu Sitemap XML als motors de cerca

Google Search Console

  1. Inicia sessió a Google Search Console.
  2. Selecciona la propietat del lloc web.
  3. A la barra lateral esquerra, fes clic a "Sitemaps".
  4. Introdueix la URL del teu Sitemap XML i fes clic a "Enviar".

Bing Webmaster Tools

  1. Inicia sessió a Bing Webmaster Tools.
  2. Selecciona la propietat del lloc web.
  3. A la barra lateral esquerra, fes clic a "Sitemaps".
  4. Introdueix la URL del teu Sitemap XML i fes clic a "Enviar".

Exercici Pràctic

Objectiu

Crear i enviar un Sitemap XML per a un lloc web fictici.

Passos

  1. Crear el Sitemap XML:

    • Utilitza un generador en línia com XML-Sitemaps.com per crear un Sitemap XML per al lloc web fictici http://www.exemple.com.

  2. Revisar el Sitemap XML:

    • Assegura't que el fitxer conté totes les pàgines importants del lloc web.

  3. Enviar el Sitemap XML a Google Search Console:

    • Inicia sessió a Google Search Console.
    • Selecciona la propietat del lloc web.
    • Afegeix la URL del Sitemap XML i envia'l.

Solució

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>http://www.exemple.com/</loc>
    <lastmod>2023-10-01</lastmod>
    <changefreq>monthly</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>http://www.exemple.com/about</loc>
    <lastmod>2023-09-15</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>http://www.exemple.com/contact</loc>
    <lastmod>2023-09-20</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.7</priority>
  </url>
</urlset>

Errors Comuns i Consells

  • Error: No incloure totes les pàgines importants.

    • Solució: Revisa el teu lloc web i assegura't que totes les pàgines importants estan incloses al Sitemap XML.

  • Error: No actualitzar el Sitemap XML després de fer canvis al lloc web.

    • Solució: Actualitza el Sitemap XML cada vegada que afegeixis o eliminis pàgines del teu lloc web.

  • Consell: Utilitza eines automàtiques per mantenir el teu Sitemap XML actualitzat, especialment si tens un lloc web gran o amb contingut dinàmic.

Conclusió

Els Sitemaps XML són una eina essencial per a la SEO tècnica, ja que ajuden els motors de cerca a indexar el teu lloc web de manera més eficient. Crear i mantenir un Sitemap XML és una tasca senzilla però crucial per assegurar-te que totes les pàgines importants del teu lloc web són descobertes i indexades pels motors de cerca.
