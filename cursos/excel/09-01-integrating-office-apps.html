Integrar Excel amb Altres Aplicacions d'Office

Següent ►
►

Introducció

Integrar Excel amb altres aplicacions d'Office pot millorar significativament la teva productivitat i eficiència. Aquesta secció t'ensenyarà com connectar i utilitzar Excel amb altres eines d'Office com Word, PowerPoint, Outlook i Access.

Objectius d'Aprenentatge

  • Entendre com enllaçar dades d'Excel amb Word i PowerPoint.
  • Aprendre a importar i exportar dades entre Excel i Access.
  • Utilitzar Excel per gestionar correus electrònics i contactes a Outlook.

  1. Enllaçar Dades d'Excel amb Word

1.1. Copiar i Enganxar Dades

La manera més senzilla d'integrar Excel amb Word és copiar i enganxar dades.

  1. Selecciona les dades a Excel: Arrossega el cursor per seleccionar les cel·les que vols copiar.
  2. Copia les dades: Prem Ctrl + C o fes clic dret i selecciona "Copiar".
  3. Enganxa les dades a Word: Obre un document de Word, prem Ctrl + V o fes clic dret i selecciona "Enganxar".

1.2. Enllaçar Dades

Per mantenir les dades actualitzades automàticament entre Excel i Word, pots enllaçar-les.

  1. Copia les dades a Excel: Selecciona les cel·les i prem Ctrl + C.
  2. Enganxa com a enllaç a Word:
    • A Word, fes clic a la pestanya "Inici".
    • Fes clic a la fletxa sota "Enganxar" i selecciona "Enganxar especial".
    • Tria "Enllaç de full de càlcul de Microsoft Excel Objecte" i fes clic a "D'acord".

Exemple Pràctic

// Excel
A1: Producte
A2: Vendes
B1: A
B2: 100
C1: B
C2: 150
// Word
Producte | Vendes
A        | 100
B        | 150

  1. Enllaçar Dades d'Excel amb PowerPoint

2.1. Copiar i Enganxar Gràfics

  1. Crea un gràfic a Excel: Selecciona les dades i insereix un gràfic.
  2. Copia el gràfic: Selecciona el gràfic i prem Ctrl + C.
  3. Enganxa el gràfic a PowerPoint: Obre una presentació de PowerPoint, prem Ctrl + V.

2.2. Enllaçar Gràfics

  1. Copia el gràfic a Excel: Selecciona el gràfic i prem Ctrl + C.
  2. Enganxa com a enllaç a PowerPoint:
    • A PowerPoint, fes clic a la pestanya "Inici".
    • Fes clic a la fletxa sota "Enganxar" i selecciona "Enganxar especial".
    • Tria "Enllaç de full de càlcul de Microsoft Excel Objecte" i fes clic a "D'acord".

Exemple Pràctic

// Excel
A1: Mes
A2: Gener
A3: Febrer
A4: Març
B1: Vendes
B2: 200
B3: 250
B4: 300
// PowerPoint
[Gràfic enllaçat amb les dades d'Excel]

  1. Importar i Exportar Dades entre Excel i Access

3.1. Importar Dades d'Excel a Access

  1. Obre Access: Crea una nova base de dades o obre una existent.
  2. Importa dades:
    • Fes clic a "Dades externes" > "Excel".
    • Selecciona el fitxer d'Excel i segueix l'assistent per importar les dades.

3.2. Exportar Dades d'Access a Excel

  1. Obre Access: Selecciona la taula o consulta que vols exportar.
  2. Exporta dades:
    • Fes clic a "Dades externes" > "Excel".
    • Selecciona la ubicació i el nom del fitxer d'Excel i segueix l'assistent per exportar les dades.

Exemple Pràctic

// Access
Taula: Clients
ID | Nom    | Correu Electrònic
1  | Joan   | joan@example.com
2  | Maria  | maria@example.com
// Excel
A1: ID
A2: 1
A3: 2
B1: Nom
B2: Joan
B3: Maria
C1: Correu Electrònic
C2: joan@example.com
C3: maria@example.com

  1. Utilitzar Excel amb Outlook

4.1. Exportar Contactes d'Outlook a Excel

  1. Obre Outlook: Ves a "Fitxer" > "Obre i exporta" > "Importa/Exporta".
  2. Exporta contactes:
    • Selecciona "Exporta a un fitxer" > "Valors separats per comes".
    • Tria la carpeta de contactes i segueix l'assistent per exportar.

4.2. Enviar Correus des d'Excel

  1. Crea una macro a Excel:
    • Obre l'Editor de VBA (Alt + F11).
    • Insereix un nou mòdul i afegeix el codi següent:
Sub EnviarCorreu()
    Dim OutlookApp As Object
    Dim Correu As Object
    Set OutlookApp = CreateObject("Outlook.Application")
    Set Correu = OutlookApp.CreateItem(0)
    
    With Correu
        .To = "destinatari@example.com"
        .Subject = "Assumpte del correu"
        .Body = "Cos del correu"
        .Send
    End With
End Sub

Exercicis Pràctics

Exercici 1: Enllaçar Dades d'Excel amb Word

  1. Crea una taula a Excel amb les dades següents:
    • Producte: A, B, C
    • Vendes: 100, 150, 200
  2. Enllaça aquesta taula a un document de Word.

Exercici 2: Enllaçar un Gràfic d'Excel amb PowerPoint

  1. Crea un gràfic a Excel amb les dades següents:
    • Mes: Gener, Febrer, Març
    • Vendes: 200, 250, 300
  2. Enllaça aquest gràfic a una presentació de PowerPoint.

Exercici 3: Importar i Exportar Dades entre Excel i Access

  1. Crea una taula a Excel amb les dades següents:
    • ID: 1, 2
    • Nom: Joan, Maria
    • Correu Electrònic: joan@example.com, maria@example.com
  2. Importa aquesta taula a Access.
  3. Exporta una taula d'Access a Excel.

Exercici 4: Enviar un Correu des d'Excel

  1. Crea una macro a Excel que enviï un correu electrònic a destinatari@example.com amb l'assumpte "Hola" i el cos "Aquest és un correu de prova".

Conclusió

Integrar Excel amb altres aplicacions d'Office pot simplificar moltes tasques i millorar la teva eficiència. Ara que has après a enllaçar dades amb Word i PowerPoint, importar i exportar dades amb Access, i gestionar correus amb Outlook, pots començar a aplicar aquestes habilitats en els teus projectes diaris.
