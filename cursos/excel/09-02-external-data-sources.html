Utilitzar Excel amb Fonts de Dades Externes

Següent ►
►

En aquest tema, aprendrem com connectar i utilitzar dades externes a Excel. Aquesta habilitat és essencial per a l'anàlisi de dades avançada, ja que permet integrar informació de diverses fonts en un sol full de càlcul. Explorarem diferents tipus de fonts de dades externes i com importar-les a Excel.

  1. Tipus de Fonts de Dades Externes

Excel permet connectar-se a diverses fonts de dades externes, incloent:

  • Bases de dades SQL: Com Microsoft SQL Server, MySQL, Oracle, etc.
  • Arxius de text: Com CSV, TXT.
  • Arxius XML.
  • Arxius JSON.
  • Serveis web: APIs RESTful.
  • Altres aplicacions d'Office: Com Access.
  • Fonts de dades en línia: Com SharePoint, Azure, etc.

  1. Importar Dades des d'un Arxiu de Text (CSV)

Pas a Pas:

  1. Obrir Excel.
  2. Anar a la pestanya "Dades".
  3. Seleccionar "Des d'un Text/CSV".
  4. Seleccionar l'arxiu CSV que voleu importar.
  5. Configurar les opcions d'importació:
    • Delimitador: Seleccioneu el caràcter que separa els camps (normalment una coma).
    • Codificació: Seleccioneu la codificació correcta (normalment UTF-8).
  6. Fer clic a "Carregar".

Exemple:

Nom, Edat, Ciutat
Joan, 28, Barcelona
Maria, 34, Madrid
Pere, 45, València

Resultat a Excel:

Nom Edat Ciutat
Joan 28 Barcelona
Maria 34 Madrid
Pere 45 València

  1. Connectar-se a una Base de Dades SQL

Pas a Pas:

  1. Anar a la pestanya "Dades".
  2. Seleccionar "Des d'una Base de Dades".
  3. Seleccionar el tipus de base de dades (p. ex., "Des de SQL Server").
  4. Introduir les credencials de connexió:
    • Servidor: Nom del servidor SQL.
    • Base de dades: Nom de la base de dades.
    • Autenticació: Tipus d'autenticació (Windows o SQL Server).
  5. Seleccionar les taules o vistes que voleu importar.
  6. Fer clic a "Carregar".

Exemple de Connexió a SQL Server:

SELECT Nom, Edat, Ciutat
FROM Usuaris
WHERE Edat > 30;

Resultat a Excel:

Nom Edat Ciutat
Maria 34 Madrid
Pere 45 València

  1. Importar Dades des d'una API RESTful

Pas a Pas:

  1. Anar a la pestanya "Dades".
  2. Seleccionar "Des d'altres fonts".
  3. Seleccionar "Des d'un servei web (OData)".
  4. Introduir l'URL de l'API.
  5. Configurar les opcions d'autenticació si és necessari.
  6. Seleccionar les dades que voleu importar.
  7. Fer clic a "Carregar".

Exemple d'URL d'API:

https://api.exemple.com/usuaris

Resultat a Excel:

Nom Edat Ciutat
Joan 28 Barcelona
Maria 34 Madrid
Pere 45 València

Objectiu:

Importar dades d'un arxiu CSV i d'una base de dades SQL a Excel i combinar-les en un sol full de càlcul.

  1. Importar l'arxiu CSV "usuaris.csv".
  2. Connectar-se a la base de dades SQL "empresa" i importar la taula "empleats".
  3. Combinar les dades en un sol full de càlcul utilitzant la funció VLOOKUP.

Solució:

Importar l'arxiu CSV:

  1. Seguir els passos descrits a la secció 2.

Connectar-se a la base de dades SQL:

  1. Seguir els passos descrits a la secció 3.

Combinar les dades:

  1. Resum

En aquest tema, hem après com importar dades externes a Excel des de diverses fonts, incloent arxius de text, bases de dades SQL i APIs RESTful. També hem vist com combinar aquestes dades utilitzant funcions d'Excel. Aquestes habilitats són essencials per a l'anàlisi de dades avançada i la integració de múltiples fonts d'informació en un sol full de càlcul.
