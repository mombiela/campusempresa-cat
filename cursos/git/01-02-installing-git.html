Instal·lant Git

Següent ►
►

En aquesta secció, aprendrem com instal·lar Git en diferents sistemes operatius. Git és una eina de control de versions distribuïda que és essencial per a qualsevol desenvolupador. La instal·lació de Git és el primer pas per començar a utilitzar aquesta poderosa eina.

Contingut

  1. Instal·lant Git en Windows
  2. Instal·lant Git en macOS
  3. Instal·lant Git en Linux
  4. Verificant la Instal·lació

Instal·lant Git en Windows

Pas 1: Descarregar l'Instal·lador

  1. Visita la pàgina oficial de descàrrega de Git: https://git-scm.com/download/win.
  2. Descarrega l'instal·lador per a Windows.

Pas 2: Executar l'Instal·lador

  1. Obre el fitxer descarregat (.exe).
  2. Segueix les instruccions de l'assistent d'instal·lació. Les opcions per defecte són adequades per a la majoria dels usuaris.

Pas 3: Configuració de l'Entorn

  1. Durant la instal·lació, se't demanarà que triïs l'editor de text per defecte que Git utilitzarà. Pots seleccionar l'editor que prefereixis.
  2. També se't demanarà que configuris altres opcions com la línia d'ordres i la configuració de la terminal. Les opcions per defecte són recomanades.

Pas 4: Finalitzar la Instal·lació

  1. Completa l'assistent d'instal·lació i fes clic a "Finish".

Instal·lant Git en macOS

Pas 1: Utilitzar Homebrew

Homebrew és un gestor de paquets per a macOS que facilita la instal·lació de programari.

  1. Si no tens Homebrew instal·lat, pots instal·lar-lo executant la següent comanda a la terminal: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Pas 2: Instal·lar Git

  1. Un cop tinguis Homebrew instal·lat, pots instal·lar Git executant la següent comanda a la terminal: 
    brew install git

Instal·lant Git en Linux

Pas 1: Utilitzar el Gestor de Paquets

La majoria de distribucions de Linux tenen Git disponible als seus repositoris de programari. A continuació es mostren les comandes per a les distribucions més comunes:

Debian/Ubuntu

  1. Actualitza l'índex de paquets: 
    sudo apt update
  2. Instal·la Git: 
    sudo apt install git

Fedora

  1. Instal·la Git: 
    sudo dnf install git

Arch Linux

  1. Instal·la Git: 
    sudo pacman -S git

Verificant la Instal·lació

Després d'instal·lar Git, és important verificar que la instal·lació s'ha completat correctament.

  1. Obre una terminal o línia d'ordres.
  2. Executa la següent comanda: 
    git --version

Si Git s'ha instal·lat correctament, hauràs de veure una sortida similar a aquesta:

git version 2.30.1

Conclusió

Ara que has instal·lat Git al teu sistema, estàs preparat per començar a utilitzar-lo. En la següent secció, aprendrem sobre la terminologia bàsica de Git, que és essencial per comprendre com funciona aquesta eina.
