Introducció

En aquesta secció, cobrim els conceptes bàsics de la gestió d'entrada i sortida en PowerShell. Entendre com interactuar amb els usuaris i mostrar informació és crucial per escriure scripts efectius. Explorarem diverses maneres de llegir l'entrada dels usuaris i mostrar la sortida a la consola.

Sortida de dades

Utilitzant `Write-Output`

El cmdlet Write-Output és la manera principal d'enviar sortida a la consola o a la següent comanda en la pipeline.

Write-Output "Hello, World!"
  • Explicació: Aquesta comanda mostrarà "Hello, World!" a la consola.

Utilitzant `Write-Host`

El cmdlet Write-Host s'utilitza per mostrar informació directament a la consola. Sovint s'utilitza per mostrar missatges a l'usuari.

Write-Host "This is a message to the user."
  • Explicació: Aquesta comanda mostrarà el missatge "This is a message to the user." a la consola.

Utilitzant `Write-Verbose`, `Write-Debug` i `Write-Warning`

Aquests cmdlets s'utilitzen per proporcionar diferents nivells de missatgeria:

  • Write-Verbose: Per a sortida detallada.
  • Write-Debug: Per a informació de depuració.
  • Write-Warning: Per a missatges d'advertència.
Write-Verbose "This is a verbose message."
Write-Debug "This is a debug message."
Write-Warning "This is a warning message."
  • Explicació: Aquestes comandes mostraran missatges si les variables de preferència corresponents ($VerbosePreference, $DebugPreference, $WarningPreference) estan configurades adequadament.

Taula comparativa

| Cmdlet | Propòsit | Exemple | |----------------|----------------------------------|-----------------------------------------| | Write-Output | Enviar sortida a consola/pipeline | Write-Output "Hello, World!" | | Write-Host | Mostrar missatge a la consola | Write-Host "Message to user" | | Write-Verbose| Mostrar missatges detallats | Write-Verbose "Verbose message" | | Write-Debug | Mostrar missatges de depuració | Write-Debug "Debug message" | | Write-Warning| Mostrar missatges d'advertència | Write-Warning "Warning message" |

Llegir entrada

Utilitzant `Read-Host`

El cmdlet Read-Host s'utilitza per llegir l'entrada de l'usuari.

$name = Read-Host "Enter your name"
Write-Output "Hello, $name!"
  • Explicació: Aquest script demana a l'usuari que introdueixi el seu nom i després el saluda amb el nom introduït.

Llegir entrada segura

Per a informació sensible com contrasenyes, pots utilitzar el paràmetre -AsSecureString amb Read-Host.

$password = Read-Host "Enter your password" -AsSecureString
  • Explicació: Aquesta comanda demanarà a l'usuari que introdueixi una contrasenya, que es guardarà com una cadena segura.

Exemple d'script

Aquí tens un script senzill que combina entrada i sortida:

# Demana a l'usuari el seu nom
$name = Read-Host "Enter your name"

# Demana a l'usuari la seva edat
$age = Read-Host "Enter your age"

# Mostra un missatge de benvinguda
Write-Output "Hello, $name! You are $age years old."
  • Explicació: Aquest script demana a l'usuari el seu nom i edat, després mostra un missatge de benvinguda amb la informació proporcionada.

Conclusió

En aquesta secció, hem cobert els conceptes bàsics d'entrada i sortida en PowerShell. Hem après a utilitzar diversos cmdlets per mostrar missatges i llegir l'entrada dels usuaris. Aquestes habilitats fonamentals són essencials per crear scripts interactius i fàcils d'utilitzar. A mesura que progressis, trobaràs aquestes tècniques inavaluables per desenvolupar scripts de PowerShell més complexos.
