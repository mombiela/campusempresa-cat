Introducció

Les cadenes són un tipus de dada fonamental en PowerShell, utilitzades per representar text. Aquesta secció cobrirà els conceptes bàsics de treballar amb cadenes, incloent la creació, manipulació i format de cadenes.

Creació de Cadenes

En PowerShell, les cadenes es poden crear utilitzant cometes simples (') o cometes dobles (").

Cadenes amb Cometes Simples

Les cadenes amb cometes simples són cadenes literals. Les variables i les seqüències d'escapament no són avaluades.

$singleQuotedString = 'Aquesta és una cadena amb cometes simples: $variable'
Write-Output $singleQuotedString

Cadenes amb Cometes Dobles

Les cadenes amb cometes dobles permeten la interpolació de variables i seqüències d'escapament.

$variable = "PowerShell"
$doubleQuotedString = "Aquesta és una cadena amb cometes dobles: $variable"
Write-Output $doubleQuotedString

Manipulació de Cadenes

PowerShell proporciona diversos mètodes i operadors per manipular cadenes.

Concatenació

Les cadenes es poden concatenar utilitzant l'operador + o simplement col·locant-les una al costat de l'altra.

$string1 = "Hola"
$string2 = "Món"
$concatenatedString = $string1 + " " + $string2
Write-Output $concatenatedString

Subcadena

Extreu una part d'una cadena utilitzant el mètode Substring.

$fullString = "PowerShell"
$subString = $fullString.Substring(0, 5)
Write-Output $subString

Longitud de la Cadena

Obté la longitud d'una cadena utilitzant la propietat Length.

$string = "PowerShell"
$length = $string.Length
Write-Output $length

Substituir

Substitueix les ocurrències d'una subcadena per una altra subcadena utilitzant el mètode Replace.

$string = "Hola, Món"
$replacedString = $string.Replace("Món", "PowerShell")
Write-Output $replacedString

Dividir

Divideix una cadena en una matriu de subcadenes basant-se en un delimitador.

$string = "poma,plàtan,cirera"
$array = $string.Split(',')
Write-Output $array

Format de Cadenes

PowerShell suporta diverses maneres de formatar cadenes.

Format Compost

Utilitza espais reservats i l'operador -f per al format compost.

$name = "Joan"
$age = 30
$formattedString = "Nom: {0}, Edat: {1}" -f $name, $age
Write-Output $formattedString

Interpolació de Cadenes

Utilitza cadenes amb cometes dobles per a la interpolació de cadenes.

$name = "Joan"
$age = 30
$interpolatedString = "Nom: $name, Edat: $age"
Write-Output $interpolatedString

Operacions Avançades amb Cadenes

Per a operacions més avançades amb cadenes, PowerShell proporciona cmdlets i mètodes addicionals.

Expressions Regulars

Utilitza expressions regulars per a la coincidència de patrons i manipulació de text.

$string = "La pluja a Espanya"
$pattern = "\b\w*ain\b"
if ($string -match $pattern) {
    Write-Output "Coincidència trobada: $($matches[0])"
}

Cadenes Multilínia

Crea cadenes multilínia utilitzant símbols @.

$multilineString = @"
Aquesta és una
cadena multilínia
en PowerShell.
"@
Write-Output $multilineString

Here-Strings

Els here-strings es poden utilitzar per a cadenes multilínia i conserven el format.

$hereString = @"
Aquesta és una here-string.
Conserva el format
    i la indentació.
"@
Write-Output $hereString

Conclusió

Treballar amb cadenes en PowerShell és essencial per a tasques de scripting i automatització. Aquesta secció ha cobert els conceptes bàsics de creació, manipulació i format de cadenes, així com algunes operacions avançades. En dominar aquests conceptes, podràs gestionar dades de text de manera més efectiva en els teus scripts de PowerShell.
