Paràmetres de l'Script
Modularitzar el Codi amb Funcions ►

En aquesta secció, explorarem com utilitzar els paràmetres de l'script en PowerShell. Els paràmetres de l'script et permeten passar dades als teus scripts, fent-los més flexibles i reutilitzables. Cobrirem els conceptes bàsics per a principiants i avançarem cap a conceptes més avançats.

Introducció als Paràmetres de l'Script

Els paràmetres de l'script són una manera de fer que els teus scripts siguin més dinàmics permetent-te passar diferents valors cada vegada que executes l'script. Això és particularment útil per a tasques d'automatització i scripting que requereixen entrades variables.

Sintaxi Bàsica

Per definir paràmetres en un script de PowerShell, utilitzes la paraula clau param seguida d'una llista de paràmetres tancats entre parèntesis.

param (
    [string]$Name,
    [int]$Age
)

Write-Output "Name: $Name"
Write-Output "Age: $Age"
  • [string]$Name: Això defineix un paràmetre anomenat Name de tipus string.
  • [int]$Age: Això defineix un paràmetre anomenat Age de tipus int.

Executar l'Script amb Paràmetres

Per executar l'script amb paràmetres, simplement proporciones els valors després del nom de l'script.

.\MyScript.ps1 -Name "Alice" -Age 30

Valors Per Defecte

També pots establir valors per defecte per als paràmetres, que s'utilitzaran si no es proporciona cap valor.

param (
    [string]$Name = "John Doe",
    [int]$Age = 25
)

Write-Output "Name: $Name"
Write-Output "Age: $Age"

Paràmetres Obligatòris

Per fer que un paràmetre sigui obligatori, pots utilitzar l'atribut Mandatory.

param (
    [Parameter(Mandatory=$true)]
    [string]$Name,
    [int]$Age
)

Write-Output "Name: $Name"
Write-Output "Age: $Age"

Si executes l'script sense proporcionar el paràmetre Name, PowerShell et demanarà que l'introdueixis.

Paràmetres Avançats de l'Script

Validació de Paràmetres

PowerShell proporciona diversos atributs per validar els valors dels paràmetres.

  • ValidateSet: Assegura que el valor del paràmetre sigui un d'un conjunt predefinit de valors.
  • ValidateRange: Assegura que el valor del paràmetre estigui dins d'un rang especificat.
  • ValidatePattern: Assegura que el valor del paràmetre coincideixi amb un patró d'expressió regular.
param (
    [ValidateSet("Red", "Green", "Blue")]
    [string]$Color,

    [ValidateRange(1, 100)]
    [int]$Percentage,

    [ValidatePattern("^[a-zA-Z]+$")]
    [string]$Name
)

Write-Output "Color: $Color"
Write-Output "Percentage: $Percentage"
Write-Output "Name: $Name"

Àlies de Paràmetres

Pots definir àlies per als paràmetres per proporcionar noms alternatius.

param (
    [Alias("FN")]
    [string]$FirstName,

    [Alias("LN")]
    [string]$LastName
)

Write-Output "First Name: $FirstName"
Write-Output "Last Name: $LastName"

Ús de `CmdletBinding`

L'atribut CmdletBinding converteix el teu script en una funció avançada, permetent-te utilitzar paràmetres comuns i altres característiques avançades.

[CmdletBinding()]
param (
    [string]$Name,
    [int]$Age
)

Write-Output "Name: $Name"
Write-Output "Age: $Age"

Conjunts de Paràmetres

Els conjunts de paràmetres et permeten definir diferents conjunts de paràmetres que es poden utilitzar en diferents escenaris.

[CmdletBinding()]
param (
    [Parameter(Mandatory=$true, ParameterSetName="Set1")]
    [string]$Name,

    [Parameter(Mandatory=$true, ParameterSetName="Set2")]
    [int]$Age
)

if ($PSCmdlet.ParameterSetName -eq "Set1") {
    Write-Output "Name: $Name"
} elseif ($PSCmdlet.ParameterSetName -eq "Set2") {
    Write-Output "Age: $Age"
}

Conclusió

Els paràmetres de l'script són una característica poderosa en PowerShell que poden millorar enormement la flexibilitat i reutilització dels teus scripts. Comprenent i utilitzant les característiques bàsiques i avançades dels paràmetres, pots crear scripts més dinàmics i robustos. Ja sigui establint valors per defecte, fent paràmetres obligatoris, o utilitzant validació avançada i conjunts de paràmetres, dominar els paràmetres de l'script és essencial per a un scripting efectiu en PowerShell.
