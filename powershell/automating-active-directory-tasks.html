Introducció

Active Directory (AD) és un servei crític per gestionar usuaris, ordinadors i altres recursos en un entorn Windows. PowerShell proporciona una manera poderosa d'automatitzar moltes de les tasques associades amb Active Directory, fent l'administració més eficient i menys propensa a errors.

En aquest tema, cobrirem:

  • Connexió a Active Directory
  • Gestió d'Usuaris
  • Gestió de Grups
  • Gestió d'Ordinadors
  • Tècniques Avançades d'Automatització

Connexió a Active Directory

Abans de poder gestionar objectes d'Active Directory, necessites establir una connexió amb el teu entorn AD.

Instal·lació del Mòdul d'Active Directory

Primer, assegura't que el mòdul d'Active Directory per PowerShell està instal·lat. Pots instal·lar-lo utilitzant la següent comanda:

Install-WindowsFeature -Name RSAT-AD-PowerShell

Importació del Mòdul d'Active Directory

Un cop instal·lat, importa el mòdul a la teva sessió de PowerShell:

Import-Module ActiveDirectory

Connexió a un Controlador de Domini

Per connectar-te a un controlador de domini específic, utilitza el cmdlet Get-ADDomainController:

$domainController = Get-ADDomainController -Discover -Service "PrimaryDC"

Gestió d'Usuaris

La gestió d'usuaris és una tasca comuna a Active Directory. PowerShell fa que sigui fàcil crear, modificar i eliminar comptes d'usuari.

Creació d'un Nou Usuari

Per crear un nou usuari, utilitza el cmdlet New-ADUser:

New-ADUser -Name "John Doe" -GivenName "John" -Surname "Doe" -SamAccountName "jdoe" -UserPrincipalName "jdoe@domain.com" -Path "OU=Users,DC=domain,DC=com" -AccountPassword (ConvertTo-SecureString "P@ssw0rd" -AsPlainText -Force) -Enabled $true

Modificació d'un Usuari Existent

Per modificar un usuari existent, utilitza el cmdlet Set-ADUser:

Set-ADUser -Identity "jdoe" -Title "Senior Developer" -Department "IT"

Eliminació d'un Usuari

Per eliminar un usuari, utilitza el cmdlet Remove-ADUser:

Remove-ADUser -Identity "jdoe"

Gestió de Grups

Els grups són essencials per gestionar permisos i control d'accés. PowerShell proporciona cmdlets per crear, modificar i eliminar grups.

Creació d'un Nou Grup

Per crear un nou grup, utilitza el cmdlet New-ADGroup:

New-ADGroup -Name "Developers" -GroupScope Global -Path "OU=Groups,DC=domain,DC=com"

Afegir un Usuari a un Grup

Per afegir un usuari a un grup, utilitza el cmdlet Add-ADGroupMember:

Add-ADGroupMember -Identity "Developers" -Members "jdoe"

Eliminar un Usuari d'un Grup

Per eliminar un usuari d'un grup, utilitza el cmdlet Remove-ADGroupMember:

Remove-ADGroupMember -Identity "Developers" -Members "jdoe" -Confirm:$false

Gestió d'Ordinadors

La gestió de comptes d'ordinador és una altra tasca crítica a Active Directory.

Creació d'un Nou Ordinador

Per crear un nou compte d'ordinador, utilitza el cmdlet New-ADComputer:

New-ADComputer -Name "PC01" -Path "OU=Computers,DC=domain,DC=com" -Enabled $true

Restabliment d'un Compte d'Ordinador

Per restablir un compte d'ordinador, utilitza el cmdlet Reset-ADComputer:

Reset-ADComputer -Identity "PC01"

Eliminació d'un Ordinador

Per eliminar un compte d'ordinador, utilitza el cmdlet Remove-ADComputer:

Remove-ADComputer -Identity "PC01"

Tècniques Avançades d'Automatització

Per a una automatització més avançada, pots combinar múltiples cmdlets i utilitzar tècniques de scripting per realitzar tasques complexes.

Creació Massiva d'Usuaris

Pots crear múltiples usuaris a partir d'un fitxer CSV utilitzant un script:

$users = Import-Csv -Path "C:\Users\new_users.csv"
foreach ($user in $users) {
    New-ADUser -Name $user.Name -GivenName $user.GivenName -Surname $user.Surname -SamAccountName $user.SamAccountName -UserPrincipalName $user.UserPrincipalName -Path $user.Path -AccountPassword (ConvertTo-SecureString $user.Password -AsPlainText -Force) -Enabled $true
}

Tasques Programades

Pots programar scripts de PowerShell per executar-se en moments específics utilitzant el Programador de Tasques.

$action = New-ScheduledTaskAction -Execute "PowerShell.exe" -Argument "-File C:\Scripts\DailyUserReport.ps1"
$trigger = New-ScheduledTaskTrigger -Daily -At 6am
Register-ScheduledTask -Action $action -Trigger $trigger -TaskName "DailyUserReport" -Description "Generates a daily report of user activities"

Conclusió

PowerShell és una eina poderosa per automatitzar tasques d'Active Directory. Dominant els cmdlets i tècniques cobertes en aquest tema, pots agilitzar significativament els teus processos de gestió d'AD. Ja sigui creant usuaris, gestionant grups o automatitzant fluxos de treball complexos, PowerShell proporciona la flexibilitat i el control necessaris per gestionar eficientment el teu entorn d'Active Directory.
